# 股票市场（Stock Market）深度知识库

> 股票市场：人类历史最悠久、监管最成熟、基础设施最完备的金融市场之一。
> 代表：纽约证券交易所(NYSE)、纳斯达克(Nasdaq)、上交所/深交所、港交所、伦交所等。

如果你读完了 [CEX](../crypto/cex/README.md) 和 [DEX](../crypto/dex/README.md)，那么股票市场会给你一个全新的视角：
**这是一个运转了几百年、被无数次危机打磨过的「成熟版交易系统」。**

它最大的学习价值在于——**股票市场是 CEX「角色集中」的反面**：

> CEX 一家公司同时做了交易所+券商+银行+托管（[见 CEX 1.1](../crypto/cex/01-行业全景/1.1-CEX是什么.md)）。
> 股票市场则把这些角色**严格分业、相互制衡**：交易所只撮合、券商只代理、做市商只提供流动性、
> 清算所只清算、中央存管只保管、银行只管钱。**没有任何一方能独自接触你的全部资产和交易。**
>
> **理解了为什么股市要这样"拆分"，你就理解了 FTX 为什么会暴雷，以及金融监管几百年血泪换来的核心智慧。**

这个知识库的目标：让**小白**理解「买股票背后到底发生了什么」，让**从业者/工程师**挖到撮合、清算交收(T+N)、做市、监管的深水细节。

## 🎯 读完应该建立的「认知锚点」⭐

1. **股市的核心智慧是「分业制衡」**：交易所、券商、做市商、清算所、存管机构各司其职、互相监督。这是几百年危机换来的，也是 CEX 缺失的。
2. **你买的股票，真正存在「中央证券存管(CSD)」里**，券商只是帮你代持记账——这和 CEX「交易所托管」表面像，但背后有独立存管 + 监管 + 投资者保护基金，信任结构完全不同。
3. **交易（秒级）和交收（T+1/T+2）是分开的**：成交只是「达成协议」，真正的「钱货两清」要过 1-2 天，靠中央对手方(CCP)担保。这是 CEX「即时内部记账」最大的不同。
4. **做市商和流动性**：和 [CEX 做市](../crypto/cex/04-运营体系/4.3-做市与流动性运营.md)、[DEX 的 LP](../crypto/dex/02-产品与机制/2.3-流动性与无常损失.md) 是同一个问题的第三种答案。
5. **信息披露是股市监管的基石**：上市公司必须公开财报。这是「保护投资者」的核心，也是加密市场长期缺失的。
6. **股市是理解一切金融市场的"标准答案"**：它最成熟、最规范，学透它，再回头看 CEX/DEX 的"野蛮"和"创新"会更清醒。

## 一张图看懂买一股股票背后发生了什么

```
你(投资者)
   │ ① 在券商 App 下单"买 1 股苹果"
   ▼
券商(Broker) ── 不自己撮合，把订单路由出去
   │ ② 路由到 交易所/做市商/暗池
   ▼
交易所(如 Nasdaq) ── ③ 订单簿撮合，和某个卖方成交
   │ ④ 成交回报(秒级，但这只是"达成协议")
   ▼
清算所/CCP ── ⑤ 成为买卖双方的中央对手方，担保交收
   │ ⑥ T+1/T+2 进行"钱货两清"
   ▼
中央证券存管(CSD) ── ⑦ 股票登记到你名下(券商代持)
                    资金从你账户划走
   你真正"拥有"了这股(记录在存管体系，券商代你持有)
```

> **对照 CEX**：CEX 把②~⑦全部压缩成「数据库改一行余额」（[见 CEX 1.1](../crypto/cex/01-行业全景/1.1-CEX是什么.md)）。
> 股市却用一整条**分工明确、互相制衡、受监管**的产业链来完成——慢，但安全、可追责、有兜底。
> **效率 vs 安全的权衡，在这里给出了和 CEX 完全相反的答案。**
