# 2.2 法币与超额抵押稳定币

> 深度：🟡 → 🔴
> 两种"有抵押"的稳定币：法币抵押（USDT/USDC）和加密超额抵押（DAI）。它们是目前最主流、最可信的。

## 一、法币抵押型（USDT、USDC）⭐

```
机制: 中心化发行方持有"法币储备"(现金 + 短期美债等), 1:1 背书发行的稳定币
  你给发行方 1 美元 → 它铸造 1 个稳定币给你
  你拿 1 个稳定币赎回 → 它给你 1 美元, 销毁该币
锚定靠: "可赎回性" + 套利(见 2.1)
代表: USDT(Tether)、USDC(Circle)
```

### 优点 vs 风险
| 优点 | 风险 |
| --- | --- |
| 最稳（有真实储备背书） | 中心化（信任发行方） |
| 流动性最深、最广泛接受 | 储备透明度（USDT 历史上被质疑） |
| 锚定记录好 | 可被冻结（发行方能冻结地址，见下） |
| | 监管风险、银行通道风险 |

### 储备透明度问题 ⚠️
```
关键: 发行方真的有 1:1 的储备吗?
  USDT(Tether): 历史上储备构成被质疑(商业票据等), 透明度争议
  USDC(Circle): 相对透明(定期披露, 主要是现金+短期美债)
→ 类比 CEX 储备金证明(PoR, 见 CEX 6.2): 都是"你得信它真有储备"
  2023 USDC 曾因储备银行(硅谷银行)倒闭短暂脱锚(后恢复)
→ 法币稳定币的风险 = 储备的真实性和安全性
```

- 💡 **中心化的双刃剑**：法币稳定币能被发行方**冻结**（配合执法或被要求时）——
  这违背了加密"抗审查"理想，但也是它能合规、被主流接受的原因。是"合规 vs 抗审查"的又一体现（见 [DEX 6.4](../../dex/06-风险与安全/6.4-监管与合规.md)）。

## 二、加密超额抵押型（DAI）⭐

```
机制: 用"加密资产"超额抵押来发行稳定币(无需中心化法币储备)
  例(MakerDAO 的 DAI):
    你抵押 150 美元的 ETH → 能借出 100 DAI(抵押率 150%)
    → 为什么超额? 因为 ETH 价格会波动, 留缓冲防止抵押不足
赎回: 还回 DAI + 利息, 取回你的 ETH 抵押品
```

### 为什么要"超额"抵押 ⭐
```
传统借贷: 靠信用 + 法律追债(还不上能起诉, 见 债券)
DeFi: 匿名 + 无法追债! → 借款人跑了怎么办?
  → 必须"超额抵押": 抵押品价值 > 借出价值
  → 借款人不还? 没关系, 清算抵押品就行(抵押品够覆盖)
→ 超额抵押是 DeFi 借贷的根基(因为无法追债, 见 03)
```

- 💡 **这是 DeFi 借贷 vs [传统借贷](../../../bonds/01-行业全景/1.1-债券是什么.md) 的根本区别**：传统靠信用+法律追债，DeFi 靠超额抵押+清算（因为匿名无法追债）。

### 清算机制（维持稳定）
```
如果抵押品(ETH)价格下跌, 抵押率跌破阈值:
  → 触发清算: 卖掉抵押品还债(见 03、08)
  → 保证每个 DAI 始终有足额抵押支撑 → 维持锚定
```

### 优点 vs 风险
| 优点 | 风险 |
| --- | --- |
| 去中心化（无中心发行方） | 资本效率低（要超额抵押） |
| 透明（抵押品链上可查） | 抵押品（加密资产）自身波动大 |
| 抗审查 | 极端行情下清算可能失灵（见 2.3、08） |
| | 复杂度高 |

## 法币 vs 超额抵押对照 ⭐

| | 法币抵押(USDC) | 超额抵押(DAI) |
| --- | --- | --- |
| 抵押品 | 法币/美债（链下） | 加密资产（链上） |
| 去中心化 | 低（中心化发行） | 高 |
| 资本效率 | 高（1:1） | 低（超额） |
| 透明度 | 靠披露/审计 | 链上可验证 |
| 可冻结 | 能 | 难 |
| 主要风险 | 储备真实性、监管 | 抵押品波动、清算失灵 |

> 💡 注意：很多"去中心化"稳定币（如 DAI）的抵押品里其实也包含了大量 USDC（法币稳定币）——
> **所以它们的"去中心化"是打折扣的**（间接依赖中心化稳定币）。纯粹的去中心化稳定币很难做大。

## 小结

- 法币抵押（USDT/USDC）：1:1 法币储备背书，最稳、流动性最深，但中心化、靠储备真实性、可被冻结（USDC 曾因硅谷银行短暂脱锚）。
- 超额抵押（DAI）：用加密资产超额抵押发行，去中心化、透明，但资本效率低、抵押品波动。
- 超额抵押是 DeFi 借贷根基——因匿名无法追债，靠"抵押品+清算"代替"信用+法律"。
- 法币稳定币高效但中心化，超额抵押去中心化但低效；很多"去中心化"稳定币其实依赖 USDC。
