## 核心收获 ⭐

```
无风险利率(国债收益率) = 一切资产估值的"地基/折现率"
  利率↑ → 折现率↑ → 所有"未来现金流资产"现值↓ → 资产价格普遍承压
  利率↓ → 折现率↓ → 资产价格普遍上涨
→ 这就是为什么"美联储一加息, 全球股债加密一起跌"
→ 利率是金融世界的"地心引力"
```

> ⭐ **这是整个 Finance Wiki 最重要的"大一统"洞察之一**：
> 股票、债券、房产、汇率、加密，看似各自独立，其实都站在"利率"这同一个地基上。
> **理解利率与万物的关系，你就拿到了理解"全球资金为何这样流动"的总钥匙。**
