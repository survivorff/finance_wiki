# 6.3 骗局识别

> 深度：🟡⭐ 实用避坑
> DEX 无许可发币 = 骗局的天堂。这篇教你识别最常见的骗局，是散户的保命课。

## 为什么 DEX 骗局这么多

```
回忆: 工厂合约让任何人无许可创建任意代币和池子(见 3.2)
→ 骗子能零成本发一个币、建个池、包装个故事
→ 没有 CEX 那样的上币审核(尽管 CEX 审核也有水分, 见 CEX 4.2)
→ DEX 上骗局币的密度远高于 CEX
```

## 一、貔貅盘（Honeypot）⭐ 能买不能卖

```
原理: 合约里藏了恶意代码 —— 你能买入, 但卖出时被阻止(只有骗子能卖)
表现: 看到一个币暴涨, 你买入, 然后发现怎么都卖不掉 → 价值归零
为什么叫貔貅: 传说中的神兽"只进不出"
```

### 识别
- 用貔貅检测工具（Token Sniffer、Honeypot.is 等）扫合约。
- 看是否有人能成功卖出（链上交易记录里有没有「卖」单）。
- 合约未开源 / 有可疑的转账限制函数 → 高危。

## 二、Rug Pull（卷地毯/撤池跑路）⭐

```
原理: 项目方/庄家在吸引大量资金后, 突然撤走流动性 或 抛售, 卷款跑路
类型:
  - 抽流动性: 项目方移除池子里的流动性 → 币瞬间归零, 你的币换不回钱
  - 砸盘跑路: 团队持有大量代币, 拉高后一次性砸盘
  - 后门铸币: 合约留后门, 偷偷增发砸盘
```

### 识别
- 流动性是否锁定（Locked Liquidity）？锁多久？
- 团队代币是否锁仓（vesting）？持仓是否过于集中？
- 合约是否有「增发」「修改」后门？是否放弃了所有权（renounce）？
- 项目是否匿名团队 + 无审计 + 一夜爆火（meme 高危）。

## 三、钓鱼（Phishing）⭐ 最高发

```
形式:
  - 假网站: 仿冒 uniswap 等的域名(uni-swap.xx), 骗你连钱包签授权
  - 假空投: "你获得空投, 点击领取"→ 连接后签恶意授权
  - 假客服: 假装官方客服, 骗你交助记词/私钥(真客服永远不会要这些!)
  - 恶意授权/签名: 见 2.6, 骗你签 Permit 把资产授权走
  - 假代币/假合约: 同名代币但合约地址是假的
```

### 防御
```
1. 永远用官方域名(收藏书签), 不点社群/私信/搜索广告里的链接
2. 任何人要你的助记词/私钥 = 100% 骗子
3. 看不懂的签名/授权绝不签(尤其 Permit、setApprovalForAll)
4. 用 Rabby 等会模拟+警示风险的钱包
5. 小额测试, 用独立钱包隔离风险
```

## 四、其他常见骗局

| 骗局 | 手法 |
| --- | --- |
| **假币/仿冒** | 同名代币不同合约，骗你买假的 |
| **拉高出货（Pump & Dump）** | 喊单拉盘后砸给散户（同 [CEX](../../cex/05-市场与博弈/5.4-市场操纵与黑幕.md)） |
| **假 KOL/名人骗局** | 盗号或仿冒名人推广骗局币 |
| **高收益庞氏** | 承诺超高 APY 的「农场」，本质拆东补西 |
| **空投钓鱼代币** | 给你钱包空投个垃圾币，诱导你去恶意网站交互 |
| **dusting 攻击** | 给你转一点币追踪/诱导你交互 |

## 实用尽调清单（买不明代币前）⭐

```
✅ 合约是否开源、审计过？
✅ 流动性是否锁定？锁多久？
✅ 持币是否极度集中(看前 10 地址占比)？
✅ 团队代币是否锁仓？是否匿名团队？
✅ 用貔貅检测工具扫一遍(能不能卖)？
✅ 合约有没有增发/修改/黑名单后门？
✅ 社区是否真实(还是全是机器人喊单)？
✅ 用独立小钱包 + 小额先试
```

> ⚠️ **黄金法则**：
> - **「保证高收益」「稳赚」「内部消息」「限时空投」= 骗局信号**。
> - **看不懂的合约/签名 = 不碰、不签。**
> - **FOMO（怕错过）是骗子最好的武器**——越是催你「再不买就晚了」，越要警惕。
> - DEX 没有客服、不可逆、无兜底——**自己就是最后一道防线。**

## 对照 CEX 💡

> CEX 也有骗局（拉高出货、垃圾币上币，见 [CEX 5.4](../../cex/05-市场与博弈/5.4-市场操纵与黑幕.md)），
> 但 CEX 至少有上币审核（哪怕掺水）、有客服、可能冻结涉案资金。
> DEX **无许可 + 不可逆 + 无客服** → 骗局密度和危害都更高。
> **去中心化的自由，伴随的是「骗子也无许可、你也无人保护」的代价。**

## 小结

- DEX 无许可发币导致骗局密度远超 CEX。
- 三大主流骗局：貔貅盘（能买不能卖）、Rug Pull（撤池/砸盘跑路）、钓鱼（假网站/假空投/假客服/恶意签名）。
- 尽调看：开源审计、流动性锁定、持币集中度、团队锁仓、貔貅检测、合约后门。
- 黄金法则：远离「稳赚/内部消息/限时」、看不懂的不签、警惕 FOMO、自己是最后防线。
