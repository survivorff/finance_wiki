# 3.4 暗池与场外交易

> 深度：🔴💡 内幕
> 不是所有股票交易都在公开交易所发生。「暗池」是股市一个低调但巨大的部分，理解它能看清市场的真实结构。

## 明池 vs 暗池

```
明池(Lit Market): 公开交易所, 订单簿对所有人可见(挂单价量公开)
暗池(Dark Pool): 私密交易场所, 订单"不公开显示", 成交后才披露
```

## 暗池是什么、为什么存在 ⭐

```
暗池 = 不公开挂单的私密撮合场所(多由大投行/机构运营)
为什么要"暗":
  机构要买卖巨量股票(如 100 万股)
  → 如果在公开订单簿挂出来, 所有人都看到"有人要买巨量"
  → 别人抢先买入抬价(抢跑), 机构被迫高价成交(冲击成本)
  → 在暗池里悄悄成交, 不暴露意图, 减少冲击
```

- 💡 **暗池主要服务机构大宗交易**——避免大单冲击市场和被抢跑。
- 成交后会上报、披露（事后透明），但成交前不可见（事前不透明）。

## 暗池的争议 ⚠️

| 好处 | 争议 |
| --- | --- |
| 减少大单冲击成本 | 降低市场透明度 |
| 保护机构交易意图 | 价格发现转移到暗处 |
| 降低交易成本 | 可能存在不公平（运营方优势） |

- ⚠️ 如果太多交易跑到暗池，公开市场的「价格发现」功能被削弱（明池价格不能反映全部供需）。
- ⚠️ 暗池运营方（投行）可能有信息优势或利益冲突，曾有暗池因违规被罚。

## 内部化（Internalization）💡

```
内部化: 券商/做市商不把订单送到交易所, 而是"自己吃下"(用自己的库存成交)
  尤其散户订单(通过 PFOF 买来, 见 2.1)
  → 批发做市商(如 Citadel Securities)内部消化大量散户订单
```

- 💡 这就是 [PFOF（订单流付款）](../02-参与者与产品/2.1-交易所与券商.md) 的下游：散户订单被卖给做市商，做市商内部成交赚价差。
- 大量散户订单根本没到公开交易所，而是在做市商内部成交了。

## 暗池/内部化 vs 加密的对照 ⭐

```
股市暗池 ≈ CEX 的 OTC 大宗交易(见 CEX 2.2)
  → 都是"大单私下成交, 避免冲击盘面"

内部化 ≈ CEX 的自营/做市内部消化订单(见 CEX 4.3)
  → 都是"订单不进公开撮合, 在机构内部成交"
  → 都有"利益冲突 + 透明度"的争议

DEX 的对照: DEX 几乎全透明(链上), 没有真正的"暗池"
  → 但这反而暴露给 MEV(见 DEX 5.2)
  → "透明"和"被抢跑"是一体两面!
```

> 💡 **深刻洞察**：暗池的存在揭示了一个悖论——
> **完全透明的市场，会让大额交易者无处藏身（被抢跑/冲击）。**
> 所以股市发展出暗池（事前不透明），DEX 发展出私有交易通道（防 MEV）。
> **"透明"不总是好事**——这是 DEX 极致透明带来 MEV 问题的另一面镜子。

## 监管对暗池的态度

- 暗池受监管（要注册、报告、披露成交），但事前的不透明被持续审视。
- 监管在平衡：**保护大额交易者（允许暗池）vs 维护市场透明和价格发现（限制暗池占比）**。

## 小结

- 明池公开挂单，暗池私密撮合（成交后才披露），暗池主要服务机构大宗交易避免冲击和抢跑。
- 内部化是 PFOF 下游：散户订单被做市商内部消化，大量订单不进公开交易所。
- 暗池≈CEX 的 OTC，内部化≈CEX 自营做市，都有透明度和利益冲突争议。
- 深刻悖论：完全透明会让大额交易被抢跑，所以股市有暗池、DEX 有私有通道——透明是双刃剑。
