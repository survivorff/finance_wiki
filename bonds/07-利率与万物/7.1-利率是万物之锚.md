# 7.1 利率是万物之锚

> 深度：🟡 → 🔴 ⭐
> 这是整个 Finance Wiki 的核心洞察之一。理解了它，所有市场的涨跌都有了统一的解释。

## 核心命题：无风险利率是一切资产的"地基"⭐

```
任何资产的价值 = 它未来现金流的"现值"(见 3.2 DCF)
现值 = 未来现金流 / (1 + 折现率)^t
折现率 = 无风险利率 + 风险溢价
       ↑
   无风险利率(国债收益率)是所有折现率的"基准/地基"
→ 无风险利率一变, 所有资产的折现率都变, 所有资产价格都重估
```

## 巴菲特的比喻 💡

> 巴菲特说：**「利率之于资产价格，如同地心引力之于物体。」**
> - 利率低 → 引力小 → 万物轻飘飘往上涨（高估值能被容忍）。
> - 利率高 → 引力大 → 万物被往下拽（估值受压）。

```
为什么? 因为利率是"无风险就能拿到的回报":
  无风险利率 5% 时: 你凭什么买一个回报预期才 6% 的高风险资产? 要求更高回报 → 压低其价格
  无风险利率 0% 时: 6% 回报的资产"真香" → 资金涌入 → 推高价格
→ 所有资产都在和"无风险利率"竞争 → 利率是所有资产的"机会成本基准"
```

## 利率如何定价各类资产 ⭐

### 股票
```
股票估值 = 未来股息/盈利的折现(见 股票/DCF)
  利率↑ → 折现率↑ → 估值↓(尤其高估值成长股, 现金流在远期, 对折现率最敏感)
  → 这就是 2022 加息时科技/成长股暴跌的原因(久期长的"成长股"像长期债)
```

- 💡 **成长股 ≈ 长久期债券**：现金流都在遥远的未来，对利率极度敏感（同 [久期](../03-利率与定价/3.3-久期与凸性.md) 逻辑）。利率一涨，跌得最惨。

### 房地产
```
房价 ←→ 房贷利率:
  利率↑ → 房贷月供↑ → 购买力↓ + 折现率↑ → 房价承压
  利率↓ → 买得起的人多 + 折现率↓ → 房价上涨
```

### 汇率
```
利率↑ → 资金涌入该货币赚利息(见 FX 4.1) → 货币升值
→ 利率差驱动套息和全球资金流(见 FX 4.3、本模块 7.2)
```

### 加密 ⭐
```
加密虽"去中心化", 但同样受利率支配:
  利率低(2020-2021 QE): 流动性泛滥 + 无风险回报低 → 资金涌入高风险高回报的加密 → 牛市
  利率高(2022 加息): 流动性收紧 + 无风险都有 5% → 资金撤离加密 → 熊市
→ 加密大牛大熊, 高度同步于"全球利率/流动性周期"
```

- 💡 **这戳破了一个迷思**：很多人以为加密"独立于传统金融"，但实际上**加密是"风险资产里风险最高的一类"，对利率/流动性最敏感**。利率宽松时它涨最多，收紧时它跌最狠。加密没有逃离"利率引力"。

## 2022：万物重估的实证 ⭐

```
2022 美联储激进加息(无风险利率从~0 升到 5%+):
  → 债券大跌(久期, 见 3.3)
  → 成长股/科技股暴跌(长久期估值)
  → 加密暴跌(最高风险资产)
  → 房地产降温
  → 几乎所有资产同步下跌!
→ "利率是万物之锚"最生动、最惨痛的近期实证
  (打破了"资产分散就能避险"的幻想 — 利率一动, 万物同跌)
```

## 实际利率 vs 名义利率 💡

```
真正影响资产的是"实际利率"(名义利率 − 通胀预期):
  实际利率 = 你的钱"扣除通胀后"的真实回报
  实际利率↑ → 持有现金/国债的真实回报上升 → 风险资产更没吸引力 → 压力大
  实际利率为负(通胀>名义利率): 持有现金亏 → 资金被"赶"向风险资产/实物/加密(抗通胀叙事)
→ TIPS 收益率(见 2.2)反映实际利率, 是观察这个的关键
```

- 💡 **负实际利率是风险资产和加密的"温床"**：当存银行/买国债都跑不赢通胀，资金被迫寻找出路 → 涌入股票、房产、加密。2020-2021 的资产狂欢就有这个背景。

## 把所有市场连起来 ⭐⭐

```
                 央行货币政策(见 5.2)
                        │
                        ▼
              无风险利率(国债收益率)
                        │ "地心引力"
        ┌───────────────┼───────────────┬──────────────┐
        ▼               ▼               ▼              ▼
      股票            房地产           汇率           加密
   (折现率)          (房贷)         (套息/资金流)    (流动性/风险偏好)
        │               │               │              │
        └─────── 利率↑ → 万物承压; 利率↓ → 万物上涨 ──────┘
```

> ⭐ **这张图是整个 Finance Wiki 的"大一统"**：
> 你学的股票、外汇、加密（CEX/DEX），看似独立，其实都挂在"利率"这根弦上。
> **央行 → 利率 → 万物。** 这就是"全球资金如何被搅动"的最底层答案。

## 小结

- 无风险利率（国债收益率）是一切资产估值的折现率基准/地基。
- 巴菲特："利率之于资产价格如地心引力"——利率低万物涨，利率高万物跌（都在和无风险利率竞争）。
- 利率定价股票（成长股≈长久期债，最敏感）、房产（房贷）、汇率（套息）、加密（流动性/风险偏好）。
- 2022 加息引发"万物同跌"（债/股/加密/房），是"利率是万物之锚"的惨痛实证，打破分散避险幻想。
- 真正起作用的是实际利率（名义−通胀），负实际利率是风险资产和加密的温床。
- 大一统：央行→利率→万物，是"全球资金如何流动"的最底层答案。
