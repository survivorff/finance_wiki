> 深度：🟡 进阶 → 🔴 硬核
> 目标：理解股票到底怎么成交——订单簿、竞价机制、订单类型、暗池、订单路由。

## 核心对照 ⭐

| | 股市 | [CEX](../../crypto/cex/02-产品体系/2.1-现货交易.md) | [DEX](../../crypto/dex/02-产品与机制/2.1-AMM是什么.md) |
| --- | --- | --- | --- |
| 撮合规则 | 价格-时间优先 | 价格-时间优先 | AMM 公式 |
| 开收盘 | 集合竞价 | 无（7×24连续） | 无 |
| 市场结构 | 碎片化（多交易所+暗池）| 相对集中 | 碎片化（多池） |
| 特殊 | 订单路由+最优执行义务 | — | MEV |

> 💡 股市的撮合内核和 CEX 几乎一样（都是价格-时间优先订单簿），但因为**分业 + 多交易所并存**，多出了「订单路由」「最优执行」「暗池」这些 CEX 没有的复杂性。
