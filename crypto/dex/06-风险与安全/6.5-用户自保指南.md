# 6.5 用户自保指南

> 深度：🟢⭐ 实用清单
> 把所有 DEX 风险知识，落成「我该怎么做」的可执行清单。DEX 没有客服，自己就是唯一的安全部门。

> ⚠️ 本文是风险教育，不构成投资建议。链上操作不可逆、无兜底，风险自负。

## 一、钱包安全（最重要）⭐

```
🔐 助记词/私钥:
   - 离线手抄/钢板备份, 绝不拍照、不存云盘、不发任何地方
   - 任何人(包括"客服""官方")要助记词 = 100% 骗子
   - 大额用硬件钱包(Ledger/Trezor), 私钥永不触网
🔐 钱包分层:
   - 大额"金库钱包": 硬件钱包, 只收不乱动, 不连接任何 DApp
   - 日常"热钱包": 放少量资金, 用来交互 DApp
   - 高风险操作(撸空投/试新协议): 专用"隔离钱包"
🔐 防钓鱼:
   - 官网收藏书签, 绝不点社群/私信/搜索广告的链接
   - 仔细核对域名(假站差一个字母)
```

## 二、授权管理（DEX 特有的命门）⭐

```
⚠️ approve 授权 = 给合约动你币的权限, 是最大盗币入口(见 2.6)
做法:
  1. 尽量授权"精确额度"而非"无限授权"
  2. 定期用 revoke.cash 等工具检查并撤销不用的授权
  3. 看不懂的签名(尤其 Permit/permit2/setApprovalForAll)绝不签
  4. 用 Rabby 等会"模拟交易+风险提示"的钱包
```

## 三、交互前的尽调（碰新协议/新币前）

```
✅ 协议: 是否老牌、审计过、TVL 健康、没出过事？
✅ 代币: 开源？流动性锁定？持币集中度？团队锁仓？貔貅检测？
✅ 用小额先测试整个"买入→卖出"流程能否走通
✅ 用独立隔离钱包碰高风险新东西
```
（详细骗局识别见 [6.3](./6.3-骗局识别.md)）

## 四、交易操作

```
⚠️ 转账: 反复核对地址和链(转错不可逆!) → 先转小额测试
⚠️ 滑点: 主流币用默认; 别设太高(被夹, 见 5.2); 太低会失败
⚠️ MEV: 大额交易用防 MEV RPC / 抗 MEV 聚合器(CoW)
⚠️ Gas: 留足原生币(ETH/SOL)付 Gas
⚠️ 确认细节: 钱包弹窗仔细看"实际转出什么、给谁", 别盲签
```

## 五、当 LP / 玩 DeFi 的风险控制

```
⚠️ 无常损失: 选低波动池(稳定币对)、高交易量池; 算清"真实收益"(扣 IL 和币价)
⚠️ 高 APR 警惕: 看是不是通胀发币 + 币价暴跌的幻觉(见 4.1)
⚠️ 不过度套娃: 嵌套越多协议, 风险叠加越多
⚠️ 衍生品: 远离高杠杆(同 CEX, 见 CEX 6.5), 注意预言机/清算风险
⚠️ 分散: 别把所有资金压一个协议(合约风险)
```

## 六、骗局红线（看到就跑）🚨

```
🚨 "保证高收益/稳赚/内部消息/限时空投" → 骗局
🚨 要你提供助记词/私钥 → 100% 骗子
🚨 看不懂的授权/签名 → 不签
🚨 能买不能卖的币 → 貔貅
🚨 匿名团队 + 无审计 + 一夜爆火 + 流动性没锁 → Rug 风险
🚨 催你 FOMO("再不买就晚了") → 越催越要冷静
```

## 七、心法 ⭐

> **DEX 世界的铁律：不可逆、无客服、无兜底。你就是自己资产唯一的守护者。**
>
> - 把「自托管」当成「自己当银行」——既享自由，也担全责。
> - 大额冷存（硬件钱包），日常小额热用，高风险隔离钱包。
> - 看不懂的，不碰、不签、不投。
> - 慢就是快：DEX 上「冲动」和「FOMO」是亏钱的头号原因。
> - 永远假设「每个新链接、新协议、新签名都可能是陷阱」，验证后再操作。

## DEX vs CEX 自保的核心区别 💡

| | CEX（[对照](../../cex/06-风险与监管/6.5-用户自保指南.md)） | DEX |
| --- | --- | --- |
| 主要防什么 | 平台跑路/挪用 | 自己操作失误 + 骗局 + 合约风险 |
| 资产保管 | 别囤大额在所 | 自己管私钥（硬件钱包） |
| 兜底 | 可能有客服/赔付 | ❌ 完全没有 |
| 核心动作 | 选合规所、看 PoR | 管好私钥、管好授权、尽调 |

## 小结

- 钱包安全第一：助记词离线备份、大额硬件钱包、钱包分层、防钓鱼。
- 授权管理是 DEX 命门：精确授权、定期撤销、看不懂的签名绝不签。
- 碰新东西前尽调（审计/流动性锁定/持币集中/貔貅检测），小额隔离钱包先试。
- 交易核对地址链、合理滑点、防 MEV、不盲签。
- 当 LP/玩 DeFi 控好无常损失、警惕高 APR、不过度套娃、分散。
- 心法：不可逆无兜底，慢就是快，看不懂就不碰，你是自己唯一的守护者。
