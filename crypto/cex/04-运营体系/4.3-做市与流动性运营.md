# 4.3 做市与流动性运营

> 深度：🔴💡 行业内幕
> 流动性是交易所的命根子。这篇讲交易所怎么「制造」和「维护」流动性，以及其中的灰色操作。

## 为什么流动性是命根子

- 用户选交易所，本质选「**流动性最好**的地方」：滑点小、能即时成交、大单不冲击价格。
- **流动性飞轮**：流动性好 → 吸引交易者 → 交易量大 → 做市商更愿来 → 流动性更好。
- 这个正反馈造就了**头部极度垄断**——新所最大的难题就是「冷启动流动性」。

```
        ┌─────────────────────────────┐
        │  好流动性 → 吸引交易者         │
        │     ▲                ▼        │
        │  做市商愿来 ◀── 交易量大       │
        └─────────────────────────────┘
              强者愈强的飞轮
```

## 一、做市商（Market Maker）是谁

- 在盘口**两侧持续挂买卖单**的专业玩家，赚**买卖价差（spread）**，同时为市场提供深度。
- **专业做市商**：Wintermute、GSR、Jump Crypto、DWF Labs 等，服务多家交易所和项目。
- 没有做市商，一个新交易对就是「一潭死水」——挂单稀疏、价差巨大、根本没法正常交易。

```
        做市商挂买          做市商挂卖
买方 ──▶ │买 100.0│ spread │卖 100.2│ ◀── 卖方
         └────────┘  0.2   └────────┘
   做市商两边都挂，赚价差，让买卖双方都能即时成交
```

## 二、交易所怎么激励做市商

| 激励方式 | 说明 |
| --- | --- |
| **Maker 返佣** | maker 不仅免费，还能拿返佣（负手续费） |
| **VIP 费率** | 大做市商享最低费率档 |
| **API/技术优待** | 更高频率限制、更低延迟接入、专属通道 |
| **做市商协议** | 签约：承诺维持一定深度和价差，换取激励 |

> 💡 maker/taker 模型（见 [5.1](../05-市场与博弈/5.1-盈利模式拆解.md)）的本质就是「**补贴提供流动性的人，向消耗流动性的人收费**」。这是激励做市的制度设计。

## 三、新所/新币的流动性冷启动

新交易所或新上的币，没有自然流动性，怎么办？

```
1. 引入专业做市商(付费请他们来做市)
2. 项目方提供做市资金/代币给做市商
3. 交易所自营做市(自己下场提供初始流动性) ← 灰色
4. 流动性挖矿激励(LP 奖励)
5. 跨所搬砖套利者自然会来抹平价差(借力外部流动性)
```

## 四、灰色地带：自营做市与刷量 ⚠️

### 1. 交易所自营做市的利益冲突
- 交易所自己/关联方下场做市，能掌握**用户订单信息**，存在「**看底牌、抢跑（front-running）、对赌用户**」的嫌疑。
- ⭐ **FTX-Alameda 就是终极反面教材**：Alameda 作为 FTX 的关联做市商，既能看到用户数据，又挪用用户资金，是利益冲突的总爆发。

### 2. 刷量（Wash Trading）⚠️ 行业顽疾
- **自买自卖**制造虚假交易量，目的是：
  - 骗高 CoinMarketCap/CoinGecko 排名（吸引用户和项目方）。
  - 让新币显得「有人气、有流动性」。
  - 骗项目方付更高上币费。
- 💡 Bitwise 曾向 SEC 提交报告，指出当时大量交易所的报告交易量**绝大部分是假的**。
- 真实流动性 vs 虚假交易量：看「深度和滑点」比看「成交量」靠谱（假量做不出真深度）。

### 3. 拉盘砸盘配合
- ⚠️ 做市商可能和项目方/交易所配合，人为拉抬或打压价格（操纵）。

## 五、流动性的「真假」怎么辨别 💡

| 看什么 | 真流动性 | 假繁荣 |
| --- | --- | --- |
| 盘口深度 | 各价位挂单厚实 | 挂单稀疏，一推就动 |
| 滑点 | 大单滑点小 | 大单滑点巨大 |
| 价差 | 紧(spread 小) | 宽 |
| 成交量 vs 深度 | 匹配 | 量大但深度薄(刷量特征) |

## 六、流动性与价格发现

- 头部所流动性最深，所以**主流币的「市场价」基本由头部所的成交决定**（价格发现中心）。
- 小所价格跟随头部所，靠套利者（搬砖）保持一致。
- 这也是为什么指数价/标记价要取**头部所加权**（见 [3.5](../03-技术架构/3.5-清算与风控.md)）。

## 小结

- 流动性是交易所命根子，有「强者愈强」的飞轮效应，造就头部垄断。
- 做市商提供深度、赚价差；交易所用 maker 返佣、VIP 费率、技术优待激励他们。
- 新所/新币靠引入做市商、自营做市、流动性挖矿冷启动。
- 灰色地带：自营做市的利益冲突（FTX 教训）、刷量造假、拉砸配合。
- 辨别真假流动性看「深度和滑点」，而非成交量。
