# 1.4 发展简史

> 深度：🟢 小白友好
> FX 的历史 = 国际货币制度的演化史。理解它，才懂今天的汇率为什么这样运转。

## 时间线总览

```
金本位时代(19世纪-1914): 货币和黄金挂钩, 汇率基本固定
两次世界大战: 金本位崩溃, 货币动荡
1944 布雷顿森林体系: 美元挂钩黄金, 各国货币挂钩美元(固定汇率) ⭐
1971 尼克松冲击: 美元与黄金脱钩, 布雷顿森林瓦解 ⭐
1973~ 浮动汇率时代: 主要货币自由浮动, 现代 FX 市场诞生
1990s 电子化: 电子交易平台兴起
1999 欧元诞生: 欧洲统一货币
2000s~ 算法/高频、零售外汇兴起
2010s~ 数字货币、稳定币开始冲击传统跨境支付
```

## 关键节点

### 金本位（Gold Standard）
- 货币价值和黄金绑定，各国货币间汇率基本固定。
- 💡 优点：稳定、可信；缺点：货币供应受制于黄金，无法灵活应对经济波动。

### 1944 布雷顿森林体系 ⭐
```
二战后建立的国际货币秩序:
  美元和黄金挂钩(35 美元 = 1 盎司黄金)
  其他货币和美元挂钩(固定汇率)
  → 美元成为世界中心货币(美元霸权的制度起点, 见 1.3)
  → 同时成立 IMF、世界银行
```

### 1971 尼克松冲击 ⭐
```
背景: 美国印太多美元(越战+福利), 黄金不够兑付
1971: 尼克松宣布"美元和黄金脱钩" → 布雷顿森林体系崩溃
→ 货币不再有黄金支撑, 变成纯"信用货币(fiat)"
```

- 💡 **这是现代货币史的分水岭**：从此货币价值靠"国家信用"而非黄金。
- 💡 **加密的起源动机之一就是反对这个**：比特币白皮书暗含对"央行无限印钞的信用货币"的不信任，想用"算法稀缺"替代"国家信用"。理解尼克松冲击，就理解了加密"硬通货"叙事的历史背景。

### 1973~ 浮动汇率时代（现代 FX 诞生）
```
主要货币改为"自由浮动"(由市场供需决定汇率)
→ 汇率开始波动 → 产生了对冲汇率风险和投机的需求
→ 现代外汇市场(投机、做市、衍生品)由此繁荣
```

### 1999 欧元诞生
- 多个欧洲国家放弃本国货币，统一用欧元——人类最大的货币一体化实验。
- EUR/USD 成为最大货币对。

### 电子化与零售化（2000s~）
- 电子平台让交易高效化；零售外汇经纪商让散户能参与（也带来高杠杆亏损问题，见 [06](../06-风险与监管/README.md)）。

## 汇率制度的光谱 💡

历史演化留下了不同的汇率制度，今天各国仍在用不同的：

```
完全固定 ◀──────────────────────────────▶ 完全浮动
联系汇率      钉住        有管理浮动        自由浮动
(港币钉美元)  (钉一篮子)  (中国等)         (美/欧/日)
```

- 详见 [05 汇率制度](../05-央行与政策/5.2-汇率制度.md)。

## 贯穿全史的主线 ⭐

> **货币史 = 「锚定什么」的演化**：
> - 金本位：锚定黄金（实物稀缺）
> - 布雷顿森林：锚定美元（美元锚黄金）
> - 1971 后：锚定"国家信用"（纯信用货币）
> - 加密尝试：锚定"算法/代码"（比特币）或"美元"（稳定币）
>
> 💡 **每个时代都在回答"货币凭什么有价值"**。这条主线把 FX 和 [加密稳定币](../../crypto/cex/02-产品体系/2.4-理财与赚币.md)、[DeFi 稳定币](../../crypto/defi/README.md) 连了起来——稳定币本质是"在加密世界重新锚定法币"，是这个几百年命题的最新篇章。

## 小结

- FX 史 = 国际货币制度演化史：金本位→布雷顿森林（美元锚黄金）→1971 脱钩→浮动汇率。
- 1971 尼克松冲击是分水岭：货币变成纯信用货币（fiat），也是加密"硬通货"叙事的历史背景。
- 1973 浮动汇率催生现代 FX 市场（投机/对冲/衍生品），1999 欧元是最大货币一体化实验。
- 主线是「货币锚定什么」的演化（黄金→美元→国家信用→算法/稳定币），把 FX 和加密稳定币连起来。
