# 05 · 发行与公司行动

> 深度：🟡 进阶
> 目标：理解股票怎么「诞生」（IPO/发行）以及上市后公司的各种「行动」（分红、拆股、回购、并购）。

这是「一级市场」（公司和投资者之间）的故事，对照 [CEX 的上币/Launchpad](../../crypto/cex/02-产品体系/2.6-生态产品.md)。

## 核心对照 ⭐

| 股市 | CEX/加密 |
| --- | --- |
| IPO（首次公开发行） | 项目方发币 / 交易所 Launchpad（[CEX 2.6](../../crypto/cex/02-产品体系/2.6-生态产品.md)） |
| 投行承销 | 交易所/做市商辅助发行 |
| 招股书 + 尽职调查 | 白皮书（监管弱很多） |
| 锁定期 | 代币 vesting 解锁 |
| 分红 | 代币质押收益/回购销毁（[CEX 平台币](../../crypto/cex/02-产品体系/2.6-生态产品.md)） |
| 回购注销 | 代币回购销毁 |

> 💡 加密的发币机制，几乎是股票发行的「无监管简化版」——同样的逻辑，但缺了招股书、尽调、承销商责任、锁定期监管。理解 IPO 的严谨，就理解了加密发币的随意和风险。
