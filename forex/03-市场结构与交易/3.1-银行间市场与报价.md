# 3.1 银行间市场与报价

> 深度：🟡 进阶
> 没有中央交易所，价格怎么形成？答案在银行间的报价机制。

## 银行间市场（Interbank Market）

```
FX 的核心 = 大银行之间互相报价、成交的网络(见 1.2)
  - 银行互相有"授信额度"(信任对方能履约)
  - 通过电子平台(EBS/Reuters)或直接(电话/聊天)交易
  - 这一层点差最小、流动性最深 = "批发价"
```

## 报价机制：双向报价 ⭐

```
做市银行同时报"买价"和"卖价":
  EUR/USD  1.1000 / 1.1002
           ↑买价(bid)  ↑卖价(ask)
  bid: 银行愿意"买入欧元"的价
  ask: 银行愿意"卖出欧元"的价
  点差(spread) = ask − bid = 2 个点 → 做市银行的利润
```

- 客户"买欧元"按 ask 成交，"卖欧元"按 bid 成交——和 [股票/加密的 bid/ask](../../crypto/cex/02-产品体系/2.1-现货交易.md) 完全一样。

## 价格怎么趋于统一（无中央交易所）⭐

```
问题: 没有中央交易所, 花旗报 1.1000, 德银可能报 1.1001, 怎么统一?
答案: 套利 + 电子聚合
  - 如果价差大到能套利, 套利者立刻买低卖高 → 价格被拉平
  - 电子平台(EBS/Reuters/ECN)聚合多家银行报价 → 显示"最优买卖价"
  → 主要货币对的价格在全球高度一致(套利保证)
```

- 💡 **这和 [DEX 多池套利校准](../../crypto/dex/02-产品与机制/2.1-AMM是什么.md)、[股票 NBBO](../../stocks/03-交易与市场结构/3.5-订单路由与最优执行.md) 是同一个原理**：
  分散的报价，靠套利和聚合形成"事实上的统一价格"。**没有中央交易所不等于没有统一价格——套利就是那只看不见的手。**

## 流动性分层（价格的"批发零售"）⭐

```
银行间(批发): EUR/USD 点差可能 < 1 个点(流动性极深)
   ▼ 加价
大客户(企业/基金): 点差稍大
   ▼ 加价
零售(散户经纪商): 点差几个点甚至更多
→ 散户拿到的是"零售价", 经过层层加价(见 1.2)
```

- 💡 这解释了为什么散户在 FX 处于劣势——**不仅信息差，连价格本身都是被加价的零售价**。

## 报价的两种方式

```
直接报价(Direct): 1 单位外币 = 多少本币(多数国家)
间接报价(Indirect): 1 单位本币 = 多少外币(英镑、欧元、澳元等惯例)
→ 这是 EUR/USD(欧元在前) 和 USD/JPY(美元在前) 报价方向不同的历史原因
```

## 流动性的时段变化 💡

```
流动性随交易时段变化(见 1.2 全球接力):
  伦敦/纽约重叠时段(欧美都开市): 流动性最好、点差最小、波动最大
  亚洲早盘/周末前后: 流动性差、点差大、易剧烈波动(流动性差时一笔大单冲击大)
→ 重大数据/央行发布时, 流动性可能瞬间枯竭(点差暴涨)
```

- 💡 对照 [加密极端行情流动性枯竭](../../crypto/cex/06-风险与监管/6.1-风险地图.md)：FX 同样在重大事件时流动性骤降、点差暴涨——流动性不是恒定的。

## 小结

- 银行间市场是 FX 核心，大银行靠授信互相双向报价（bid/ask），点差是做市利润。
- 没有中央交易所，但靠套利+电子平台聚合形成统一价格（同 DEX 套利、股票 NBBO）。
- 价格分层（银行间批发→大客户→零售），散户拿被层层加价的零售价。
- 流动性随时段变化（欧美重叠时段最好），重大事件时可能枯竭、点差暴涨。
