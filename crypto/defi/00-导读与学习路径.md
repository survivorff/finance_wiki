# 00 · 导读与学习路径

> 花 5 分钟理解怎么用这个知识库，以及 DeFi 与传统金融的映射。

## 为谁而写

- **小白**：用过 CEX/DEX，听过稳定币、挖矿、借贷，想搞懂。
- **DeFi 从业/投研**：想系统理解稳定币、借贷、衍生品、收益。
- **智能合约工程师**：想理解清算、利率模型、稳定币机制。
- **对照学习者**：读完传统金融，想看它们如何在链上重演。

## 阅读约定（全知识库一致）

- **🟢🟡🔴** 深度，**⭐** 核心，**💡 洞察**，**⚠️ 误区/风险**。
- **不构成投资建议。** DeFi 风险极高、不可逆、无兜底。

## ⭐ DeFi ↔ 传统金融映射表（核心）

| DeFi | 传统对应 | 关键区别 |
| --- | --- | --- |
| 稳定币 | [货币/外汇](../../forex/README.md) | 链上锚定法币，但无央行背书（靠抵押/算法） |
| 借贷协议 | [银行/债券](../../bonds/README.md) | 超额抵押（无法追债），算法化利率 |
| DEX | [交易所](../../stocks/README.md) | AMM 替代订单簿（见 [DEX](../dex/README.md)） |
| 衍生品 DEX | [合约/期货](../cex/02-产品体系/2.2-合约衍生品.md) | 链上自托管玩永续 |
| 收益聚合 | 资管/理财 | 自动化、可组合 |
| 流动性挖矿 | （无直接对应） | 代币激励冷启动（见 [DEX 4.1](../dex/04-生态与运营/4.1-流动性挖矿.md)） |
| 闪电贷 | （传统不可能） | 原子性独有的无抵押瞬时贷 |
| DeFi 整体 | [影子银行](../../bonds/05-市场与参与者/5.3-回购市场.md) | 链上、无监管、可组合 |

> 💡 **学习钥匙**：DeFi 的每个东西，几乎都能在传统金融找到"原型"。
> 用传统金融的知识做地图，你能极快理解 DeFi——同时也能看清它"重新发明"时引入的新风险。

## DeFi 在加密知识库里的位置

```
CEX(中心化交易) → DEX(去中心化交易) → DeFi(去中心化的"完整金融体系")
  DEX 是 DeFi 的一部分(交易那块), DeFi 还包括 借贷/稳定币/衍生品/收益
→ 本知识库聚焦 DEX 之外的 DeFi: 稳定币、借贷、衍生品、收益、可组合性
```

## 一个重要心态

> DeFi 是加密世界创新最快、也最危险的角落。
> 「高 APY」的诱惑背后，是合约风险 + 清算风险 + 脱锚风险 + 套娃传染 + 不可逆 + 无兜底的多重叠加。
> [Luna 崩盘、Celsius 暴雷](../cex/06-风险与监管/6.4-经典案例复盘.md) 都源于此。
> 学 DeFi 是为了理解"链上金融如何运转和为何崩溃"，而不是盲目追逐 APY 当韭菜。
