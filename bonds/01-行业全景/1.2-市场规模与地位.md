# 1.2 市场规模与地位

> 深度：🟢🟡
> 债券市场比股市更大、更重要，却更"低调"。理解它的地位，才理解金融世界的真实权力结构。

## 市场规模：比股市更大 ⭐

```
全球债券市场规模: 上百万亿美元(量级超过全球股市)
其中:
  政府债(国债)是最大的一块, 美国国债市场是全球最大、最重要的单一债券市场
  企业债、市政债、机构债等
对比:
  全球股市市值: ~100+ 万亿(见 股票 1.2)
  全球债市: 规模相当甚至更大, 且交易更活跃(尤其国债)
```

> 💡 **反直觉**：大众觉得股市是金融中心，但债券市场体量更大、对经济影响更深。
> 「股票是新闻，债券是真相」——债市更大、更机构化、更理性，常更早反映经济真相。

## 美国国债：全球金融的"地基" ⭐⭐

```
美国国债(US Treasuries)的特殊地位:
  - 被视为全球"无风险资产"的标杆(美国政府"不会违约"的假设)
  - 全球央行外汇储备的主要持有对象(见 FX 1.3 美元霸权)
  - 美债收益率 = 全球资产定价的"无风险利率"基准(见 07)
  - 流动性极深, 是全球最安全的"避风港"
→ 美债收益率一动, 全球资产重新定价
```

- 💡 **美债 = [美元霸权](../../forex/01-行业全景/1.3-主要货币与货币对.md) 的金融载体**：
  各国把美元储备投资于美债 → 美债成为全球资金的"蓄水池" → 美国能低成本借债（"嚣张特权"）。
- ⚠️ 这也意味着：美债收益率（利率）的变动，通过全球资产定价，影响每一个市场（见 [07](../07-利率与万物/README.md)）。

## 债券市场的参与者（机构主导）💡

```
债券市场高度"机构化"(散户参与远少于股市):
  - 政府(发行国债)
  - 央行(买卖国债执行货币政策, 见 05)
  - 银行、保险、养老金(大买家, 需要稳定收益)
  - 主权基金、共同基金、债券基金
  - 对冲基金(利率投机/套利)
→ 散户主要通过"债券基金/ETF"间接参与
```

- 💡 对照 [股票散户化](../../stocks/07-现代演化/7.1-电子化与被动投资.md)：债市散户参与少，更多是机构的游戏，所以更"理性"、波动相对小（除危机时）。

## 债券市场的功能

```
1. 政府融资: 国家靠发债弥补财政赤字、搞建设(国债)
2. 企业融资: 不稀释股权地借钱(企业债, 见 02)
3. 货币政策传导: 央行通过国债买卖调节利率和流动性(见 05)
4. 无风险基准: 提供"无风险利率", 给一切资产定价(见 07)
5. 避险与配置: 危机时的避风港、资产配置的稳定器
```

## 为什么债市"低调"却重要 ⭐

```
股市: 涨跌刺激、暴富故事、媒体爱报 → 高关注
债市: 收益率小数点后的变动、机构主导 → 低关注
但是:
  债市规模更大、利率影响一切(见 07)
  债市危机(国债崩、信用冻结)往往是系统性危机的核心(见 06、08)
→ "债市才是金融世界真正的权力中心"
```

## 小结

- 全球债券市场上百万亿美元，规模超过股市，但更低调、更机构化。
- 美国国债是全球"无风险资产"基石、央行储备主要对象、全球资产定价基准——美元霸权的金融载体。
- 债市高度机构化（政府/央行/银行/保险/养老金/基金），散户多通过债券基金间接参与。
- 债市功能：政府/企业融资、货币政策传导、提供无风险基准、避险配置。
- "股票是新闻，债券是真相"——债市低调但是金融世界真正的权力中心。
