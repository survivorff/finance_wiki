# 6.1 风险地图

> 深度：🟡 进阶
> DEX 的风险分四类：合约风险、经济风险、操作风险、系统性风险。对照 [CEX 风险地图](../../cex/06-风险与监管/6.1-风险地图.md)。

## 风险全景

```
┌──────────────── DEX 风险地图 ────────────────┐
│                                               │
│ 1. 合约风险    代码漏洞、被黑、可升级权限被盗    │
│ 2. 经济风险    无常损失、清算、预言机操纵、脱锚  │
│ 3. 操作风险    私钥丢失、转错、签错授权、钓鱼 ⭐ │
│ 4. 系统性风险  连环清算、桥被黑、稳定币崩、链故障│
│                                               │
└───────────────────────────────────────────────┘
```

## 1. 合约风险（协议层）

- **代码漏洞**：智能合约有 bug，被黑客利用掏空（重入、整数溢出、逻辑错误等，见 [6.2](./6.2-智能合约安全.md)）。
- **可升级权限被盗**：若合约可升级，升级私钥（多签）被盗 → 协议被改成恶意。
- **预言机依赖**：依赖的预言机被操纵/失灵（见 [3.4](../03-技术架构/3.4-预言机与价格.md)）。
- **特点**：⚠️ 合约一旦被黑，资金**瞬间、批量、不可逆**地损失，且通常无法追回。

## 2. 经济风险（机制层）

- **无常损失**：LP 的核心风险（见 [2.3](../02-产品与机制/2.3-流动性与无常损失.md)）。
- **清算风险**：借贷/衍生品仓位被清算（见 [2.5](../02-产品与机制/2.5-衍生品DEX.md)）。
- **MEV 被夹**：交易被三明治攻击（见 [5.2](../05-市场与博弈/5.2-MEV详解.md)）。
- **稳定币脱锚**：依赖的稳定币失去锚定 → 池子和仓位崩塌。
- **代币归零/通胀**：挖矿代币、治理代币价值崩塌。

## 3. 操作风险（用户层）⭐ DEX 最高发

> ⚠️ **这是 DEX 上散户损失最多的来源，而且全部不可逆、无人兜底。**

- **私钥/助记词丢失或泄露**：丢了 = 永久失去，泄露 = 立即被盗。
- **转错地址/转错链**：不可逆，钱没了。
- **签错授权**：签了恶意授权/钓鱼签名 → 资产被卷走（见 [2.6](../02-产品与机制/2.6-钱包与交互.md)）。
- **钓鱼**：假网站、假空投、假客服骗你签名或交私钥。
- **踩骗局币**：貔貅、Rug Pull（见 [6.3](./6.3-骗局识别.md)）。

## 4. 系统性风险（生态层）

- **连环清算**：价格暴跌 → 大量 DeFi 仓位同时清算 → 抛售加剧下跌（链上版踩踏，对照 [CEX 连环爆仓](../../cex/06-风险与监管/6.1-风险地图.md)）。
- **跨链桥被黑**：桥是被黑重灾区（见 [3.5](../03-技术架构/3.5-跨链与桥.md)）。
- **稳定币崩盘**：大型稳定币脱锚会引发整个 DeFi 连锁危机（如算法稳定币崩盘历史）。
- **可组合性传染**：协议互相嵌套，一个底层协议出事 → 上面叠的全遭殃（乐高崩塌）。
- **链本身故障**：公链宕机/拥堵/重组 → DEX 瘫痪、清算失败。

## 风险的「可见性 + 可逆性」分级 ⭐

| 风险 | 可见性 | 可逆性 | 防御 |
| --- | --- | --- | --- |
| 操作失误（转错/签错） | 自己可控 | ❌ 不可逆 | 极度谨慎、小额测试、看懂再签 |
| 骗局币（貔貅/Rug） | 中（可尽调） | ❌ 不可逆 | 不碰不明项目、查审计 |
| 合约被黑 | 低（漏洞难察） | ❌ 通常不可逆 | 用老牌审计过的协议 |
| 无常损失 | 中（可计算） | ⚠️ 撤出即实现 | 选对池、算清收益 |
| MEV 被夹 | 低（隐形） | ❌ 已发生 | 防 MEV 工具、合理滑点 |

> ⭐ **DEX 风险的共同特点：大多不可逆、无客服、无赔付。** 这和 CEX 完全不同——
> CEX 还可能有客服、赔付、监管介入；DEX 出事，代码就是法律，钱没了就是没了。

## 对照 CEX 风险 ⭐

| | CEX | DEX |
| --- | --- | --- |
| 平台跑路/挪用 | ⭐ 最致命 | ✅ 无（自托管） |
| 被黑 | 交易所被盗 | 合约漏洞 + 私钥被盗 |
| 操作失误 | 较少（有客服兜底） | ⚠️ 极高发且不可逆 |
| 骗局密度 | 中（有审核） | ⚠️ 极高（无许可） |
| 兜底 | 可能有 | ❌ 无 |

## 小结

- DEX 四类风险：合约（漏洞/权限/预言机）、经济（无常损失/清算/MEV/脱锚）、操作（私钥/转错/签错/钓鱼，最高发）、系统性（连环清算/桥/稳定币/传染/链故障）。
- 操作风险是散户损失最多的来源，且全部不可逆、无人兜底。
- DEX 消灭了 CEX 的「跑路挪用」风险，但新增大量「不可逆、无兜底」的新风险。
- 去中心化 ≠ 更安全，只是风险换形态且责任全在自己。
