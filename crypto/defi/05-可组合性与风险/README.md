## 核心收获 ⭐

```
可组合性(货币乐高): DeFi 最强的创新引擎, 也是最独特的风险源
  威力: 协议自由拼接 → 创新爆炸、资金高效
  代价: 风险层层叠加、互相依赖 → 一处出事全线传染
→ "乐高"既能拼出摩天大楼, 也能一块塌全塌
```

> ⭐ 可组合性是 DeFi 区别于一切传统金融的"超能力"——但也是它系统性风险的根源。理解这个张力，是理解 DeFi 的关键。
