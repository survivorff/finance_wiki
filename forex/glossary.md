# 外汇市场术语表（Glossary）

> 带 ⭐ 的是核心高频术语。通用交易术语（bid/ask、点差、做市）可对照 [CEX 术语表](../crypto/cex/glossary.md)、[股票术语表](../stocks/glossary.md)。

## 基础
- **外汇 / Forex / FX** ⭐：货币兑换市场。
- **汇率 / Exchange Rate** ⭐：两种货币的相对价格。
- **货币对 / Currency Pair** ⭐：如 EUR/USD（base/quote 结构）。
- **基础货币 / 报价货币 (base/quote)**：前者你买卖的、后者标价的。
- **点 / Pip** ⭐：汇率最小变动单位（通常小数点后第4位）。
- **点差 / Spread**：买卖价差，做市商利润。
- **OTC（场外）** ⭐：无中央交易所的交易方式（FX 的本质）。
- **银行间市场 / Interbank**：大银行间交易的核心层。

## 货币
- **主要货币 / Majors**：USD/EUR/JPY/GBP/CHF/CAD/AUD。
- **主要货币对 / 交叉盘 / 新兴市场对**：含美元/不含美元/含新兴货币。
- **避险货币**：USD/JPY/CHF（危机时受追捧）。
- **商品货币**：AUD/CAD（与大宗相关）。
- **离岸货币**：如 CNH（离岸人民币）vs CNY（在岸）。
- **美元霸权** ⭐：美元的全球中心地位。

## 产品
- **即期 / Spot** ⭐：当前汇率换汇，T+2 交收。
- **远期 / Forward** ⭐：约定未来汇率（对冲用）。
- **远期点 / Forward Points**：远期与即期之差（由利率差定）。
- **外汇掉期 / FX Swap** ⭐：即期+反向远期（交易量最大）。
- **货币掉期 / Currency Swap**：长期+交换利息。
- **NDF（无本金交割远期）**：受管制货币用，只结差额。
- **外汇期权 / FX Options**：汇率权利。
- **外汇期货 / FX Futures**：交易所标准化远期（有 CCP）。

## 市场结构与交收
- **做市 / Market Making**：双向报价提供流动性。
- **Last Look（最后观望）** ⭐：FX 做市商最后一刻拒单的权利（争议）。
- **ECN**：电子通信网络（聚合报价）。
- **赫斯塔特风险 / Herstatt Risk** ⭐：跨时区交收的本金损失风险。
- **CLS** ⭐：FX 交收机构，用 PvP 消除赫斯塔特风险。
- **PvP（支付对支付）** ⭐：两种货币同步结算（≈股票 DVP）。

## 汇率原理 ⭐
- **利率平价 / IRP** ⭐：利率差锁定远期汇率。
- **覆盖利率平价 / CIP**：用远期锁定，几乎严格成立。
- **非覆盖利率平价 / UIP**：不锁定，现实常不成立（套息根源）。
- **购买力平价 / PPP** ⭐：长期汇率向购买力相等靠拢。
- **大麦克指数 / Big Mac Index**：PPP 的通俗版。
- **套息交易 / Carry Trade** ⭐：借低息换高息赚利差。
- **远期溢价之谜**：UIP 系统性失效的现象。
- **risk-on/risk-off** ⭐：市场风险偏好的切换（避险情绪）。

## 央行与政策 ⭐
- **央行 / Central Bank** ⭐：货币发行和管理者（Fed/ECB/BOJ/PBOC）。
- **货币政策**：利率、QE/QT 等。
- **QE / QT** ⭐：量化宽松（印钞）/紧缩。
- **前瞻指引 / Forward Guidance**：央行沟通暗示政策。
- **鹰派 / 鸽派**：倾向收紧/宽松。
- **汇率制度**：固定/浮动/钉住等。
- **不可能三角 / 三元悖论** ⭐：固定汇率、资本自由、独立货币政策三选二。
- **联系汇率**：本币严格钉住外币（港币）。
- **外汇干预 / Intervention**：央行买卖本币影响汇率。
- **外汇储备 / FX Reserves**：央行干预的弹药。

## 风险
- **汇率风险 / 杠杆风险 / 交收风险 / 流动性风险**。
- **零售外汇 / Retail FX** ⭐：散户炒外汇（亏损率极高）。
- **差价合约 / CFD**：不实际交割的杠杆产品。
- **隔夜利息 / Swap Fee**：持仓过夜的利息成本。
- **跳空 / Gap**：价格跳跃（止损可能失效）。

## 现代演化
- **稳定币 / Stablecoin** ⭐：链上锚定法币（民间跨境美元）。
- **CBDC** ⭐：央行数字货币。
- **去美元化 / De-dollarization**：寻求美元替代。
- **SWIFT**：跨境银行报文系统（制裁战场，详见后续支付模块）。
- **FX Global Code**：FX 行业自律准则。
- **cross-currency basis**：CIP 的持续偏离。
