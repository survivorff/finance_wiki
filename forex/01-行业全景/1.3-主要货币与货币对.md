# 1.3 主要货币与货币对

> 深度：🟢 入门
> 理解哪些货币重要、货币对怎么分类，是看懂 FX 行情的基础。

## 美元霸权 ⭐

```
美元(USD)是绝对的中心货币:
  - 全球大部分外汇交易都涉及美元(一边是美元)
  - 国际贸易、大宗商品(石油等)主要用美元计价
  - 各国央行储备的主要货币是美元
→ "美元本位"的全球货币体系
```

- 💡 **这给了美国独特的"嚣张特权"**：能印全球都需要的货币、美元政策影响全世界。
- 美元强弱是全球资金流动的总开关——美元走强，资金回流美国，新兴市场承压（见 [05](../05-央行与政策/README.md)）。

## 货币对分类 ⭐

```
主要货币对(Majors): 含美元的主流货币对, 流动性最好、点差最小
  EUR/USD(最大)、USD/JPY、GBP/USD、USD/CHF、AUD/USD、USD/CAD

交叉盘(Crosses): 不含美元的货币对
  EUR/JPY、GBP/JPY、EUR/GBP...
  历史上要通过美元换算, 现在可直接交易

新兴市场货币对(EM/Exotics): 含新兴市场货币, 点差大、波动大、流动性差
  USD/TRY(土耳其里拉)、USD/ZAR(南非兰特)、USD/MXN...
```

- 💡 **EUR/USD 是全球交易量最大的货币对**——欧美两大经济体的货币，流动性极深。

## 货币的"性格"分类 💡

不同货币有不同的市场角色：

| 类型 | 特点 | 例子 |
| --- | --- | --- |
| **避险货币** | 危机时资金涌入（被认为安全） | USD、JPY、CHF |
| **商品货币** | 与大宗商品价格相关 | AUD、CAD、NOK |
| **高息货币** | 利率高，套息交易目标（见 [04](../04-汇率原理/README.md)） | 部分新兴市场货币 |
| **储备货币** | 各国央行储备持有 | USD（主导）、EUR |

- 💡 **避险逻辑**：市场恐慌时，资金从高风险货币流向避险货币（日元、瑞郎、美元）——这是 FX 独有的「risk-on/risk-off」情绪切换。

## 离岸货币 💡（以人民币为例）

```
人民币有两个市场:
  CNY(在岸): 中国大陆境内, 受管制
  CNH(离岸): 香港等地, 相对市场化
→ 因为资本管制, 在岸和离岸汇率可能略有差异
→ 对照: 加密的"同一币不同市场价差"(见 CEX 跨所套利)
```

## 小结

- 美元是绝对霸权（多数交易涉及美元，全球计价/储备主导），赋予美国"嚣张特权"。
- 货币对分主要（含美元，流动性最好）、交叉盘（不含美元）、新兴市场（点差大波动大）。
- EUR/USD 是最大货币对；货币有避险/商品/高息/储备等"性格"。
- 避险货币（USD/JPY/CHF）在危机时受追捧，体现 FX 的 risk-on/risk-off 切换。
