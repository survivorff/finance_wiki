## 核心收获

- 外汇是交易「货币对」的市场，汇率是两种货币的相对价格。
- 全球最大市场（日交易量 7-8 万亿美元），但没有中央交易所，是 OTC 银行间网络。
- 美元是绝对霸权（多数交易对都含美元）。
- FX 历史是「货币制度演化史」——从金本位到浮动汇率。
