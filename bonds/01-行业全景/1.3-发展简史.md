# 1.3 发展简史

> 深度：🟢 小白友好
> 债券史 = 国家融资史，尤其和战争紧密相连。理解历史，才懂国债为什么是金融的基石。

## 时间线总览

```
中世纪意大利城邦: 早期政府债务(威尼斯、佛罗伦萨为战争借钱)
17世纪 荷兰/英国: 现代国债诞生(英格兰银行 1694, 为战争融资)
18-19世纪: 国债市场成熟, 罗斯柴尔德等债券家族崛起
20世纪: 战争债(一战二战靠发债融资)、布雷顿森林后美债崛起
1970s-80s: 高通胀 → 利率飙升 → 债券大熊市; 后利率长期下行
2008: 金融危机, 央行 QE 大量买债(见 05)
2020: 疫情, 利率降至零/负, 央行天量购债
2022~: 通胀回归, 央行激进加息 → 债券大跌、全球资产重估(见 07)
```

## 关键节点

### 国债与战争 ⭐
```
历史上, 国债主要为"战争"融资:
  打仗烧钱 → 政府收的税不够 → 发债向公众借钱
  英格兰银行(1694)就是为给英国政府筹战争资金而设立
→ "现代国债 + 中央银行"制度, 起源于国家的战争融资需求
```

- 💡 **深刻**：债券和国家权力深度绑定——能发债借到钱的国家，才能打得起仗、搞得起建设。国债是"国家信用"的金融化。

### 罗斯柴尔德与债券家族
- 19 世纪，罗斯柴尔德家族通过跨国债券承销成为金融巨头。
- 💡 早期的"债券承销"≈ 现代 [投行承销 IPO](../../stocks/05-发行与公司行动/5.1-IPO上市.md)，只是承销的是国家债务。

### 1970s-80s 大通胀与债券熊市 ⭐
```
1970s: 高通胀 → 利率飙升(美联储一度加息到近 20%)
  → 债券价格暴跌(利率涨债券跌, 见 03) → 债券大熊市
1980s后: 通胀被控制, 利率开启 40 年长期下行
  → 债券长期大牛市(利率跌债券涨)
```

- 💡 这段历史是理解 [债券价格与利率反向](../03-利率与定价/README.md) 的最好案例。

### 2008 与 2020：QE 时代 ⭐
```
央行为救经济, 大规模"印钞买债"(QE, 见 FX 5.1):
  → 压低利率、注入流动性
  → 利率降到历史低位(甚至负利率!)
  → 资产价格狂欢(低利率=万物上涨, 见 07)
```

### 2022：利率回归，万物重估 ⭐
```
通胀回归 → 央行激进加息 → 利率飙升
  → 债券大跌(40 年牛市终结)
  → 股票、加密同步暴跌(万物重估, 见 07)
→ 这是"利率是万物之锚"最生动的近期实证
```

## 负利率：颠覆认知的现象 💡

```
2010s 部分国家出现"负利率":
  → 你借钱给政府, 到期拿回的比借出的还少(利率为负!)
  → 反常识: 为什么有人愿意"付费"借钱给政府?
  原因: 避险需求、央行政策、通缩预期、监管要求持有国债
→ 负利率是货币政策走到极端的产物, 也催生了对"另类资产"(含加密)的兴趣
```

## 贯穿全史的主线 ⭐

> **债券史 = "国家信用"的金融化史 + 利率的长期周期史**：
> - 国债把"国家信用"变成可交易的资产，是现代金融的基石。
> - 利率有长周期（1980s-2020 的 40 年下行，2022 的反转），驱动了资产价格的长期大潮。
>
> 💡 **理解利率的长周期，就理解了资产价格的"大时代"**：
> 40 年利率下行 = 资产价格的长期牛市（包括最后催生了加密的狂热）；
> 利率反转 = 一个时代的结束。这是宏观投资者最关心的"大局"。

## 小结

- 债券史 = 国家融资史，尤其与战争紧密相连（英格兰银行为战争融资而设）。
- 国债是"国家信用的金融化"，能发债的国家才有实力。
- 1970s-80s 高通胀引发债券熊市，之后 40 年利率下行（债券长牛）。
- 2008/2020 QE 时代利率降至零/负（资产狂欢），2022 加息引发万物重估。
- 主线是"国家信用金融化 + 利率长周期"——理解利率长周期就理解资产价格的大时代。
