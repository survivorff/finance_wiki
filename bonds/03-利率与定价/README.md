## 核心收获 ⭐

```
1. 债券价格 = 未来所有现金流(票息+本金)的"现值之和"
2. 折现率 = 市场利率 → 利率涨, 现值跌, 债券价格跌(反向!)
3. 收益率(yield) = 市场给债券定价后隐含的"真实回报率"
4. 久期(duration) = 债券价格对利率变动的敏感度(久期越长, 利率风险越大)
```

> ⭐ **最反直觉、最核心的一句话**：**利率涨，债券价格跌。**
> 因为已发行债券的票息是固定的，市场利率涨了，你这张低息债就"不香了"，价格得跌到让它的收益率追上市场利率。理解这个，就懂了债券的一切。
