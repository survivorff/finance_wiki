# 2.1 按发行人分类

> 深度：🟢🟡
> 谁发的债，决定了它的信用风险和收益率。这是债券最重要的分类。

## 信用风险阶梯 ⭐

```
最安全(收益率最低) ──────────────────────────▶ 最危险(收益率最高)
主权国债(发达国家) → 机构债 → 投资级企业债 → 高收益债 → 新兴市场/困境债
   信用风险递增, 收益率递增(高风险要高回报补偿)
```

## 一、政府债 / 国债（Government Bonds）⭐

```
发行人: 国家政府
信用: 发达国家国债被视为"无风险"(政府能征税、能印钞还债)
代表: 美国国债(Treasuries)、中国国债、德国国债(Bunds)、日本国债(JGB)
分类(按期限):
  短期(国库券 T-Bills): 1 年以内, 通常零息(折价发行)
  中期(T-Notes): 2-10 年
  长期(T-Bonds): 10-30 年
```

- 💡 国债收益率是 [无风险利率基准](../07-利率与万物/README.md)——一切资产定价的锚。
- ⚠️ "无风险"是相对的：本国货币国债信用风险低（能印钞还），但有[利率风险、通胀风险](../06-风险与危机/README.md)；外币债或弱国国债仍可能违约。

## 二、企业债（Corporate Bonds）

```
发行人: 公司(借钱不稀释股权, 见 1.1)
信用: 取决于公司财务, 由评级机构打分
分两档:
  投资级(Investment Grade): 评级高(如 AAA~BBB), 违约风险低, 收益率较低
  高收益债/垃圾债(High Yield/Junk): 评级低(BB 及以下), 违约风险高, 收益率高
```

- 💡 企业债收益率 = 国债收益率 + 信用利差（补偿违约风险）。

## 三、市政债（Municipal Bonds）

```
发行人: 地方政府/市政机构(修桥铺路、公共项目)
特点: 部分有税收优惠(如美国市政债利息免联邦税)
信用: 取决于地方财政
```

## 四、其他

| 类型 | 说明 |
| --- | --- |
| **机构债** | 政府支持机构发行（如房利美/房地美的住房抵押相关） |
| **超主权债** | 国际组织发行（世界银行等） |
| **新兴市场债** | 新兴国家发行，收益率高、风险高（汇率+信用，见 [FX 5.3](../../forex/05-央行与政策/5.3-干预与货币危机.md)） |

## 信用评级（Credit Rating）⭐

```
评级机构(穆迪 Moody's、标普 S&P、惠誉 Fitch)给债券打分:
  AAA(最高) → AA → A → BBB → BB → B → CCC → D(违约)
  BBB- 及以上 = 投资级; 以下 = 高收益/垃圾债
作用: 帮投资者判断违约风险, 影响收益率
```

- ⚠️ **评级机构的争议**（2008 教训）：评级机构曾给次贷相关产品过高评级（AAA），误导投资者，是 [2008 危机](../06-风险与危机/README.md) 的重要原因之一。**评级不等于真相**，评级机构有利益冲突（发行人付费评级）。
- 💡 对照 [加密的审计/评级](../../crypto/dex/06-风险与安全/6.2-智能合约安全.md)：都是"第三方背书"，都有"背书未必可靠"的问题（审计过的合约仍被黑，评级 AAA 仍违约）。

## 信用利差（Credit Spread）⭐ 债市的"恐慌指数"

```
信用利差 = 企业债收益率 − 同期限国债收益率
  → 衡量"市场对该发行人违约风险的定价"
利差扩大: 市场恐慌(怕违约) → 要求更高补偿 → 风险资产承压
利差收窄: 市场乐观 → 风险偏好上升
→ 信用利差是判断市场风险情绪的关键指标(债市版的"恐慌/贪婪")
```

- 💡 对照 [FX 的 risk-on/risk-off](../../forex/04-汇率原理/4.1-什么决定汇率.md)、[加密情绪](../../crypto/cex/05-市场与博弈/5.5-资金流动全景.md)：信用利差是债市表达"风险情绪"的方式。利差骤然扩大常是危机前兆。

## 小结

- 按发行人分：国债（最安全）→机构债→投资级企业债→高收益债→新兴市场债，风险和收益率递增。
- 国债收益率是无风险利率基准；企业债=国债+信用利差。
- 信用评级（AAA→D）帮判断违约风险，但评级机构有利益冲突、不等于真相（2008 教训）。
- 信用利差是债市的"恐慌指数"——扩大=怕违约、市场恐慌，收窄=乐观。
