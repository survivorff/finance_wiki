# 6.2 债务危机案例

> 深度：🟡 → 🔴
> 债券/债务引发的危机往往是系统性的。这些案例揭示"安全资产"如何变成危机的导火索。

## 一、主权债务危机

### 机制
```
主权债务危机: 一国政府还不起债(尤其外币债):
  1. 政府借太多债(财政赤字累积)
  2. 经济恶化 / 利率上升 → 偿债成本飙升
  3. 市场担忧违约 → 抛售该国债 → 收益率飙升(价格跌)
  4. 借新还旧成本暴增 → 恶性循环 → 违约/求援
```

### 案例
```
欧债危机(2010-2012): 希腊等南欧国家债务危机
  → 希腊国债收益率飙升、濒临违约 → 欧盟/IMF 救助 + 紧缩
  → 暴露了"统一货币(欧元)但财政不统一"的缺陷
新兴市场: 阿根廷(多次违约)、其他高外债国家反复上演(见 FX 5.3)
```

- 💡 对照 [FX 货币危机](../../forex/05-央行与政策/5.3-干预与货币危机.md)：主权债务危机和货币危机常常孪生——弱国货币贬值 + 外币债负担暴增 + 资本外逃，互相强化。

### 本币债 vs 外币债的关键区别 ⭐
```
本币债(能印钞还): 不会"名义违约"(大不了印钞), 但印钞→通胀/贬值(软违约)
外币债(不能印别国钞): 真的可能违约(还不出美元就是还不出)
→ 这就是为什么"借外债"对弱国特别危险(失去了印钞这个最后手段)
```

## 二、2008 金融危机（债券是核心）⭐

```
2008 本质是一场"债券/信用危机":
  1. 次贷(给信用差的人的房贷)被打包成 MBS/CDO(见 2.2)
  2. 评级机构给这些产品高评级(AAA), 误导投资者(见 2.1)
  3. 房价跌 → 次贷违约潮 → MBS/CDO 价值暴跌
  4. 持有这些"安全资产"的机构巨亏
  5. Repo 市场冻结(见 5.3) → 流动性危机 → 雷曼倒闭 → 全球危机
教训: "把垃圾打包+评级+分层" 不能消灭风险, 只是隐藏和放大
```

- 💡 **核心教训**：被认为"安全"的资产（AAA 的 MBS）恰恰是危机的核心。**当"安全"是被包装出来的假象时，崩塌最猛烈。**
- 💡 对照 [FTX"安全"叙事崩塌](../../crypto/cex/06-风险与监管/6.4-经典案例复盘.md)、[算法稳定币崩盘](../../crypto/defi/README.md)：都是"看似安全→实则脆弱→信心崩塌→踩踏"。

## 三、利率冲击：2022 与英国养老金危机 ⭐

```
2022 利率飙升(央行加息), 引发:
1. 债券全面大跌(久期风险, 见 3.3) — 连"安全"国债基金巨亏
2. 英国养老金危机(LDI 危机):
   养老金用了"负债驱动投资(LDI)"+ 杠杆(借 Repo 持国债)
   → 英国国债收益率突然飙升 → 国债价格暴跌
   → 杠杆头寸被追加保证金 → 被迫抛售国债 → 收益率更飙升(踩踏)
   → 英格兰银行紧急入市买债救场
```

- 💡 **这是"利率风险 + 杠杆 + 踩踏"的经典**：连最保守的养老金，用了杠杆后也在利率冲击下濒临崩溃。
- 💡 对照 [CEX 连环爆仓](../../crypto/cex/03-技术架构/3.5-清算与风控.md)、[FX 套息踩踏](../../forex/04-汇率原理/4.3-套息交易.md)：**杠杆 + 拥挤 + 价格冲击 = 踩踏**，是跨所有市场的系统性风险通用公式（再次出现）。

## 四、债务的长期问题：政府债务累积 💡

```
全球政府债务持续累积(尤其发达国家):
  低利率时: 借债成本低, 债务可持续
  利率上升时: 偿债成本(利息)飙升 → 财政压力 → 可能的长期风险
担忧: 高债务 + 高利率 → 债务可持续性问题
  极端出路: 通胀稀释债务、央行印钞买债(赤字货币化, 见 5.2)、违约/重组
```

- 💡 这是 [加密"硬通货"叙事](../01-行业全景/1.3-发展简史.md) 的宏观背景——对"政府债务无限膨胀 + 央行印钞稀释"的担忧，推动了对比特币等"抗通胀"资产的兴趣。

## 危机的共同模式（再次印证）⭐

```
债务危机的配方(和股票/加密危机同构, 见 股票 6.3):
  过度债务/杠杆 + 利率上升/触发点 + 拥挤交易 + 信心崩塌 → 踩踏/违约
→ 无论债券、股票、FX、加密, 系统性危机的剧本惊人相似
```

## 小结

- 主权债务危机：政府借太多+经济恶化+利率升→抛售→收益率飙升→恶性循环（欧债/阿根廷）。
- 本币债能印钞还（软违约/通胀），外币债真会违约——借外债对弱国尤其危险。
- 2008 本质是债券/信用危机（次贷→MBS/CDO→评级误导→违约潮→Repo 冻结），"被包装的安全"崩塌最猛。
- 2022 利率冲击引发债券大跌和英国养老金 LDI 危机（利率+杠杆+踩踏）。
- 危机配方（过度杠杆+利率/触发点+拥挤+信心崩→踩踏）跨债券/股票/FX/加密惊人相似。
