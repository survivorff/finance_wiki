# 6.1 风险地图

> 深度：🟡 进阶
> FX 的风险类型，对照其他市场。

## 风险全景

```
1. 汇率风险(市场风险): 汇率波动导致损失(最基本)
2. 杠杆风险: 高杠杆放大盈亏, 小波动就爆仓(见 6.2)
3. 利率风险: 利率变动影响套息和远期(见 4.3)
4. 交收风险: 赫斯塔特风险(见 3.3, 由 CLS 缓解)
5. 流动性风险: 重大事件时点差暴涨、流动性枯竭(见 3.1)
6. 对手方/信用风险: OTC 双边交易, 对手违约(银行间靠授信, 零售靠经纪商信用)
7. 国家/政治风险: 资本管制、政变、制裁导致货币冻结或暴跌
```

## 各风险详解

### 汇率风险（企业最关心）
- 企业有真实汇率敞口（出口收外币），用 [远期/期权对冲](../02-参与者与产品/2.2-即期与远期.md)。
- 投机者主动承担汇率风险博收益。

### 杠杆风险 ⭐
- FX 杠杆极高（50:1、100:1 甚至更高），见 [6.2](./6.2-零售外汇陷阱.md)。
- 对照 [CEX 合约强平](../../crypto/cex/03-技术架构/3.5-清算与风控.md)：同样的"保证金不足→强平"逻辑。

### 黑天鹅与流动性枯竭 ⚠️
```
极端事件(央行突然行动、地缘冲突): 
  → 流动性瞬间枯竭、点差暴涨、跳空(价格跳跃)
  → 止损可能失效(跳过止损价直接成交在更差的价)
著名: 2015 年瑞士央行突然放弃瑞郎上限 → 瑞郎瞬间暴涨 30%+
  → 大量交易者爆仓, 一些经纪商倒闭
```

- 💡 对照 [加密插针/极端行情](../../crypto/cex/05-市场与博弈/5.4-市场操纵与黑幕.md)：FX 同样有"跳空、止损失效、瞬间爆仓"——杠杆市场的共同风险。

### 国家/政治风险（FX 独有）
```
FX 交易的是"国家货币", 所以有股票/加密较少的风险:
  - 资本管制: 突然限制资金进出 → 钱被困
  - 制裁: 货币被冻结、被踢出 SWIFT(见后续支付模块)
  - 政治动荡: 政变/战争 → 货币暴跌
```

## 风险对照表 ⭐

| 风险 | FX | 股票 | 加密 |
| --- | --- | --- | --- |
| 市场波动 | 汇率 | 股价 | 币价 |
| 杠杆爆仓 | ⭐ 极高杠杆 | 融资融券 | 合约 |
| 交收 | 赫斯塔特/CLS | CCP/DVP | 链上原子 |
| 国家/政治 | ⭐ 突出（货币=主权） | 较少 | 监管打击 |
| 平台风险 | 黑经纪商 | 受监管券商 | 交易所跑路 |

## 小结

- FX 风险：汇率、杠杆、利率、交收（赫斯塔特）、流动性、对手方、国家/政治。
- 杠杆风险突出（50-100x），黑天鹅时流动性枯竭+跳空+止损失效（2015 瑞郎暴涨是经典）。
- 国家/政治风险是 FX 独有（货币=主权，有资本管制/制裁/动荡风险）。
- 与股票/加密的风险结构相通，但杠杆和国家风险尤为突出。
