# 2.3 买方与卖方

> 深度：🟡 进阶
> 华尔街把金融机构分成「买方」和「卖方」——这个划分能帮你看清谁在股市里扮演什么角色。

## 买方 vs 卖方（核心概念）⭐

```
卖方(Sell Side): "卖服务/产品"给投资者的机构
  - 投行、券商、研究机构
  - 卖什么: 承销发行、经纪交易、研究报告、做市
  - 收入: 佣金、承销费、价差

买方(Buy Side): 用钱"买资产做投资"的机构
  - 基金、资管、对冲基金、养老金、保险
  - 做什么: 管理资金, 买卖证券赚投资回报
  - 收入: 管理费 + 业绩提成
```

> 💡 记忆法：**卖方卖"服务"（赚手续费/佣金），买方买"资产"（赚投资收益）。**
> 一个投研报告分析师在卖方（卖研究），一个基金经理在买方（用研究做投资决策）。

## 主要参与者

### 买方机构
| 类型 | 特点 |
| --- | --- |
| **共同基金 / 公募** | 面向大众，受严格监管，如指数基金 |
| **对冲基金** | 面向富人/机构，策略灵活（做多做空、杠杆、衍生品） |
| **养老金 / 主权基金** | 巨量长期资金，风格保守 |
| **保险公司** | 用保费做投资 |
| **私募 / VC / PE** | 投未上市公司（一级市场为主） |

### 卖方机构
| 类型 | 特点 |
| --- | --- |
| **投资银行(投行)** | 承销 IPO/增发、并购顾问、做市（见 [2.1 券商](./2.1-交易所与券商.md)、[05 IPO](../05-发行与公司行动/README.md)） |
| **券商** | 经纪交易、研究 |
| **研究机构** | 卖方分析师，出研报、评级、目标价 |

### 散户（Retail）
- 个人投资者。信息、工具、资金都弱于机构。
- 💡 和 [CEX/DEX 的散户](../../crypto/cex/05-市场与博弈/5.2-多方博弈格局.md) 一样，**散户在股市同样是食物链相对底层**——但股市有信息披露、投资者保护、禁止操纵等制度，对散户的保护远强于加密市场。

## 投资银行（Investment Bank）的特殊地位 💡

投行是卖方的核心，做几件大事：
- **承销（Underwriting）**：帮公司 IPO/增发，把股票卖给投资者，收承销费（见 [05](../05-发行与公司行动/5.1-IPO上市.md)）。
- **并购顾问（M&A）**：帮公司买卖、合并。
- **做市与交易**：自营和代客交易。
- **研究**：发布行业/公司研究。

> 💡 对照 [CEX 的上币 + Launchpad](../../crypto/cex/02-产品体系/2.6-生态产品.md)：
> 投行承销 IPO ≈ 交易所帮项目方发币上市。都是「帮发行方把证券/代币卖给公众并收费」。
> 区别：投行承销受严格监管（尽职调查、招股书、责任追究），加密发币长期是灰色地带。

## 机构 vs 散户的博弈（同三个市场）

```
机构优势: 信息(研究/调研)、资金、工具(算法/衍生品)、低成本、纪律
散户劣势: 信息滞后、情绪化、成本高、易追涨杀跌
→ 长期看, 散户直接博弈难胜机构(同 CEX/DEX)
→ 这也是"指数基金被动投资"流行的原因(打不过就买整个市场, 见 07)
```

> 💡 但股市给散户的「保护」远好于加密：
> - 信息披露让散户能看到公司真实情况（不像加密很多项目啥都不披露）。
> - 禁止内幕交易/操纵（虽不能根除，但有威慑和追责）。
> - 投资者保护基金兜底券商失败。

## 小结

- 卖方卖服务（投行/券商/研究，赚手续费），买方买资产（基金/资管/对冲，赚投资收益）。
- 买方：公募/对冲/养老金/保险/私募；卖方核心是投行（承销/并购/做市/研究）。
- 投行承销 IPO ≈ 加密交易所帮项目方发币，但前者受严格监管。
- 散户在股市同样相对弱势，但受信息披露、禁操纵、保护基金等制度保护，远好于加密。
