# 1.4 产业链分工

> 深度：🟡
> CEX 不是孤岛，它处在一条完整产业链的中心。理解上下游，才能看懂资金怎么流入流出。

## 产业链全景图

```
上游                          CEX 核心                       下游/配套
──────────────────────────────────────────────────────────────────────
项目方/发币方  ──上币──▶    ┌──────────────┐    ◀──做市──   做市商(MM)
                           │              │
法币支付通道  ──入金──▶     │   交易所平台   │    ──行情──▶   数据/行情商
(银行/三方支付)             │              │                (CMC/CoinGecko)
                           └──────────────┘    ◀──托管──   托管/钱包技术商
散户/机构     ──交易──▶          │ ▲                        (Fireblocks等)
                               │ │
量化/套利基金 ──流动性─▶         │ │
                           监管/审计/储备金证明机构
```

## 各生态位详解

### 1. 项目方 / 发币方（上游：内容供给）
- 想让自己的代币能被买卖，需要在 CEX「**上币（Listing）**」。
- 头部所上币 = 巨大流量和价格利好，所以项目方挤破头想上币。
- 这催生了 CEX 重要的权力和收入（上币费），也是灰色地带（详见 [04 运营](../04-运营体系/README.md)）。

### 2. 做市商（Market Maker, MM）⭐ 流动性的命脉
- 在交易对盘口两侧持续挂买卖单，保证「有人接盘、价差小、深度好」。
- 没有做市商，新上的币就是「一潭死水」，没人能正常买卖。
- 分两类：
  - **专业做市商**（如 Wintermute, GSR, Jump Crypto）：服务多家交易所和项目方。
  - **交易所关联做市商**：交易所自己或关联方做市（FTX-Alameda 就是反面教材）。
- 交易所用**返佣、低费率、API 优待**抢顶级做市商。详见 [04 做市运营](../04-运营体系/4.3-做市与流动性运营.md)。

### 3. 法币通道（On/Off Ramp）⭐ 资金的水龙头
- 银行、第三方支付（如 MoonPay, Simplex）、C2C 网络，决定用户能否顺畅出入金。
- **这是合规最敏感、最容易被掐断的环节**：银行一旦拒绝服务加密交易所，整个法币通道就断了。
- 很多交易所被监管打击，第一刀就是切断它的银行通道。

### 4. 托管 / 钱包技术商
- 如 **Fireblocks、Copper、BitGo**，提供机构级私钥管理（MPC、多签）。
- 中小交易所往往不自建钱包系统，而是采购这些服务商的方案。
- 头部所（Binance、Coinbase）多为自研。

### 5. 行情 / 数据商
- **CoinMarketCap（被币安收购）、CoinGecko**：面向散户的价格聚合。
- **Kaiko、Amberdata、Coinglass**：面向机构的专业市场数据。
- 它们聚合并分发各交易所数据，也是「交易量排名」的裁判（所以交易所有动机刷量讨好它们）。

### 6. 量化 / 套利基金（特殊的「用户」）
- 跨所套利、资金费率套利、高频交易。
- 它们既是交易所的大客户（贡献交易量），又是市场效率的推动者，还常常是散户的对手盘。

### 7. 监管 / 审计 / 安全
- 牌照发放机构、储备金证明审计方、智能合约/安全审计公司（如 CertiK、慢雾 SlowMist）。

## 资金从哪来、到哪去（产业链视角）

```
法币(用户的钱) ──通道──▶ 进入CEX ──交易/合约──▶ 在用户间/与做市商间流动
                                            │
                                            ├─▶ 手续费 → 交易所收入
                                            ├─▶ 价差 → 做市商收入
                                            ├─▶ 资金费率 → 多空之间转移
                                            └─▶ 提币 → 流向链上/其他平台
```

> 💡 **关键洞察**：在这条链上，**交易所是「抽水」的一方**——无论用户赚还是亏，
> 交易所通过手续费稳定抽成。这就是为什么「卖铲子的最赚钱」。这个商业本质在 [05 盈利模式](../05-市场与博弈/5.1-盈利模式拆解.md) 详谈。

## 小结

- CEX 处在产业链中心，上游是项目方和法币通道，下游是做市商、数据商、托管商。
- 做市商和法币通道是两条「命脉」：前者决定流动性，后者决定资金能否进出。
- 在整条产业链里，交易所是稳定「抽水」的一方，旱涝保收。
