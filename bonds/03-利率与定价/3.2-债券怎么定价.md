# 3.2 债券怎么定价

> 深度：🟡 → 🔴
> 债券价格 = 未来现金流的现值之和。这是金融估值最基础的原理——现金流折现（DCF）。

## 核心原理：现金流折现（DCF）⭐

```
债券价格 = 把"未来要收到的所有钱"(票息 + 到期本金)
          按"市场利率"折算成"今天的价值"(现值), 再加总
```

### 为什么要"折现" ⭐
```
"今天的 100 元" > "1 年后的 100 元"
  因为: 今天的 100 元可以存银行/投资赚利息
  → 未来的钱要"打折"换算成今天的价值 → 这就是"折现"
折现率 = 市场利率(你的钱本可以赚到的回报率)
```

- 💡 **这是整个金融估值的基石**：不只债券，[股票估值](../07-利率与万物/README.md)、项目投资、一切"未来现金流"的资产，都用这个原理定价。

## 定价公式 ⭐

```
价格 P = Σ [ 每期现金流 / (1 + r)^t ]

P = C/(1+r)^1 + C/(1+r)^2 + ... + C/(1+r)^n + F/(1+r)^n
其中:
  C = 每期票息
  F = 面值(到期还的本金)
  r = 折现率(市场利率/收益率)
  n = 期数
  t = 第几期
```

### 算例
```
债券: 面值 100, 票息率 3%(每年 3 元), 3 年期, 市场利率 r = 5%
P = 3/(1.05)^1 + 3/(1.05)^2 + 3/(1.05)^3 + 100/(1.05)^3
  = 2.857 + 2.721 + 2.591 + 86.384
  ≈ 94.55
→ 因为市场利率(5%) > 票息率(3%), 债券折价(94.55 < 100)
  验证了 3.1: 利率高于票息率 → 折价
```

## 折现率怎么定 ⭐

```
折现率 r = 无风险利率 + 风险溢价
  无风险利率: 同期限国债收益率(见 02、07)
  风险溢价: 信用利差(违约风险, 见 2.1) + 流动性溢价等
→ 越risky 的债, 折现率越高 → 同样现金流, 价格越低
```

- 💡 **这把所有债券的定价统一了**：都是"未来现金流 ÷ (无风险利率 + 风险补偿)"。无风险利率（国债）是地基，风险越高加的越多。

## 收益率 = 反解出来的折现率 💡

```
实务中: 已知"市场价格", 反解出"折现率" → 这个折现率就是 YTM(到期收益率, 见 3.1)
  价格 ⇄ 收益率 是一一对应的(给定价格能算收益率, 反之亦然)
  债市报价主要报"收益率", 因为它更直观可比
```

## 为什么"利率涨→价格跌"（数学解释）⭐

```
看定价公式: P = Σ C/(1+r)^t + F/(1+r)^n
  r(折现率/利率)↑ → 分母 (1+r)^t ↑ → 每一项现值↓ → 价格 P↓
  → 利率涨, 价格跌(数学上必然, 不是市场情绪)
反之 r↓ → 价格↑
→ 这就是 3.1 那个反向关系的数学本质
```

## 对照其他市场的估值 💡

```
债券定价(DCF) 是金融估值的"母版":
  股票估值: 未来股息/现金流折现(DDM/DCF) — 同样的折现思想
  房产: 未来租金折现
  加密: 大多没有现金流 → 难用 DCF → 估值更靠"叙事/供需"(这是加密估值难的根源)
→ 有现金流的资产能 DCF 定价(债/股/房), 无现金流的(多数加密)只能靠供需博弈
```

- 💡 **这解释了为什么加密估值这么难、波动这么大**：大多数加密资产没有现金流，无法用 DCF 这种"客观锚"定价，只能靠供需和叙事——所以更投机、更波动。而债券有明确现金流，估值最"硬"。

## 小结

- 债券价格 = 未来所有现金流（票息+本金）按市场利率折现的现值之和（DCF）。
- 折现因为"今天的钱比未来的钱值钱"；折现率 = 无风险利率 + 风险溢价。
- 价格与收益率一一对应（反解折现率即 YTM）。
- 利率涨→分母变大→现值变小→价格跌（数学必然，是 3.1 反向关系的本质）。
- DCF 是金融估值母版（债/股/房都用），无现金流的加密难 DCF 定价，所以更靠供需叙事、更波动。
