# 02 · 产品体系

> 深度：🟢→🔴 从入门到进阶
> 目标：把 CEX 卖给用户的每一个产品都拆开讲透——是什么、怎么用、怎么设计、怎么赚钱、有什么坑。

CEX 不是一个产品，是一个**产品矩阵**。从最简单的现货买卖，到复杂的期权、结构化理财，覆盖了几乎传统金融的所有形态。本模块逐个拆解。

## 一个贯穿全模块的认知 ⭐

CEX 的产品矩阵不是随便堆的，而是一个精心设计的**「用户价值漏斗」**：

```
一键买币(小白入门) → 现货(学会交易) → 理财(沉淀资金) → 合约(高频高费) → 平台币/生态(绑定)
        低风险低门槛 ──────────────────────────────▶ 高风险高利润
```

每一层产品都在把用户**往「更高频、更高费率、更深绑定」的方向推**。理解这个漏斗，就理解了 CEX 的产品战略。详见 [2.7 产品设计哲学](./2.7-产品设计哲学.md)。
