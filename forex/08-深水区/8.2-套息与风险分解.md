# 8.2 套息与风险分解

> 深度：🔴🔴 极硬核
> 把套息交易（carry trade）的收益和风险用数学拆开，理解它"赚小亏大"的本质。
> 前置：[4.3 套息交易](../04-汇率原理/4.3-套息交易.md)。

## 套息收益分解 ⭐

```
持有"高息货币 vs 低息货币"头寸的总收益:
  总收益 ≈ 利率差(carry) + 汇率变动(spot return)
        = (r_high − r_low) + ΔS/S

平时(汇率平稳): 总收益 ≈ 利率差 → 稳定的正收益("捡钱")
危机(汇率剧变): ΔS/S 巨大负值 → 总收益巨亏(吞掉多年利差)
```

## 为什么"赚小亏大"（负偏态）⭐

```
收益分布特征:
  大多数时间: 小幅正收益(利差累积, 像收租)
  少数极端时刻: 巨幅负收益(高息货币暴跌 + 套息平仓踩踏, 见 4.3)
→ 收益分布"负偏态(negative skew)": 长长的左尾(大亏)
→ 用正态分布的"夏普比率"会高估套息的吸引力(掩盖了尾部风险)
```

> ⭐ **这是套息的本质风险**：它的高夏普比率（平时稳定）是"假象"，因为标准差没捕捉到罕见的巨亏尾部。
> 用一句话说：**套息是"在压路机前捡钢镚"——平时捡得很爽，被压一次全没。**

## 套息 ≈ 卖出波动率 ⭐⭐

```
套息的收益结构 ≈ 卖出期权/卖出保险:
  卖期权: 平时收权利金(稳定小收益), 极端时巨额赔付
  套息:   平时赚利差(稳定小收益), 危机时巨亏
→ 两者数学上同构: 都是"承担尾部风险, 换取平时的稳定收益"
```

- 💡 **这是金融最深刻的统一之一**。把这些放在一起看：
  | 策略 | 平时 | 尾部 |
  | --- | --- | --- |
  | FX 套息 | 赚利差 | 套息平仓踩踏巨亏 |
  | [卖期权](../../crypto/cex/02-产品体系/2.2-合约衍生品.md) | 收权利金 | 被行权巨亏 |
  | [DEX 当 LP](../../crypto/dex/08-深水区/8.2-无常损失数学推导.md) | 收手续费 | 无常损失/极端行情 |
  | [CEX 做市](../../crypto/cex/08-深水区/8.7-做市与市场微观结构.md) | 赚价差 | 逆向选择/库存爆炸 |
  | 卖信用保险(CDS) | 收保费 | 违约赔付 |

  > **它们全是"卖波动率/卖保险"的变体**——平时稳赚小钱，承担罕见的巨大尾部风险。
  > 理解了这个统一结构，你就理解了金融里一大类"看起来稳赚"策略的真实风险。**世界上没有免费的午餐，"稳定收益"的背后往往是"卖出了尾部保险"。**

## 套息的风险度量

```
不能只看夏普比率(被负偏态误导), 还要看:
  - 最大回撤(Max Drawdown): 历史最惨亏多少
  - 偏度(Skewness): 负偏度越大越危险
  - 尾部风险(VaR/ES): 极端损失
  - 与"风险情绪"的相关性: 套息在 risk-off 时集体爆(见 4.3)
```

## 套息的拥挤与系统性风险 💡

```
套息是"拥挤交易"(很多人同方向):
  → 平时大家一起赚利差
  → 一旦反转, 大家一起平仓 → 踩踏自我强化(见 4.3)
  → 杠杆放大踩踏 → 可能引发跨市场连锁(套息平仓 → 抛售风险资产)
→ 套息平仓是全球市场的一个系统性风险源
```

- 💡 对照 [CEX 连环爆仓](../../crypto/cex/03-技术架构/3.5-清算与风控.md)、[套息平仓踩踏](../04-汇率原理/4.3-套息交易.md)：**拥挤 + 杠杆 + 反转 = 踩踏**，是跨市场的系统性风险通用公式。

## 小结

- 套息总收益 ≈ 利率差 + 汇率变动；平时约等于利差（稳赚），危机时汇率剧变巨亏。
- 收益是负偏态（赚小亏大），夏普比率会高估吸引力（掩盖尾部）——"压路机前捡钢镚"。
- 套息 ≈ 卖波动率/卖保险，和卖期权、DEX 当 LP、CEX 做市、卖 CDS 数学同构——都是承担尾部风险换平时稳定收益。
- 要用最大回撤/偏度/尾部风险度量，而非只看夏普。
- 套息是拥挤交易，反转时踩踏（拥挤+杠杆+反转），是全球系统性风险源。
