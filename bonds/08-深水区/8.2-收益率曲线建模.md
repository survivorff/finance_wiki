# 8.2 收益率曲线建模

> 深度：🔴🔴 极硬核
> 即期利率、远期利率、如何从债券价格"剥"出收益率曲线。
> 前置：[04 收益率曲线](../04-收益率曲线/README.md)、[8.1](./8.1-债券定价与收益率.md)。

## 三种利率：别混淆 ⭐

```
1. 到期收益率(YTM): 一只"附息债"持有到期的单一回报率(见 3.1)
   → 但它"混合"了不同时间现金流的利率, 不够纯
2. 即期利率(Spot Rate / Zero Rate): "零息债"的收益率
   → 从今天到某个时点 t 的"纯利率"(不含中间票息再投资问题)
   → 不同 t 的即期利率连成"即期曲线"(zero curve), 是定价的基础
3. 远期利率(Forward Rate): 隐含的"未来某段时间"的利率
   → 例: 从即期利率推出"1 年后开始的 1 年期利率"
```

> 💡 **关键**：实务定价用"即期利率曲线"折现每一笔现金流（每笔按对应期限的即期利率），比用单一 YTM 更准。YTM 只是"综合平均"。

## 即期利率与折现 ⭐

```
用即期利率给债券定价(更精确):
  P = C/(1+z₁) + C/(1+z₂)² + ... + (C+F)/(1+zₙ)ⁿ
  其中 zₜ = 期限 t 的即期利率(每笔现金流用对应期限的即期利率折现!)
对比 8.1 用单一 y(YTM): 这里每个时点用不同的 zₜ → 更反映真实利率期限结构
```

## Bootstrapping：从债券价格剥出即期曲线 ⭐

```
问题: 市场只直接报"附息债"的价格/YTM, 怎么得到"即期利率曲线"?
方法: Bootstrapping(自举法) — 从短到长逐步"剥"出即期利率

步骤(简化):
1. 最短期(零息/国库券): 直接得到 z₁(即期利率)
2. 下一期附息债: 已知 z₁, 用它折现第一期票息,
   剩下的用方程解出 z₂
3. 再下一期: 用已知的 z₁,z₂ 折现前面的票息, 解出 z₃
4. ...逐步向后, 剥出整条即期曲线
```

```
伪代码(bootstrapping):
spot = []
for each bond (从短到长):
    # 用已剥出的 spot 折现该债前面的票息
    pv_known = sum(coupon/(1+spot[i])**(i+1) for i in range(len(spot)))
    # 剩余现值 = 价格 − 已知部分, 对应最后一期现金流
    remaining = price - pv_known
    last_cf = coupon + face
    n = 该债期数
    spot_n = (last_cf/remaining)**(1/n) - 1   # 解出该期即期利率
    spot.append(spot_n)
```

## 远期利率推导 ⭐

```
从即期利率推远期(无套利):
  (1+z₂)² = (1+z₁)(1+f₁,₂)
  → "存 2 年" 必须等于 "存 1 年 再 存 1 年(远期锁定)"(否则套利)
解出 1 年后的 1 年期远期利率:
  f₁,₂ = (1+z₂)²/(1+z₁) − 1
```

- 💡 **这和 [FX 的覆盖利率平价](../../forex/08-深水区/8.1-远期掉期定价.md) 是同一个"无套利"思想**：
  - 债券：长期即期利率 = 短期即期利率"链式相乘"远期利率。
  - FX：远期汇率 = 即期汇率 × 利率差。
  - **都是"两条路径的回报必须相等，否则套利"——无套利定价的统一逻辑。**

## 曲线的参数化模型 💡

```
离散的即期利率点 → 需要"平滑成连续曲线"用于定价:
  插值法: 线性/样条插值
  参数模型: Nelson-Siegel、Svensson(用几个参数拟合整条曲线的形状)
    → 用"水平 + 斜率 + 曲率"等因子描述曲线
央行/机构用这些模型构建和分析收益率曲线
```

- 💡 Nelson-Siegel 把曲线分解成"水平、斜率、曲率"三因子——和分析 [曲线形状（陡/平/倒挂）](../04-收益率曲线/4.2-倒挂与经济预测.md) 的直觉对应。

## 应用：曲线交易 💡

```
理解了即期/远期, 就能做"曲线交易":
  - 赌曲线变陡/变平(而非单一利率方向)
  - 远期利率 vs 实际利率的偏差套利
  - 用久期(见 8.1)中性化方向风险, 只押注曲线形状
→ 这是固收交易员的高阶玩法
```

## 小结

- 三种利率：YTM（附息债综合回报）、即期利率（零息债的纯利率，定价基础）、远期利率（隐含的未来利率）。
- 实务用即期利率曲线折现每笔现金流（比单一 YTM 更准）。
- Bootstrapping 从短到长逐步"剥"出即期曲线；远期利率由即期利率无套利推出。
- 远期利率推导（链式相乘）和 FX 覆盖利率平价是同一无套利思想。
- 曲线用 Nelson-Siegel 等参数模型平滑（水平/斜率/曲率三因子），支持曲线交易。
