# 1.2 DeFi 生态全景

> 深度：🟢🟡
> DeFi 是一个分层的生态。理解各层和它们如何叠在一起，才看懂"货币乐高"。

## DeFi 的分层版图 ⭐

```
┌─────────────────────────────────────────────┐
│ 应用层: 收益聚合、保险、资管(Yearn 等)         │ ← 把下面的乐高拼起来
├─────────────────────────────────────────────┤
│ 核心协议: DEX(交易) · 借贷 · 衍生品           │ ← DeFi 的"金融原语"
├─────────────────────────────────────────────┤
│ 资产层: 稳定币 · LSD(质押凭证) · 代币          │ ← 链上的"钱"和资产
├─────────────────────────────────────────────┤
│ 基础设施: 区块链 · 预言机 · 钱包(见 DEX 03)    │ ← 底座
└─────────────────────────────────────────────┘
```

## 用 TVL 衡量 DeFi 💡

```
TVL(Total Value Locked, 总锁仓量, 见 DEX 1.3):
  锁定在 DeFi 协议里的资产总值 → 衡量 DeFi 规模
  在 DefiLlama 等可查
注意: TVL 受币价影响大, 且"套娃"会重复计算(同一笔钱在多个协议被计多次)
```

## 可组合性的实例：一笔操作串多个协议 ⭐

```
"杠杆做多 ETH"的 DeFi 玩法(可组合):
  1. 存 ETH 到借贷协议(Aave)作抵押
  2. 借出稳定币(USDC)
  3. 去 DEX(Uniswap)把 USDC 换成 ETH
  4. 再存入 Aave 作抵押
  5. 再借 USDC... (循环 → 加杠杆)
→ 用借贷 + DEX 两个乐高, 拼出"杠杆"功能
→ 这就是货币乐高的威力(也是风险叠加, 见 05)
```

- 💡 对照 [债券回购加杠杆](../../../bonds/05-市场与参与者/5.3-回购市场.md)：DeFi 循环借贷 = Repo 加杠杆的链上版——**用抵押品反复借钱放大头寸**，逻辑完全一样。

## 各公链的 DeFi 生态 💡

```
DeFi 生态按公链分布(同 DEX 1.3):
  以太坊: DeFi 大本营(Aave/Curve/Maker 等老牌)
  L2(Arbitrum/Base): 更便宜, DeFi 迁移
  Solana: 高速低费, 自己的 DeFi 生态
  各链都有本地 DeFi 龙头
→ 跨链 DeFi + 桥连接各生态(桥是高危, 见 DEX 3.5)
```

## 小结

- DeFi 是分层生态：基础设施（链/预言机/钱包）→资产层（稳定币/LSD）→核心协议（DEX/借贷/衍生品）→应用层（收益聚合）。
- 各板块：稳定币（血液）、DEX（交易）、借贷、衍生品、收益聚合、LSD。
- TVL 衡量规模（但受币价影响、套娃重复计算）。
- 可组合性实例：借贷+DEX 循环拼出杠杆（=Repo 加杠杆的链上版）。
- 生态按公链分布（以太坊为大本营，L2/Solana 各有生态），靠桥连接。
