# 1.1 外汇与汇率是什么

> 深度：🟢 小白友好

## 外汇是什么

> **外汇（Foreign Exchange / Forex / FX）= 一种货币兑换成另一种货币。**
> 外汇市场 = 全球买卖各国货币的市场。

你出国旅游换钱、跨境网购、公司进出口结算、投资海外股票——背后都要「换汇」，都是外汇市场的一部分。

## 汇率：两种货币的「相对价格」⭐

```
汇率 = 一种货币用另一种货币表示的价格
EUR/USD = 1.10  意思是: 1 欧元 = 1.10 美元
USD/JPY = 150   意思是: 1 美元 = 150 日元
```

### 货币对的读法 ⭐
```
EUR/USD 这个"货币对":
  EUR = 基础货币(base, 前面的) — 你买卖的"那一个单位"
  USD = 报价货币(quote, 后面的) — 用来标价的

EUR/USD 上涨 = 欧元变强/美元变弱(1 欧元能换更多美元)
EUR/USD 下跌 = 欧元变弱/美元变强
```

> 💡 对照 [股票/加密的交易对](../../crypto/cex/02-产品体系/2.1-现货交易.md)：BTC/USDT 和 EUR/USD 是一样的「base/quote」结构。
> **汇率永远是"成对"的相对价格——没有"美元的绝对价格"，只有"美元相对欧元/日元/...的价格"。**

## 买和卖（双向）

```
"买入 EUR/USD" = 买欧元、卖美元(赌欧元涨)
"卖出 EUR/USD" = 卖欧元、买美元(赌欧元跌)
→ 外汇天然是"买一个货币同时卖另一个货币" — 因为你必须用某种货币去买另一种
```

- 💡 这点和 [DEX 的 swap](../../crypto/dex/02-产品与机制/2.1-AMM是什么.md) 一样——总是「用 A 换 B」，买一个必然卖另一个。

## 点差与点（Spread & Pip）

```
点(Pip): 汇率的最小变动单位, 通常是小数点后第 4 位
  EUR/USD 从 1.1000 → 1.1001 = 涨了 1 个点
点差(Spread): 买价和卖价之差(做市商赚的, 同股票/加密的 spread)
  买价 1.1000 / 卖价 1.1002 → 点差 2 个点
```

- 主要货币对点差极小（流动性极好），小众货币对点差大。

## 为什么有外汇市场（功能）

```
1. 国际贸易: 进出口要换汇结算(中国出口商收美元, 要换人民币)
2. 跨境投资: 买外国股票/债券要先换成当地货币
3. 旅游/汇款: 个人跨境用钱
4. 对冲: 企业/投资者对冲汇率风险
5. 投机: 赌汇率涨跌获利
```

> 💡 **FX 是全球经济的"血液循环系统"**：任何跨国的钱流动，几乎都要经过外汇市场换汇。
> 你买一台日本相机、投资美股、海外留学——背后都有 FX 在运转。

## ⚠️ 小白须知

- **汇率是相对的**：「美元涨了」必须问「相对谁涨」——相对日元涨可能相对欧元跌。
- **报价方向易混**：USD/JPY 上涨是「美元变强」，EUR/USD 上涨是「美元变弱」——因为美元在前/在后不同。
- **零售外汇高危**：散户炒外汇用高杠杆，亏损率极高（见 [06](../06-风险与监管/README.md)）。

## 小结

- 外汇是货币兑换，汇率是两种货币的相对价格（成对报价，base/quote 结构同股票/加密）。
- 买一个货币必然卖另一个；点差是做市商收益，点是最小变动单位。
- FX 是全球经济的血液循环——一切跨境资金流动都要经过换汇。
