# 7.2 利率与全球资金流

> 深度：🟡 进阶 ⭐
> 利率不只定价资产，还驱动全球资金的跨市场、跨国流动。这是回答"全球资金如何搅动"的核心。

## 利率差驱动跨国资金流 ⭐

```
资金追逐"风险调整后的更高回报":
  哪里利率高(风险可控) → 资金流向哪里(套息, 见 FX 4.3)
  美国加息 → 美元资产回报高 → 全球资金回流美国
    → 新兴市场资本外流、货币贬值、被迫跟随加息(见 FX 5.3)
  美国降息 → 资金流向新兴市场/风险资产找收益
→ 美联储利率 = 全球资金流动的"总开关"(见 FX 5.1)
```

- 💡 这把 [债券利率](../03-利率与定价/README.md) 和 [FX 套息](../../forex/04-汇率原理/4.3-套息交易.md) 直接连起来——**利率差是全球资金流动的"势能差"，资金从低利率"高处"流向高利率"低处"。**

## 跨市场资金轮动（Risk On/Off）⭐

```
利率/流动性环境决定资金在"风险资产"和"安全资产"间轮动:
Risk-On(宽松/降息/乐观):
  资金从国债/现金 → 流向 股票/高收益债/新兴市场/加密(追逐回报)
Risk-Off(收紧/危机/恐慌):
  资金从风险资产 → 逃回 国债/美元/黄金(避险, 见 FX 4.1)
→ "股债跷跷板": 经典上, 股市跌时资金避险买国债(债涨)
  (但 2022 例外: 利率主导下股债同跌, 见 7.1)
```

- 💡 对照 [FX risk-on/off](../../forex/04-汇率原理/4.1-什么决定汇率.md)、[加密资金流](../../crypto/cex/05-市场与博弈/5.5-资金流动全景.md)：**整个金融世界的资金，在"贪婪（risk-on）"和"恐惧（risk-off）"间反复轮动，而利率/流动性环境是这个开关的总闸。**

## 流动性周期与资产大潮 ⭐

```
"流动性"(钱的松紧)由央行利率 + QE/QT 决定(见 5.2):
  宽松周期(低利率+QE): 流动性泛滥 → 万物上涨(包括加密牛市, 见 7.1)
  收紧周期(高利率+QT): 流动性退潮 → 万物承压(包括加密熊市)
巴菲特: "退潮时才知道谁在裸泳"
  → 流动性退潮时, 高杠杆/庞氏/脆弱的东西暴露(FTX、各种暴雷都在 2022 收紧期)
```

- 💡 **加密的牛熊本质是流动性周期的放大版**：[FTX/Celsius/Luna 暴雷](../../crypto/cex/06-风险与监管/6.4-经典案例复盘.md) 集中在 2022 流动性收紧期——不是巧合，是"退潮"让脆弱的裸泳者暴露。

## "美元微笑"与全球美元周期 💡

```
美元的特殊地位(见 FX 1.3)让它的周期主导全球:
  强美元周期(美国加息/避险): 全球美元回流美国, 新兴市场承压、大宗(美元计价)下跌
  弱美元周期: 资金外溢到全球, 新兴市场/大宗/风险资产受益
→ "全球金融周期"很大程度是"美元/美国利率周期"
```

## 把"全球资金搅动"讲清楚 ⭐⭐

回到你最初想懂的问题——「世界上的钱如何被搅动、流动」：

```
最底层的驱动 = 央行利率(尤其美联储)
        │
        ▼
  ① 利率差 → 跨国资金流动(套息, 资金往高息处流, 见 FX 4.3)
  ② 折现率 → 跨资产重估(利率定价万物, 见 7.1)
  ③ 流动性松紧 → risk-on/off 轮动(风险资产 vs 安全资产)
  ④ 美元周期 → 全球美元的潮汐(强美元抽水, 弱美元放水)
        │
        ▼
  股票、债券、汇率、大宗、房产、加密 — 全部被这套机制驱动
```

> ⭐ **这就是"全球资金如何搅动"的完整答案**（综合了 FX 和债券两个赛道）：
> **央行（尤其美联储）通过利率，驱动了①跨国流动②跨资产重估③风险轮动④美元潮汐——
> 这四条线，搅动了全世界所有市场的钱。** 利率是那只最根本的"看不见的手"。

## 对照加密：以为逃离，实则同步 💡

```
加密叙事: "去中心化、独立于央行、对冲法币"
现实: 加密是对全球利率/流动性最敏感的资产之一(见 7.1)
  → 美联储宽松 → 加密狂欢; 美联储收紧 → 加密寒冬
  → 加密非但没逃离"利率引力", 反而是引力下"弹跳最高/摔得最狠"的那个
→ 真正的"独立"很难 — 只要资金在传统和加密间流动, 利率就传导过去
```

## 小结

- 利率差驱动跨国资金流动（资金往高息处流，美联储是总开关），连接债券利率和 FX 套息。
- 资金在风险资产和安全资产间随 risk-on/off 轮动，利率/流动性是总闸（股债跷跷板，2022 例外）。
- 流动性周期驱动资产大潮（宽松→万物涨/加密牛，收紧→退潮/暴雷暴露）。
- 美元周期主导"全球金融周期"（强美元抽水，弱美元放水）。
- "全球资金搅动"的完整答案：央行利率→①跨国流动②跨资产重估③风险轮动④美元潮汐→驱动所有市场。
- 加密以为逃离央行，实则对利率最敏感（引力下弹跳最高摔最狠）。
