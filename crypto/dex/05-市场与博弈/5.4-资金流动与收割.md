# 5.4 资金流动与收割

> 深度：🟡⭐ 把 DEX 串起来的「链上资金地图」
> 对照 [CEX 5.5 资金流动全景](../../cex/05-市场与博弈/5.5-资金流动全景.md)，回答：链上世界的钱如何被搅动、谁在收割谁。

## 一、钱怎么进入链上世界

```
法币世界 ──▶ 链上世界(DEX 玩不了法币, 钱多从 CEX/通道进来)
├── CEX 提币: 在 CEX 买币 → 提到自己钱包 → 进入 DeFi (最主流路径)
├── 法币 On-Ramp: MoonPay 等直接买币到钱包(费率高)
├── 稳定币: USDC/USDT 是链上的"美元", 一切以它计价
└── 跨链桥: 从其他链转入(见 3.5)
```

> 💡 **关键**：DEX 自己没有法币通道（见 [1.1](../01-行业全景/1.1-DEX是什么.md)）。
> 所以 **CEX 实际上是 DeFi 的「资金入口/法币闸门」**——大部分链上的钱，是先经过 CEX 才进来的。
> 这呼应了 [CEX 5.5 的「CEX 是中央水库，链上是灌溉网」](../../cex/05-市场与博弈/5.5-资金流动全景.md)。

## 二、钱在链上怎么流动

```
                  钱包里的稳定币/币
                        │
        ┌───────────────┼───────────────┐
        ▼               ▼               ▼
     DEX 交易        当 LP          DeFi 套娃
    (换币/打新)    (赚手续费       (借贷/质押/
        │           -无常损失)      收益耕作)
        │               │               │
        ▼               ▼               ▼
   ┌────────────────────────────────────────┐
   │  钱在协议间高速流动(可组合性, 一笔交易串多协议)│
   └────────────────────────────────────────┘
```

## 三、钱最终流向谁（收割链）⭐

```
链上资金的"抽水口":
├── 协议手续费 → LP + 协议金库 (旱涝保收)
├── MEV → 搜索者/构建者/验证者 (隐形税, 见 5.2)
├── 套利者 → 赚价差(部分来自 LP 和慢交易者)
├── 项目方/早期 → 发币、加池、拉盘出货(同 CEX)
├── 骗局 → 貔貅/Rug Pull 直接卷走(见 06)
└── Gas → 矿工/验证者 + 销毁
散户净流出: 滑点 + MEV + 无常损失 + 骗局 + Gas 多重消耗
```

## 四、链上食物链 ⭐

```
顶端  ┌─────────────────────────────────┐
      │ 协议/MEV基础设施(抽手续费和MEV)    │  旱涝保收
      ├─────────────────────────────────┤
      │ 专业玩家(MEV搜索者/量化/高频套利)  │  技术+速度+信息优势
      ├─────────────────────────────────┤
      │ 项目方/早期资本(发币出货)          │  低成本筹码变现
      ├─────────────────────────────────┤
      │ 专业 LP / 做市机构(主动管理对冲)   │  专业能力
      ├─────────────────────────────────┤
底层  │ 普通散户(被多重消耗)              │  信息/工具/速度全劣势
      └─────────────────────────────────┘
```

> ⚠️ 和 [CEX 食物链](../../cex/05-市场与博弈/5.2-多方博弈格局.md) 惊人地相似——**换了去中心化的皮，食物链结构没变。**

## 五、链上独有的资金流动特征

| 特征 | 说明 |
| --- | --- |
| **完全透明** | 所有资金流链上可查（Nansen/Arkham 追踪巨鲸） |
| **可组合高速流动** | 一笔交易串多个协议，资金高速腾挪 |
| **无国界、无许可** | 全球任何人随时可参与，无 KYC |
| **MEV 隐形税** | 链上独有的「排序税」 |
| **骗局密度高** | 无许可建池 → 貔貅/Rug 泛滥 |
| **追踪但伪匿名** | 能追踪地址行为，但不一定知道是谁 |

## 六、回到核心问题 ⭐

> **「链上世界的资金如何被搅动、流动？」**

DEX 给出的答案：
1. **钱主要经 CEX/通道/桥进入链上**，以稳定币为计价血液。
2. **在 DEX/借贷/衍生品间靠可组合性高速流动**，一笔交易可串多个协议。
3. **被协议、MEV、套利者、项目方多方再分配**，散户多重消耗后净流出。
4. **全程链上透明可追踪**（这是和 CEX 黑盒的根本区别），但透明不等于公平。
5. 食物链结构和 CEX 高度同构——**去中心化改变了「信任谁」，没改变「谁收割谁」。**

## 把 CEX 和 DEX 连起来看 ⭐

```
法币 ──▶ CEX(中央水库, 法币闸门) ──提币──▶ 链上钱包 ──▶ DEX/DeFi(灌溉网)
                ▲                                          │
                └──────── CEX-DEX 套利/提现回流 ────────────┘
```
> CEX 和 DEX 不是对立的，而是**一个资金大循环的两端**：CEX 管法币进出和大额，DEX 管链上原生流动和长尾。资金在两者间通过提币、套利、桥不断往复。理解了这个大循环，你就理解了整个加密资金的流动骨架。

## 小结

- 钱主要经 CEX/通道/桥进入链上，稳定币是计价血液；CEX 是 DeFi 的法币闸门。
- 钱在 DEX/借贷/衍生品间靠可组合性高速流动，被协议/MEV/套利/项目方再分配。
- 链上食物链与 CEX 高度同构：协议和 MEV 在顶端，散户在底层被多重消耗。
- 链上独有特征：完全透明可追踪、可组合高速流动、MEV 隐形税、骗局密度高。
- CEX 与 DEX 是同一资金大循环的两端，资金通过提币/套利/桥往复——这是加密资金流动的骨架。
