# 1.3 发展简史

> 深度：🟢 小白友好
> DeFi 史 = 「狂热挖矿 → 暴雷出清 → 走向成熟」的循环，和 [CEX](../../cex/01-行业全景/1.5-发展简史与里程碑.md)、[股票](../../../stocks/01-行业全景/1.4-发展简史.md) 的危机-监管循环同构。

## 时间线总览

```
2017-2018: MakerDAO 上线(DAI 稳定币)、早期 DeFi 雏形
2020 夏: DeFi Summer — 流动性挖矿引爆(见 DEX 1.5、4.1) ⭐
2021: DeFi 狂热, TVL 暴涨, 各种"农场"和高 APY
2022.5: Luna/UST 崩盘 — 算法稳定币归零, 数百亿蒸发 ⭐
2022: Celsius/3AC/FTX 连环暴雷(见 CEX 6.4), DeFi 寒冬
2023~: 走向"真实收益(Real Yield)"、LSD(质押)、RWA
2023-24: 链上美债(RWA)兴起, 传统资产上链
2024~: 再质押(Restaking)、意图、DeFi 与传统融合
```

## 关键节点

### MakerDAO 与 DAI（DeFi 的奠基）
- MakerDAO 创造了 **DAI**——第一个成功的去中心化超额抵押稳定币（见 [02](../02-稳定币/README.md)）。
- 💡 证明了"无需中心化机构，用智能合约+超额抵押也能发行稳定币"，是 DeFi 的奠基石。

### DeFi Summer（2020）⭐ 引爆
- **流动性挖矿**（见 [DEX 4.1](../../dex/04-生态与运营/4.1-流动性挖矿.md)）引爆 DeFi。
- Compound 发 COMP 代币奖励用户 → 各协议争相发币挖矿 → TVL 和 APY 暴涨。
- 资金疯狂在各"农场"间追逐高 APY（"挖提卖"）。

### Luna/UST 崩盘（2022.5）⭐ 最惨痛教训
```
Terra 的 UST 是"算法稳定币"(见 02), 靠和 Luna 代币的套利机制维持锚定:
  正常: 套利机制维持 UST = 1 美元
  崩盘: 信心动摇 → UST 脱锚 → 套利机制反而加速崩溃(死亡螺旋)
       → UST 和 Luna 几天内归零, 数百亿美元蒸发
→ 算法稳定币"无足额抵押"的致命缺陷被彻底暴露
→ 引发连锁: 3AC、Celsius 等(持有大量相关资产)接连暴雷(见 CEX 6.4)
```

- 💡 **DeFi 史上最惨痛的教训**：算法稳定币的"算法维持锚定"在信心崩塌时失效，且机制反而加速崩溃（死亡螺旋）。详见 [02 稳定币](../02-稳定币/2.3-算法稳定币.md)。

### DeFi 寒冬与出清（2022）
- Luna 崩盘 + Celsius/FTX 连环暴雷 → DeFi TVL 暴跌、信心受挫。
- 💡 **流动性退潮"让裸泳者暴露"**（见 [债券 7.2](../../../bonds/07-利率与万物/7.2-利率与全球资金流.md)）——2022 利率收紧期，DeFi 的庞氏/脆弱项目集中暴雷。

### 走向真实收益与 RWA（2023~）⭐
```
痛定思痛, DeFi 转向:
  真实收益(Real Yield): 用真实手续费/收入而非通胀代币给收益(见 DEX 4.1)
  LSD(流动性质押): 质押 ETH 拿凭证, 安全的真实收益(见 04)
  RWA(真实世界资产): 把美债等传统资产上链, 提供"链上无风险收益"(见 07)
→ 从"庞氏式高 APY"转向"可持续的真实收益"
```

## 贯穿全史的主线 ⭐

> **DeFi 史 = 「创新狂热 → 风险爆发 → 出清反思 → 走向成熟」的循环**，
> 和 [CEX 史](../../cex/01-行业全景/1.5-发展简史与里程碑.md)、[股票史](../../../stocks/01-行业全景/1.4-发展简史.md) 的"危机→监管→完善"完全同构：
> - Luna 崩盘 ≈ 股票的 1929 / 加密的 FTX —— 用惨痛教训催生对"足额抵押/透明"的重视。
> - 从挖矿狂热到真实收益 ≈ 市场从投机走向成熟。
>
> 💡 **DeFi 正在用几年时间，重走传统金融几百年"从蛮荒到规范"的路**（见 [股票 1.4](../../../stocks/01-行业全景/1.4-发展简史.md)），只是更快、更剧烈。

## 小结

- MakerDAO/DAI 奠基（证明去中心化超额抵押稳定币可行）。
- DeFi Summer（2020）流动性挖矿引爆狂热，2021 高 APY 农场泛滥。
- Luna/UST 崩盘（2022）是最惨痛教训——算法稳定币死亡螺旋归零，引发连环暴雷。
- 2022 寒冬出清（退潮暴露裸泳者），2023~ 转向真实收益、LSD、RWA。
- 主线是"创新狂热→风险爆发→出清→成熟"，与传统金融危机-监管循环同构（更快更剧烈）。
