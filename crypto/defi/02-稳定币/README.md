## 核心收获 ⭐

```
稳定币 = 锚定法币(通常美元)的加密币, 是 DeFi 的"计价货币和血液"
三大类型(按锚定机制):
  法币抵押(USDT/USDC): 1:1 法币储备背书 — 最稳, 但中心化
  超额抵押(DAI): 用加密资产超额抵押 — 去中心化, 但资本效率低
  算法(已多次失败): 靠算法维持 — 资本效率高, 但极易崩(Luna)
稳定币三难: 价格稳定、资本效率、去中心化 — 三者难兼得(见 2.4)
```

> ⭐ **稳定币是连接法币世界和加密世界的桥，也是 DeFi 内部的计价单位（见 [FX 7.2](../../../forex/07-现代演化/7.2-稳定币CBDC与未来.md)）。**
> 理解稳定币如何维持"1 美元"以及为什么会脱锚，是理解 DeFi 风险的核心。
