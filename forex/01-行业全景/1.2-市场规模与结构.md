# 1.2 市场规模与结构

> 深度：🟢🟡
> 注：数字来自国际清算银行(BIS)三年一次的调查量级，随时间变化，本文给量级感受。

## 市场规模：全球最大 ⭐

```
外汇日交易量: 约 7-8 万亿美元(BIS 调查量级)
对比:
  全球股市日交易量: 几千亿美元量级
  加密日交易量: 牛市千亿美元量级
→ FX 日交易量 ≈ 全球股市的 10 倍以上, 加密的几十倍
→ 当之无愧的"全球最大金融市场"
```

> 💡 为什么这么大？因为 FX 不只是投机——**所有国际贸易、跨境投资、对冲都要经过它**，
> 而且大量是「掉期（swap）」等短期滚动交易（见 [02](../02-参与者与产品/README.md)），重复换汇放大了交易量。

## 结构：没有中央交易所的 OTC 市场 ⭐⭐

```
股票: 有中央交易所(NYSE/Nasdaq), 所有订单汇集
FX:   没有中央交易所! 是"场外(OTC, Over-The-Counter)"市场
      → 由全球各大银行用电子网络/电话互相报价、成交
      → 一张分散的"银行间网络"
```

### 市场的分层结构
```
第1层 — 银行间市场(Interbank): 大银行之间互相交易(最核心、量最大、点差最小)
   ▼ (大银行向下报价)
第2层 — 大客户: 大型企业、对冲基金、其他金融机构
   ▼
第3层 — 零售: 散户(通过零售外汇经纪商)、小企业、旅游换汇
        → 越往下, 点差越大、价格越差(层层加价)
```

- 💡 **关键**：散户拿到的价格，是经过层层加价的「零售价」，远不如银行间的「批发价」。
- 对照 [股票的分业产业链](../../stocks/01-行业全景/1.3-分业制衡的产业链.md)：FX 也是分层的，但没有中央交易所和 CCP，而是「信任链条 + 双边信用」。

## 主要交易中心（地理）

```
FX 是 24 小时市场, 随地球自转在各金融中心"接力":
  悉尼 → 东京 → 伦敦 → 纽约 → (回到悉尼)
伦敦: 最大的外汇中心(占全球交易量最大份额)
纽约、新加坡、香港、东京: 其他主要中心
→ 周一到周五几乎 24 小时连续交易(周末休市)
```

- 💡 这是 FX 「24×5」的来源——不是一个市场开 24 小时，而是全球中心接力。
- 对照 [加密 24×7](../../crypto/cex/03-技术架构/3.7-高可用与安全工程.md)：加密真正全年无休，FX 周末休市。

## 电子化的聚合 💡

```
虽然没有中央交易所, 但有"电子交易平台"聚合银行间报价:
  EBS、Reuters Dealing(传统银行间平台)
  各种 ECN(电子通信网络)、单一银行平台
→ 让分散的报价趋于统一(套利抹平差异)
```

- 💡 **这和 [DEX 聚合器/多池套利](../../crypto/dex/02-产品与机制/2.4-聚合器与意图.md) 异曲同工**：
  没有中央价格，但靠电子平台聚合 + 套利者，让分散的报价收敛到一致。**FX 是「去中心化市场如何自发形成统一价格」的古老范例。**

## 为什么 FX 能没有中央交易所还运转 ⭐

```
关键: 参与者是"少数高信用的大银行"
  → 银行间互相有授信额度(信任对方能履约)
  → 不需要中央交易所/CCP 来担保(靠双边信用 + 后来的 CLS 银行)
  → 散户通过受监管的经纪商接入
对比 DEX: DEX 没有"高信用机构"这个前提, 所以用代码+超额抵押替代信任
```

## 小结

- FX 是全球最大市场（日交易量 7-8 万亿美元，远超股市和加密）。
- 没有中央交易所，是 OTC 银行间网络，分「银行间→大客户→零售」三层，越往下价格越差。
- 24×5 连续交易（全球金融中心接力，伦敦最大），周末休市。
- 靠电子平台聚合+套利形成统一价格（类似 DEX），靠大银行的高信用和双边授信运转（区别于 DEX 用代码）。
