# 04 · 生态与运营

> 深度：🟡 进阶
> 目标：DEX 没有「公司运营」，但有一套独特的「协议运营」——流动性挖矿、代币经济、治理、增长。

## 核心对照：CEX 运营 vs DEX 运营 ⭐

| CEX（[对照](../../cex/04-运营体系/README.md)） | DEX |
| --- | --- |
| 公司团队运营 | 协议 + DAO + 社区 |
| 返佣拉新 | 空投/积分预期拉新 |
| 激励做市商 | 流动性挖矿激励 LP |
| 上币审批（中心化权力） | 工厂合约无许可建池 |
| 平台币（公司发行） | 治理代币（协议发行） |
| 烧钱买量 | 代币激励（也是另一种"烧钱"） |

## 一个核心命题 ⭐

> DEX 和 CEX 一样面临「鸡生蛋」难题（[对照 CEX 4.6](../../cex/04-运营体系/4.6-交易所如何活下去.md)）：
> **没流动性没人来，没人来没流动性。**
>
> CEX 用「请做市商 + 补贴」解决。
> DEX 的杀手锏是**「流动性挖矿」——用协议代币激励任何人来当 LP**，
> 把「冷启动流动性」这件事用代币众包出去。这是 DeFi 最强的增长引擎，也是泡沫的根源。
