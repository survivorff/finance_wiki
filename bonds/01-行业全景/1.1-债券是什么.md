# 1.1 债券是什么

> 深度：🟢 小白友好

## 一句话定义

> **债券（Bond）= 一张"借条"。** 你买债券，就是借钱给发行人（政府/企业），
> 发行人承诺：**定期付你利息（票息），到期还你本金。**

## 用生活化的例子理解

```
你朋友找你借 10000 元, 写了张借条:
  "借 10000 元(本金/面值), 每年付你 5% 利息(500元), 5 年后还本金"
→ 这张借条, 本质就是一张"债券"
债券市场, 就是这种"借条"能被买卖、转让的市场
```

## 债券的核心要素 ⭐

```
面值(Par/Face Value): 借的本金, 到期要还的金额(如 100 元/1000 元)
票息率(Coupon Rate): 每期付的利息比例(如 5%/年)
票息(Coupon): 每期实际付的利息(面值 × 票息率)
到期日(Maturity): 还本金的日期(1年/10年/30年...)
发行人(Issuer): 借钱的一方(政府/企业)
```

### 一个例子
```
某国债: 面值 100, 票息率 3%, 10 年期
  → 你花约 100 买入(发行时)
  → 每年收 3 元利息(票息)
  → 10 年后拿回 100 本金
  总共: 10年×3 + 100 = 130 元(如果持有到期且不违约)
```

## 债券 vs 股票（本质区别）⭐

```
股票 = 股权(你是"老板"之一, 分享利润, 也承担风险)
债券 = 债权(你是"债主", 收固定利息, 到期还本, 不分享公司成长)
关键: 公司破产时, 先还债主(债券), 剩下的才轮到股东(股票)
  → 债券求偿顺序优先 → 风险比股票低 → 回报也相对低且固定
```

- 💡 详见 [00 导读的债券 vs 股票对照表](../00-导读与学习路径.md)。

## 为什么发行债券（发行人视角）

```
政府: 财政赤字(花的比收的多) → 发国债借钱(基建/福利/还旧债)
企业: 需要资金扩张, 但不想稀释股权(发股票会稀释)
  → 发债券借钱(到期还, 不让出公司所有权)
对照: 加密项目发币融资(见 CEX 2.6) ≈ 某种"股权融资"
     而债券是"债权融资"(借了要还)
```

## 为什么买债券（投资者视角）

```
1. 稳定收益: 固定票息, 比股票稳(适合养老金、保险、求稳的人)
2. 保本(相对): 持有到期且不违约 → 拿回本金
3. 分散风险: 债券和股票often 反向(股跌时资金避险买债)
4. 投机: 赌利率变动, 低买高卖债券赚价差(见 03)
```

## ⚠️ 小白须知

- **债券不是"绝对保本"**：发行人可能违约（尤其企业债、垃圾债）；中途卖出价格会波动（见 [03](../03-利率与定价/README.md)）。
- **"无风险"是相对的**：国债被认为"无风险"，但那是指"信用风险低"（政府不赖账），仍有利率风险和通胀风险。

## 小结

- 债券是"借条"：买债券=借钱给发行人，收票息+到期还本。
- 核心要素：面值、票息率、票息、到期日、发行人。
- 债券是债权（债主，求偿优先，收益固定），股票是股权（老板，求偿最后，收益不限）。
- 发行人用债券"借钱不稀释股权"，投资者买债券求稳定收益和分散风险。
