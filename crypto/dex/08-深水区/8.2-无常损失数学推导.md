# 8.2 无常损失数学推导

> 深度：🔴🔴 极硬核
> 把无常损失从公式推导出来，并讲清 LVR 和对冲。前置：[2.3 无常损失](../02-产品与机制/2.3-流动性与无常损失.md)、[8.1 AMM 数学](./8.1-AMM数学全解.md)。

## 设定

```
LP 在 x*y=k 池存入: x₀ 个 X, y₀ 个 Y, 初始价格 P₀ = y₀/x₀
之后 X 的价格变为 P₁, 价格比 r = P₁/P₀
比较两种策略在价格变为 P₁ 时的价值:
  策略 HODL: 一直拿着 x₀ 和 y₀ 不动
  策略 LP:   存进池子, 让套利者把池子价格拉到 P₁
无常损失 = (LP 价值 − HODL 价值) / HODL 价值
```

## 推导池子在价格 P₁ 时的储备 ⭐

```
不变量:  x·y = k = x₀·y₀
新价格:  y/x = P₁  (套利者把池价拉到 P₁)

联立解出新储备:
  x₁ = √(k/P₁),   y₁ = √(k·P₁)
```

## 计算两种策略的价值（以 Y 计价）

```
HODL 价值:  V_hodl = x₀·P₁ + y₀
LP 价值:    V_lp   = x₁·P₁ + y₁ = √(k·P₁) + √(k·P₁) = 2√(k·P₁)
```

代入 `k = x₀y₀`、`P₀=y₀/x₀`，用价格比 `r = P₁/P₀` 化简（设初始两边等值便于表达）：

```
V_lp / V_hodl = 2√r / (1 + r)
```

## 无常损失公式 ⭐⭐

```
            2√r
IL(r) = ─────────  − 1        (r = 价格变化倍数)
           1 + r
```

> 这个简洁公式是 DeFi 最重要的公式之一。`IL ≤ 0` 永远成立（LP 永远不优于 HODL，单边价格变动就有损失）。

### 验证几个点（呼应 [2.3 的表](../02-产品与机制/2.3-流动性与无常损失.md)）
```
r=1   (不变):   2·1/2 − 1 = 0       → 无损失 ✓
r=2   (翻倍):   2√2/3 − 1 ≈ −0.0572 → 约 −5.7% ✓
r=4   (4倍):    2·2/5 − 1 = −0.20   → −20% ✓
r=0.5 (腰斩):   2·0.707/1.5 −1 ≈ −5.7% → 涨跌对称 ✓
```

### 关键性质
- **永远 ≤ 0**：只要价格偏离（无论涨跌），LP 就不如 HODL。
- **对称**：涨到 2x 和跌到 0.5x 的 IL 相同（对价格比的对数对称）。
- **偏离越大，IL 越大**，但增速放缓（凹函数）。

## 为什么会这样：套利者的视角 ⭐

```
价格上涨时(r>1): 池价滞后于市场 → 套利者来池子"买走便宜的 X"
  → 池子的 X 被买走(在涨的时候卖掉了 X), Y 增多
  → LP 相当于"在上涨途中不断卖出 X" → 踏空了 X 的上涨
价格下跌时: 套利者把跌的 X"塞给"池子 → LP 接了下跌的 X
→ 无论涨跌, LP 总是站在"被套利者占便宜"的一边
```

> 💡 这正是 [CEX 8.7 逆向选择](../../cex/08-深水区/8.7-做市与市场微观结构.md) 的被动版：
> **LP = 永不调价、永不撤单的做市商，必然被知情的套利者系统性收割。** 无常损失就是这个收割的量化。

## LVR（Loss-Versus-Rebalancing）💡 更精确的刻画

```
无常损失对比的是"HODL"(不动)
LVR 对比的是"持续再平衡的做市商"(在 CEX 实时调价做市)
  → LVR 衡量 LP 因"只能在链上被动成交、无法实时调价"而比"主动做市"少赚的钱
  → 是学术界更认可的"LP 真实成本"度量
直觉: LVR ≈ 套利者每次套利从 LP 身上赚走的钱的累积
     与波动率² 成正比(波动越大, LP 被套利收割越多)
```

- 💡 LVR 和波动率²正相关，呼应 [CEX 8.7 的 Avellaneda-Stoikov](../../cex/08-深水区/8.7-做市与市场微观结构.md)（价差与波动率相关）——**做市的成本本质都来自波动**。

## 对冲无常损失

```
方法1: 选低波动池(稳定币对) → r≈1 → IL≈0
方法2: 用期权/合约对冲 → IL 的形状类似"卖出跨式期权"(short straddle)
       → LP 本质是在"卖波动率", 可用买入波动率(期权)对冲
方法3: V3 主动管理区间 + delta 对冲(专业做市)
方法4: 要求手续费收益 > IL: 高交易量 + 适当费率才划算
```

> 💡 **深刻洞察**：**当 LP ≈ 卖出波动率（short gamma/straddle）。**
> 你赚手续费（像收期权金），但价格剧烈波动时亏（像期权被行权）。
> 所以「LP 划不划算」= 「手续费收入能否覆盖你卖出的波动率的成本」。这把 DeFi 的 LP 和传统期权定价连起来了。

## 小结

- 无常损失公式 `IL(r) = 2√r/(1+r) − 1`，恒 ≤ 0，涨跌对称，偏离越大越亏。
- 根源：LP 是被动做市商，套利者在正确方向系统性收割（逆向选择的被动版）。
- LVR 是更精确的 LP 成本度量，与波动率²正相关。
- LP 本质是「卖出波动率」（short straddle），赚手续费=收期权金，波动大时亏——可用期权对冲。
- 实务：选低波动池、要求手续费覆盖 IL、V3 主动管理+对冲。
