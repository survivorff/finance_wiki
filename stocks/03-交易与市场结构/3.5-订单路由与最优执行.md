# 3.5 订单路由与最优执行

> 深度：🔴 硬核 ⭐
> 同一只股票在多个交易所 + 多个暗池交易，你的订单该去哪成交？这是股市独有的「碎片化」难题，和 [DEX 聚合器](../../crypto/dex/02-产品与机制/2.4-聚合器与意图.md) 异曲同工。

## 市场碎片化（Fragmentation）⭐

```
同一只股票(如 AAPL)可以在很多地方交易:
  - 多个公开交易所(NYSE, Nasdaq, 各电子交易所...)
  - 多个暗池
  - 做市商内部化
→ 流动性被"打散"在十几个场所
→ 没有一个统一的"全市场订单簿"
```

- 💡 对照：[CEX 通常是单一平台内的统一订单簿](../../crypto/cex/02-产品体系/2.1-现货交易.md)（流动性集中）。
- 股市的碎片化是历史 + 监管 + 竞争的产物（鼓励交易所竞争，但代价是流动性分散）。

## NBBO：全国最优买卖价 ⭐

```
NBBO(National Best Bid and Offer):
  把所有交易所的最优买价和最优卖价汇总
  → 得到"全市场最优买价"和"全市场最优卖价"
  → 这是订单路由和最优执行的基准
```

- 美国用 **SIP（证券信息处理器）** 汇总各交易所行情，计算 NBBO。
- 💡 这相当于给碎片化的市场拼出一个「虚拟的统一盘口」。

## 订单路由（Order Routing）⭐

```
你在券商下单 → 券商要决定"把订单送到哪个场所成交":
  - 哪个交易所价格最好?
  - 哪里有暗池流动性?
  - 内部化给做市商(PFOF)?
→ 用"智能订单路由(SOR)"系统决定
```

- 💡 **这和 [DEX 聚合器的智能路由](../../crypto/dex/02-产品与机制/2.4-聚合器与意图.md) 几乎是同一件事**！
  - 股市：订单路由器在多个交易所/暗池间找最优执行。
  - DEX：聚合器在多个 AMM 池间找最优执行。
  - **碎片化的流动性 → 需要"路由/聚合"找最优——跨市场的共同问题和共同解法。**

## 最优执行义务（Best Execution）⭐

```
最优执行义务: 券商有"法律义务"为客户争取最优的成交结果
  → 不能为了自己赚 PFOF 就把订单送到差的地方
  → 要综合考虑: 价格、速度、成交概率、成本
```

- ⚠️ 这和 [PFOF（订单流付款）](../02-参与者与产品/2.1-交易所与券商.md) 存在**潜在冲突**：
  - 券商把订单卖给出价最高的做市商（赚 PFOF），还是送到成交价最好的地方（对客户最优）？
  - 监管要求「最优执行」来约束这种冲突，但执行和监督是难点（GME 事件后焦点）。

## Reg NMS（美国市场结构的核心规则）💡

```
Reg NMS(全国市场系统监管规则): 美国股市结构的基础法规, 关键有:
  - 订单保护规则: 不能在"比别处更差的价格"成交(防止穿价)
  - 必须路由到 NBBO 最优价
→ 把碎片化的市场用规则"粘合"成一个相对统一、公平的整体
```

- 💡 这是「用监管规则解决碎片化」的典范：市场可以分散竞争，但规则保证你总能拿到全市场最优价。

## 三个市场的「碎片化与聚合」对照 ⭐

| | 碎片化来源 | 聚合/路由方案 |
| --- | --- | --- |
| 股市 | 多交易所+暗池 | 订单路由 + NBBO + Reg NMS + 最优执行义务 |
| [CEX](../../crypto/cex/01-行业全景/1.3-市场格局与玩家.md) | 多个交易所 | 套利者跨所抹平 + 用户自己选所 |
| [DEX](../../crypto/dex/02-产品与机制/2.4-聚合器与意图.md) | 多个 AMM 池/多链 | 聚合器（1inch/Jupiter）+ 意图/求解器 |

> 💡 **流动性碎片化是所有成熟市场的共同宿命**（竞争必然导致多场所），
> 而「路由/聚合 + 规则/套利」是共同的解法。
> 股市用**监管规则（Reg NMS、最优执行义务）**强制公平，
> DEX 用**聚合器和套利**市场化解决，CEX 介于两者之间。
> 这是三个市场最深刻的结构性对照之一。

## 小结

- 股市流动性碎片化在多交易所+暗池，没有统一订单簿。
- NBBO（全国最优买卖价）由 SIP 汇总，是路由和执行的基准。
- 订单路由（SOR）在多场所找最优执行——和 DEX 聚合器是同一件事。
- 最优执行义务（法律强制券商为客户争取最优）与 PFOF 存在冲突，靠 Reg NMS 等规则约束。
- 碎片化是成熟市场共同宿命，股市靠监管规则、DEX 靠聚合器套利、CEX 居中——深刻的三方对照。
