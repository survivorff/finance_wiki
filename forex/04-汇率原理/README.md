## 核心收获 ⭐

```
汇率的根本驱动(从短到长):
短期: 资金流、避险情绪、消息/数据、央行干预
中期: 利率差(套息) ← FX 最重要的驱动
长期: 通胀差(购买力平价)、经济基本面、国际收支
```

> ⭐ **一句话钥匙**：**汇率短期看情绪和资金流，中期看利率，长期看通胀和基本面。**
> 而「利率」是贯穿始终的核心——这也是为什么 FX 交易员最盯央行的利率决议。

## 为什么这是全知识库的关键模块 💡

> 汇率不只关乎 FX 交易者——它是**全球资金流动的"总阀门"**：
> - 美元利率↑ → 全球资金回流美国 → 新兴市场货币贬值、资本外流
> - 汇率波动影响股票（外资进出）、债券、大宗（美元计价）、甚至加密
>
> 理解汇率原理，你就拿到了理解「**世界上的钱为什么这样流动**」的总钥匙——
> 这正是你最初想搞懂的「全球资金如何搅动」的核心答案之一。
