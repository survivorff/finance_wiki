## 核心收获 ⭐

```
链上衍生品: 见 DEX 2.5(永续 DEX 已详述), 本模块补充期权/合成资产
收益来源(关键!):
  真实收益: 交易手续费、借贷利息、质押奖励、RWA 利息 → 可持续
  庞氏收益: 纯通胀代币激励、拆东补西 → 不可持续(见 1.3 Luna)
→ 看懂"收益从哪来", 是 DeFi 避坑的核心能力
```
