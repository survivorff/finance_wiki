# 8.1 AMM 数学全解

> 深度：🔴🔴 极硬核
> 把 AMM 的定价、滑点、手续费、V3 集中流动性的数学讲到「能自己算」的程度。
> 前置：[2.1 AMM](../02-产品与机制/2.1-AMM是什么.md)、[2.2 AMM 进阶](../02-产品与机制/2.2-AMM进阶.md)。

## 一、恒定乘积公式

```
池子: x 个 token X, y 个 token Y
不变量: x · y = k  (k 恒定)
即时价格(Y 计价的 X 价格): P = y / x
```

> 💡 「即时价格」是曲线在当前点的斜率，是「无穷小交易」的价格。实际交易有量，会产生滑点。

## 二、一笔 swap 的精确计算（无手续费）⭐

用 Δx 个 X 换出 Δy 个 Y。交易后 X 增加，Y 减少，乘积不变：

```
(x + Δx)(y − Δy) = k = x·y

解出换出的 Δy:
        y · Δx
Δy = ───────────
        x + Δx
```

### 算例（呼应 [2.1](../02-产品与机制/2.1-AMM是什么.md)）
```
池: x=10 ETH, y=20000 USDC, k=200000, 即时价 P=2000
要卖出 1 ETH(Δx=1):
  Δy = 20000·1 / (10+1) = 20000/11 ≈ 1818 USDC
  → 卖 1 ETH 只得 1818 USDC, 均价 1818(低于即时价 2000)
要买 1 ETH(从池取出 Δy 方向相反):
  需投入 USDC: Δin = x·Δout/(x−Δout) 形式 → 200000/9 − 20000 ≈ 2222 USDC
  → 买 1 ETH 花 2222, 均价 2222(高于 2000)
```

## 三、滑点（Slippage）公式 ⭐

```
卖出 Δx 的有效均价 = Δy/Δx = y/(x+Δx)
即时价 = y/x
滑点 = 1 − 有效均价/即时价 = 1 − x/(x+Δx) = Δx/(x+Δx)

→ 滑点 ≈ Δx / x  (当交易量远小于池子时)
```

> ⭐ **核心结论**：**滑点 ≈ 你的交易量 / 池子深度**。
> 池子越大（x 越大）、你交易越小（Δx 越小），滑点越小。这就是「深度=低滑点」的数学。

## 四、加入手续费

```
手续费率 f(如 0.3% = 0.003): 输入先扣费再参与计算
有效输入 Δx_eff = Δx · (1 − f)
Δy = y · Δx_eff / (x + Δx_eff)
扣下的 f·Δx 留在池子 → 使 k 缓慢增大 → LP 赎回时分到手续费
```

- 💡 手续费让 k 随交易缓慢增长，这就是 LP 收益的来源（赎回时按份额分到增大的池子）。

## 五、LP 份额数学

```
首次注入: 铸造 LP token 数量 = √(x·y)  (几何平均, Uniswap V2)
后续注入: 按比例 mint, LP_minted = total_LP · min(Δx/x, Δy/y)
赎回: 按份额取回 (share · x, share · y) + 累积手续费
```

## 六、V3 集中流动性数学 ⭐⭐

V3 让 LP 在价格区间 `[Pa, Pb]` 内提供流动性。核心是「虚拟储备」概念。

```
在区间 [Pa, Pb] 提供流动性 L, 等价于:
  虚拟储备满足 (x + L/√Pb)(y + L·√Pa) = L²

流动性 L 与实际投入的关系(价格 P 在区间内时):
  x = L(1/√P − 1/√Pb)
  y = L(√P − √Pa)
```

### 直觉解读
```
- LP 只需在 [Pa,Pb] 区间内备货, 不用为区间外的价格准备资金
  → 同样的钱, 在窄区间内提供 大得多 的"等效深度"(资本效率↑)
- 价格在区间内移动: 像一个"放大版"的小 x*y=k 池
- 价格触及 Pb: 你的 X 全部换成 Y(全是 Y)
- 价格触及 Pa: 你的 Y 全部换成 X(全是 X)
- 价格冲出区间: 流动性"失活", 不再赚手续费, 全是单一资产
```

### 资本效率提升倍数
```
区间越窄, 效率越高。集中在 [Pa,Pb] 相对全区间的效率放大约:
  效率 ≈ 1 / (1 − ⁴√(Pa/Pb))   (量级感受: 窄区间可达几十~上百倍)
代价: 区间越窄, 价格越易冲出 + 无常损失越剧烈(见 8.2)
```

> 💡 V3 的 tick 机制把价格离散成 `1.0001^i` 的刻度，LP 在 tick 区间提供流动性。
> 这和 [CEX 撮合的 tick size + 价格档位](../../cex/08-深水区/8.1-撮合引擎实现细节.md) 异曲同工——都是「把连续价格离散化管理」。

## 七、StableSwap 曲线（Curve）简述

```
Curve 不变量(简化): A·n^n·Σxᵢ + D = A·D·n^n + D^(n+1)/(n^n·Πxᵢ)
  A = "放大系数", 控制曲线在均衡点附近有多"平"
  A 越大 → 中间段越接近恒定和(x+y=const, 零滑点)
  → 在 1:1 附近极低滑点, 远离时退化为恒定乘积(防掏空)
```
- 不用记公式，记住结论：**A 控制「平坦区间」的大小，这就是稳定币低滑点的来源**（见 [2.2](../02-产品与机制/2.2-AMM进阶.md)）。

## 小结

- 恒定乘积：`x·y=k`，即时价 `P=y/x`，换出量 `Δy = y·Δx/(x+Δx)`。
- 滑点 ≈ 交易量/池子深度——这是「深度=低滑点」的数学本质。
- 手续费让 k 缓慢增大，是 LP 收益来源；LP 份额按几何平均铸造。
- V3 集中流动性用虚拟储备让 LP 在区间内做市，资本效率放大几十倍，代价是 IL 放大+冲出区间失活。
- StableSwap 用放大系数 A 控制平坦区间，实现稳定币低滑点。
