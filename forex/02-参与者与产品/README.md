## 核心对照 ⭐

| 产品 | FX | 加密对应 |
| --- | --- | --- |
| 即期 Spot | 当前汇率即时换汇 | 现货交易 |
| 远期 Forward | 锁定未来汇率 | 交割合约 |
| 掉期 Swap | 即期+远期组合 | （加密较少，类似资金费率套利结构） |
| 期权 Options | 汇率期权 | 加密期权 |

> 💡 FX 的「掉期（swap）」是其独有且交易量最大的产品——它把「即期换汇」和「远期换回」打包，本质是「用一种货币的资金换另一种货币的资金一段时间」，是跨境资金管理的核心工具。
