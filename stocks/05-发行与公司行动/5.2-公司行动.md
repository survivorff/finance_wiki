# 5.2 公司行动（Corporate Actions）

> 深度：🟡 进阶
> 上市后，公司会有各种影响股票的「动作」。理解它们，才不会在分红、拆股时一脸懵。

## 什么是公司行动

```
公司行动(Corporate Action): 上市公司发起的、影响股票/股东的事件
由 CSD/券商(见 4.3)自动处理, 按你的持股执行
```

## 一、分红派息（Dividend）

```
公司把利润的一部分分给股东:
  现金分红: 直接发钱(每股 X 元)
  股票分红: 送股(每持 10 股送 1 股)
关键日期:
  除息日(Ex-dividend date): 这天之后买入不享受本次分红
  股权登记日: 这天持有的股东有资格领分红
分红后股价"除权除息": 股价相应下调(因为公司发出了钱)
```

- 💡 **对照加密**：股票分红 ≈ [代币质押收益/某些"分红型"代币](../../crypto/dex/04-生态与运营/4.2-代币经济与治理.md)。
  但股票分红来自**真实利润**，很多加密"收益"来自通胀发币（见 [DEX 4.1](../../crypto/dex/04-生态与运营/4.1-流动性挖矿.md)）——本质不同。

## 二、股票拆分/合并（Split / Reverse Split）

```
拆股(Split): 1 股拆成 N 股, 股价相应除以 N
  → 总价值不变! (拆股不创造价值, 只是"切更小份")
  目的: 股价太高(如 1 股 3000 美元)拆低, 便于小散户买
合股/反向拆股(Reverse Split): N 股并成 1 股, 股价乘 N
  目的: 股价太低(仙股)拉高, 避免退市/改善形象
```

- ⚠️ **误区**：拆股不让你更富！1 股变 10 股但每股价值变 1/10，总值不变。
- 💡 对照加密：类似代币的「面额调整」（rebase 代币、改变小数位），同样不创造价值。

## 三、股票回购（Buyback / Repurchase）⭐

```
公司用自有现金从市场买回自己的股票:
  → 减少流通股数 → 每股价值/每股收益(EPS)上升 → 通常利好股价
  → 是公司"回报股东"的方式(替代/补充分红)
```

- 💡 **直接对照 [CEX 平台币的回购销毁](../../crypto/cex/02-产品体系/2.6-生态产品.md)**！
  - 公司回购股票 ≈ 交易所用利润回购销毁平台币（BNB 等）。
  - **完全是同一个逻辑**：用利润减少流通量、支撑价格、回报持有者。
  - 加密平台币的回购销毁，就是直接借鉴了股票回购。
- ⚠️ 争议：回购可能被用来「人为抬高股价」而非投资发展，也可能配合高管减持。

## 四、增发/配股（Secondary Offering / Rights Issue）

```
增发: 上市后再发新股融资 → 稀释老股东股份(蛋糕被切更多份)
配股: 给老股东"按比例低价认购新股"的权利
→ 通常短期利空(稀释 + 公司缺钱信号), 但看融资用途
```

- 💡 对照加密：增发 ≈ [代币增发/解锁](../../crypto/dex/04-生态与运营/4.2-代币经济与治理.md)（稀释持有者）。

## 五、并购（M&A）

```
收购(Acquisition): 一家公司买下另一家
合并(Merger): 两家合成一家
对股东影响: 被收购方股东可能拿到现金 或 换成收购方股票(按比例)
套利: "并购套利"(merger arbitrage)是一种专门策略(赌并购能否完成)
```

## 六、其他公司行动

| 行动 | 含义 |
| --- | --- |
| **分拆（Spin-off）** | 把子公司独立上市，老股东按比例获得新公司股票 |
| **退市（Delisting）** | 股票被摘牌（主动私有化 或 被动不达标，对照 [CEX 下币](../../crypto/cex/04-运营体系/4.2-上币机制内幕.md)） |
| **股东大会投票** | 重大事项股东投票（对照 [DEX DAO 治理](../../crypto/dex/04-生态与运营/4.2-代币经济与治理.md)） |

## 公司行动的处理（基础设施视角）💡

- 这些行动都由 [CSD/券商](../04-清算交收/4.3-中央存管CSD.md) **自动按持股执行**——你不用手动操作，分红自动到账、拆股自动调整。
- 💡 对照加密：链上的「空投、快照、治理投票」也是按持仓自动执行（智能合约），思路一致，只是一个靠中心化存管、一个靠链上合约。

## 小结

- 公司行动是上市公司影响股票/股东的事件，由 CSD/券商按持股自动处理。
- 分红（来自真实利润，区别于加密通胀收益）、拆股（不创造价值，只切更小份）。
- 股票回购≈CEX 平台币回购销毁（完全同一逻辑：减少流通、支撑价格、回报持有者）。
- 增发稀释股东（≈代币增发），并购涉及现金/换股，还有分拆、退市、股东投票（≈DAO 治理）。
