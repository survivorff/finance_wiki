# 1.1 DeFi 是什么

> 深度：🟢 小白友好

## 一句话定义

> **DeFi（Decentralized Finance，去中心化金融）= 用智能合约在区块链上重建金融功能，无需银行/券商/交易所等中介。**

借贷、交易、稳定币、衍生品、理财——传统金融里银行和券商做的事，DeFi 用**部署在链上的代码（智能合约）**自动完成。

## DeFi vs CeFi ⭐

```
CeFi(中心化金融): 传统银行/券商 + 加密的中心化平台(CEX, 见 CEX 知识库)
  → 信任机构(公司)
DeFi(去中心化金融): 智能合约自动运行, 用户自托管
  → 信任代码(见 DEX 1.1)
```

| | CeFi（含 CEX） | DeFi |
| --- | --- | --- |
| 谁运营 | 公司/机构 | 智能合约 + 社区 |
| 资产保管 | 机构托管 | 用户自托管 |
| 准入 | KYC | 无许可（任何人） |
| 透明度 | 黑盒 | 链上公开 |
| 信任 | 信任机构 | 信任代码 |
| 兜底 | 可能有 | 无 |

## DeFi 的核心特征

### 1. 无许可（Permissionless）⭐
- 任何人有钱包就能用，无需 KYC、无需开户、无需批准。
- 任何人都能基于现有协议搭新协议（见可组合性）。

### 2. 可组合性（Composability）⭐ 货币乐高
```
DeFi 协议像乐高积木, 可以自由拼接、互相调用:
  在借贷协议借出稳定币 → 去 DEX 换币 → 存入收益协议挖矿
  全部能在"一笔交易"里原子完成(见 DEX 1.4)
→ "货币乐高(Money Lego)" — DeFi 最强大的特性
```

### 3. 自托管 + 透明
- 资产在自己钱包，所有合约和交易链上公开可验证（同 [DEX](../../dex/01-行业全景/1.1-DEX是什么.md)）。

## DeFi 包含什么（和 DEX 的关系）⭐

```
DEX(去中心化交易) 是 DeFi 的一部分(交易那块, 见 DEX 知识库)
DeFi 的完整版图还包括:
  稳定币(链上货币, 见 02)
  借贷协议(链上银行, 见 03)
  衍生品(链上合约, 见 04)
  收益聚合(链上理财, 见 04)
→ DeFi = 链上的"完整金融体系", DEX 只是其中的"交易所"
```

## 为什么 DeFi 重要

```
1. 金融普惠: 任何人(无银行账户者)都能用金融服务
2. 抗审查: 无许可、难关停
3. 创新速度: 可组合性让创新爆炸式增长
4. 透明: 链上可验证, 不像传统金融黑盒
5. 7×24 全球: 无国界、不停歇
```

## ⚠️ 小白须知

- **高 APY 陷阱**：DeFi 常宣传超高收益，但背后是高风险（合约/清算/脱锚/通胀代币贬值），见 [06](../06-安全与监管/README.md)。
- **不可逆 + 无客服**：操作失误、踩中骗局、合约被黑——钱没了就是没了（同 [DEX](../../dex/06-风险与安全/README.md)）。
- **复杂度高**：套娃协议风险叠加，普通人很难看懂自己承担了什么风险。

## 小结

- DeFi 用智能合约在链上重建金融功能，信任代码而非机构（区别于 CeFi/CEX）。
- 核心特征：无许可、可组合（货币乐高）、自托管、透明。
- DeFi = 链上完整金融体系（稳定币+借贷+DEX+衍生品+收益），DEX 只是其中的交易所部分。
- 价值是普惠/抗审查/创新/透明，代价是高 APY 陷阱、不可逆无兜底、复杂度高。
