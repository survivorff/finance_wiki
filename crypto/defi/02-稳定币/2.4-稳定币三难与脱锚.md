# 2.4 稳定币三难与脱锚

> 深度：🔴 硬核 ⭐
> 稳定币为什么没有"完美方案"？因为"稳定币三难"。理解它，就理解了所有稳定币的取舍。

## 稳定币三难困境（Stablecoin Trilemma）⭐

```
稳定币想要三个目标, 但难同时实现:
        ┌─────────────────────┐
        │   ① 价格稳定         │
        │        ╱   ╲         │
        │      ╱       ╲       │
        │ ②资本      ③去中心化  │
        │ 效率                 │
        └─────────────────────┘
  通常只能较好地实现两个, 牺牲第三个
```

### 三种稳定币的取舍 ⭐
```
法币抵押(USDC): ①稳定 + ②资本效率高(1:1) → 牺牲 ③去中心化(中心化发行)
超额抵押(DAI): ①稳定 + ③去中心化 → 牺牲 ②资本效率(要超额抵押)
算法(UST): ②资本效率 + ③去中心化 → 牺牲 ①稳定(死亡螺旋, 见 2.3)
→ 没有一种能三全! 这是稳定币设计的根本约束
```

> ⭐ **这和 [FX 的不可能三角](../../../forex/05-央行与政策/5.2-汇率制度.md)、[区块链不可能三角](../../cex/08-深水区/8.5-真实架构案例拆解.md) 是同一种"三选二"约束**：
> 复杂系统里，"既要又要还要"往往在数学/机制上不可能。
> 稳定币三难解释了为什么"完美稳定币"不存在——每种都是一组取舍。

## 脱锚（De-pegging）⭐

```
脱锚: 稳定币价格偏离锚定值(如 USDC 跌到 0.95, UST 归零)
程度:
  小幅短暂脱锚: 流动性紧张/恐慌, 套利通常能拉回(如 USDC 硅谷银行事件后恢复)
  永久脱锚/归零: 机制失败(如 UST 死亡螺旋, 见 2.3)
```

## 各类型的脱锚风险 ⭐

| 类型 | 脱锚触发 | 后果 |
| --- | --- | --- |
| 法币抵押 | 储备出问题（银行倒闭/储备造假）、监管冻结 | 通常可恢复（若储备真实），USDC 硅谷银行事件后恢复 |
| 超额抵押 | 抵押品暴跌+清算失灵、依赖的法币稳定币脱锚 | 较稳健但极端行情有风险 |
| 算法 | 信心崩塌 | 死亡螺旋、可能归零（UST） |

## 脱锚的连锁传染 ⚠️

```
稳定币是 DeFi 的血液(见 2.1), 脱锚会连锁传染:
  某稳定币脱锚 → 用它做抵押/交易的协议受损
  → 借贷协议坏账、DEX 池子失衡、套娃协议崩塌(见 05)
  → 恐慌蔓延到整个 DeFi
UST 崩盘就引发了连锁: 3AC、Celsius 等接连暴雷(见 CEX 6.4)
```

- 💡 对照 [债券信用传染](../../../bonds/06-风险与危机/6.2-债务危机案例.md)、[CEX 挤兑](../../cex/06-风险与监管/6.1-风险地图.md)：稳定币脱锚是 DeFi 版的"系统性传染源"——因为它是血液，一处出血全身受影响。

## 如何判断稳定币的安全性 💡

```
看几个关键:
1. 抵押类型: 有足额抵押吗?(法币/超额抵押 > 算法)
2. 储备透明度: 储备真实、可验证吗?(法币型看审计/披露, 同 PoR)
3. 抵押品质量: 抵押的是什么?(美债/现金 > 波动加密 > 自家代币)
4. 历史记录: 经历过压力测试吗?脱锚后恢复了吗?
5. 收益异常: 提供"不合理高息"的(如 UST 的 20%)= 危险信号(庞氏特征)
```

> ⚠️ **黄金法则**：和 [CEX 理财](../../cex/02-产品体系/2.4-理财与赚币.md) 一样——**不合理的高收益 = 危险信号**。
> 稳定币本身不该有高收益（它只是"稳定的钱"），如果某个稳定币靠超高息吸引你存入，警惕死亡螺旋。

## 稳定币 vs 货币锚定的历史 💡

```
回顾 FX 1.4 货币锚定的演化:
  金本位(锚黄金) → 布雷顿森林(锚美元) → 信用货币(锚国家信用)
  → 稳定币(锚美元: 用储备/抵押/算法)
→ 稳定币是"货币锚定"这个几百年命题的最新篇章
  法币抵押稳定币 = "数字美元代币"(强化美元, 见 FX 7.2)
  算法稳定币 = "想凭空创造锚定" → 失败(没有足够实力支撑锚定, 同 FX 固定汇率崩溃)
```

## 小结

- 稳定币三难：价格稳定、资本效率、去中心化三者难兼得（法币抵押牺牲去中心化，超额抵押牺牲效率，算法牺牲稳定）——同 FX/区块链不可能三角。
- 脱锚分小幅短暂（套利可拉回，如 USDC 硅谷银行）和永久归零（机制失败，如 UST）。
- 稳定币是 DeFi 血液，脱锚会连锁传染（UST 引发 3AC/Celsius 暴雷）。
- 判断安全性看：抵押类型、储备透明度、抵押品质量、历史、收益是否异常（高息=危险）。
- 稳定币是"货币锚定"几百年命题的最新篇章，算法稳定币的失败同固定汇率崩溃。
