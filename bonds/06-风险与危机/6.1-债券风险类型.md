# 6.1 债券风险类型

> 深度：🟡 进阶
> 债券"安全"是相对的。这一篇拆解债券的各类风险。

## 债券风险全景 ⭐

```
1. 利率风险: 利率涨, 债券价格跌(久期越长越严重, 见 3.3) — 最普遍, 连国债都有
2. 信用/违约风险: 发行人还不上钱(企业债、弱国债) — 国债相对低
3. 通胀风险: 通胀吃掉固定票息的实际购买力
4. 流动性风险: 小众债难卖、点差大
5. 再投资风险: 收到的票息只能按更低利率再投资
6. 提前赎回风险: 可赎回债被提前收回(见 2.2)
7. 汇率风险: 外币债的汇率波动(见 FX)
```

## 一、利率风险 ⭐ 最普遍

```
利率涨 → 债券价格跌(见 3.1)
  久期越长, 跌得越狠(见 3.3)
关键认知: 连"无风险"国债都有利率风险!
  → "信用安全"(政府不赖账) ≠ "价格不波动"
  → 2022 长期国债跌 20-30%+ 就是利率风险(不是违约)
```

- 💡 **这是最多人忽视的风险**：以为买国债"绝对安全"，结果利率一涨，长期国债基金巨亏。**信用安全和价格安全是两回事。**

## 二、信用/违约风险

```
发行人还不上本息(违约):
  国债(本币): 风险极低(政府能印钞还, 但印钞会贬值/通胀, 是另一种"软违约")
  外币国债: 风险较高(不能印别国货币, 弱国可能违约, 见 6.2)
  企业债: 看公司财务(高收益债风险高, 见 2.1)
衡量: 信用评级、信用利差(见 2.1)、CDS(信用违约掉期, 给违约风险定价/对冲)
```

- 💡 **CDS（信用违约掉期）**：本质是"债券违约保险"——付保费，违约时获赔。卖 CDS = [卖保险/卖波动率](../../forex/08-深水区/8.2-套息与风险分解.md)（平时收保费，违约时巨赔）。2008 年 AIG 因大量卖 CDS 而濒临崩溃。

## 三、通胀风险 ⭐

```
通胀吃掉固定票息的实际价值:
  你的债券每年付 3%, 但通胀 5% → 实际购买力每年缩水 2%
  → 名义回报 3%, 实际回报 −2%
对抗: 通胀挂钩债(TIPS, 见 2.2)、浮息债
```

- 💡 **实际利率 = 名义利率 − 通胀**。这是债券投资者最该关心的——拿到的票息能不能跑赢通胀。高通胀时期，固定利率债券是"温水煮青蛙"。

## 四、其他风险

| 风险 | 说明 |
| --- | --- |
| **流动性风险** | 小众债/企业债难卖、点差大（同 [FX 流动性](../../forex/03-市场结构与交易/3.1-银行间市场与报价.md)） |
| **再投资风险** | 票息和到期本金只能按更低利率再投（利率下行时） |
| **提前赎回风险** | 可赎回债被提前收回（见 [2.2](../02-债券类型/2.2-按结构分类.md)） |
| **汇率风险** | 外币债的汇率波动（见 [FX](../../forex/README.md)） |

## 风险对照其他市场 💡

```
债券的核心风险(利率+信用), 在其他市场都有回响:
  利率风险: 利率影响一切资产估值(见 07) — 不只债券
  信用风险: ≈ 加密的"对手方/协议违约"(CEX 跑路、DeFi 坏账)
  通胀风险: ≈ 加密"通胀型代币"稀释持有者(见 DEX 4.1)
→ 风险的本质跨市场相通, 只是表现形式不同
```

## 小结

- 债券风险：利率（最普遍，连国债都有）、信用/违约、通胀、流动性、再投资、提前赎回、汇率。
- 利率风险：信用安全≠价格安全，2022 长期国债大跌是利率风险（非违约）。
- 信用风险用评级/利差/CDS 衡量；CDS 是"违约保险"，卖 CDS=卖波动率（AIG 2008 教训）。
- 通胀风险：实际利率=名义利率−通胀，高通胀时固定债券是温水煮青蛙。
- 风险本质跨市场相通（利率影响一切、信用≈对手方违约、通胀≈代币稀释）。
