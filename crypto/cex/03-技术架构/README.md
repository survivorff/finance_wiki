## 几个贯穿全模块的工程主题 ⭐

| 主题 | 核心选择 | 为什么 |
| --- | --- | --- |
| 撮合 | 单线程 + 全内存 + 事件溯源 | 极致性能 + 强一致 + 可恢复 |
| 账户 | 复式记账 + 原子事务 | 账目永远平衡、可审计 |
| 钱包 | 冷热分离 + 多签/MPC | 资产安全、降低被盗损失 |
| 行情 | WebSocket 增量推送 + 多级缓存 | 百万连接低延迟 |
| 风控 | 标记价 + 强平 + 保险基金 + ADL | 防穿仓、防系统性坏账 |

> **一句话**：CEX 技术的核心是一台**确定性、高性能、可恢复的撮合状态机**，外加一套**复式记账总账**和**冷热分离的钱包系统**。这些理念和传统证券交易所一脉相承。
