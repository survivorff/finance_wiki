## 核心收获

- DEX = 自托管 + 链上智能合约撮合，是对 CEX 托管原罪的回应。
- 主要类型：AMM（主流）、链上订单簿、聚合器、衍生品 DEX。
- 市场被「公链生态」切割（以太坊、Solana、BSC、各 L2 各有头部 DEX）。
- DEX 是 DeFi 乐高的「流动性底座」，可被其他协议自由组合。
