# 06 · 监管与风险

> 深度：🟡 进阶
> 目标：理解股市监管的基石（信息披露、禁止操纵）、风险类型，以及历史危机的教训。

股市监管是几百年危机换来的，也是加密市场正在追赶的方向。理解股市监管，就理解了 [CEX/DEX 监管](../../crypto/cex/06-风险与监管/6.3-全球监管格局.md) 的「成熟版目标」。

## 核心对照 ⭐

| | 股市 | CEX | DEX |
| --- | --- | --- | --- |
| 监管成熟度 | 几百年，最成熟 | 追赶中（FTX 后加速） | 刚起步、难监管 |
| 信息披露 | 强制、详尽 | 弱（PoR 起步） | 链上透明但无"公司披露" |
| 禁止操纵 | 严格立法+执法 | 存在黑幕 | MEV/操纵 |
| 投资者保护 | 保护基金+适当性 | 弱 | 几乎无 |

> 💡 股市监管的两大基石——**信息披露**和**禁止操纵**——正是加密市场最缺、也最该补的。学股市监管，等于看到加密监管的未来路线图。
