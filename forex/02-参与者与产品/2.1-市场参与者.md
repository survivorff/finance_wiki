# 2.1 市场参与者

> 深度：🟢🟡
> FX 里谁在玩？从能"印钱"的央行，到被收割的散户，是一条清晰的"权力 + 信息"递减链。

## 参与者金字塔 ⭐

```
        ┌──────────────────────────────┐
        │ 央行(Central Banks)           │  能印钞、能干预 — 终极玩家
        ├──────────────────────────────┤
        │ 大银行(做市/银行间)           │  报价、做市、最深流动性
        ├──────────────────────────────┤
        │ 对冲基金/资管(大投机者)       │  套利、宏观投机
        ├──────────────────────────────┤
        │ 跨国企业(真实需求)            │  贸易结算、对冲汇率风险
        ├──────────────────────────────┤
        │ 零售经纪商 + 散户              │  最差价格、最高杠杆、最易亏
        └──────────────────────────────┘
        权力/信息/价格优势 从上到下递减
```

## 各参与者详解

### 1. 央行（Central Banks）⭐ 终极玩家
- **独有能力**：印钞、设定利率、直接干预汇市（买卖本币）。
- **目的**：不是赚钱，而是稳定汇率、控制通胀、实现政策目标。
- 💡 **这是 FX 独有的存在**——股票/加密市场没有"能改变游戏规则、印出无限弹药"的玩家。央行一句话能让汇率剧烈波动（见 [05](../05-央行与政策/README.md)）。

### 2. 大银行（Interbank / 做市商）
- 花旗、摩根大通、德银、瑞银等——FX 的核心做市商。
- 互相报价、成交，构成 [银行间市场](../01-行业全景/1.2-市场规模与结构.md)。
- 为客户（企业、基金）做市，赚点差。
- 💡 对照 [股票做市商](../../stocks/02-参与者与产品/2.2-做市商与流动性.md)、[CEX 做市商](../../crypto/cex/04-运营体系/4.3-做市与流动性运营.md)——同样的做市逻辑（赚价差、管库存），只是标的是货币。

### 3. 对冲基金/资管（大投机者）
- 全球宏观对冲基金（如历史上索罗斯狙击英镑）。
- 做利率/汇率的宏观押注、套息交易（见 [04](../04-汇率原理/4.3-套息交易.md)）、套利。
- 💡 著名案例：1992 年索罗斯做空英镑，逼英镑退出欧洲汇率机制，「击垮英格兰银行」——散户对手盘里有这种顶级玩家。

### 4. 跨国企业（真实需求方）
- 进出口商、跨国公司：因真实贸易/投资需要换汇。
- 主要需求是**对冲汇率风险**（锁定成本），不是投机。
- 💡 例：中国出口商收美元，担心美元贬值，用远期锁定汇率（见 [2.2](./2.2-即期与远期.md)）。

### 5. 零售经纪商 + 散户
- 零售外汇经纪商：给散户提供交易通道（高杠杆）。
- 散户：炒外汇的个人。
- ⚠️ **食物链最底层**：拿最差的价格、用最高的杠杆、对手是央行/银行/顶级基金——亏损率极高（见 [06](../06-风险与监管/README.md)）。

## 谁影响汇率最大 💡

```
短期: 大银行做市 + 对冲基金投机 + 央行干预 → 主导日内波动
中长期: 央行货币政策(利率) + 经济基本面 → 主导趋势
散户: 几乎不影响汇率(体量太小), 只是被趋势裹挟
```

## 对照其他市场的参与者 ⭐

| 角色 | FX | 股票 | 加密 |
| --- | --- | --- | --- |
| 终极干预者 | 央行 | 无 | 无（项目方影响有限） |
| 做市 | 大银行 | 做市商 | 做市商/LP |
| 大投机者 | 宏观对冲基金 | 对冲基金 | 巨鲸/量化 |
| 真实需求 | 跨国企业 | （较少） | （较少） |
| 底层 | 散户 | 散户 | 散户 |

> 💡 FX 和其他市场最大的不同：**有央行这个"非营利、能印钞、能改规则"的超级玩家**。
> 在股票/加密里，没有任何一方能"印出无限弹药"或"直接设定价格规则"。这让 FX 的博弈多了一个其他市场没有的维度。

## 小结

- FX 参与者金字塔：央行（终极玩家，能印钞干预）→大银行（做市）→对冲基金（投机）→企业（真实需求/对冲）→散户（底层）。
- 央行是 FX 独有的超级玩家，目的是政策而非赚钱，能剧烈影响汇率。
- 做市逻辑和股票/加密相通，但标的是货币；散户同样是食物链底层且亏损率极高。
- 短期由银行/基金/央行主导，中长期由央行政策和基本面决定。
