# 债券与利率市场术语表（Glossary）

> 带 ⭐ 的是核心高频术语。通用交易术语可对照 [股票术语表](../stocks/glossary.md)、[CEX 术语表](../crypto/cex/glossary.md)。

## 基础
- **债券 / Bond** ⭐：借条/债权凭证。
- **面值 / Par / Face Value** ⭐：本金，到期偿还金额。
- **票息 / Coupon** ⭐：定期支付的利息。
- **票息率 / Coupon Rate**：票面固定的利率。
- **到期日 / Maturity** ⭐：偿还本金的日期。
- **发行人 / Issuer**：借钱方（政府/企业）。
- **国债 / Government Bond / Treasuries** ⭐：政府发行，无风险基准。
- **企业债 / Corporate Bond**：公司发行。

## 收益率与定价 ⭐
- **收益率 / Yield** ⭐：市场给债券的实际回报率。
- **到期收益率 / YTM** ⭐：持有到期的年化总回报。
- **当前收益率**：年票息/价格。
- **即期利率 / Spot Rate**：零息债收益率（定价基础）。
- **远期利率 / Forward Rate**：隐含的未来利率。
- **折现 / Discounting** ⭐：未来现金流换算成现值。
- **现金流折现 / DCF** ⭐：估值的基础方法。
- **折现率 / Discount Rate**：= 无风险利率 + 风险溢价。
- **折价/平价/溢价**：市价低于/等于/高于面值。
- **价格与利率反向** ⭐：利率涨债券跌（核心铁律）。

## 风险指标 ⭐
- **久期 / Duration** ⭐：价格对利率的敏感度。
- **麦考利久期**：加权平均回收时间。
- **修正久期** ⭐：价格敏感度（ΔP/P≈−ModD×Δy）。
- **凸性 / Convexity** ⭐：久期的二阶修正（曲率）。
- **利率风险 / 信用风险 / 通胀风险 / 流动性风险 / 再投资风险**。

## 债券类型
- **零息债 / Zero-Coupon**：折价发行，不付票息，久期最长。
- **浮息债 / Floating Rate**：票息随基准浮动，抗利率上升。
- **可赎回债 / Callable**：发行人可提前赎回。
- **可转债 / Convertible**：可转成股票（股债混合）。
- **通胀挂钩债 / TIPS** ⭐：本息随通胀调整（反映实际利率）。
- **永续债 / Perpetual**：无到期日。
- **ABS / MBS**：资产/抵押贷款支持证券（2008 核心）。

## 信用
- **信用评级 / Credit Rating** ⭐：AAA→D（投资级/垃圾债）。
- **投资级 / 高收益债(垃圾债)** ⭐。
- **信用利差 / Credit Spread** ⭐：企业债与国债收益率之差（债市恐慌指数）。
- **CDS（信用违约掉期）**：违约保险（卖 CDS=卖波动率）。
- **违约 / Default**：还不上本息。

## 收益率曲线 ⭐
- **收益率曲线 / Yield Curve** ⭐：不同期限收益率连成的线。
- **正常/倒挂/平坦曲线** ⭐：向上/向下/水平。
- **倒挂 / Inversion** ⭐：短期收益率>长期（衰退警报）。
- **10Y-2Y / 10Y-3M**：常看的曲线斜率指标。
- **变陡/变平 (steepening/flattening)**：曲线形状变化。
- **期限溢价 / Term Premium**：借更久要的额外补偿。
- **Bootstrapping**：从债券价格剥出即期曲线。
- **Nelson-Siegel**：曲线参数化模型（水平/斜率/曲率）。

## 市场与货币政策 ⭐
- **一级/二级市场**：发行/交易。
- **国债拍卖 / Auction**：政府发债方式。
- **一级交易商 / Primary Dealers**：参与拍卖、央行操作对手的大银行。
- **公开市场操作 / OMO** ⭐：央行买卖国债调节流动性。
- **QE / QT** ⭐：量化宽松（印钞买债）/紧缩。
- **政策利率**：央行设定的短期利率。
- **回购 / Repo** ⭐：债券抵押的超短期借钱。
- **逆回购 / Reverse Repo**：放钱收抵押。
- **影子银行 / Shadow Banking**：类银行但少监管（DeFi 是链上版）。

## 利率与万物 ⭐
- **无风险利率 / Risk-Free Rate** ⭐：国债收益率，万物定价之锚。
- **实际利率 / Real Rate** ⭐：名义利率−通胀。
- **名义利率 / Nominal Rate**：未扣通胀的利率。
- **利率是万物之锚** ⭐：利率定价一切资产（"地心引力"）。
- **流动性周期** ⭐：钱的松紧周期，驱动资产大潮。
- **Risk-On/Off** ⭐：风险偏好轮动。
- **主权债务危机**：政府还不起债。
- **赤字货币化**：央行印钞支持财政赤字。
