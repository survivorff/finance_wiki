# 股票市场术语表（Glossary）

> 按主题分组的股市常用术语。带 ⭐ 的是最核心、最高频的。
> 通用金融/交易术语（订单簿、做市、滑点等）可对照 [CEX 术语表](../crypto/cex/glossary.md)。

## 基础概念
- **股票 / Stock / Share** ⭐：公司所有权的碎片。
- **普通股 / Common Stock**：有投票权+浮动分红。
- **优先股 / Preferred Stock**：无投票权+固定分红，求偿优先。
- **一级市场 / Primary Market** ⭐：公司首次发行（IPO/增发），公司拿钱。
- **二级市场 / Secondary Market** ⭐：投资者间买卖，公司不拿钱。
- **市值 / Market Cap**：股价 × 总股本。
- **市盈率 / P/E**：股价 / 每股收益，估值指标。
- **指数 / Index**：一篮子股票的加权（标普500等）。

## 参与者
- **交易所 / Exchange** ⭐：撮合场所（NYSE/Nasdaq）。
- **券商 / Broker** ⭐：代理你下单的中介。
- **做市商 / Market Maker** ⭐：提供流动性、报价赚价差。
- **指定做市商 / DMM / Specialist**：NYSE 给每只股票指定、有维持秩序义务的做市商。
- **买方 / Buy Side** ⭐：基金/资管/对冲（买资产做投资）。
- **卖方 / Sell Side** ⭐：投行/券商/研究（卖服务）。
- **投资银行 / 投行**：承销、并购顾问、做市、研究。
- **PFOF（订单流付款）** ⭐：券商把订单流卖给做市商换钱（零佣金背后）。

## 交易与市场结构
- **订单簿 / Order Book**：买卖挂单集合（同 CEX）。
- **价格-时间优先**：撮合规则（同 CEX/DEX）。
- **连续竞价**：盘中逐笔实时撮合。
- **集合竞价 / Call Auction** ⭐：开收盘集中统一定价撮合。
- **NBBO（全国最优买卖价）** ⭐：汇总各交易所的最优价。
- **订单路由 / SOR**：在多场所找最优执行（≈DEX 聚合器）。
- **最优执行义务 / Best Execution** ⭐：券商为客户争取最优成交的法律义务。
- **Reg NMS**：美国市场结构核心规则（订单保护）。
- **暗池 / Dark Pool** ⭐：私密撮合场所，成交后才披露。
- **内部化 / Internalization**：做市商内部消化订单。
- **熔断 / Circuit Breaker**：暴跌时暂停交易。
- **涨跌幅限制**：限制单日波动。
- **MOO/MOC**：开盘/收盘市价单（按集合竞价价）。

## 清算交收 ⭐（股市最独特）
- **交易 / Trade**：达成成交协议（秒级）。
- **清算 / Clearing** ⭐：计算轧差、担保。
- **交收 / Settlement** ⭐：钱货两清（T+1/T+2）。
- **T+N**：成交后第 N 日交收。
- **轧差 / Netting** ⭐：把多笔交易压缩成净额。
- **DVP（券款对付）** ⭐：交股票和付钱同时发生。
- **中央对手方 / CCP** ⭐：插入交易中间、担保交收的机构。
- **合约更替 / Novation**：CCP 成为双方对手方的机制。
- **违约瀑布 / Default Waterfall** ⭐：CCP 分层兜底违约（≈CEX 保险基金+ADL）。
- **中央证券存管 / CSD** ⭐：登记保管证券的机构。
- **名义持有 / Street Name / Nominee**：券商代客户持有证券。
- **客户资产隔离** ⭐：客户资产与券商自有资产分开（投资者保护基石）。
- **投资者保护基金 / SIPC**：券商破产时兜底客户损失。
- **初始/变动保证金**：开仓预存 / 每日盯市追加。
- **每日盯市 / Mark-to-Market**：按收盘价重算盈亏。
- **SPAN / VaR**：CCP 保证金风险模型。

## 发行与公司行动
- **IPO（首次公开发行）** ⭐：公司首次公开卖股票。
- **承销 / Underwriting** ⭐：投行帮公司卖股票（含法律责任）。
- **招股书 / Prospectus**：IPO 的详尽披露文件。
- **锁定期 / Lock-up**：内部人上市后限售期（≈代币 vesting）。
- **直接上市 / SPAC**：非传统 IPO 的上市方式。
- **分红 / Dividend** ⭐：公司把利润分给股东。
- **除息日 / 股权登记日**：分红资格的关键日期。
- **拆股 / Split**：1 股拆 N 股（不创造价值）。
- **回购 / Buyback** ⭐：公司买回自己股票（≈平台币回购销毁）。
- **增发 / 配股**：再发新股融资（稀释）。
- **并购 / M&A**：收购/合并。
- **分拆 / Spin-off**：子公司独立上市。

## 产品
- **ETF（交易所交易基金）** ⭐：一篮子资产、可交易的基金。
- **指数基金 / 被动投资** ⭐：跟踪指数、低费、躺赢市场平均。
- **期权 / Options**：买卖未来权利（同 CEX）。
- **股指期货**：指数的期货合约。
- **融资融券 / Margin** ⭐：借钱买（做多）/ 借券卖（做空）。
- **卖空 / Short Selling**：借股票卖出、跌了买回。
- **逼空 / Short Squeeze** ⭐：空头被迫平仓推高价格（GME）。
- **可转债 / REITs**：股债混合 / 房地产信托。
- **ADR（美国存托凭证）**：外国公司在美交易的凭证（≈跨链包装）。

## 监管与风险
- **SEC** ⭐：美国证券监管机构。
- **信息披露 / Disclosure** ⭐：强制公司公开真实信息（股市信任基石）。
- **内幕交易 / Insider Trading** ⭐：用未公开信息获利（违法）。
- **市场操纵 / Manipulation** ⭐：拉高出货/幌骗/对敲等（违法）。
- **适当性 / Suitability**：确保产品适合客户。
- **熔断 / 涨跌停**：波动控制工具。
- **系统性风险**：连锁崩溃风险。
- **Reg FD（公平披露）**：信息须向所有人公平披露。

## 现代演化
- **高频交易 / HFT** ⭐：超高速算法交易（延迟套利≈MEV）。
- **托管 / Colocation**：服务器放交易所机房旁减少延迟。
- **减速带 / Speed Bump**：故意加延迟抹平速度优势（IEX）。
- **批量拍卖 / Frequent Batch Auction**：批量撮合消除速度优势（≈集合竞价/CoW）。
- **代币化 / Tokenization** ⭐：用区块链重构证券（≈用链上账本替代 CSD）。
- **执行算法 / VWAP / TWAP / IS** ⭐：把大单拆开执行（防冲击）。
- **Almgren-Chriss**：最优执行模型。
- **Implementation Shortfall**：执行落差，衡量执行质量。
