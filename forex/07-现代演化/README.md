## 核心趋势

```
1. 全电子化: FX 几乎全电子, 算法主导
2. 稳定币冲击: USDT/USDC 成为新的跨境美元渠道(绕开传统银行)
3. CBDC: 各国央行数字货币, 可能重塑跨境支付和 FX
4. 去美元化: 地缘驱动的替代支付渠道探索
```

> 💡 FX 和加密在跨境支付领域**正面相遇**：稳定币正在成为"民间的跨境美元"，挑战传统 FX/银行的跨境结算。这是加密对传统金融最实质的冲击点之一。
