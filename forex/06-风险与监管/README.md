## 核心认知 ⭐

> **零售外汇是全球金融亏损率最高的领域之一。** 100:1 杠杆 + 24 小时盯盘 + 对手是央行/银行/顶级算法 → 散户几乎注定被收割。
> 这和 [CEX 高杠杆合约](../../crypto/cex/02-产品体系/2.2-合约衍生品.md)、[DEX 高风险](../../crypto/dex/06-风险与安全/README.md) 是同一类"散户绞肉机"——**杠杆是放大器，散户是燃料。**
