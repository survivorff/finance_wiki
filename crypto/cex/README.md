# 加密货币 · CEX（中心化交易所）深度知识库

> Centralized Exchange，中心化加密货币交易所。
> 代表：Binance（币安）、Coinbase、OKX、Bybit、Upbit、Kraken 等。

CEX 是绝大多数人接触加密世界的第一个入口，也是整个加密资金流动的核心枢纽：
法币在这里进出、币在这里交易、价格在这里被发现。**理解 CEX，就理解了加密市场资金搅动的「主战场」。**

这个知识库的目标：让**完全的小白**能按路径入门，也让**行业从业者**能挖到产品、技术、运营、内幕层面的深度细节。

## 🎯 这个知识库想让你获得的「认知锚点」

读完后，你应该能脱口而出这几个本质判断：

1. **CEX ≈「交易所 + 券商 + 银行 + 托管行」的合体**——这在传统金融是被分业监管禁止集于一身的。角色集中是它效率的来源，也是它原罪的来源。
2. **你在 CEX 看到的余额，大多是数据库里的一行记录，不是链上的币。**「Not your keys, not your coins」。
3. **撮合引擎是一台确定性状态机**：相同输入序列→相同状态。这是金融级一致性的根基。
4. **CEX 的真正利润大头是合约（衍生品），不是现货。** 高杠杆是收入放大器，也是散户的绞肉机。
5. **流动性是命根子，且有飞轮效应**——这造就了头部极度垄断的格局。
6. **CEX 全部的技术和监管设计，本质都在回答一个问题：如何让一个中心化的钱袋子值得被信任。** 这个问题没被完美解决，正是 DEX 叙事的起点。

## 一张图看懂 CEX 的资金流

```
                       ┌─────────────────────────────────────┐
   法币世界            │              CEX 内部                 │      链上世界
 ┌──────────┐  入金    │  ┌──────────┐      ┌──────────────┐  │  提币  ┌──────────┐
 │ 银行卡    │ ───────▶ │  │ 法币账户  │      │ 撮合引擎      │  │ ─────▶ │ 链上钱包  │
 │ 第三方支付 │ ◀─────── │  │ (内部记账)│◀────▶│ (订单簿撮合)  │  │ ◀───── │ (用户自持) │
 └──────────┘  出金    │  └──────────┘      └──────────────┘  │  充币  └──────────┘
                       │        ▲                  ▲          │
                       │        │   ┌──────────┐   │          │
                       │        └──▶│ 用户币账户 │◀──┘          │
                       │            │ (内部记账)│              │
                       │            └──────────┘              │
                       │   交易所热钱包 / 冷钱包 实际托管资产    │
                       └─────────────────────────────────────┘
```

> **关键洞察**：用户之间的交易只是数据库里余额数字的增减（内部记账），不上链；
> 只有充值/提现才真正动用区块链。这是 CEX 高性能的来源，也是它最大的信任风险所在。
