# 5.1 IPO 上市

> 深度：🟡 进阶 ⭐
> IPO 是公司从「私有」走向「公众」的关键一跃。对照 [CEX 发币/上币](../../crypto/cex/02-产品体系/2.6-生态产品.md)，能看清「规范发行」和「加密发币」的差距。

## 什么是 IPO

```
IPO(Initial Public Offering, 首次公开发行):
  公司第一次把股票卖给公众, 从私有公司变成上市公司
目的:
  - 融资(拿到发展资金, 一级市场, 见 1.1)
  - 早期投资者/创始人退出变现
  - 提升品牌、便于后续融资、用股票激励员工
```

## IPO 流程 ⭐

```
1. 准备: 公司找投行(承销商)、审计、律师, 整理财务和合规
2. 尽职调查(Due Diligence): 投行/律师深挖公司, 确保信息真实
3. 招股书(Prospectus): 编制详尽的法律文件, 披露业务/财务/风险
   → 提交监管(如 SEC)审核
4. 路演(Roadshow): 向机构投资者推介, 收集需求(询价)
5. 定价(Pricing): 根据需求确定发行价
6. 发行 + 上市: 股票卖给投资者, 开始在交易所交易
```

## 承销商（投行）的角色 ⭐

```
承销(Underwriting): 投行帮公司把股票卖出去
  - 包销: 投行先买下全部股票再卖(承担卖不掉的风险)
  - 代销: 投行尽力卖, 卖不掉退回(风险小)
承销商责任: 尽职调查 + 对招股书真实性负责(法律责任!)
收入: 承销费(发行额的百分比)
```

- 💡 对照 [CEX Launchpad/交易所帮项目方发币](../../crypto/cex/02-产品体系/2.6-生态产品.md)：交易所也帮项目「把币卖出去」，但**几乎不承担承销商那样的法律责任和尽职调查义务**。

## IPO 定价的博弈 💡

```
定价太低: 公司"贱卖"(留了钱在桌上), 但首日大涨, 打新者爽
定价太高: 破发(上市即跌), 公司难看, 投资者亏
→ 投行要在"公司想多融资"和"投资者想低价买"之间平衡
现象: 很多 IPO 故意略微低定价 → 首日上涨 → 皆大欢喜(但公司少融了钱)
```

- 💡 对照 [CEX 打新（Launchpad）](../../crypto/cex/02-产品体系/2.6-生态产品.md)：「上线即暴涨」的打新逻辑，和 IPO 首日上涨异曲同工——都是「故意留点甜头给打新者」。

## 上市方式的变体

| 方式 | 特点 |
| --- | --- |
| **传统 IPO** | 投行承销、路演定价 |
| **直接上市（Direct Listing）** | 不发新股、不承销，直接让老股流通（如 Spotify、Coinbase） |
| **SPAC（特殊目的收购公司）** | 借壳上市，先成立空壳公司上市再并购目标 |
| **荷兰式拍卖** | 按需求拍卖定价（如 Google IPO） |

## 关键概念

```
锁定期(Lock-up): 内部人(创始人/早期投资者)上市后一段时间(如 90-180 天)不能卖
  → 防止上市就砸盘
  → 对照加密的代币 vesting/解锁(同样防砸盘, 但加密解锁更不透明)
绿鞋机制(Greenshoe): 承销商可超额配售, 稳定上市初期价格
破发(Break issue): 上市价跌破发行价
```

## IPO vs 加密发币的核心差距 ⭐

| | 股票 IPO | 加密发币(ICO/IDO/Launchpad) |
| --- | --- | --- |
| 信息披露 | 详尽招股书 + 监管审核 | 白皮书（质量参差，无强制审核） |
| 尽职调查 | 投行/律师严格尽调 | 基本没有 |
| 中介责任 | 承销商承担法律责任 | 交易所基本免责 |
| 锁定期 | 法律/规则约束 | vesting（项目方自定，可不透明） |
| 投资者门槛 | 适当性管理 | 基本无门槛 |
| 欺诈追责 | 证券法严惩 | 长期灰色地带 |

> 💡 **加密发币 = IPO 的「去监管简化版」**。同样是「把证券/代币卖给公众」，
> 但加密缺了招股书、尽调、承销责任、锁定期监管、适当性这一整套保护。
> 这就是为什么加密一级市场骗局横行（[貔貅、Rug](../../crypto/dex/06-风险与安全/6.3-骗局识别.md)），
> 而 IPO 欺诈相对少（不是没有，但有严格的事前审核和事后追责）。
> **理解 IPO 的严谨，就理解了加密发币为什么风险那么高。**

## 小结

- IPO 是公司首次公开发行股票，从私有变上市，目的是融资+退出+提升品牌。
- 流程：准备→尽职调查→招股书（监管审核）→路演询价→定价→发行上市。
- 承销商（投行）帮卖股票并承担法律责任，与加密交易所帮发币（基本免责）形成对比。
- IPO 定价博弈和打新逻辑与 CEX Launchpad 异曲同工。
- 加密发币是 IPO 的去监管简化版，缺了招股书/尽调/承销责任/锁定期/适当性——这是加密一级市场骗局多的根源。
