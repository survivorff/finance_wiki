## 一、做市报价（和 CEX/DEX 同源）

做市的核心数学（库存风险、逆向选择、Avellaneda-Stoikov 模型）已在 [CEX 8.7](../../crypto/cex/08-深水区/8.7-做市与市场微观结构.md) 讲透。这里强调：

> 💡 **这套做市理论最初就是为股市/传统市场建立的**（Avellaneda-Stoikov 2008、Glosten-Milgrom、Kyle 模型都源自股市微观结构研究）。
> [CEX 做市](../../crypto/cex/08-深水区/8.7-做市与市场微观结构.md) 和 [DEX 无常损失](../../crypto/dex/08-深水区/8.2-无常损失数学推导.md) 都是这套理论的应用/变体。
> **股市是市场微观结构学的故乡。**

### 股市做市的特殊点
```
- 指定做市商(DMM)有"义务"(见 2.2): 报价义务、参与集合竞价、极端时接盘
- 受监管约束: 报价宽度、持续性有要求
- 与 CEX/DEX 做市的"无义务"形成对比(做市责任递减光谱, 见 2.2)
```

## 二、最优执行算法（机构交易的核心）⭐

机构要交易巨量股票（如买 500 万股），不能直接砸市场（冲击成本巨大）。怎么办？用执行算法把大单拆开。

### 核心矛盾：冲击成本 vs 时间风险 ⭐
```
执行太快(一次砸出去): 冲击成本大(推动价格不利于自己, 同 DEX 滑点)
执行太慢(拖很久): 时间风险大(期间价格可能大幅不利变动)
→ 执行算法就是在"冲击成本"和"时间风险"之间找最优平衡
→ 这个框架叫"执行落差(Implementation Shortfall)"
```

### 主流执行算法

#### 1. TWAP（时间加权平均价）
```
把大单在时间上均匀切片执行:
  例: 100万股, 在 4 小时内每 5 分钟下 ~2万股
目标: 成交均价贴近"时段时间均价"
优点: 简单、可预测  缺点: 可被预测和抢跑(模式固定)
```

#### 2. VWAP（成交量加权平均价）⭐ 最常用
```
按市场"成交量分布"切片执行:
  市场放量时多下, 缩量时少下(跟随市场节奏)
目标: 成交均价贴近市场 VWAP(机构常用 VWAP 作为执行质量基准)
优点: 隐蔽性好(混在市场量里)、贴近市场均价
```

```
VWAP 基准:  VWAP = Σ(成交价 × 成交量) / Σ成交量
执行评价: 你的成交均价 vs 市场 VWAP, 越接近(或更好)说明执行越好
```

#### 3. Implementation Shortfall（IS，执行落差）⭐ 最优化框架
```
IS = 决策时的"账面价格" 与 "实际成交均价 + 未成交机会成本" 的差
目标: 最小化 IS = 最小化(冲击成本 + 时间风险 + 机会成本)
Almgren-Chriss 模型: 给出"最优执行轨迹"
  → 在冲击成本(执行快则大)和时间风险(执行慢则大)间求最优解
  → 风险厌恶越高 → 执行越快(宁可付冲击成本也要降低时间风险)
```

> 💡 **Almgren-Chriss 最优执行模型**是机构交易的理论基石。核心结论：
> **最优执行速度取决于「冲击成本、波动率、风险厌恶」的平衡**——风险厌恶高就执行快，波动大就执行快。

#### 4. POV（参与率）/ 狙击算法等
```
POV(Percentage of Volume): 始终占市场成交量的固定比例(如 10%)
狙击/流动性寻找(Liquidity Seeking): 主动找暗池/大单流动性, 伺机成交
```

## 三、执行算法的跨市场对照 ⭐

```
股市执行算法(VWAP/TWAP/IS): 把大单拆开, 防冲击、防被预测
  ↕ 同一个问题
CEX 大单: 走 OTC(见 CEX 2.2) 或 算法拆单
DEX 大单: 聚合器拆单跨池(见 DEX 2.4) + 防 MEV(见 DEX 5.2)
```

> 💡 **「如何把大单不动声色地执行掉」是所有市场的共同难题**：
> - 股市：VWAP/TWAP/IS 算法 + 暗池。
> - CEX：OTC + 算法单。
> - DEX：聚合器拆单 + 私有通道防 MEV。
> 核心都是平衡「冲击成本 vs 时间风险 vs 暴露风险」。**这是市场微观结构的终极实战问题，三个市场殊途同归。**

## 四、TWAP 的另一面：预言机 💡

```
有趣: 同一个 TWAP(时间加权均价):
  - 在股市/执行: 是"执行算法的基准"(目标成交价)
  - 在 DEX: 是"抗操纵的预言机"(见 DEX 3.4)
→ 同一个数学工具, 在不同场景有完全不同的用途
```

## 小结

- 做市理论（库存风险/逆向选择/AS 模型）源自股市微观结构研究，CEX/DEX 都是其应用——股市是这门学问的故乡。
- 最优执行的核心矛盾是「冲击成本 vs 时间风险」，框架是 Implementation Shortfall。
- 主流算法：TWAP（时间均匀）、VWAP（跟随成交量，最常用）、IS/Almgren-Chriss（最优化轨迹）、POV。
- 「大单如何不动声色地执行」是股市/CEX/DEX 共同难题，殊途同归（算法拆单+暗池/OTC/聚合器+防MEV）。
- TWAP 在股市是执行基准、在 DEX 是抗操纵预言机——同一工具不同用途。
