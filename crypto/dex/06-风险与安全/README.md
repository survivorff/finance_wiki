# 06 · 风险与安全

> 深度：🟡 进阶 → 🔴 硬核
> 目标：搞清 DEX 有哪些风险、怎么被黑、怎么识别骗局、怎么自保，以及监管态度。

DEX 没有「平台跑路」风险，但有一整套**全新的、不可逆的、无人兜底的**风险。这个模块是 DEX 的「保命指南」。

## 核心认知：风险变了形态，没有消失 ⭐

```
CEX 风险(对照 CEX 06)          DEX 风险
─────────────────────────────────────────────
平台挪用/跑路        ──消失──▶ (自托管, 没有这个风险)
被黑(交易所被盗)     ──变形──▶ 合约漏洞被黑(协议层)
                              私钥丢失/被盗(用户层)
市场风险(爆仓)       ──保留──▶ 清算/无常损失
                    ──新增──▶ MEV 被夹
                    ──新增──▶ 貔貅/Rug Pull(无许可发币)
                    ──新增──▶ 操作不可逆(转错/签错)
监管风险             ──变形──▶ 抗审查 vs 被监管打击前端
```

> ⭐ **最重要的一句话**：DEX 消灭了「信任机构」的风险，但**新增了「信任代码 + 自己操作 + 无人兜底」的风险**。
> 而且 DEX 的风险大多**不可逆、无客服、无赔付**——踩了就是踩了。
> 「去中心化」不等于「更安全」，只是「风险换了一种形态，且全部责任在你」。
