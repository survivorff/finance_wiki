# 4.3 套息交易（Carry Trade）

> 深度：🔴 硬核 ⭐
> FX 最重要、规模最大的策略之一。理解它，就理解了全球资金为什么在国家间流动。

## 什么是套息交易

```
套息交易(Carry Trade):
  借入"低利率货币" → 换成"高利率货币" → 赚利率差
经典例子(历史上的日元套息):
  1. 借日元(利率接近 0)
  2. 换成澳元/高息货币(利率较高)
  3. 赚利率差(carry)
  → 只要汇率不大幅不利变动, 就能持续赚利差
```

## 为什么套息能赚钱：UIP 不成立 ⭐

```
理论(UIP, 见 4.2): 高息货币应该贬值, 恰好抵消利率优势 → 套息不该赚钱
现实: 高息货币often 没有如期贬值(甚至升值, 因为资金涌入)
  → 套息交易者既赚利差, 有时还赚汇率升值 → 双重收益
→ 这个"理论与现实的偏离"是套息的根本盈利来源
```

- 💡 这是金融学著名的「**远期溢价之谜（forward premium puzzle）**」——UIP 在现实中系统性失效，套息策略长期有正回报。

## 套息交易的全球影响 ⭐⭐

套息不只是个交易策略，它是**全球资金流动的主引擎之一**：

```
低息国(如日本)→ 资金借出 → 流向高息国(新兴市场/澳洲等)
  → 高息国货币升值、资产价格上涨、流动性充裕
  → 全球资金从"低息洼地"流向"高息高地"
→ 这就是"利率差驱动全球资金流动"(呼应 4.1)
```

> ⭐ **这正是你想理解的「全球资金如何搅动」的核心机制之一**：
> **钱从低利率的地方流向高利率的地方（追逐 carry），利率差就是全球资金流动的"势能差"。**
> 美联储/日本央行的利率决议，能引发万亿美元的全球资金重新配置。

## 套息交易的致命风险：平仓踩踏 ⚠️⭐

```
套息"赚小钱、亏大钱"的特征:
  平时: 慢慢赚利差(稳定的小收益)
  危机时: 突然反转 → 灾难
风险事件(避险情绪 risk-off, 见 4.1):
  → 高息货币(风险货币)暴跌、低息货币(避险, 如日元)暴涨
  → 套息交易者同时亏"汇率" + 被迫平仓
  → 大量套息平仓 → 抢着买回日元 → 日元暴涨 → 更多人爆仓
  → "套息平仓踩踏"(carry trade unwind)
```

- 💡 **这和 [CEX 合约连环爆仓](../../crypto/cex/03-技术架构/3.5-清算与风控.md)、[股票逼空](../../stocks/02-参与者与产品/2.5-相关产品.md) 是同构的踩踏**：
  - 一个方向的拥挤交易 → 反转时争相平仓 → 自我强化的踩踏。
  - 套息用了杠杆，平仓时杠杆放大踩踏。
- 历史上多次「日元套息平仓」引发全球市场剧烈波动（如某些危机时刻日元单日暴涨）。

## 套息的收益结构 💡

```
套息收益 = 利率差(carry) + 汇率变动
  平时: 利率差是稳定的正收益(像"捡钱")
  尾部: 汇率剧烈反向 → 巨亏(像"在压路机前捡钢镚")
→ 收益分布: 大多数时候小赚, 偶尔巨亏(负偏态)
→ 和 卖期权/卖波动率 的收益结构一样!
```

- 💡 **深刻联系**：套息交易的收益结构 ≈ [卖出波动率](../../crypto/dex/08-深水区/8.2-无常损失数学推导.md)（同 DEX 的 LP、CEX 的做市）——
  **平时稳赚小钱（收"保险费"），尾部风险事件巨亏（"保险赔付"）。** 这种"赚小亏大"的负偏态结构在金融里反复出现（套息、做市、LP、卖期权），本质都是"承担尾部风险换取平时的稳定收益"。

## 套息 vs 加密的"挖矿/质押收益"💡

```
FX 套息: 借低息货币赚高息货币的利差
加密: "稳定币理财/质押"赚收益(见 CEX 2.4、DeFi)
共同点: 都是"追逐收益率"的资金流动
  → 资金从低收益处流向高收益处(无论是国家利率还是 DeFi APR)
  → 都有"收益率突然消失/反转"的尾部风险(套息平仓 ≈ DeFi 挤兑/暴雷)
```

## 小结

- 套息交易：借低息货币换高息货币赚利差，是 FX 规模最大的策略之一。
- 能赚钱因为 UIP 现实中不成立（高息货币没如期贬值）——著名的"远期溢价之谜"。
- 它是全球资金流动主引擎：钱从低息流向高息，利率差是资金流动的"势能差"。
- 致命风险是"平仓踩踏"（危机时高息货币暴跌、避险货币暴涨、争相平仓自我强化），同 CEX 连环爆仓/股票逼空。
- 收益结构是"赚小亏大"的负偏态，本质同卖波动率（LP/做市/卖期权），承担尾部风险换平时稳定收益。
