# 04 · 运营体系

> 深度：🟡 进阶 → 🔴 内幕
> 目标：揭开交易所「台前光鲜、幕后博弈」的运营机器——怎么拉新、怎么做市、怎么上币、怎么活下去。

技术和产品决定了交易所「能做什么」，运营决定了它「能不能赚到钱、活下去、做大」。这个模块包含大量行业内幕视角。

## 一个核心认知 ⭐

> 交易所的运营本质是在解决一个**「鸡生蛋」难题**：
> **没有用户就没有流动性，没有流动性就留不住用户。**
>
> 所有运营动作——补贴拉新、激励做市、上热门币、刷量、空投——
> 本质都是在强行启动这个飞轮，直到网络效应自我维持。理解这一点，所有运营套路都能想通。
