## 核心：风险-收益的阶梯 ⭐

```
风险/收益从低到高(收益率从低到高):
  国债(主权信用, 最安全) → 机构债 → 高评级企业债 → 高收益债(垃圾债) → 新兴市场债
利差(spread): 风险越高的债, 收益率比国债高出越多(补偿信用风险)
```

> 💡 「信用利差」是债券世界的核心概念——它衡量"市场对违约风险的定价"。利差扩大=市场恐慌（怕违约），利差收窄=市场乐观。这是债市的"恐慌指数"。
