# 2.2 按结构分类

> 深度：🟡 进阶
> 债券的"现金流结构"五花八门。理解结构，才懂不同债券的风险特征。

## 一、按票息结构

### 固定利率债（Fixed Rate）
- 票息固定（如每年 3%）。最常见。
- 利率风险大（市场利率变动，固定票息的债价格波动，见 [03](../03-利率与定价/README.md)）。

### 浮动利率债（Floating Rate）
- 票息随基准利率浮动（如 基准利率 + 1%）。
- 💡 利率涨时票息也涨 → 对利率上升的"免疫力"强（价格波动小）。

### 零息债（Zero-Coupon）⭐
```
不付票息! 折价发行, 到期按面值还本
  例: 现在花 80 买, 5 年后拿回 100 → 那 20 的差价就是"利息"
特点: 久期长(对利率最敏感, 见 03/08)、无再投资风险
```

- 💡 国库券（短期国债）就是典型零息债（折价买入，到期拿面值）。

## 二、按嵌入期权

### 可赎回债（Callable）
```
发行人有权"提前赎回"(还钱):
  利率下降时, 发行人想低息再融资 → 提前赎回旧的高息债
  → 对投资者不利(利率跌时你的高息债被收回)
  → 所以可赎回债收益率更高(补偿这个风险)
```

### 可回售债（Putable）
- 投资者有权提前要求发行人还钱（对投资者有利）。

### 可转债（Convertible Bonds）⭐
```
可转换成股票的债券:
  平时当债券收利息, 也可在条件满足时转成发行公司的股票
  → "债性 + 股性"的混合(下有保底, 上有股票上涨空间)
```

- 💡 对照 [股票相关产品](../../stocks/02-参与者与产品/2.5-相关产品.md)、[加密的混合代币](../../crypto/dex/04-生态与运营/4.2-代币经济与治理.md)：可转债是"股债混合"，加密里也有各种"混合权益"代币的尝试。

## 三、特殊债券

### 通胀挂钩债（如美国 TIPS）⭐
```
本金/票息随通胀(CPI)调整:
  → 保护投资者不被通胀侵蚀(通胀高时本金也涨)
  → 反映"实际利率"(名义利率 − 通胀)
重要: TIPS 收益率 ≈ 市场对"实际利率"的定价
     名义国债收益率 − TIPS 收益率 ≈ 市场的"通胀预期"(盈亏平衡通胀率)
```

- 💡 **这是观察"市场通胀预期"的重要工具**——把名义债和通胀债收益率一减，就得到市场隐含的通胀预期。

### 永续债（Perpetual）
- 没有到期日，永远付息（理论上）。像"债券版的优先股"。

### 资产支持证券（ABS/MBS）⚠️
```
把一堆贷款(房贷/车贷/信用卡)打包, 用其现金流支撑发行的债券
  MBS(抵押贷款支持证券): 用房贷打包
⚠️ 2008 危机的核心: 次级房贷打包成 MBS/CDO, 被评级机构给高分,
   房价跌→违约潮→这些"安全"产品崩盘→全球危机(见 06)
```

- 💡 **2008 的教训**：把高风险贷款"打包+分层+评级",看起来变安全了，实则风险被隐藏和放大。
- 💡 对照 [DeFi 的可组合套娃](../../crypto/dex/05-市场与博弈/5.3-套利与链上玩法.md)：把资产层层打包嵌套，复杂度掩盖风险——2008 的 CDO 和某些 DeFi 套娃有相似的"复杂性风险"。

## 小结

- 按票息：固定利率（最常见，利率风险大）、浮动利率（抗利率上升）、零息（折价发行，久期最长）。
- 按嵌入期权：可赎回（利率跌时被收回，收益率高补偿）、可回售、可转债（股债混合）。
- 特殊：通胀挂钩债（TIPS，保护通胀，名义-TIPS 收益率差=通胀预期）、永续债、ABS/MBS。
- ABS/MBS 把贷款打包，是 2008 危机核心（打包+分层+评级掩盖放大风险），类似 DeFi 套娃的复杂性风险。
