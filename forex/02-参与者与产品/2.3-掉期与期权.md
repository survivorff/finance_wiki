# 2.3 掉期与期权

> 深度：🟡 → 🔴
> 外汇掉期是 FX 交易量最大的产品（很多人没听过却是市场主力）。期权则提供灵活的风险管理。

## 一、外汇掉期（FX Swap）⭐ 市场主力

```
外汇掉期 = 即期 + 反向远期 的组合(打包成一笔):
  现在: 按即期汇率, 用 A 货币换 B 货币
  未来: 按远期汇率, 把 B 货币换回 A 货币
本质: "用 A 货币的资金, 借用 B 货币的资金一段时间"(货币互换式的短期资金调度)
```

### 为什么外汇掉期交易量最大 💡
```
场景: 一家银行手里有美元, 但短期需要欧元(几天):
  不想承担汇率风险(只想"临时用一下欧元")
  → 做 FX swap: 现在美元换欧元, 约定几天后换回(汇率锁定)
  → 等于"用美元抵押借欧元一段时间", 成本就是利率差
用途: 银行间资金管理、流动性调度、对冲展期
→ 因为是"滚动的短期资金调度", 重复交易量巨大 → 占 FX 交易量大头
```

- 💡 这就是为什么 [FX 日交易量那么大](../01-行业全景/1.2-市场规模与结构.md)——大量是这种短期滚动的掉期，不是方向性投机。
- 💡 对照 [DeFi 的借贷/抵押](../../crypto/defi/README.md)：FX swap 本质也是"抵押一种资产借另一种"，只是用远期锁定而非超额抵押。

## 二、货币掉期（Currency Swap）

```
货币掉期: 比 FX swap 更长期, 还交换"利息流"
  双方交换两种货币的本金 + 在存续期内交换利息 + 到期换回本金
用途: 企业跨境长期融资、对冲长期汇率和利率风险
例: 中国公司想要美元贷款但只能在国内借人民币
  → 和对手做货币掉期, 换成美元负债
```

- 💡 这是央行之间「货币互换协议」的基础——危机时各国央行互相提供本币流动性（如美联储和各国的美元互换额度，是全球美元流动性的"安全阀"）。

## 三、外汇期权（FX Options）

```
外汇期权: 未来以某汇率买/卖货币的"权利"(非义务)
  看涨/看跌期权, 付权利金获得选择权
用途:
  对冲(像买保险): 锁定最坏汇率, 但保留有利方向的收益
  投机: 用有限成本博汇率大幅波动
```

- 概念同 [股票期权](../../stocks/02-参与者与产品/2.5-相关产品.md)、[加密期权](../../crypto/cex/02-产品体系/2.2-合约衍生品.md)，定价用 Black-Scholes 的外汇版（Garman-Kohlhagen 模型）。

### 期权 vs 远期对冲的区别 💡
```
远期对冲: 锁死汇率(消除不确定, 但也放弃了有利方向的收益)
期权对冲: 付权利金 → 锁定最坏情况, 但保留有利方向收益(像买保险)
→ 远期免费但"锁死", 期权要花钱但"留有上行"
```

## 四、外汇期货（FX Futures）

```
和远期类似, 但"标准化 + 在交易所交易 + 有 CCP 清算"
  远期: OTC、定制、双边信用风险
  期货: 交易所标准合约、CCP 担保(见 股票 4.2)、每日盯市
→ FX 期货是 FX 里少数"有中央交易所"的部分(如 CME 的货币期货)
```

- 💡 有意思：FX 整体是 OTC（无中央交易所），但 FX 期货是例外（在交易所交易）——给了散户/小机构一个标准化、有 CCP 担保的参与方式。

## 产品对照总表 ⭐

| 产品 | 本质 | 主要用途 | 加密/股票对应 |
| --- | --- | --- | --- |
| 即期 Spot | 即时换汇 | 真实换汇 | 现货 |
| 远期 Forward | 锁定未来汇率 | 对冲 | 交割合约 |
| 掉期 FX Swap | 即期+反向远期 | 短期资金调度 | （抵押借币结构） |
| 货币掉期 | 长期+换利息 | 跨境长期融资 | （较少对应） |
| 期权 Options | 汇率权利 | 对冲/投机 | 期权 |
| 期货 Futures | 标准化远期 | 投机/对冲 | 交割合约(交易所版) |

## 小结

- 外汇掉期（FX swap，即期+反向远期）是 FX 交易量最大的产品，本质是短期跨币种资金调度，类似抵押借币。
- 货币掉期更长期且交换利息流，是央行间货币互换协议的基础（全球美元流动性安全阀）。
- 外汇期权提供"买保险"式对冲（锁定最坏、保留上行），区别于远期的"锁死"。
- 外汇期货是 FX 里少数有中央交易所+CCP 的部分，给小机构标准化通道。
