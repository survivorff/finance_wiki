# 4.1 什么是收益率曲线

> 深度：🟡 进阶 ⭐
> 收益率曲线是债券世界的核心图表，浓缩了市场对未来的全部预期。

## 定义

```
收益率曲线(Yield Curve) = 同一发行人(通常是国债)、不同期限的债券收益率连成的曲线
  X 轴: 到期期限(3个月 → 2年 → 5年 → 10年 → 30年)
  Y 轴: 对应的收益率
最常看: 美国国债收益率曲线(全球基准)
```

## 三种典型形状 ⭐

```
1. 正常曲线(向上倾斜) — 最常见
   收益率
    │        ___ 30年
    │     __/
    │  __/ 10年
    │_/ 2年
    └──────────── 期限
   含义: 期限越长收益率越高(正常: 借得久要更高补偿)
   → 经济正常、温和增长预期

2. 倒挂曲线(向下倾斜) — 衰退警报 ⚠️
   收益率
    │_  2年(短期反而高!)
    │ \__
    │    \___ 10年(长期反而低)
    └──────────── 期限
   含义: 短期利率 > 长期利率(反常!)
   → 市场预期未来要降息(因为预期衰退) → 著名的衰退先行指标(见 4.2)

3. 平坦曲线 — 转折期
   各期限收益率接近 → 经济不确定/政策转折
```

## 为什么正常曲线"向上倾斜"⭐

```
1. 期限溢价: 借得越久, 不确定性越大(通胀/违约/利率变动), 要更高补偿
2. 增长/通胀预期: 经济正常时, 预期未来利率会更高(增长+通胀)
→ 所以长期收益率通常 > 短期收益率(向上倾斜是"常态")
```

## 收益率曲线浓缩了什么信息 ⭐

```
曲线的每个点和整体形状, 反映市场对:
  - 未来的利率路径(央行会加息还是降息)
  - 通胀预期(通胀高→长期收益率高)
  - 经济增长预期
  - 风险偏好
→ 是市场"集体智慧"对未来的定价
```

> 💡 **为什么它是"水晶球"**：收益率曲线是几万亿美元真金白银投票出来的对未来的预期。
> 它不是某个专家的观点，而是整个市场用钱表达的判断——所以信息含量极高，且常常很准。

## 曲线如何形成 💡

```
短端(短期利率): 主要由央行政策利率决定(见 05、FX 5.1)
  → 央行加息, 短端上移
长端(长期利率): 由市场对"长期增长 + 通胀 + 期限溢价"的预期决定
  → 市场买卖长期国债, 形成长端收益率
→ 短端"政策驱动", 长端"市场驱动" → 两端的相对位置决定曲线形状
```

- 💡 这解释了倒挂：当央行猛加息（短端被推高），但市场预期未来衰退要降息（长端走低）→ 短端超过长端 → 倒挂（见 [4.2](./4.2-倒挂与经济预测.md)）。

## 曲线的运动 💡

```
平移(整体上移/下移): 利率整体变化
变陡(steepening): 长端比短端涨更多(预期增长/通胀回升)
变平/倒挂(flattening/inversion): 短端比长端高(央行收紧+衰退预期)
→ 交易员有专门的"曲线交易"(赌曲线形状变化, 而非单一利率)
```

## 小结

- 收益率曲线 = 不同期限国债收益率连成的线，浓缩市场对未来利率/通胀/增长的全部预期。
- 三种形状：正常（向上倾斜，常态）、倒挂（向下，衰退警报）、平坦（转折期）。
- 正常向上倾斜因期限溢价+增长通胀预期；短端由央行政策驱动，长端由市场预期驱动。
- 它是几万亿美元投票的"水晶球"，信息含量极高；交易员有专门的曲线交易。
