# 2.2 做市商与流动性

> 深度：🟡 → 🔴
> 做市商是股市流动性的核心。这是 [CEX 做市](../../crypto/cex/04-运营体系/4.3-做市与流动性运营.md) 和 [DEX 的 LP](../../crypto/dex/02-产品与机制/2.3-流动性与无常损失.md) 的「传统金融原版」。

## 做市商是谁

```
做市商(Market Maker) = 持续报出买价和卖价、随时准备成交的专业机构
作用: 保证你想买时有人卖、想卖时有人买(提供流动性)
盈利: 赚买卖价差(spread) + 返佣/各种激励
```

- 没有做市商，冷门股票可能挂单稀疏、买卖困难——这点三个市场完全一样。

## 股市做市的几种角色

### 1. 指定做市商（DMM / Specialist，NYSE 模式）
```
NYSE 给每只股票指定一个"主做市商(DMM)":
  - 有义务维持该股票的"公平有序"市场
  - 开盘/收盘集合竞价时负责定价
  - 极端波动时有义务用自有资金"逆势"接盘, 平滑波动
→ 权利(优先信息)和义务(维持秩序)并存
```

### 2. 电子做市商 / 高频做市（现代主流）
```
Citadel Securities、Virtu 等: 用算法高速报价
  - 在多个交易所/平台同时做市
  - 极低延迟、海量报价、快速调整
→ 现代股市流动性的主要提供者(见 07 高频交易)
```

### 3. 批发做市商（接散户订单流）
- 接收券商卖来的散户订单流（PFOF，见 [2.1](./2.1-交易所与券商.md)），在内部成交（内部化）。

## 做市的经济学（和 CEX/DEX 同源）⭐

做市商面对的核心问题，三个市场一模一样：

```
赚: 买卖价差 spread
亏: 1. 库存风险 — 持仓在手, 价格不利波动就亏
    2. 逆向选择 — 和"知情交易者"成交时系统性吃亏
```

> 💡 这正是 [CEX 8.7 做市与微观结构](../../crypto/cex/08-深水区/8.7-做市与市场微观结构.md) 讲的库存风险和逆向选择，
> 也是 [DEX 无常损失](../../crypto/dex/02-产品与机制/2.3-流动性与无常损失.md) 的根源。
> **做市的数学本质（价差补偿库存风险和逆向选择）跨越所有市场——股市是这套理论的发源地。**
> Avellaneda-Stoikov 等做市模型最初就是为股市/传统市场建立的（见 [08 深水区](../08-深水区/README.md)）。

## 做市义务 vs DEX 的 LP（关键对照）⭐

```
股市 DMM:    有"义务"维持秩序(法律/交易所规则要求), 极端时要接盘
CEX 做市商:  商业行为, 靠返佣激励, 无强制义务(可随时撤)
DEX LP:      纯被动公式, 不会"主动调价", 也无义务(可随时撤资)
```

- 💡 越往「去中心化」走，做市的「义务」越弱、越被动：
  - 股市做市商有义务、有牌照、受监管。
  - CEX 做市商是商业合作。
  - DEX 的 LP 是「无义务、被动、人人可当」的极端——代价是承担无常损失。
- 这是一条清晰的「做市责任递减」光谱。

## 价差与流动性指标（同 CEX/DEX）

- **买卖价差（Bid-Ask Spread）**：越小流动性越好。
- **市场深度（Depth）**：各价位的挂单量。
- **流动性好的股票**（大盘蓝筹）：价差极小、深度厚。
- **流动性差的股票**（小盘冷门）：价差大、易被操纵——和 [加密小币](../../crypto/cex/05-市场与博弈/5.4-市场操纵与黑幕.md) 一样。

## 小结

- 做市商持续双边报价提供流动性，赚价差，是股市流动性核心。
- 角色：指定做市商(DMM，有维持秩序义务)、电子高频做市商(现代主流)、批发做市商(接订单流)。
- 做市经济学（价差补偿库存风险+逆向选择）跨越股市/CEX/DEX，股市是这套理论发源地。
- 「做市义务」呈递减光谱：股市做市商有义务 > CEX 商业激励 > DEX 的 LP 纯被动无义务。
