## 核心收获 ⭐

```
链上借贷(Aave/Compound 等):
  存款人: 把币存入资金池, 赚利息
  借款人: 超额抵押后, 从池子借币, 付利息
  利率: 由"资金利用率"算法决定(用得多→利率高)
  清算: 抵押率跌破阈值 → 清算人卖抵押品还债(对照 CEX 强平)
关键: 超额抵押(因匿名无法追债, 见 2.2) + 算法利率 + 清算机器人
```

> ⭐ DeFi 借贷 = 链上的"银行 + 债券市场"，但用**超额抵押**替代信用、用**算法**替代人定利率、用**清算机器人**替代追债。理解它，就理解了 DeFi 怎么在没有信用体系的匿名世界里实现借贷。
