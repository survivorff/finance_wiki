# 5.3 套利与链上玩法

> 深度：🔴 硬核
> 链上独有的套利和玩法。对照 [CEX 套利](../../cex/05-市场与博弈/5.3-典型玩法与套利.md)，看资金如何在链上和链间流动。
> ⚠️ 原理科普，非投资建议。链上玩法门槛高、风险大，描述中的「套利」均指扣除 Gas/MEV/风险前的理想情况。

## 一、DEX 间套利（链上原生）

```
同一币对在 Uniswap 和 Curve 价格不同:
  在便宜的池子买, 贵的池子卖, 赚差价
→ 把两个池子价格拉平(校准 AMM, 见 2.1)
```
- ⚠️ 这本身就是一种 MEV（套利 MEV，见 [5.2](./5.2-MEV详解.md)），竞争极激烈，普通人抢不过机器人。

## 二、闪电贷套利（链上独有）⭐

```
利用闪电贷(见 1.4)无需本金:
  1. 闪电贷借入 100万 USDC(无抵押)
  2. 在 A DEX 买入被低估的 ETH
  3. 在 B DEX 卖出 ETH(价格高)
  4. 归还 100万 USDC + 手续费
  5. 剩下的就是利润
  全在一笔原子交易内 → 不成功就回滚(等于没发生), 无本金风险
```

- 💡 **这是传统金融不可能的玩法**：无抵押借巨款做套利，靠原子性兜底。
- 门槛：要会写合约、抢得过 MEV 竞争、算清 Gas 成本。

## 三、CEX-DEX 套利 ⭐ 连接两个世界

```
CEX(如币安) 和 DEX(如 Uniswap) 价格有差异时:
  在便宜的一边买, 贵的一边卖
难点:
  - 跨越中心化和去中心化(要在 CEX 有账户 + 链上钱包)
  - DEX 这边有 Gas、MEV、出块延迟
  - 要管理两边的库存和资金调度
```

- 💡 这是连接 [CEX](../../cex/05-市场与博弈/5.3-典型玩法与套利.md) 和 DEX 两个市场资金流的关键通道。专业机构主导。
- CEX 价格通常「领先」（流动性深、快），DEX 跟随——套利者把 DEX 价格拉向 CEX。

## 四、资金费率 / 基差套利（衍生品）

- 和 [CEX 资金费率套利](../../cex/05-市场与博弈/5.3-典型玩法与套利.md) 同理，但用衍生品 DEX（如 Hyperliquid、GMX）。
- 可以「CEX 现货/合约 + DEX 合约」组合对冲，跨平台吃资金费差。

## 五、收益耕作（Yield Farming）的套利化

```
资金在各协议间追逐最高真实收益:
  在 A 协议 LP 挖矿 → APR 降了 → 转到 B 协议
  组合: 借贷 + LP + 质押 多层嵌套(可组合性, 见 1.4)放大收益
  "收益聚合器"(如 Yearn)自动帮你找最优、自动调仓
```
- ⚠️ 嵌套越多，风险叠加越多（每层都有合约风险、清算风险）。

## 六、链上套利的特殊约束 ⭐

| 约束 | 影响 |
| --- | --- |
| **Gas 成本** | 小额套利不划算（Gas 吃掉利润） |
| **MEV 竞争** | 机会被搜索者抢跑，普通人吃不到 |
| **出块延迟** | 机会可能在你的交易上链前消失 |
| **滑点** | 大额套利自身会移动价格 |
| **原子性** | 闪电贷套利可无风险（不成就回滚） |

## 一个清醒认知 ⭐

> 和 [CEX 套利](../../cex/05-市场与博弈/5.3-典型玩法与套利.md) 一样的结论：
> **链上「看起来稳赚」的套利，几乎都被专业机器人（MEV 搜索者）和高频机构抢光了。**
> 普通人手动套利：要么被抢跑，要么 Gas 吃掉利润，要么算错被夹。
> 套利的存在让市场更有效（价格更一致），但超额收益早被竞争压到极薄——这是「有效市场」在链上的体现。

## 小结

- 链上套利：DEX 间套利、闪电贷套利（无本金，链上独有）、CEX-DEX 套利（连接两世界）、资金费率/基差套利、收益耕作。
- 闪电贷靠原子性实现「无抵押无风险套利」，是传统金融不可能的玩法。
- CEX-DEX 套利是连接中心化与去中心化资金流的关键通道，CEX 价格通常领先。
- 链上套利受 Gas、MEV 竞争、出块延迟约束；超额收益被专业机器人压薄，散户难吃到。
