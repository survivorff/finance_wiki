# 5.2 汇率制度

> 深度：🟡 进阶 ⭐
> 不同国家用不同的汇率制度，背后是「不可能三角」这个深刻的约束。

## 汇率制度光谱 ⭐

```
完全固定 ◀──────────────────────────────────▶ 完全浮动
│            │              │              │           │
美元化      联系汇率       钉住一篮子      有管理浮动   自由浮动
(用美元)   (港币钉美元)   (钉货币篮子)    (中国等)    (美/欧/日)
```

| 制度 | 特点 | 例子 |
| --- | --- | --- |
| **美元化** | 直接用美元，放弃本币 | 部分小国 |
| **联系汇率** | 本币严格钉住某外币 | 港币钉美元 |
| **钉住一篮子** | 钉住多种货币的加权 | 部分国家 |
| **有管理浮动** | 市场决定为主，央行干预 | 中国（参考一篮子） |
| **自由浮动** | 完全由市场决定 | 美元、欧元、日元 |

## 不可能三角（三元悖论）⭐⭐

这是国际金融最重要的理论之一：

```
三个目标, 只能同时实现两个:
        ┌─────────────────────┐
        │   ① 固定汇率         │
        │        ╱   ╲         │
        │      ╱       ╲       │
        │ ②自由      ③独立     │
        │ 资本流动   货币政策   │
        └─────────────────────┘
  你只能选两个, 必然放弃第三个
```

### 三种组合
```
选 ①+② (固定汇率 + 自由资本流动) → 放弃 ③独立货币政策
  例: 港币(钉美元 + 资本自由) → 香港利率必须跟随美国, 没有独立货币政策

选 ①+③ (固定汇率 + 独立货币政策) → 放弃 ②自由资本流动
  例: 用资本管制(限制资金进出)来同时固定汇率和独立定利率

选 ②+③ (自由资本流动 + 独立货币政策) → 放弃 ①固定汇率
  例: 美国/欧洲(汇率自由浮动 + 资本自由 + 独立货币政策)
```

> ⭐ **为什么不可能三个都要**：
> 如果你又想固定汇率、又想资本自由流动、又想独立定利率——
> 一旦你的利率和别国不同，资金就会大量套利流动（[套息](../04-汇率原理/4.3-套息交易.md)），
> 冲击你的固定汇率，你要么放弃固定汇率，要么管制资本，要么放弃独立利率。**数学上无解。**

## 不可能三角的现实选择 💡

```
中国: 选 ①固定(有管理) + ③独立货币政策 → 用资本管制(放弃②)
  → 这就是为什么人民币有在岸(CNY)和离岸(CNH)之分(见 1.3)
美国/欧元区: 选 ②自由 + ③独立 → 汇率自由浮动(放弃①)
香港: 选 ①固定(钉美元) + ②自由 → 放弃独立货币政策(利率跟美国)
```

- 💡 **这个三角解释了很多现象**：为什么中国有资本管制、为什么香港利率跟美国、为什么发达国家汇率自由浮动。每个国家都在三角里做了取舍。

## 固定 vs 浮动的权衡

| | 固定汇率 | 浮动汇率 |
| --- | --- | --- |
| 优点 | 稳定、利于贸易和投资预期 | 自动调节、独立货币政策、缓冲外部冲击 |
| 缺点 | 易被投机攻击、丧失货币政策独立性 | 汇率波动、不确定性 |
| 风险 | 维持成本高，崩溃时剧烈（见 [5.3](./5.3-干预与货币危机.md)） | 波动带来对冲需求 |

## 对照加密/DeFi 💡

```
"不可能三角"思维在加密里也有回响:
  区块链不可能三角: 去中心化、安全、可扩展, 三者难兼得(见 CEX 8.5)
  稳定币的"三难": 价格稳定、资本效率、去中心化, 难兼得
→ "三选二"的约束是复杂系统的普遍规律
```

- 💡 货币的不可能三角和区块链/稳定币的三难困境，是同一种"系统性约束"的不同体现——**鱼与熊掌的取舍无处不在**。

## 小结

- 汇率制度是光谱：从美元化/联系汇率（固定）到自由浮动，各国按需选择。
- 不可能三角：固定汇率、自由资本流动、独立货币政策三者只能取其二（数学上无解三全）。
- 现实选择：中国（固定+独立，资本管制）、美欧（自由+独立，浮动汇率）、香港（固定+自由，利率跟美）。
- 固定汇率稳定但易被攻击、丧失货币政策独立性；浮动汇率能自动调节但有波动。
- "三选二"约束在区块链不可能三角、稳定币三难中同样出现——系统性约束的普遍规律。
