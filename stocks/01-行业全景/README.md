## 核心收获

- 股票是「公司所有权的碎片」，股市是买卖这些碎片的场所 + 公司融资的渠道。
- 全球股市市值达百万亿美元量级，美国(NYSE/Nasdaq)最大。
- 股市的灵魂是**分业制衡的产业链**——这是和 CEX 最根本的区别。
- 股市几百年历史 = 一部「危机 → 监管 → 制度完善」的演化史。
