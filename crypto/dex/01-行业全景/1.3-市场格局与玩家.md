# 1.3 市场格局与玩家

> 深度：🟢🟡
> 注：DEX 数据链上公开（可在 DefiLlama、Dune 查），但随市场波动大，本文给量级和相对格局。

## 衡量 DEX 的关键指标（链上可查）⭐

和 CEX 不同，DEX 数据**几乎全部链上公开、难以造假**（这是去中心化的好处）：

| 指标 | 含义 | 在哪看 |
| --- | --- | --- |
| **TVL（Total Value Locked）** | 锁定在协议里的总资产（流动性规模） | DefiLlama |
| **交易量（Volume）** | 链上成交额 | DefiLlama, Dune |
| **DEX/CEX 现货量比** | DEX 占整体现货交易的比例 | The Block |
| **手续费/收入** | 协议产生的真实手续费 | Token Terminal |
| **活跃地址** | 真实使用人数（地址≠人，但难造假） | Dune |

> 💡 **TVL** 是 DeFi 最常用的指标，类似「这个协议管了多少钱」。但要注意 TVL 受币价影响大，且可被「自家代币锁仓」虚高。

| DEX | 类型 | 链 | 特点 |
| --- | --- | --- | --- |
| **Uniswap** | AMM | 以太坊 + 多链 | DEX 鼻祖与龙头，V2/V3/V4 引领创新 |

## 格局被「公链」切割 ⭐

DEX 的市场不是按公司分，而是**按公链生态分**——每条链都有自己的头部 DEX：

```
以太坊主网:  Uniswap, Curve, Balancer     (老牌、安全、Gas 贵)
L2(Arbitrum/Base/OP):  Uniswap, Aerodrome, GMX  (便宜、快)
Solana:      Jupiter, Raydium, Orca       (高速、低费、meme 币天堂)
BSC:         PancakeSwap                   (散户多、便宜)
其他:        各链都有本地龙头
```

> 💡 这和 CEX 的「头部通吃全球」很不同。DEX 因为依附于公链，所以**格局更碎片化、更随公链兴衰起落**。
> Solana 火 → Jupiter/Raydium 火；Base 火 → Aerodrome 火。**选对链，比选对 DEX 更重要。**

## DEX vs CEX 的市场份额

- 现货交易量：**CEX 仍远大于 DEX**，但 DEX 占比长期上升趋势。
- 衍生品：DEX 份额增长更快，Hyperliquid 等已能在永续合约领域对头部 CEX 构成实质竞争。
- 💡 每次 CEX 出事（FTX 暴雷、某所被查），DEX 的份额和 TVL 就会跳涨——**DEX 是 CEX 信任危机的直接受益者**。

## 玩家生态（谁在 DEX 里活动）

| 玩家 | 角色 |
| --- | --- |
| **散户交易者** | 换币、打新、玩 meme |
| **流动性提供者(LP)** | 存币进池子赚手续费（对应 CEX 做市商，但人人可当） |
| **套利者/MEV 搜索者** | 抹平价差、抢跑夹击（见 [05](../05-市场与博弈/README.md)） |
| **协议开发者** | 部署和维护 DEX 合约 |
| **DAO/治理代币持有者** | 参与协议治理投票 |
| **项目方** | 在 DEX 上发币、建初始流动性池 |
| **机构/做市商** | 专业 LP、跨 CEX-DEX 套利 |

## 小结

- DEX 数据链上公开难造假，核心指标是 TVL、交易量、手续费、活跃地址。
- 头部：Uniswap/Curve（AMM）、Hyperliquid/dYdX/GMX（衍生品）、Jupiter（聚合）。
- 格局按「公链生态」切割且碎片化，随公链兴衰起落——选对链很关键。
- CEX 现货量仍领先，但 DEX 占比上升，是 CEX 信任危机的直接受益者。
