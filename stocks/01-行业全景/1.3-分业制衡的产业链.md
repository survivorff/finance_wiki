# 1.3 分业制衡的产业链 ⭐

> 深度：🟡 进阶 ⭐ 本知识库最核心的一篇
> 这是股市和 CEX 最根本的区别，也是几百年金融监管智慧的结晶。

## 核心思想：把交易所、券商、做市、清算、托管「拆开」⭐

回忆 [CEX 的"原罪"](../../crypto/cex/01-行业全景/1.1-CEX是什么.md)：一家公司同时是交易所+券商+银行+托管。
**股票市场用一整条分工明确、相互制衡的产业链，把这些角色彻底拆开。**

```
你(投资者)
   │
   ▼
┌─────────────┐  下单/代理
│ 券商 Broker  │  只负责"代理你买卖"，不撮合、不保管(资产在存管)
└─────────────┘
   │ 路由订单
   ▼
┌─────────────┐  撮合
│ 交易所       │  只负责"撮合买卖"，不碰你的钱和股票
│ Exchange    │
└─────────────┘
   │ 成交
   ▼
┌─────────────┐  做市
│ 做市商 MM    │  只负责"提供流动性"，赚价差
└─────────────┘
   │
   ▼
┌─────────────┐  清算(担保)
│ 清算所 CCP   │  成为买卖双方对手方，担保交收，管理违约风险
└─────────────┘
   │
   ▼
┌─────────────┐  存管(保管)
│ 中央存管 CSD │  真正"保管"股票的登记，券商代你持有
└─────────────┘
   │
   ▼
┌─────────────┐  存钱
│ 银行         │  保管资金
└─────────────┘
```

## 各角色「只做一件事」⭐

| 角色 | 唯一职责 | 不能做什么 |
| --- | --- | --- |
| **交易所** | 撮合订单 | 不碰客户资产、不代理下单 |
| **券商(Broker)** | 代理客户下单、开户 | 不撮合、不（独自）保管资产 |
| **做市商(MM)** | 提供流动性、报价 | 受规则约束，行为被监控 |
| **清算所(CCP)** | 担保交收、管违约 | 不撮合、不代理 |
| **中央存管(CSD)** | 登记保管证券 | 不交易、不撮合 |
| **银行** | 保管资金 | 不碰证券 |
| **监管机构** | 监督所有人 | 制定规则、执法 |

> ⭐ **为什么要这样拆？核心是"没有任何一方能独自接触你的全部资产 + 全部交易信息 + 规则制定权"。**
> - 交易所想作弊？它碰不到你的钱（钱在银行、券商、存管）。
> - 券商想挪用？你的股票在独立的中央存管，有独立记录可核对。
> - 做市商想操纵？交易所和监管在盯着。
> - 任何一方出问题，其他方和监管能发现、能制衡、能追责。

## 这正是 FTX 暴雷的反面教材 ⭐

```
FTX(CEX): 交易所 + 自营做市(Alameda) + 托管 集于一身
  → 能挪用用户资产、能看用户底牌、能自我交易
  → 没有任何外部制衡 → 暴雷(见 CEX 6.4)

股票市场: 这些角色被法律强制分开
  → 券商不能挪用你的股票(在独立存管, 且有客户资产隔离规定)
  → 即使券商破产, 你的股票还在存管, 能转移到别的券商
  → 这就是"分业制衡"用几百年危机换来的保护
```

> 💡 **如果 FTX 是在股票市场的监管框架下运营，它根本不可能挪用用户资产去做自营**——
> 因为托管、交易、自营会被强制分离，且有独立审计和监管。
> **CEX 的"高效一体化"，恰恰丢掉了股市用血泪换来的"分业制衡"保护。**

## 客户资产隔离（投资者保护的基石）

```
券商必须把"客户资产"和"券商自有资产"严格分开:
  → 客户的股票/资金不能被券商挪用、不能用于偿还券商的债务
  → 券商破产时, 客户资产受保护, 可转移
  → 还有"投资者保护基金"(如美国 SIPC)兜底券商失败造成的损失
```

- 💡 对照：[CEX 直到 FTX 后才开始强调"客户资产隔离"](../../crypto/cex/06-风险与监管/6.3-全球监管格局.md)——而股市几十年前就是法律强制了。

## 这条产业链的代价

- **慢**：层层中介、T+2 交收（见 [04](../04-清算交收/README.md)）。
- **贵/复杂**：多方分润、流程繁琐。
- **门槛高**：每个环节都要牌照、合规。

> 💡 这就是「**效率 vs 安全**」的经典权衡，股市选择了安全，CEX 选择了效率。
> 两者没有绝对优劣，但理解这个权衡，你就理解了金融基础设施设计的核心矛盾。

## 小结

- 股市把交易所/券商/做市/清算/存管/银行/监管严格分业、相互制衡。
- 每个角色「只做一件事」，没有任何一方能独自接触你的全部资产和交易。
- 这是 FTX 暴雷的反面教材——分业制衡从制度上杜绝了挪用和自我交易。
- 客户资产隔离 + 投资者保护基金是投资者保护的基石。
- 代价是慢、贵、复杂——这是「安全 vs 效率」权衡中股市选择了安全。
