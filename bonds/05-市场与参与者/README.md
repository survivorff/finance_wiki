## 核心收获

- 债券一级市场（政府拍卖/企业承销）发行，二级市场以 OTC（类似 [FX](../../forex/01-行业全景/1.2-市场规模与结构.md)）为主。
- 央行通过买卖国债（公开市场操作/QE）调节利率和流动性——债市是货币政策的传导枢纽。
- 回购市场（Repo）是金融体系的"水管"，2008 危机就是从这里冻结的。
