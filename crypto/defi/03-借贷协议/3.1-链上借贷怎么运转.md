# 3.1 链上借贷怎么运转

> 深度：🟡 进阶 ⭐
> 链上借贷是 DeFi 仅次于稳定币和 DEX 的核心。理解资金池模式和超额抵押，就懂了链上银行。

## 资金池模式（Pool-Based）⭐

```
传统借贷: 一个借款人 对 一个贷款人(点对点匹配)
DeFi 主流(Aave/Compound): "资金池"模式
  存款人 → 把币存入一个共享的"资金池" → 赚利息
  借款人 → 从池子借币(超额抵押) → 付利息
  → 不用点对点匹配, 池子自动调节(像 AMM 的池子思想, 见 DEX 2.1)
```

- 💡 对照 [AMM 资金池](../../dex/02-产品与机制/2.1-AMM是什么.md)：借贷协议也用"池子"思想——大家把钱汇集到池子，算法自动撮合供需，无需点对点。

## 存款方：赚利息 ⭐

```
你把 USDC 存入 Aave:
  → 获得"存款凭证"(如 aUSDC), 代表你在池子的份额 + 累积利息
  → 利息来自借款人付的利息(扣除协议抽成)
  → 随时可取回(只要池子有足够流动性)
对照: 银行存款(见 债券)、CEX 理财(见 CEX 2.4)
  区别: 链上透明、无许可、利率算法化、但无存款保险
```

## 借款方：超额抵押借币 ⭐

```
你想借 USDC, 但你是匿名的, 协议怎么敢借给你?
→ 超额抵押(见 2.2):
  1. 你存入价值 150 美元的 ETH 作抵押
  2. 能借出最多 ~100 美元的 USDC(抵押率 150%, 或说 LTV 66%)
  3. 付利息
  4. 还清 USDC + 利息 → 取回 ETH
为什么超额: 匿名无法追债(见 2.2), 留缓冲防抵押品(ETH)价格波动
```

### 关键参数 ⭐
```
LTV(Loan-to-Value, 贷款价值比): 能借出占抵押品的比例(如 66% → 抵押 150 借 100)
清算阈值(Liquidation Threshold): 抵押率跌到这, 触发清算(如 80%)
健康因子(Health Factor): 衡量你离清算多远(< 1 触发清算, 见 3.2)
```

## 为什么有人要"超额抵押借款"💡

```
看似奇怪: 我有 150 的 ETH, 为啥要抵押它借 100 USDC?(还不如直接卖 ETH)
原因:
1. 不想卖 ETH(看好长期), 但要流动性(借 USDC 用) — 同 债券抵押借款逻辑
2. 加杠杆: 借 USDC 买更多 ETH 再抵押再借(循环, 见 1.2)
3. 套利/收益耕作: 借出来去别处赚更高收益
4. 避税(某些地区借款不算应税事件)
```

- 💡 对照 [CEX 杠杆/借贷](../../cex/02-产品体系/2.3-杠杆交易与借贷.md)、[债券回购](../../../bonds/05-市场与参与者/5.3-回购市场.md)：链上借贷的用途（不卖资产拿流动性、加杠杆、套利）和传统抵押借款完全一样。

## 闪电贷（Flash Loan）⭐ DeFi 独有

```
闪电贷: "无抵押"借巨款, 但必须在"同一笔交易内"归还(否则整笔回滚)
  → 因为不还就当没发生过, 所以无需抵押(见 DEX 1.4)
用途: 套利、清算、抵押品置换(都在一笔原子交易内)
⚠️ 也是攻击武器: 借巨款操纵价格/攻击漏洞(见 06、DEX 3.4)
```

- 💡 这是传统借贷不可能的——只有链上原子性才能实现"不还就回滚"的无抵押贷。

## 链上借贷 vs 传统借贷 ⭐

| | DeFi 借贷 | 传统银行/债券 |
| --- | --- | --- |
| 凭什么借给你 | 超额抵押 | 信用 + 法律追债 |
| 利率谁定 | 算法（利用率，见 3.2） | 银行/市场 |
| 撮合 | 资金池 | 点对点/市场 |
| 违约处理 | 清算抵押品 | 起诉/催收 |
| 准入 | 无许可、匿名 | KYC、信用审核 |
| 兜底 | 无 | 存款保险等 |
| 闪电贷 | 有（独有） | 不可能 |

> 💡 **核心差异**：DeFi 借贷**没有信用概念**（匿名），完全靠"超额抵押+清算"。
> 这让它无许可、高效，但也意味着**不能像传统那样靠信用放贷**（无法做无抵押信贷）——这是 DeFi 借贷的能力边界。

## 小结

- 链上借贷用"资金池"模式：存款人存币赚息，借款人超额抵押借币付息，无需点对点（同 AMM 池思想）。
- 超额抵押（如抵押 150 借 100）是根基——因匿名无法追债，靠抵押品+清算代替信用+法律。
- 关键参数：LTV、清算阈值、健康因子。
- 借款用途和传统一样（不卖资产拿流动性、加杠杆、套利）。
- 闪电贷是 DeFi 独有（原子性实现无抵押瞬时贷）；DeFi 借贷无信用概念，做不了无抵押信贷。
