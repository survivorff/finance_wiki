# 4.2 利率平价与购买力平价

> 深度：🔴 硬核
> FX 两大经典理论：利率平价（短中期、决定远期汇率）和购买力平价（长期）。

## 一、利率平价（Interest Rate Parity, IRP）⭐

### 核心思想
```
两种投资方式应该回报相等(否则有无风险套利):
  方式 A: 把钱存成本币, 赚本币利率
  方式 B: 换成外币, 存成外币赚外币利率, 到期换回
→ 如果两者回报不等, 就能无风险套利 → 市场会消除差异
→ 结果: 远期汇率 由 两国利率差 锁定
```

### 覆盖利率平价（CIP, Covered）⭐
```
"覆盖" = 用远期合约锁定未来汇率(无汇率风险)
公式:  远期汇率 / 即期汇率 = (1 + 本币利率) / (1 + 外币利率)

直觉: 高利率货币 → 远期"贴水"(远期比即期便宜)
     低利率货币 → 远期"升水"(远期比即期贵)
     → 远期的升贴水 恰好抵消 利率差 → 无套利
```

- 💡 这就是 [远期汇率定价](../02-参与者与产品/2.2-即期与远期.md) 的来源。CIP 在正常市场几乎严格成立（套利保证），详细推导见 [08 深水区](../08-深水区/8.1-远期掉期定价.md)。
- 💡 对照 [CEX 期现套利/资金费率](../../crypto/cex/08-深水区/8.2-永续合约数学全解.md)：完全同一类"无套利定价"——期货/远期价格被现货+利率/资金成本锁死。

### 非覆盖利率平价（UIP, Uncovered）
```
"非覆盖" = 不用远期锁定, 赌未来即期汇率
UIP 理论: 高利率货币"应该"贬值, 恰好抵消利率优势
现实: UIP 经常【不成立】! 高利率货币往往没有如期贬值
  → 这个"偏离"正是套息交易(carry trade)能赚钱的原因(见 4.3)
```

- 💡 **CIP 成立（套利保证），但 UIP 经常不成立**——这个理论与现实的差距，是套息交易这个万亿美元策略的根基。

## 二、购买力平价（Purchasing Power Parity, PPP）⭐

### 核心思想
```
长期看, 汇率应该使"同样的钱在两国买到同样多的东西":
  如果一个汉堡在美国 5 美元、在英国 4 英镑
  → 长期 GBP/USD 应趋向 5/4 = 1.25(购买力相等)
→ 汇率长期向"购买力相等"的水平靠拢
```

### 巴拉萨-萨缪尔森 与「大麦克指数」💡
```
大麦克指数(Big Mac Index): 经济学人用麦当劳巨无霸价格比较各国货币高估/低估
  → PPP 的通俗版: 同一个汉堡各国价格换算, 看货币是否"便宜/贵"
现实: PPP 长期大致成立, 但短期严重偏离
  原因: 很多东西不可贸易(理发/房租)、关税、运输成本、品牌等
```

- 💡 PPP 是**长期**理论——短期汇率可以长期偏离 PPP（甚至几年），但极长期趋向回归。

## 三、两个理论的时间尺度 ⭐

```
利率平价(IRP): 短中期 — 决定远期汇率、驱动套息
购买力平价(PPP): 长期 — 通胀差决定长期汇率趋势
→ 呼应 4.1: 中期看利率, 长期看通胀
```

## 为什么这些理论重要（即使不完美）💡

```
理论 ≠ 精确预测, 但提供"锚":
  - CIP: 几乎严格成立 → 远期定价、掉期定价的基石(见 08)
  - UIP 偏离 → 套息交易的机会(见 4.3)
  - PPP: 判断货币长期"高估/低估"的参考
→ 懂这些, 你能理解"为什么远期这么定价""为什么套息能赚钱""为什么某货币被认为高估"
```

## 小结

- 利率平价（IRP）：两种投资回报应相等否则套利；覆盖版（CIP）几乎严格成立，锁定远期汇率（高息货币远期贴水）。
- 非覆盖版（UIP）理论上高息货币该贬值，但现实经常不成立——这是套息交易能赚钱的根基。
- 购买力平价（PPP）：长期汇率向"购买力相等"靠拢（大麦克指数是通俗版），短期可严重偏离。
- 时间尺度：IRP 管短中期和远期定价，PPP 管长期趋势；理论提供"锚"而非精确预测。
