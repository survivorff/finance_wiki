# DeFi 术语表（Glossary）

> 带 ⭐ 的是核心高频术语。通用加密术语见 [CEX 术语表](../cex/glossary.md)、[DEX 术语表](../dex/glossary.md)。

## 基础
- **DeFi（去中心化金融）** ⭐：智能合约重建金融功能。
- **CeFi（中心化金融）**：传统/中心化机构（含 CEX）。
- **可组合性 / Composability** ⭐：协议像乐高自由拼接（货币乐高）。
- **无许可 / Permissionless** ⭐：任何人可用、可搭建。
- **TVL（总锁仓量）** ⭐：协议锁定的资产总值。
- **货币乐高 / Money Lego**：可组合的协议拼接。

## 稳定币 ⭐
- **稳定币 / Stablecoin** ⭐：锚定稳定价值（通常 1 美元）的加密币。
- **法币抵押型**：USDT/USDC，1:1 法币储备背书。
- **超额抵押型**：DAI，加密资产超额抵押。
- **算法稳定币** ⭐：靠算法维持锚定（多已失败，UST）。
- **脱锚 / De-peg** ⭐：偏离锚定值。
- **死亡螺旋 / Death Spiral** ⭐：算法稳定币崩溃的正反馈。
- **稳定币三难** ⭐：稳定/资本效率/去中心化三选二。
- **CDP（抵押债仓）** ⭐：抵押资产铸造稳定币的金库。
- **稳定费 / Stability Fee**：借 DAI 的利息。
- **PSM（锚定稳定模块）**：1:1 兑换稳定币的强力锚定。

## 借贷 ⭐
- **借贷协议**：Aave/Compound 等链上银行。
- **超额抵押 / Over-collateralization** ⭐：抵押品价值>借款（因匿名无法追债）。
- **LTV（贷款价值比）** ⭐：能借出占抵押品的比例。
- **健康因子 / Health Factor** ⭐：离清算的距离（<1 触发清算）。
- **清算阈值 / Liquidation Threshold**：触发清算的抵押率。
- **清算 / Liquidation** ⭐：抵押不足时强制处置抵押品。
- **清算人 / Liquidator** ⭐：执行清算赚奖励的机器人。
- **利用率 / Utilization** ⭐：借出/存入，决定利率。
- **拐点利率模型 / Kinked Model**：利用率超拐点后利率陡升。
- **闪电贷 / Flash Loan** ⭐：原子交易内的无抵押贷。
- **资金池模式 / Pool-Based**：共享资金池借贷。

## 衍生品与收益
- **链上永续**：见 [DEX 2.5](../dex/02-产品与机制/2.5-衍生品DEX.md)。
- **合成资产 / Synthetic Assets**：用抵押+预言机复刻资产价格。
- **收益耕作 / Yield Farming**：追逐高收益（同 DEX 5.3）。
- **收益聚合器**：Yearn 等自动找最优收益。
- **真实收益 / Real Yield** ⭐：来自手续费/利息/质押的可持续收益。
- **LSD（流动性质押衍生品）** ⭐：stETH 等，质押的同时保持流动性。
- **再质押 / Restaking** ⭐：已质押资产再用于保障其他协议（收益+风险叠加）。
- **APY / APR**：年化收益率（注意通胀代币贬值幻觉）。

## 可组合性与风险
- **套娃 / Nesting** ⭐：协议层层嵌套（风险叠加）。
- **传染 / Contagion** ⭐：一处出事全线传导。
- **预言机操纵** ⭐：闪电贷+操纵价格攻击（见 [DEX 3.4](../dex/03-技术架构/3.4-预言机与价格.md)）。
- **治理攻击**：借/买治理代币通过恶意提案。
- **复杂性风险**：套娃后无法评估的风险。

## 安全与未来
- **智能合约漏洞**：见 [DEX 6.2](../dex/06-风险与安全/6.2-智能合约安全.md)。
- **重入 / 整数溢出**：经典漏洞。
- **储备金/安全模块**：协议兜底坏账的缓冲（≈CEX 保险基金）。
- **RWA（真实世界资产）** ⭐：传统资产代币化上链。
- **链上美债** ⭐：代币化美债，给 DeFi 提供真实无风险收益。
- **黑色星期四**：2020.3 ETH 暴跌+链拥堵致 MakerDAO 坏账事件。

## 经典案例
- **Luna / UST** ⭐：算法稳定币死亡螺旋归零（2022，约 400 亿）。
- **Celsius / 3AC**：Luna 后连环暴雷（传染）。
- **DeFi Summer**：2020 流动性挖矿引爆。
- **Anchor**：UST 的 20% 高息（庞氏诱因）。
