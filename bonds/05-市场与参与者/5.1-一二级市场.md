# 5.1 一二级市场

> 深度：🟡 进阶
> 债券怎么发行、怎么交易。和股票的一二级市场对照。

## 一级市场：债券怎么发行 ⭐

### 政府债：拍卖（Auction）
```
政府定期"拍卖"国债:
  - 一级交易商(Primary Dealers, 指定的大银行)有义务参与竞标
  - 投标方式: 竞争性投标(报收益率) + 非竞争性投标(接受拍卖结果)
  - 价高者得(收益率低者优先) → 确定发行收益率
→ 透明、定期、规模巨大(政府持续融资)
```

- 💡 对照 [股票 IPO](../../stocks/05-发行与公司行动/5.1-IPO上市.md)：国债拍卖是高度标准化、定期的"批发发行"，不像 IPO 那样一次性、需路演。

### 企业债：承销（Underwriting）
```
企业发债: 找投行承销(同 股票 IPO 的承销, 见 股票 5.1)
  投行帮定价、找买家、承销
  → 卖给机构投资者(保险/养老金/基金)
```

## 二级市场：债券怎么交易 ⭐

```
债券二级市场 = 以 OTC(场外)为主, 不像股票有中央交易所!
  → 通过做市商(大银行)报价交易(类似 FX, 见 FX 1.2)
  → 国债流动性极好, 企业债/小债流动性差
→ 债券是"机构间 OTC 市场", 散户主要通过基金/ETF 参与(见 1.2)
```

- 💡 **又见 OTC**：债券和 [FX](../../forex/01-行业全景/1.2-市场规模与结构.md) 一样，主要是"做市商报价的 OTC 市场"，而非 [股票的中央交易所](../../stocks/01-行业全景/1.3-分业制衡的产业链.md)。
- 💡 这意味着债券也有"流动性分层"和"碎片化"问题，靠做市商和电子平台聚合。

## 一级交易商（Primary Dealers）💡

```
一级交易商: 央行/财政部指定的大银行(如各大投行)
  权利: 直接参与国债拍卖、和央行交易
  义务: 必须参与拍卖(保证国债能卖出去)、为市场做市
→ 是连接"政府发债"和"市场"的核心中介
→ 也是央行执行货币政策的"操作对手"(见 5.2)
```

## 债券交易的特点

```
- 报价用"收益率"(不是价格, 见 3.1)
- 国债流动性极深(随时能买卖大量)
- 企业债/小众债流动性差(可能很难卖、点差大)
- 电子化程度低于股票(但在提升)
- 大额、机构主导
```

## 对照三个市场的发行+交易 ⭐

| | 债券 | 股票 | 加密 |
| --- | --- | --- | --- |
| 一级发行 | 国债拍卖/企业承销 | IPO 承销 | 发币/Launchpad |
| 二级交易 | OTC 做市（同 FX） | 中央交易所 | CEX/DEX |
| 散户参与 | 间接（基金/ETF） | 直接 | 直接 |

## 小结

- 一级市场：政府债靠拍卖（一级交易商竞标定收益率），企业债靠投行承销（同股票 IPO）。
- 二级市场以 OTC 为主（做市商报价，同 FX），无中央交易所；国债流动性极好，小债差。
- 一级交易商是连接"政府发债→市场"和"央行货币政策操作"的核心中介。
- 债券报价用收益率、机构主导、电子化程度低于股票、散户多通过基金间接参与。
