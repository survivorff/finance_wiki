# 2.1 交易所与券商

> 深度：🟢🟡
> 这是普通人最常接触的两个角色，也最容易混淆。搞清它们的分工，是理解 [分业制衡](../01-行业全景/1.3-分业制衡的产业链.md) 的第一步。

## 券商（Broker）：你的「代理人」

```
券商 = 帮你买卖股票的中介, 你的"经纪人"
你不能直接去交易所下单(交易所只对会员开放)
→ 必须通过券商: 开户、入金、下单, 券商把订单送到交易所/做市商
```

### 券商做什么
- **开户与 KYC**：帮你开证券账户，做实名和适当性审核。
- **接收订单、路由**：把你的买卖指令送到合适的成交场所（订单路由，见 [3.5](../03-交易与市场结构/3.5-订单路由与最优执行.md)）。
- **代持资产**：你的股票通过券商在中央存管登记（券商代持），资金在券商的客户资金账户。
- **提供服务**：行情、研究、融资融券、投顾等。

### 券商的类型
| 类型 | 特点 | 例子 |
| --- | --- | --- |
| **全服务券商** | 投顾、研究、财富管理，费高 | 传统大行 |
| **折扣券商** | 低佣金、自助交易 | Charles Schwab、Fidelity |
| **零佣金券商** | 免佣金，靠其他方式赚钱 | Robinhood |
| **在线券商** | 纯线上、低成本 | 各类互联网券商 |

### 💡 零佣金券商怎么赚钱：PFOF
```
Robinhood 等"零佣金"靠什么活? 主要靠 PFOF(订单流付款):
  券商把你的订单"卖"给做市商(如 Citadel Securities)
  做市商付钱给券商, 换取"优先成交你的订单"的机会
→ 你以为免费, 其实做市商从你的成交价差里赚钱, 分一部分给券商
```
- ⚠️ **PFOF（Payment for Order Flow）** 是争议焦点：可能导致你的订单没被路由到最优价格。是 GME 事件后监管关注的重点。
- 💡 对照 [CEX maker/taker 返佣](../../crypto/cex/05-市场与博弈/5.1-盈利模式拆解.md)——都是「流动性/订单流」的商业化，只是形式不同。

## 交易所（Exchange）：撮合的「场所」

```
交易所 = 提供撮合的场所, 只做一件事: 把买卖订单匹配成交
关键: 交易所不碰你的钱和股票, 不代理你下单
     它只是个"中立的撮合平台", 对券商(会员)开放
```

### 交易所做什么
- **撮合交易**：运行撮合引擎，匹配买卖订单（[原理同 CEX 撮合](../../crypto/cex/03-技术架构/3.2-撮合引擎.md)）。
- **维护市场秩序**：上市审核、交易规则、熔断、监控异常。
- **发布行情数据**：实时行情（卖数据是重要收入）。
- **上市服务**：公司 IPO 上市的平台（见 [05](../05-发行与公司行动/README.md)）。

### 交易所不做什么 ⭐
- ❌ 不代理你下单（那是券商的事）。
- ❌ 不保管你的资产（那是存管的事）。
- ❌ 不清算交收（那是清算所的事）。

> ⭐ **这就是和 [CEX](../../crypto/cex/01-行业全景/1.1-CEX是什么.md) 的根本区别**：
> CEX 一家把「撮合 + 代理 + 托管 + 清算」全做了；
> 股票交易所**只做撮合**，其他交给券商、存管、清算所——这就是 [分业制衡](../01-行业全景/1.3-分业制衡的产业链.md)。

## 券商 vs 交易所（一句话区分）

```
券商 = 你的代理人(你只能通过它下单, 它帮你跑腿)
交易所 = 撮合的场所(券商把订单送到这里成交)
你 → 券商 → 交易所
```

## 托管/清算的角色（这里先点到，详见 04）

- 你的股票最终登记在**中央证券存管(CSD)**，券商代你持有。
- 成交后由**清算所(CCP)**担保交收。
- 这些都是独立机构——又是分业制衡的体现。

## 小结

- 券商是你的代理人：开户、接单、路由、代持资产；类型有全服务/折扣/零佣金。
- 零佣金券商常靠 PFOF（卖订单流给做市商）赚钱，是争议焦点。
- 交易所只做撮合 + 维护秩序 + 卖数据 + 上市服务，不碰资产、不代理、不清算。
- 「券商代理 + 交易所撮合 + 存管保管 + 清算交收」的分工，正是和 CEX 一体化的根本区别。
