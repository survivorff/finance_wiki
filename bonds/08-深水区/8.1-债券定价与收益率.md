# 8.1 债券定价与收益率

> 深度：🔴🔴 极硬核
> 把债券定价、YTM 求解、久期凸性的公式讲到能算的程度。
> 前置：[3.2 债券定价](../03-利率与定价/3.2-债券怎么定价.md)、[3.3 久期凸性](../03-利率与定价/3.3-久期与凸性.md)。

## 一、债券定价公式

```
P = Σ_{t=1}^{n} C/(1+y)^t + F/(1+y)^n

P = 价格, C = 每期票息, F = 面值, y = 每期收益率(YTM), n = 期数
(半年付息则: C 用半年票息, y 用半年收益率, n 用半年期数)
```

## 二、YTM 求解（数值方法）⭐

```
问题: 已知价格 P, 求 y(YTM) → 这是个非线性方程, 没有解析解
方法: 数值迭代(牛顿法或二分法)

伪代码(二分法):
def solve_ytm(price, coupons, face, n):
    lo, hi = 0.0, 1.0          # 收益率搜索区间
    for _ in range(100):
        y = (lo + hi) / 2
        pv = sum(c/(1+y)**t for t,c in enumerate(coupons,1)) + face/(1+y)**n
        if pv > price:          # 算出的现值偏高 → 收益率要调高
            lo = y
        else:
            hi = y
    return y
```

- 💡 价格和收益率是单调反向关系（见 [3.1](../03-利率与定价/3.1-收益率与价格.md)），所以二分法稳定收敛。
- 💡 对照 [CEX 强平价/AMM 求解](../../crypto/cex/08-深水区/8.2-永续合约数学全解.md)：金融里大量"已知价格反解隐含参数"都用数值迭代。

## 三、久期公式 ⭐

### 麦考利久期（加权平均回收时间）
```
                Σ t × [CF_t/(1+y)^t]
Macaulay D = ─────────────────────────
                       P
  = 各期现金流现值 按"时间 t"加权的平均
```

### 修正久期（价格敏感度）
```
              Macaulay D
Modified D = ─────────────
               (1 + y)

价格变动近似:  ΔP/P ≈ − Modified D × Δy
  例: 修正久期 7, 利率涨 0.01(1%) → ΔP/P ≈ −7 × 0.01 = −7%(跌 7%)
  → 这就是 3.3 说的"久期 7, 利率动 1% 价格动约 7%"
```

## 四、凸性公式 ⭐

```
              1      Σ t(t+1) × [CF_t/(1+y)^t]
Convexity = ────── × ─────────────────────────
            (1+y)^2            P

更精确的价格变动(久期 + 凸性修正):
  ΔP/P ≈ − ModD × Δy + ½ × Convexity × (Δy)^2
              ↑ 一阶(久期)      ↑ 二阶(凸性修正)
```

- 💡 **久期是一阶导、凸性是二阶导**——这就是 [3.3](../03-利率与定价/3.3-久期与凸性.md) 说的"久期是切线近似，凸性是曲率修正"的数学。
- 💡 对照 [期权 Delta/Gamma](../../crypto/cex/08-深水区/8.2-永续合约数学全解.md)、[做市 gamma](../../crypto/cex/08-深水区/8.7-做市与市场微观结构.md)：一阶敏感度 + 二阶修正是跨资产通用的泰勒展开框架。

## 五、算例（完整）

```
债券: 面值 100, 年票息 5(票息率5%), 3 年期, YTM y = 6%
定价:
  P = 5/1.06 + 5/1.06² + 5/1.06³ + 100/1.06³
    = 4.717 + 4.450 + 4.198 + 83.962
    = 97.33

麦考利久期:
  D = [1×4.717 + 2×4.450 + 3×(4.198+83.962)] / 97.33
    = [4.717 + 8.900 + 264.48] / 97.33
    ≈ 2.86 年

修正久期 = 2.86 / 1.06 ≈ 2.70
→ 利率涨 1%(0.01), 价格约跌 2.70% → P 约跌到 94.7
```

## 六、关键收益率指标汇总 💡

```
YTM(到期收益率): 持有到期的年化总回报(最常用, 见 3.1)
当前收益率: 年票息/价格(简单, 忽略价差)
即期利率(spot rate): 零息债的收益率(见 8.2)
远期利率(forward rate): 隐含的未来利率(见 8.2)
到期收益率曲线 vs 即期曲线: 实务区分(见 8.2)
```

## 小结

- 债券定价 P = 各期现金流按 YTM 折现求和；YTM 无解析解，用二分法/牛顿法数值求解。
- 麦考利久期=现金流按时间加权平均回收期；修正久期=价格敏感度（ΔP/P≈−ModD×Δy）。
- 凸性是二阶修正（ΔP/P≈−ModD×Δy+½×Conv×Δy²），久期一阶+凸性二阶=泰勒展开。
- 久期/凸性 ≈ 期权 Delta/Gamma，是跨资产通用的敏感度框架。
