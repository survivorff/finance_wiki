# 2.3 流动性提供与无常损失

> 深度：🟡 → 🔴 ⭐ LP 必懂、DEX 最反直觉的概念
> 前置：[2.1 AMM](./2.1-AMM是什么.md)，并强烈建议先读 [CEX 8.7 做市与逆向选择](../../cex/08-深水区/8.7-做市与市场微观结构.md)。

## 当 LP 能赚什么

把两种币存进池子当 LP，收益来自：
```
LP 总收益 = 交易手续费分成 + (流动性挖矿代币奖励) − 无常损失 − Gas
```
- **手续费**：池子每笔交易抽 X%（如 0.3%）按份额分给 LP。交易量越大，LP 赚越多。
- **挖矿奖励**：很多协议额外发治理代币奖励 LP（见 [04 流动性挖矿](../04-生态与运营/4.1-流动性挖矿.md)）。
- **减项**：无常损失 + Gas 成本。

> ⭐ 关键问题：**手续费 + 奖励，能不能盖过无常损失？** 这决定了当 LP 到底赚不赚钱。

## 无常损失（Impermanent Loss, IL）⭐ 核心概念

### 一句话定义
> **无常损失 = 「把币存进池子当 LP」 相比 「单纯拿着这些币不动（HODL）」 少赚/多亏的那部分。**

注意：是和「拿着不动」对比的**机会成本**，不是绝对亏损。

### 为什么会发生（直觉）⭐
AMM 的机制决定了：**价格变动时，池子会自动「卖出上涨的币、买入下跌的币」**——
正好和「追涨杀跌」相反，听起来像优点，但结果是：

```
ETH 价格上涨时:
  套利者来池子买 ETH(因为池子价格滞后) → 池子 ETH 变少、USDC 变多
  → 你作为 LP, 在 ETH 上涨过程中被动卖掉了一部分 ETH(卖早了)
  → 结果: 你持有的总价值 < 当初直接拿着 ETH 不动
        这个差额就是无常损失
```

> 💡 **本质联系**：无常损失 = [CEX 做市的「逆向选择 + 库存风险」的被动版](../../cex/08-深水区/8.7-做市与市场微观结构.md)。
> 套利者（知情者）总是在「正确方向」和你（LP）成交：价涨时从你这买走涨的，价跌时塞给你跌的。
> LP 是「永远不撤单、不调价的做市商」，所以系统性地被套利者占便宜。**这就是无常损失的根源。**

### 为什么叫「无常」（Impermanent）
- 如果价格**回到**你存入时的水平，无常损失会**消失**（所以叫"无常/暂时"）。
- 但如果你在价格偏离时**撤出**，损失就**实现（变成永久）**了。

## 无常损失有多大（量级感受）⭐

价格变动幅度 vs 无常损失（相对 HODL，恒定乘积池）：

| 价格变化 | 无常损失 |
| --- | --- |
| 1.25x（涨25%） | ~0.6% |
| 1.5x | ~2.0% |
| 2x（翻倍） | ~5.7% |
| 3x | ~13.4% |
| 4x | ~20.0% |
| 5x | ~25.5% |

> ⭐ **关键直觉**：价格偏离越大，无常损失越大，而且是**单边偏离就亏**（涨和跌都会有 IL）。
> 翻倍才亏约 5.7% 看着不多，但如果手续费收益盖不过它，当 LP 就不如直接拿着。
> 数学推导（公式 `IL = 2√r/(1+r) − 1`）见 [8.2 无常损失推导](../08-深水区/8.2-无常损失数学推导.md)。

## LP 到底赚不赚钱？⭐

```
当 LP 划算的条件:  手续费收入 + 挖矿奖励  >  无常损失 + Gas

→ 适合当 LP: 高交易量 + 价格相对稳定的池子(如稳定币对、强相关资产)
→ 不适合:   低交易量 + 价格大幅波动的池子(IL 吃掉一切)
```

### 不同池子的 LP 风险
| 池子类型 | 无常损失 | 适合当 LP？ |
| --- | --- | --- |
| 稳定币对(USDC/USDT) | 极小(价格几乎不变) | ✅ 相对安全 |
| 强相关资产(ETH/stETH) | 小 | ✅ 较好 |
| 主流币对(ETH/USDC) | 中等 | ⚠️ 看手续费能否覆盖 |
| 主流/山寨币 | 大 | ⚠️ 高风险 |
| 山寨/meme 币 | 极大(剧烈波动) | ❌ 常常巨亏 |

## V3 集中流动性的无常损失放大 ⚠️

- V3 把流动性集中在区间内，资本效率高 → **同样价格变动下，无常损失被放大**。
- 区间越窄，效率越高，IL 越剧烈；价格冲出区间，你的钱全变成单一资产且不再赚费。
- 所以 V3 LP 需要主动管理、对冲，已经是「专业做市」的活了（见 [2.2](./2.2-AMM进阶.md)）。

## 如何应对无常损失（LP 策略）

```
1. 选低波动池(稳定币对、相关资产) → 从源头减少 IL
2. 选高交易量池 → 手续费盖过 IL
3. 用对冲(在别处做空等量资产) → 抵消价格波动(delta 中性 LP)
4. V3 主动管理区间 / 用 LP 管理协议
5. 算清楚再进: 别只看 APR(年化收益率), APR 没扣 IL!
```

> ⚠️ **最大的 LP 陷阱**：很多协议宣传超高 APR（年化几百%），那是「手续费+挖矿奖励」的名义收益，**没扣无常损失和代币贬值**。
> 实际到手可能远低于宣传，甚至亏损。**永远要把 IL 和奖励代币的贬值算进去。**

## 小结

- LP 收益 = 手续费 + 挖矿奖励 − 无常损失 − Gas。
- 无常损失 = 当 LP 相比「拿着不动」少赚的部分，根源是套利者总在正确方向和 LP 成交（逆向选择的被动版）。
- 价格偏离越大 IL 越大（翻倍约 5.7%，单边偏离就亏）；价格回归则 IL 消失。
- 适合当 LP：高交易量 + 低波动池；V3 放大 IL，需主动管理。
- 警惕高 APR 宣传——它没扣无常损失和奖励代币贬值。
