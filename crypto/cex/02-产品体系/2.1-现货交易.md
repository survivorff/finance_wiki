# 2.1 现货交易（Spot Trading）

> 深度：🟢 入门 → 🟡 进阶
> 现货是 CEX 最基础的产品，也是理解一切交易的起点。

## 一句话

**用一种资产直接买卖另一种资产，一手交钱一手交货，无杠杆。** 如用 USDT 买 BTC。

## 核心概念逐个拆

### 1. 交易对（Trading Pair）
如 **BTC/USDT**：
- 前面的 **BTC** 是**基础币（base）**——你想买卖的东西。
- 后面的 **USDT** 是**计价币（quote）**——用来标价和支付的东西。
- `BTC/USDT = 60000` 意思是「1 个 BTC 值 60000 USDT」。

常见计价币：USDT、USDC（稳定币），以及 BTC、ETH、平台币、各国法币。

> ⚠️ 同一个币，用不同计价币会有不同的交易对（BTC/USDT 和 BTC/USDC 是两个独立市场，价格略有差异）。

### 2. 订单簿（Order Book）⭐
当前所有未成交买卖挂单的集合。这是中心化交易所的灵魂。

```
卖单 asks (想卖的人，从低到高排)
  60005  | ████ 2.0 BTC
  60003  | ██ 1.0
  60001  | ███ 1.5      ← 最优卖价 best ask（你想买，最低要这个价）
─────────────────────── 价差 spread = 2 USDT
  59999  | ████ 2.0     ← 最优买价 best bid（你想卖，最高能卖这个价）
  59997  | ██ 1.0
  59995  | █████ 3.0
买单 bids (想买的人，从高到低排)
```

- **最优买价（best bid）**：当前买方愿意出的最高价。
- **最优卖价（best ask）**：当前卖方愿意接受的最低价。
- **价差（spread）**：两者之差。价差越小，市场越「健康/流动性越好」。
- **深度（depth）**：每个价位挂单的厚度。深度好 = 大额交易不会引起价格剧烈跳动。

### 3. 订单类型
| 类型 | 含义 | 适用场景 |
| --- | --- | --- |
| **限价单 Limit** | 指定价格，不满足就挂在盘口等 | 想按理想价格成交，不急 |
| **市价单 Market** | 不指定价，立即吃掉盘口最优价 | 急着成交，不在乎一点滑点 |
| **止损单 Stop** | 价格触发某点后才下单 | 止损/突破追单 |
| **止盈止损 TP/SL** | 设定目标价自动卖/止损 | 自动化离场 |

进阶订单：
- **IOC**（Immediate-Or-Cancel）：立即成交能成的部分，剩下取消。
- **FOK**（Fill-Or-Kill）：要么全部立即成交，要么全部取消。
- **Post-Only**：只做挂单方（maker），如果会立即成交就取消（为了拿 maker 低费率/返佣）。

### 4. 滑点（Slippage）
你下市价单买，实际成交均价 vs 你看到的价格之间的偏差。
- 盘口深度差 + 你单子大 → 滑点大（你的大单「吃穿」了好几档价格）。
- 这是大户最头疼的问题，所以大额交易走 OTC（见 [2.5](./2.5-法币与出入金.md)）。

## 现货交易的完整流程（产品视角）

```
1. 用户下单 ──▶ 2. 风控校验(余额/限额) ──▶ 3. 进入撮合引擎
                                              │
6. 用户看到成交 ◀── 5. 清算(改双方余额、扣手续费) ◀── 4. 撮合匹配
```
（技术实现细节见 [03 技术架构](../03-技术架构/README.md)）

## 现货怎么赚钱

- 收**交易手续费**：通常 0.1% 量级，按 maker/taker 区分（详见 [05 盈利](../05-市场与博弈/5.1-盈利模式拆解.md)）。
- 现货是「基础盘」，利润率不如合约，但用户基数大、是其他产品的入口。

## 产品设计的细节（从业者视角）

- **交易对的选择**：上哪些交易对是运营决策，影响流动性和热度。
- **手续费阶梯**：按 30 天交易量和平台币持仓分 VIP 等级，激励高频用户。
- **交易界面分层**：「简易模式」给小白（一键买卖），「专业模式」给老手（完整 K 线、盘口、多种订单）。
- **交易大赛/返佣**：运营手段，刺激交易量。

## ⚠️ 小白常见误区

- **「市价单更划算」**：错，市价单吃单是 taker，费率更高，还可能滑点大。
- **「挂单一定能成交」**：限价单只有价格被触及才成交，可能一直挂着不成交。
- **「现货也会爆仓」**：错，现货无杠杆不会爆仓，最多归零。爆仓是合约/杠杆才有的。

## 小结

- 现货 = 无杠杆的即时买卖，基于订单簿撮合。
- 核心概念：交易对、订单簿、盘口深度、价差、滑点、订单类型。
- 现货是 CEX 的基础盘和入口，靠手续费赚钱，利润率不如合约。
