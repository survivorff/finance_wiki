## 核心认知 ⭐

> 债券被认为"安全"，但有两类"隐形杀手"：
> - **利率风险**：利率涨，债券跌（2022 长债大跌，见 [3.3](../03-利率与定价/3.3-久期与凸性.md)）——连"无风险"国债都难逃。
> - **信用/违约风险**：发行人赖账（企业债、弱国主权债）。
>
> 而当债务规模过大、利率上升时，**主权债务危机**就可能爆发——这是"安全资产"也会引发系统性危机的根源。
