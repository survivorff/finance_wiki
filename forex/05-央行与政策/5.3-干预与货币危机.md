# 5.3 干预与货币危机

> 深度：🟡 → 🔴
> 央行如何干预汇率，以及当干预失败时——货币危机如何爆发。这些是 FX 最惊心动魄的历史时刻。

## 央行干预（Intervention）

```
干预 = 央行直接下场买卖本币影响汇率:
  本币太弱: 卖外汇储备、买本币 → 撑住汇率(消耗外储)
  本币太强: 卖本币、买外汇 → 压低汇率(积累外储)
方式:
  直接干预: 真的买卖
  口头干预(嘴炮): 官员喊话暗示要干预(不花钱就影响预期)
  协调干预: 多国央行联手(威力更大)
```

## 干预的局限 ⚠️

```
干预不是万能的:
  撑弱势货币: 受限于外汇储备(弹药有限!)
    → 外储烧光就撑不住了 → 货币崩溃(见下面的危机)
  压强势货币: 理论上"无限"(可以无限印本币), 但会带来通胀
→ "撑跌"比"压涨"难得多(弹药有限 vs 无限)
```

- 💡 这是关键不对称：**央行印本币是无限的（能压住升值），但外汇储备是有限的（撑不住贬值）**。投机者正是攻击这个弱点。

## 货币危机经典案例

### 1992 索罗斯狙击英镑 ⭐
```
背景: 英镑加入欧洲汇率机制(ERM), 要维持对马克的固定汇率
       但英国经济弱, 这个汇率被高估、难以维持
索罗斯: 判断英国撑不住, 大举做空英镑(借英镑卖出)
英格兰银行: 拼命买英镑 + 加息 来撑汇率(消耗外储)
结果: 英国外储烧不起 + 加息伤经济 → 放弃, 英镑退出 ERM、暴跌
      索罗斯赚约 10 亿美元, "击垮英格兰银行"
```

- 💡 **教训**：当固定汇率被高估、基本面不支持时，**再强的央行也撑不住投机攻击**（不可能三角的实战，见 [5.2](./5.2-汇率制度.md)）。
- 💡 对照 [DEX 稳定币脱锚](../crypto/defi/README.md)：算法稳定币崩盘的逻辑类似——当锚定不被基本面/储备支撑时，攻击者做空 + 挤兑 → 脱锚崩溃。

### 1997 亚洲金融危机
```
泰铢等亚洲货币钉住美元, 但经济过热、外债高、外储不足
投机者攻击 → 央行撑不住 → 货币接连崩溃(泰铢、印尼盾、韩元)
→ 引发区域性经济危机
教训: 固定汇率 + 大量外债 + 资本自由流动 = 危机温床(不可能三角)
```

### 新兴市场货币危机的通用剧本 💡
```
1. 高息吸引外资流入(套息)、积累外债
2. 某个触发点(美联储加息、商品价格跌、政治风险)
3. 资金开始外逃 → 货币贬值压力
4. 央行用外储撑 → 外储耗尽
5. 货币崩盘 → 外债(美元计价)负担暴增 → 经济危机
→ 反复上演(土耳其、阿根廷等)
```

## 干预 vs 市场的博弈 ⭐

```
央行干预 vs 投机者, 是一场实力博弈:
  央行赢: 当基本面支持 + 弹药充足 + 决心强(投机者不敢赌)
  投机者赢: 当汇率明显偏离基本面 + 央行弹药有限(索罗斯式狙击)
→ 央行不是无敌的, "市场"在基本面错位时能击败央行
```

- 💡 **深刻洞察**：即使是能"印钞"的央行，在"撑住一个错误的汇率"这件事上也可能输给市场——
  因为撑跌的弹药（外储）有限，而市场的做空力量可以很大。**这是"市场力量 vs 政策意志"的经典较量。**

## 汇率与全球危机传导 💡

```
汇率危机会传导:
  一国货币崩 → 周边国家被波及(传染) → 区域危机
  美元走强 → 新兴市场普遍承压(美元债负担)
  → 汇率是全球金融风险传导的重要渠道
```

## 小结

- 央行干预（买卖本币/口头/协调）影响汇率，但有局限：撑贬值受外储限制，压升值近乎无限——不对称。
- 1992 索罗斯狙击英镑、1997 亚洲金融危机：固定汇率被高估+基本面不支持时，央行撑不住投机攻击（不可能三角实战）。
- 新兴市场危机通用剧本：外资流入→触发点→资金外逃→外储耗尽→货币崩盘→美元债负担暴增。
- 央行不是无敌的，基本面错位时市场能击败央行；汇率是全球危机传导的重要渠道。
