# 1.5 发展简史

> 深度：🟢 小白友好
> DEX 的历史是一部「不断逼近 CEX 体验、同时保持去中心化」的进化史。

## 时间线总览

```
2016 ─── EtherDelta：早期链上订单簿 DEX，难用但开创
2017 ─── Bancor：提出 AMM 雏形（流动性池概念）
2018 ─── Uniswap V1/V2 上线：x*y=k，AMM 范式确立 ⭐
2020 ─── DeFi Summer：流动性挖矿引爆，DEX 交易量暴涨
         SushiSwap 吸血鬼攻击 Uniswap
2021 ─── Uniswap V3 集中流动性；多链 DEX 爆发(BSC/Polygon)
2022 ─── FTX 崩塌 → 自托管需求暴涨，DEX 受益
2023 ─── L2 生态崛起(Arbitrum/Base)，DEX 迁移到 L2
2024 ─── Solana 复兴(Jupiter/meme 热)；Uniswap V4 + Hooks
2024-25 ─ Hyperliquid 崛起：链上永续逼近 CEX 体验
         "意图(intent)"和聚合器成为新范式
```

## 关键节点深度复盘

### EtherDelta（2016）：第一代链上订单簿
- 把订单簿搬上以太坊，完全链上。
- ⚠️ **问题**：每次挂单/撤单都要发链上交易、付 Gas、等出块——慢、贵、难用。
- **教训**：直接把 CEX 的订单簿照搬上链行不通。这逼出了 AMM 的创新。

### Uniswap 与 AMM 范式（2018）⭐ 最重要的转折
- Vitalik 等人提出、Hayden Adams 实现：用 `x*y=k` 公式 + 资金池替代订单簿。
- **为什么是革命**：
  - 不需要订单簿和专业做市商——**任何人存两种币进池子就能做市**。
  - 流动性「永远在线」（公式保证任何价位都能成交，只是滑点不同）。
  - 合约极简，Gas 可控。
- AMM 成为之后所有 DEX 的范式基础。

### DeFi Summer（2020）⭐ 引爆点
- **流动性挖矿（Liquidity Mining）**：协议发代币奖励 LP，吸引海量资金。
- Compound 发 COMP 代币引爆「挖矿」狂潮，DEX TVL 和交易量暴涨。
- **SushiSwap 吸血鬼攻击**：复制 Uniswap 代码 + 发币激励，把 Uniswap 的流动性「吸」过来——经典的「分叉 + 代币激励」打法。
- 直接催生 Uniswap 发 UNI 代币空投（DeFi 史上著名空投）。

### Uniswap V3（2021）：集中流动性
- LP 可以把流动性集中在某个价格区间，**资本效率大幅提升**（同样的钱提供更深的流动性）。
- 代价：LP 要主动管理区间，无常损失更复杂（见 [2.2](../02-产品与机制/2.2-AMM进阶.md)）。

### FTX 崩塌的外溢（2022）
- CEX 信任崩塌 → 「自托管」「Not your keys」深入人心 → DEX、硬件钱包受益。
- DEX 交易量占比在 FTX 事件后明显跳升。

### L2 迁移与 Solana 复兴（2023-2024）
- 以太坊主网 Gas 太贵，DEX 大量迁移到 **L2（Arbitrum、Base、Optimism）**——便宜快。
- **Solana** 凭借高速低费在 meme 币热潮中复兴，Jupiter 成为交易入口。

### 链上永续与意图时代（2024-2025）
- **Hyperliquid**：自建 L1 做链上订单簿永续，体验逼近 CEX，是 DEX 衍生品的里程碑。
- **Uniswap V4 + Hooks**：可定制化的 AMM（插件式扩展）。
- **意图（Intent）范式**：用户表达「我想要什么结果」，由求解器（solver）竞争实现最优执行（见 [07 未来](../07-未来发展/README.md)）。

## 贯穿全史的主线 ⭐

> **DEX 的历史 = 「在保持去中心化（自托管、链上）的前提下，不断逼近 CEX 的体验和性能」。**
> - 订单簿太慢 → 发明 AMM
> - 资本效率低 → 集中流动性
> - 主网太贵 → 迁移 L2 / 自建链
> - 体验不如 CEX → Hyperliquid 链上订单簿
> - 操作太复杂 → 意图/账户抽象
>
> 每一步都在缩小和 CEX 的差距，同时守住「自托管 + 可验证」这条底线。这正是 [07 未来发展](../07-未来发展/README.md) 的延续方向。

## 小结

- 从 EtherDelta（链上订单簿太难用）→ Uniswap AMM（范式革命）→ DeFi Summer（流动性挖矿引爆）→ V3 集中流动性 → L2/Solana → Hyperliquid 链上永续。
- AMM 是最重要的发明，解决了「链上做市」难题。
- 主线是「保持去中心化的同时不断逼近 CEX 体验」。
