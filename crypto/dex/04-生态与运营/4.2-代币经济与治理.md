# 4.2 代币经济与治理

> 深度：🟡 进阶
> DEX 协议靠「治理代币」组织所有权和决策。理解代币经济（tokenomics）和 DAO 治理，是看懂 DeFi 价值和权力的关键。

## 治理代币是什么

```
治理代币(governance token): 如 UNI、CRV、SUSHI
  - 代表对协议的"治理权"(投票决定协议参数、资金分配、升级)
  - 部分还能捕获协议收入(分手续费/回购)
  - 类比: 协议的"股票"(但法律地位模糊, 通常无正式股权/分红权)
```

- 对照 [CEX 平台币](../../cex/02-产品体系/2.6-生态产品.md)：都是「把协议/平台价值代币化」，但 DEX 治理代币更强调「治理权」和去中心化。

## 治理代币的价值来源 ⭐

```
治理代币凭什么有价值?
├── 治理权: 控制协议(尤其控制激励分配 → 见 4.3 流动性战争)
├── 现金流: 分协议手续费 / 协议用收入回购代币
├── 实用性: 减免手续费、参与功能
└── 预期/投机: 对协议未来的押注
```

> ⚠️ **「无现金流的纯治理代币」价值存疑**：如果代币只能投票、不能分享协议收入，它的价值就很虚（"治理权"值多少钱？）。
> 这是 DeFi 长期争议——很多治理代币因「无法捕获价值」而长期下跌。所以「**费用开关（fee switch）**」（是否把手续费分给代币持有者）是治理的核心议题。

## ve 模型（投票托管）⭐ Curve 的创新

```
ve = vote-escrowed(投票托管):
  把治理代币(CRV)锁仓 → 得到 veCRV(不可转让)
  锁得越久, veCRV 越多 → 投票权越大 + 分红越多 + 挖矿加成
效果:
  - 鼓励长期锁仓(对抗"挖提卖", 见 4.1)
  - 把"治理权"和"长期承诺"绑定
  - veCRV 能投票决定"哪个池子获得 CRV 激励" → 引发流动性战争(见 4.3)
```

- ve(3,3) 等变体被很多协议（Aerodrome 等）采用。
- 💡 ve 模型是 DeFi 治理设计的里程碑，把「短期投机」转向「长期对齐」。

## DAO 治理 ⭐

```
DAO(去中心化自治组织): 协议由代币持有者投票治理, 而非公司董事会
治理流程(典型):
  1. 论坛讨论提案(temperature check)
  2. 正式提案(governance proposal)
  3. 代币持有者链上投票(按代币数量加权)
  4. 通过 → (经时间锁)自动执行 或 多签执行
```

### DAO 治理的理想与现实
| 理想 | 现实 ⚠️ |
| --- | --- |
| 人人参与、去中心化决策 | 投票率低，多数人不参与 |
| 代币持有者利益一致 | 大户/VC 持币多 → 实际寡头治理 |
| 透明、抗审查 | 提案复杂，普通人看不懂 |
| 代码即法律 | 紧急情况需要中心化快速响应 |

> 💡 **治理代币集中是普遍问题**：早期投资者（VC）和团队往往持有大量代币，「一币一票」实际是「钱多说了算」。
> 很多 DAO 的「去中心化治理」名不副实，真正决策权在少数巨鲸手里。这是 DeFi 去中心化叙事的软肋。

## 空投（Airdrop）⭐ 代币分发与获客

```
空投: 把代币免费发给早期用户/特定行为者
目的:
  - 奖励早期用户、建立社区
  - 把代币(治理权)分散出去(去中心化)
  - 强力获客/拉新(预期空投吸引大量用户)
```

- **UNI 空投**（2020）：给历史用户每人 400 UNI，是 DeFi 史上著名空投，开创先河。
- 💡 **「空投预期」成了获客利器**：用户为了博空投，提前去用某协议（见 [4.4](./4.4-增长与冷启动.md)）。催生了「**撸毛党/空投猎人**」这个群体。

## 代币分配（Tokenomics）

```
典型治理代币分配:
  社区/空投/挖矿  ──┐
  团队            ──┤ 各占一定比例, 团队/投资者通常有锁仓(vesting)
  投资者(VC)      ──┤
  金库(DAO treasury)─┘
关键看:
  - 团队/VC 占比是否过高(砸盘风险)
  - 解锁节奏(大量解锁=抛压)
  - 是否有真实价值捕获
```

## 小结

- 治理代币代表协议治理权，部分能捕获收入，类比「协议股票」但法律地位模糊。
- 价值来源：治理权、现金流（费用开关是核心议题）、实用性、预期；纯治理无现金流的代币价值存疑。
- ve 模型（锁仓换更大权益）对抗「挖提卖」、绑定长期承诺，是治理里程碑。
- DAO 治理理想很美，现实是投票率低、代币集中、寡头治理，去中心化常名不副实。
- 空投是分发+获客利器（UNI 开创），催生了撸毛党；看代币经济要看团队/VC 占比和解锁节奏。
