# 6.3 监管

> 深度：🟡 进阶
> FX 没有中央交易所，怎么监管？这是一个有趣的"分布式市场监管"问题。

## FX 监管的特殊性

```
FX 没有中央交易所(见 1.2), 但不是无监管:
  - 监管"参与者"而非"市场": 监管银行、经纪商、基金
  - 各国监管自己辖区的机构和零售业务
  - 国际协调(BIS、FSB 等)制定标准
→ 监管的抓手是"机构", 不是"交易所"(因为没有交易所可管)
```

- 💡 这有点像 [DEX 监管](../../crypto/dex/06-风险与安全/6.4-监管与合规.md)——没有中央实体，只能从参与者/接入点下手。**无中央交易所的市场，监管都面临类似挑战。**

## 监管什么

```
1. 银行行为: 反操纵(FX 基准操纵丑闻后收紧, 见 3.2)、合规、风控
2. 零售保护: 限制杠杆、风险警示、打击黑平台(见 6.2)
3. 反洗钱(AML): FX 是洗钱/资金转移的渠道, 严查
4. 制裁执行: FX 是制裁的关键战场(冻结、踢出 SWIFT)
5. 系统性风险: 大银行 FX 敞口、CLS 等基础设施(见 3.3)
```

## 主要监管机构

| 地区 | 机构 |
| --- | --- |
| 美国 | CFTC（商品期货）、NFA、SEC、Fed/OCC（银行） |
| 英国 | FCA |
| 欧盟 | ESMA + 各国 |
| 日本 | FSA |
| 国际协调 | BIS（国际清算银行）、FSB |

## FX 全球行为准则（自律）💡

```
FX Global Code: FX 基准操纵丑闻后, 行业建立的"全球行为准则"
  → 规范做市、Last Look(见 3.2)、信息处理等行为
  → 是"软法"(自律), 配合各国"硬法"监管
```

## 制裁与 FX 的武器化 ⚠️（重要趋势）

```
FX 和支付系统正成为"地缘武器":
  - 冻结一国的外汇储备(如冻结某国央行资产)
  - 把一国踢出美元清算/SWIFT(见后续支付模块)
  - 限制某货币的可兑换性
→ "美元霸权"(见 1.3)让美国能用 FX/支付系统施加制裁
→ 这也推动了一些国家寻求"去美元化"和替代支付渠道(包括探索 CBDC/加密)
```

- 💡 **这是理解当前全球格局的关键**：美元在 FX/支付的中心地位被"武器化"，反过来推动了去美元化、CBDC、甚至加密跨境支付的探索。地缘政治和货币体系深度交织。

## 小结

- FX 无中央交易所，监管从"参与者"（银行/经纪商/基金）下手，各国管辖区+国际协调（同 DEX 监管思路）。
- 监管重点：反操纵、零售保护（限杠杆）、反洗钱、制裁执行、系统性风险。
- FX Global Code 是行业自律准则（规范做市/Last Look），配合各国硬法。
- FX/支付系统正被"武器化"（冻结外储、踢出 SWIFT），美元霸权的另一面，推动去美元化和 CBDC/加密探索。
