# 5.3 回购市场（Repo）

> 深度：🔴💡 内幕
> 回购市场是金融体系最重要却最不为人知的"水管"。2008 危机就是从这里冻结的。

## 什么是回购（Repo）⭐

```
回购(Repurchase Agreement, Repo):
  本质 = "用债券作抵押的超短期借钱"(通常隔夜)
  操作:
    A 把国债"卖"给 B, 约定明天按稍高价"买回"
    → 等于 A 用国债抵押, 向 B 借了一晚的钱, 那点价差就是利息
逆回购(Reverse Repo): 反过来(B 的视角, 是放钱收抵押)
```

- 💡 **通俗**：Repo = "拿国债当当票去当铺借钱，明天赎回"。是金融机构管理短期现金的核心工具。

## 为什么 Repo 如此重要 ⭐

```
1. 金融体系的"短期资金水管":
   银行、券商、对冲基金 每天靠 Repo 借入巨额短期资金运转
   → 是金融机构的"日常现金流命脉"
2. 货币政策工具: 央行用回购/逆回购调节流动性(见 5.2)
3. 国债的"货币化": 让国债能快速变成现金(国债的高流动性靠 Repo 支撑)
4. 杠杆来源: 机构用 Repo 加杠杆(借钱买更多债, 再抵押再借...)
```

- 💡 Repo 市场规模巨大（每日数万亿美元），是金融体系运转的"血液循环"，但极少被大众了解。

## Repo 如何创造杠杆 💡

```
对冲基金的"债券套利"常用 Repo 加杠杆:
  1. 买入国债
  2. 把国债通过 Repo 抵押借钱
  3. 用借来的钱再买国债
  4. 再抵押再借... (循环)
→ 用很少的本金撬动巨大的国债头寸(高杠杆)
风险: 抵押品(国债)价格跌 或 Repo 借不到钱 → 被迫平仓 → 踩踏
```

- 💡 对照 [CEX 合约杠杆](../../crypto/cex/02-产品体系/2.2-合约衍生品.md)、[DeFi 循环借贷](../../crypto/dex/05-市场与博弈/5.3-套利与链上玩法.md)：Repo 加杠杆和"循环抵押借贷"是同一逻辑——**用抵押品反复借钱放大头寸**。DeFi 的循环贷其实是 Repo 杠杆的链上翻版。

## 2008：Repo 市场的冻结 ⭐

```
2008 危机的核心机制之一就是 Repo 冻结:
  1. 机构大量用 Repo 借短期钱、持有(含次贷相关的)资产
  2. 危机中, 抵押品(MBS 等)价值暴跌、被质疑
  3. 放钱方不敢再借(怕抵押品不值钱/对手违约)
  4. Repo 市场冻结 → 机构借不到钱周转 → 被迫抛售资产 → 价格更跌
  5. 雷曼等机构因借不到短期钱而崩溃
→ "挤兑"从"储户挤兑银行"变成了"Repo 挤兑"(批发融资挤兑)
```

- 💡 **这是现代金融危机的新形态**："影子银行"（非银机构）靠 Repo 等短期批发融资运转，一旦市场恐慌、不肯再借 → 流动性瞬间冻结 → 崩溃。
- 💡 对照 [CEX 挤兑](../../crypto/cex/06-风险与监管/6.1-风险地图.md)、[稳定币挤兑](../crypto/defi/README.md)：本质都是"挤兑"——**靠短期资金/信心运转的机构，一旦信心崩，资金瞬间撤离就崩溃**。Repo 挤兑、银行挤兑、CEX 提现挤兑、稳定币脱锚，是同一种现象的不同形态。

## 影子银行（Shadow Banking）💡

```
影子银行: 像银行一样"借短贷长"、做信用中介, 但不是传统银行、监管较少
  - 货币市场基金、对冲基金、特殊投资工具(SIV)等
  - 严重依赖 Repo 等短期批发融资
风险: 没有银行的存款保险和央行最后贷款人保护
  → 挤兑时更脆弱(2008 的核心风险源)
```

- 💡 对照 [DeFi](../crypto/defi/README.md)：DeFi 借贷协议本质也是"影子银行"——做信用中介但无传统监管和保护，挤兑时同样脆弱。**DeFi 是"链上影子银行"。**

## 小结

- 回购（Repo）= 用债券抵押的超短期借钱（通常隔夜），是金融体系的"短期资金水管"。
- 重要性：机构日常现金命脉、货币政策工具、国债货币化、杠杆来源。
- Repo 加杠杆（买债→抵押→再买）和 DeFi 循环借贷是同一逻辑。
- 2008 危机核心是 Repo 冻结（抵押品暴跌→不敢借→机构借不到钱崩溃），是"批发融资挤兑"。
- 影子银行靠 Repo 运转、无监管保护、挤兑时脆弱；DeFi 是"链上影子银行"。
