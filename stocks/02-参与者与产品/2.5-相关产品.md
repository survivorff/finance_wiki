# 2.5 相关产品（ETF、期权、融资融券等）

> 深度：🟡 进阶
> 围绕股票，衍生出一整套产品。它们和 [CEX 的衍生品/理财](../../crypto/cex/02-产品体系/README.md) 高度对应。

## 一、ETF（交易所交易基金）⭐ 现代最重要的产品

```
ETF = 一篮子资产打包成一个可像股票一样买卖的基金
  买 1 份标普500 ETF = 间接持有 500 家公司的一小部分
特点: 分散风险、低费率、像股票一样实时交易
```

- **被动指数 ETF**：跟踪指数（标普500等），费率极低，是普通人投资的主流选择。
- **主动 ETF、行业 ETF、债券 ETF、商品 ETF**：各种细分。
- 💡 **比特币现货 ETF**（见 [CEX 7.2 机构化](../../crypto/cex/07-未来发展/7.2-机构化与合规化.md)）就是把这个成熟工具用到了加密——让传统投资者像买股票一样买比特币敞口。

### ETF 的「申购赎回」机制（精妙设计）💡
```
授权参与者(AP)可以用"一篮子股票" ⇄ "ETF 份额"互相兑换
→ 这个套利机制让 ETF 价格紧贴其净值(NAV), 不会大幅偏离
→ 类似 DEX 里套利者维持 AMM 价格(见 DEX 2.1) — 又是套利校准价格!
```

## 二、期权（Options）

```
期权 = 未来以某价格买/卖股票的"权利"(非义务)
  看涨期权(Call): 赌涨的权利
  看跌期权(Put): 赌跌的权利
用途: 投机(杠杆)、对冲(保险)、增强收益(卖期权收权利金)
```
- 概念和 [CEX 期权](../../crypto/cex/02-产品体系/2.2-合约衍生品.md) 一致。
- 💡 期权定价（Black-Scholes 模型）是金融工程的基石——加密期权也用同一套理论。

## 三、股指期货（Index Futures）

```
股指期货 = 约定未来某日按某价格交收"股票指数"的合约
用途: 对冲股市风险、投机、套利(期现套利, 同 CEX 8.2)
```
- 和 [CEX 交割合约](../../crypto/cex/02-产品体系/2.2-合约衍生品.md) 同源。

## 四、融资融券（Margin / Short Selling）⭐

```
融资(买空): 向券商借钱买股票 → 放大做多 (杠杆)
融券(卖空): 向券商借股票卖出, 跌了再买回还 → 做空
```

- 对照 [CEX 杠杆交易](../../crypto/cex/02-产品体系/2.3-杠杆交易与借贷.md)：机制几乎一样（借钱/借券放大）。
- ⚠️ 同样有**强制平仓（追加保证金 Margin Call）**风险——股价不利时被券商强平。
- 💡 **卖空的争议**：做空者赌公司下跌，常被骂"唱衰"，但他们也是市场纠错力量（发现高估和欺诈，如浑水做空财务造假公司）。

### 逼空（Short Squeeze）💡
```
大量做空者 + 股价上涨 → 做空者被迫买回平仓止损 → 进一步推高股价 → 更多做空者被逼平
→ GME 模因股事件就是散户抱团逼空机构(见 1.4)
```
- 💡 和 [CEX 合约的连环爆仓](../../crypto/cex/03-技术架构/3.5-清算与风控.md) 是镜像（一个是空头踩踏，一个多头/空头踩踏）。

## 五、其他

| 产品 | 说明 |
| --- | --- |
| **可转债** | 可转成股票的债券，股债混合 |
| **权证 / 涡轮** | 类似期权的衍生品 |
| **结构化产品** | 挂钩股票的复杂产品（同 [CEX 结构化理财](../../crypto/cex/02-产品体系/2.4-理财与赚币.md)） |
| **REITs** | 房地产信托，像股票一样交易房产收益 |

## 三方产品对照表 ⭐

| 股市 | CEX | DEX |
| --- | --- | --- |
| 现货股票 | 现货 | AMM swap |
| 股指期货/期权 | 合约/期权 | 衍生品 DEX(GMX/Hyperliquid) |
| 融资融券 | 杠杆交易 | 链上借贷+杠杆 |
| ETF | 理财/指数产品 | LP 份额/指数代币 |
| 可转债/结构化 | 结构化理财/双币 | 结构化 DeFi |

> 💡 三个市场的产品体系**高度同构**——因为金融需求是相通的（投机、对冲、杠杆、分散、固收）。
> 只是包装和基础设施不同。学透一个市场的产品逻辑，另外两个能快速迁移理解。

## 小结

- ETF 是现代最重要的产品（一篮子资产、低费、可交易），申赎套利机制让价格贴近净值（类似 DEX 套利校准）。
- 期权、股指期货、融资融券与 CEX 衍生品/杠杆同源同理。
- 卖空有逼空风险（GME），是 CEX 连环爆仓的镜像。
- 三个市场的产品体系高度同构，因为金融需求相通，只是包装和基础设施不同。
