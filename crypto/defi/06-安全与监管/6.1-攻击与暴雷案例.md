# 6.1 攻击与暴雷案例

> 深度：🔴 硬核 ⭐
> DeFi 被黑和暴雷的金额累计达数百亿美元。理解攻击手法和经典案例，是 DeFi 风险教育的核心。

## DeFi 损失的两大类

```
1. 被黑(被攻击): 利用漏洞/机制缺陷盗取资金(技术性)
2. 暴雷(机制崩溃): 经济模型失败/挤兑/踩踏(经济性, 如 Luna)
```

## 一、攻击手法 ⭐

### 智能合约漏洞（见 [DEX 6.2](../../dex/06-风险与安全/6.2-智能合约安全.md)）
- 重入、整数溢出、访问控制、逻辑错误等——DeFi 合约管钱，bug = 直接被掏空。

### 闪电贷 + 预言机操纵 ⭐ DeFi 最经典攻击
```
组合拳(见 DEX 3.4、05):
  1. 闪电贷借入巨款(无抵押, 见 03)
  2. 在某个流动性浅的池子巨额交易 → 操纵价格
  3. 如果某协议用这个池子的价格做预言机 → 被喂错误价格
  4. 用错误价格在该协议套利/借空 → 掏空协议
  5. 归还闪电贷, 卷走利润(全在一笔交易内)
→ DeFi 史上大量被盗事件都是这个套路
防御: 用 TWAP/去中心化预言机(见 DEX 3.4), 不用单池瞬时价
```

### 治理攻击
```
借/买大量治理代币 → 通过恶意提案(如"把金库的钱转给我") → 掏空协议
→ 可组合性(闪电贷借治理代币)让这种攻击更容易
```

### 跨链桥攻击（见 [DEX 3.5](../../dex/03-技术架构/3.5-跨链与桥.md)）
- 桥锁定大量资产 + 验证逻辑复杂 → 被黑重灾区，多起数亿美元损失。

## 二、经典暴雷案例 ⭐

### Luna/UST 死亡螺旋（2022.5）⭐ 见 [2.3](../02-稳定币/2.3-算法稳定币.md)
- 算法稳定币死亡螺旋，UST/Luna 归零，约 400 亿美元蒸发。
- **机制设计缺陷**（无足额抵押 + 反身性 + 高息诱因），非技术漏洞。

### 连环暴雷：传染的威力（2022）⭐
```
Luna 崩盘的连锁反应(见 CEX 6.4):
  Luna 崩 → 重仓的三箭资本(3AC)爆仓 → 借钱给 3AC 的 Celsius/Voyager 等暴雷
  → 这些平台冻结提现、破产 → 最终波及 FTX
→ "可组合性 + 高杠杆 + 互相借贷" 让单点崩溃传染成行业危机
→ 这就是 05 讲的"传染", 在 2022 真实上演
```

- 💡 对照 [2008 雷曼连锁](../../../bonds/06-风险与危机/6.2-债务危机案例.md)：Luna→3AC→Celsius→FTX 的连环暴雷，和 2008 次贷→雷曼→全球的传染同构——**高度互联 + 高杠杆 = 系统性传染**。

### 各类协议被黑
- 大量借贷协议、跨链桥、DEX 因合约漏洞/预言机操纵被盗（累计数百亿美元）。

## 三、防御体系（同 [DEX 6.2](../../dex/06-风险与安全/6.2-智能合约安全.md)）

```
代码审计(但≠绝对安全) + 形式化验证 + 漏洞赏金 + 时间锁+多签 +
渐进部署 + 监控 + 保险(DeFi 保险) + 用 TWAP/可靠预言机
+ 经济机制压力测试(防 Luna 式机制崩溃)
```

## 四、用户自保（同 [DEX 6.5](../../dex/06-风险与安全/6.5-用户自保指南.md)）⭐

```
1. 用老牌、审计过、经过时间考验的协议
2. 警惕高 APY(尤其说不清来源的, 见 4.2)
3. 远离算法稳定币/无足额抵押的"高息"(Luna 教训)
4. 不过度套娃(风险叠加不透明, 见 05)
5. 分散, 别 all in 一个协议
6. 钱包/授权安全(见 DEX 2.6、6.5)
7. 记住: 不可逆、无客服、无兜底
```

## 危机的共同模式（再次印证）⭐

```
DeFi 暴雷配方(和股票/CEX/债券危机同构, 见 各赛道):
  高杠杆/套娃 + 不可持续的高收益 + 机制脆弱/漏洞 + 触发点 + 信心崩 → 传染/踩踏/归零
→ 无论 DeFi、CEX、股票、债券, 系统性危机的剧本惊人相似
```

## 小结

- DeFi 损失两大类：被黑（漏洞/机制缺陷）和暴雷（经济模型失败/挤兑）。
- 攻击手法：合约漏洞、闪电贷+预言机操纵（最经典）、治理攻击、跨链桥攻击。
- Luna/UST 死亡螺旋（机制缺陷，400 亿蒸发）引发 3AC→Celsius→FTX 连环暴雷（传染，同 2008 雷曼）。
- 防御和自保同 DEX：用老牌审计协议、警惕高 APY、远离算法稳定币、不过度套娃、分散、记住不可逆无兜底。
- 危机配方（高杠杆/套娃+不可持续高收益+脆弱/漏洞+触发点+信心崩→传染踩踏）跨所有市场同构。
