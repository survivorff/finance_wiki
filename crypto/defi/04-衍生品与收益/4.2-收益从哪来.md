# 4.2 收益从哪来

> 深度：🟡 → 🔴 ⭐
> 这是 DeFi 最重要的避坑能力：搞清"收益从哪来"。分不清真实收益和庞氏，是 DeFi 亏钱的头号原因。

## 灵魂拷问：这个 APY 从哪来？⭐

```
看到一个 DeFi 产品给 30% APY, 先问: "这 30% 是谁付的、从哪来的?"
  → 答得清且可持续 = 真实收益
  → 答不清 / "靠后来人的钱" = 庞氏 → 迟早崩
```

## 真实收益（Real Yield）来源 ⭐

```
1. 交易手续费: 做 LP 赚 DEX 手续费(见 DEX 2.3, 但有无常损失)
2. 借贷利息: 存款人赚借款人付的利息(见 03)
3. 质押奖励(Staking): 参与 PoS 公链质押, 赚链的出块奖励(真实, 但有锁定)
4. RWA 利息: 链上美债等真实世界资产的利息(见 07)
5. 衍生品资金费/手续费: 见 DEX 2.5
→ 这些有"真实的价值来源"(用户付费/链发奖励/真实资产利息) → 可持续
```

## 庞氏/不可持续收益 ⚠️

```
1. 纯通胀代币激励: 协议印自家代币当"收益"发给你(见 DEX 4.1)
   → 大家都挖了卖 → 币价跌 → "高 APY"是幻觉(币贬值吃掉收益)
2. 拆东补西(庞氏): 用新进资金付老用户收益(如 Anchor 的 20%, 见 2.3 Luna)
   → 新钱不进就崩
3. 不可持续补贴: 烧钱买 TVL, 补贴一停就崩
→ 这些没有"真实价值来源", 靠"印币/新钱/烧钱" → 迟早归零
```

## 关键工具：LSD（流动性质押）⭐

```
问题: 质押 ETH 到 PoS 链能赚奖励, 但 ETH 被锁住了(不能用)
LSD(Liquid Staking Derivatives, 如 Lido 的 stETH):
  你质押 ETH → 拿到 stETH(质押凭证) → stETH 可继续交易/做 LP/抵押借贷
  → 既赚质押奖励, 又保持流动性(资金利用率翻倍)
→ stETH 是 DeFi 最大的"真实收益"资产之一(以太坊质押奖励)
```

- 💡 LSD 是 DeFi 走向"真实收益"的代表——ETH 质押奖励是链原生的真实收益，不是通胀庞氏。

## 再质押（Restaking）💡

```
再质押(如 EigenLayer): 把已质押的 ETH/stETH "再"用于保障其他协议的安全, 赚额外收益
  → 一份质押, 赚多份收益(资金效率再提升)
⚠️ 风险叠加: 再质押把"质押风险"层层叠加(被罚没/slashing 风险叠加)
  → 收益越叠越高, 风险也越叠越深(见 05 可组合性风险)
```

- 💡 再质押是"收益叠加"的极致，也是"风险叠加"的典型——呼应 [05 套娃风险](../05-可组合性与风险/README.md)。

## 收益耕作（Yield Farming）与聚合 💡

```
收益耕作: 在各协议间追逐最高收益(同 DEX 5.3)
收益聚合器(Yearn 等): 自动帮你找最优收益、自动复投、自动调仓
  → 把复杂的"收益策略"打包成一键产品
⚠️ 聚合器把多个协议套在一起 → 风险也聚合(任一底层协议出事都受影响)
```

## 判断收益真伪的清单 ⭐

```
✅ 收益来源说得清吗?(手续费/利息/质押 = 真; 说不清 = 警惕)
✅ APY 用什么计价?(用通胀代币计价的高 APY 可能是幻觉)
✅ 扣除无常损失/Gas/代币贬值后, 真实到手多少?
✅ 异常高的"稳定币收益"?(稳定币本身不该高息, 高息=风险, 见 2.4)
✅ 锁定期/退出难度?(暴雷前常"暂停提现")
✅ 套了几层?(套娃越深, 风险越不透明)
```

> ⚠️ **黄金法则（再次强调，跨越所有市场）**：
> [CEX 理财](../../cex/02-产品体系/2.4-理财与赚币.md)、[FX 套息](../../../forex/04-汇率原理/4.3-套息交易.md)、[债券高息](../../../bonds/02-债券类型/2.1-按发行人分类.md)、DeFi 挖矿——
> **不合理的高收益 = 你没看到的高风险。** 真实收益有上限（被市场竞争压平），异常高息必有隐情。

## 小结

- DeFi 避坑核心：搞清"这个 APY 从哪来、可持续吗"。
- 真实收益：交易手续费、借贷利息、质押奖励、RWA 利息——有真实价值来源、可持续。
- 庞氏/不可持续：纯通胀代币激励（币贬值吃掉收益）、拆东补西（Anchor 20%）、烧钱补贴。
- LSD（stETH）是真实收益代表（保持流动性的同时赚质押奖励）；再质押叠加收益也叠加风险。
- 判断真伪看：来源、计价币、扣损后到手、稳定币异常高息、锁定期、套娃深度。
- 黄金法则：不合理高收益=没看到的高风险（跨所有市场通用）。
