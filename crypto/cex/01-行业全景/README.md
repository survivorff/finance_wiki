## 这个模块的核心收获

- CEX 是「交易所+券商+银行+托管行」的合体（记住这个类比，贯穿全书）。
- 市场高度头部集中，有强烈的地域分化和飞轮效应。
- 围绕 CEX 有一条完整产业链：上币、做市、通道、托管、数据、监管。
- 行业的核心命题始终是「中心化托管的效率 vs 用户资产的信任」。
