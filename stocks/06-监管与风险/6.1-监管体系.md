# 6.1 监管体系

> 深度：🟡 进阶 ⭐
> 股市监管的两大基石：信息披露 + 禁止操纵。这是几百年危机换来的，也是加密监管的目标蓝图。

## 监管的核心目标

```
1. 保护投资者(尤其弱势的散户)
2. 维护市场公平、有序、透明
3. 促进资本形成(让公司能融资)
→ 三者平衡, 既不能放任欺诈, 也不能管死市场
```

## 主要监管机构

| 地区 | 机构 | 职责 |
| --- | --- | --- |
| 🇺🇸 美国 | **SEC（证券交易委员会）** | 证券市场总监管 |
| 🇺🇸 美国 | FINRA | 券商行业自律组织 |
| 🇨🇳 中国 | 证监会(CSRC) | 证券市场监管 |
| 🇬🇧 英国 | FCA | 金融行为监管 |
| 🇭🇰 香港 | SFC（证监会） | 证券期货监管 |

- 💡 对照 [CEX 监管](../../crypto/cex/06-风险与监管/6.3-全球监管格局.md)：加密的 SEC/CFTC 之争，本质是「加密资产算不算证券、归谁管」——而股票是明确的证券，监管归属清晰。

## 基石一：强制信息披露（Disclosure）⭐

> **「阳光是最好的消毒剂」——监管的核心哲学不是替你判断好坏，而是强制公司"说真话、说全话"，让你自己判断。**

```
上市公司必须定期、强制披露:
  - 年报(10-K)、季报(10-Q): 详尽财务数据
  - 重大事件(8-K): 并购、高管变动、重大合同等及时披露
  - 招股书: IPO 时的详尽信息(见 5.1)
要求: 真实、准确、完整、及时 — 造假是重罪
```

- 💡 **这是股市信任的基石**：你敢买股票，是因为公司被强制公开真实财务，且造假要坐牢。
- ⚠️ **对照加密的致命缺失**：[加密项目](../../crypto/dex/06-风险与安全/6.3-骗局识别.md) 大多没有强制信息披露——
  - 没有审计的财报、没有真实业务数据、团队可匿名。
  - 这就是为什么加密骗局横行：**没有"强制说真话"的制度。**
  - [CEX 的储备金证明（PoR）](../../crypto/cex/06-风险与监管/6.2-储备金证明PoR.md) 是加密对"披露"的初步模仿，但远不及股市的强制财报体系。

## 基石二：禁止欺诈与操纵 ⭐

```
法律严格禁止:
  - 内幕交易(Insider Trading): 用未公开重大信息交易
  - 市场操纵(Manipulation): 拉抬打压、虚假交易、散布谣言
  - 财务造假: 伪造业绩骗投资者
执法: 调查、罚款、禁入、刑事起诉(可坐牢)
```

- 详见 [6.2 市场操纵](./6.2-市场操纵与违规.md)。
- 💡 对照 [加密市场操纵](../../crypto/cex/05-市场与博弈/5.4-市场操纵与黑幕.md)：加密的插针、拉砸、内幕上币长期缺乏有效执法——股市的「禁止操纵+严厉执法」正是加密在追赶的。

## 其他监管支柱

### 适当性管理（Suitability / KYC）
```
券商要了解客户、确保产品适合客户:
  - 高风险产品(期权/杠杆)要评估客户承受能力
  - 防止把复杂高风险产品卖给不懂的散户
```
- 💡 对照 [CEX 诱导高杠杆](../../crypto/cex/02-产品体系/2.7-产品设计哲学.md)：股市有适当性约束，加密则常常诱导散户上高杠杆——监管成熟度的差距。

### 投资者保护基金
```
SIPC(美国)等: 券商破产时, 一定额度内赔付客户损失
→ 给投资者最后一道兜底(见 1.3 分业制衡)
```
- 💡 对照 [CEX 无兜底](../../crypto/cex/06-风险与监管/6.1-风险地图.md)：FTX 用户没有这种保护基金兜底。

### 公平披露（Reg FD）
- 公司不能只对部分人（如大机构）透露重大信息，必须向所有人公平披露。

## 监管的演化逻辑 ⭐

```
股市监管的每一条, 几乎都是某次危机/丑闻的产物:
  1929 股灾 → SEC + 信息披露(1933/34)
  安然财务造假 → 萨班斯法案(强化审计/内控)
  2008 危机 → 多德-弗兰克法案(强化系统性风险监管)
→ "危机 → 立法补强"的循环(见 1.4)
```

> 💡 这和 [CEX「FTX → 储备金证明」](../../crypto/cex/06-风险与监管/6.4-经典案例复盘.md) 是同构的。
> **加密监管正在重走股市的路：每次大暴雷催生一波新监管。** 股市的今天 = 加密监管的明天蓝图。

## 小结

- 监管目标：保护投资者、维护公平有序、促进资本形成。
- 基石一信息披露：强制公司真实完整披露（年报/季报/重大事件），是股市信任的根基——加密最缺这个。
- 基石二禁止操纵：严禁内幕交易、操纵、造假，严厉执法（可坐牢）——加密在追赶。
- 还有适当性管理、投资者保护基金、公平披露等支柱。
- 监管是「危机→立法补强」的产物，与 CEX 同构，是加密监管的未来蓝图。
