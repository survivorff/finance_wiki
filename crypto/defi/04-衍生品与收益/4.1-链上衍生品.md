# 4.1 链上衍生品

> 深度：🟡 → 🔴
> 链上衍生品的全景。永续合约已在 [DEX 2.5](../../dex/02-产品与机制/2.5-衍生品DEX.md) 详述，本篇补充全貌和期权/合成资产。

## 链上衍生品全景

```
链上永续合约: 见 DEX 2.5(订单簿型 Hyperliquid/dYdX、预言机池型 GMX)
链上期权: Lyra、Dopex 等
合成资产: 用抵押 + 预言机, 链上"复制"任何资产的价格暴露
利率衍生品: 把"浮动利率"和"固定利率"互换(新兴)
```

## 一、永续合约（回顾）

链上永续是衍生品 DEX 的主力，已在 [DEX 2.5](../../dex/02-产品与机制/2.5-衍生品DEX.md) 详述：
- 三大流派：订单簿型（Hyperliquid/dYdX）、预言机+LP 池型（GMX）、vAMM。
- 资金费率、保证金、清算等概念同 [CEX 永续](../../cex/02-产品体系/2.2-合约衍生品.md)。
- 清算靠 [清算人机器人](../03-借贷协议/3.2-利率模型与清算.md)。

## 二、链上期权 💡

```
链上期权: 用智能合约实现期权(看涨/看跌, 见 CEX 2.2、股票 2.5)
挑战:
  - 期权定价复杂(Black-Scholes), 链上算 gas 贵
  - 流动性难(期权有很多行权价和到期日, 碎片化)
方案: AMM 式期权池、预言机定价、结构化产品
现状: 比永续小众, 但在发展
```

## 三、合成资产（Synthetic Assets）⭐

```
合成资产: 不持有真实资产, 用"抵押 + 预言机喂价"在链上复制其价格暴露
  例: 抵押加密资产 → 铸造"合成美股/合成黄金/合成外汇"
  → 链上获得对任何资产的价格暴露(无需真持有那个资产)
代表: Synthetix 等
意义: 把传统资产(股票/大宗/外汇)的价格暴露"搬上链"
  → 链上交易"美股""黄金""欧元"(以合成形式)
```

- 💡 **合成资产是"用 DeFi 复刻一切市场"的尝试**：把 [股票](../../../stocks/README.md)、[大宗、外汇](../../../forex/README.md) 的价格暴露搬上链。强依赖预言机（见 [DEX 3.4](../../dex/03-技术架构/3.4-预言机与价格.md)）——预言机出问题，合成资产就崩。

## 四、LSD 与利率衍生品 💡

```
LSD(流动性质押衍生品, 见 4.2): stETH 等本身可看作"质押收益的衍生品"
利率衍生品(新兴): 把 DeFi 的"浮动收益率"换成"固定收益率"
  → 类似 传统的利率互换(见 债券)
  → 让 DeFi 用户能"锁定固定收益"或"投机利率变动"
→ DeFi 在补齐传统金融的"利率衍生品"这块
```

## 链上衍生品 vs CEX/传统 ⭐

| | 链上衍生品 | CEX 合约 | 传统衍生品 |
| --- | --- | --- | --- |
| 托管 | 自托管 | 交易所托管 | 券商/CCP |
| 撮合/定价 | 链上/预言机 | 中心化引擎 | 交易所/OTC |
| 清算 | 清算人机器人 | 强平引擎 | CCP |
| 透明 | 链上可验证 | 黑盒 | 部分 |
| 主要风险 | 预言机/合约/清算失灵 | 平台风险 | 对手方 |

## 小结

- 链上衍生品全景：永续（DEX 2.5 详述）、期权（小众，定价/流动性难）、合成资产、利率衍生品（新兴）。
- 合成资产用"抵押+预言机"复刻任何资产价格暴露（股票/黄金/外汇上链），强依赖预言机。
- LSD 是质押收益的衍生品；利率衍生品补齐 DeFi 的"利率互换"能力。
- 链上衍生品 vs CEX/传统：自托管+链上+清算人机器人，但有预言机/合约/清算失灵风险。
