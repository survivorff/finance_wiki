# 2.4 股票的种类

> 深度：🟢 入门
> 股票不是只有一种。按权利、风格、上市方式分类，帮你看懂市场里五花八门的标的。

## 按权利分类

### 普通股（Common Stock）
- 最常见。有投票权 + 分红权（分红不固定，公司决定）。
- 公司清算时，求偿顺序排在债权人和优先股之后（风险最高，潜在回报也高）。

### 优先股（Preferred Stock）
- 通常**无投票权**，但**分红优先且固定**（像债券）。
- 清算时求偿顺序优先于普通股。
- 💡 介于股票和债券之间的「混合证券」。

### 同股不同权（Dual-Class）💡
```
创始人持有"超级投票权"股(如 1 股 10 票), 公众持普通股(1 股 1 票)
→ 创始人用少量股权控制公司(谷歌、Meta 等科技公司常见)
→ 争议: 保护创始人愿景 vs 损害中小股东话语权
```

## 按市值/风格分类

| 分类 | 特点 |
| --- | --- |
| **大盘股 / 蓝筹** | 市值大、稳定、流动性好（如苹果、可口可乐） |
| **中小盘股** | 市值小、波动大、流动性差 |
| **成长股** | 高增长预期、少分红、估值高（科技股） |
| **价值股** | 估值低、稳定分红（传统行业） |
| **周期股 / 防御股** | 随经济周期波动 / 抗周期（必需消费、公用事业） |

## 按上市地/形式分类

### ADR（美国存托凭证）💡
```
ADR = 外国公司在美国上市交易的"凭证"
  外国公司股票存在托管行 → 在美国发行代表这些股票的凭证(ADR)
  → 美国投资者能像买美股一样买外国公司(如阿里巴巴、台积电的 ADR)
```
- 💡 这是「跨境持有」的一种巧妙设计——和 [跨链桥的"锁定-铸造"](../../crypto/dex/03-技术架构/3.5-跨链与桥.md) 思路异曲同工（本地锁定，异地发凭证）！

### 其他
- **GDR（全球存托凭证）**：类似 ADR 但面向全球。
- **AH 股**：同一公司在 A 股和港股两地上市，价格可能不同（受资本管制）。

## 几个关键概念

```
流通股 vs 总股本: 不是所有股票都能交易(有锁定期、内部持股)
市值 = 股价 × 总股本
自由流通市值 = 股价 × 可流通股本 (更反映实际可交易规模)
```

## 对照加密代币 💡

| 股票 | 加密代币 |
| --- | --- |
| 普通股（投票+分红） | 治理代币（投票，少有"分红"，见 [DEX 4.2](../../crypto/dex/04-生态与运营/4.2-代币经济与治理.md)） |
| 优先股（固定收益） | 某些质押/收益代币 |
| 同股不同权 | 团队/VC 的代币份额和投票权集中 |
| ADR（跨境凭证） | 跨链包装代币(wBTC 等) |

> 💡 加密世界其实在「重新发明」很多股票早就有的结构（治理权、收益权、跨境凭证），
> 只是用代币和合约实现，且大多缺乏股票那样的法律地位和监管保护。

## 小结

- 按权利：普通股（投票+浮动分红）、优先股（无投票+固定分红）、同股不同权（创始人超级投票权）。
- 按风格：大盘蓝筹/中小盘、成长/价值、周期/防御。
- 按上市形式：ADR/GDR（跨境凭证，类似跨链桥思路）、AH 股。
- 加密代币在重新发明股票的很多结构（治理权/收益权/跨境凭证），但缺乏法律地位和监管。
