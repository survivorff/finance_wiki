# 2.3 算法稳定币

> 深度：🔴 硬核 ⭐
> 算法稳定币是 DeFi 最大胆也最危险的实验。理解它的机制和 Luna 的崩溃，是 DeFi 风险教育的核心一课。

## 什么是算法稳定币

```
算法稳定币: 不靠(足额)抵押, 而靠"算法 + 套利机制"维持锚定
  目标: 既要稳定, 又要资本效率高(不用锁那么多抵押品), 又要去中心化
  → 想同时拿下"稳定币三难"(见 2.4)的全部 → 极难, 多数失败
```

## 典型机制：双代币 + 套利（以 Terra UST 为例）⭐

```
Terra 系统有两个币:
  UST: 算法稳定币(目标 1 美元)
  Luna: 治理/吸收波动的代币
锚定机制(套利):
  UST > 1 美元: 允许"用价值 1 美元的 Luna 铸造 1 UST" → 套利者铸 UST 卖出 → 压回 1
  UST < 1 美元: 允许"用 1 UST 销毁换价值 1 美元的 Luna" → 套利者买便宜 UST 换 Luna → 拉回 1
→ 理论上套利让 UST 维持 1 美元
关键: 没有外部足额抵押! "价值"靠 Luna 的市值支撑
```

## 死亡螺旋：Luna 如何归零 ⭐⭐

```
致命缺陷: UST 的"价值"靠 Luna 撑, 而 Luna 的价值又靠人们对 UST 的信心
  → 一旦信心崩, 互相拖累, 加速崩溃(死亡螺旋)

崩溃过程(2022.5):
1. 大额抛售 UST → UST 轻微脱锚(跌破 1)
2. 套利机制启动: 销毁 UST 铸造 Luna → Luna 供应暴增
3. Luna 供应暴增 → Luna 价格暴跌
4. Luna 暴跌 → 市场对"UST 还能换回 1 美元价值"失去信心
5. 更多人恐慌抛售 UST → 回到第 2 步, 但更猛烈
6. 死亡螺旋: UST 越脱锚 → 铸越多 Luna → Luna 越跌 → UST 越没支撑
→ 几天内 UST 和 Luna 双双归零, 约 400+ 亿美元蒸发
```

> ⭐ **死亡螺旋的本质**：算法稳定币的"稳定机制"在信心崩塌时**反而加速崩溃**——
> 正常时套利维持锚定，恐慌时同样的机制变成"印钞机+踩踏放大器"。
> **它没有外部抵押作"地基"，全靠信心，信心一塌就万劫不复。**

## 为什么算法稳定币这么脆弱 ⭐

```
1. 无外部锚: 价值靠"自家代币"循环支撑(左脚踩右脚, 类似 FTX 的 FTT, 见 CEX 6.4)
2. 反身性: 信心 → 价格 → 信心 的正反馈, 上行时狂飙, 下行时崩塌
3. 套利机制双刃: 维稳的机制在崩溃时变成加速器
4. 银行挤兑逻辑: 本质是"无足额储备"的挤兑(同 CEX/Repo 挤兑, 见 债券 5.3)
→ 高收益诱惑(Terra 曾用 Anchor 提供~20% UST 收益吸引存款) + 无抵押 = 庞氏式脆弱
```

- 💡 **Anchor 的 20% 收益**是诱因：用不可持续的高息吸引海量资金存入 UST，规模越大越脆弱——和 [Celsius 高息揽储](../../cex/02-产品体系/2.4-理财与赚币.md)、[庞氏](../../cex/06-风险与监管/6.4-经典案例复盘.md) 同理。

## 算法稳定币 vs 固定汇率崩溃 💡

```
UST 死亡螺旋 ≈ FX 固定汇率被攻击崩溃(见 FX 5.3 索罗斯狙击英镑):
  固定汇率: 央行用有限外储撑锚, 撑不住就崩
  算法稳定币: 用算法+自家代币撑锚, 信心崩就死亡螺旋
共同点: "维持一个没有足够实力支撑的锚定" → 被市场力量击垮
→ UST 是"加密版的固定汇率崩溃", 只是更快更彻底
```

- 💡 **深刻对照**：算法稳定币崩盘和 [固定汇率被狙击](../../../forex/05-央行与政策/5.3-干预与货币危机.md) 是同构的——**都是"锚定缺乏足够实力支撑，被市场力量击垮"**。几百年的货币危机教训，在加密里又重演了一遍。

## 有没有成功的算法稳定币 💡

```
纯算法(无抵押)稳定币: 几乎全部失败(UST 是最惨的)
部分抵押/混合: 一些尝试加入部分抵押缓冲(但纯算法的诱惑和风险都还在)
现实: 经过 UST 教训, 市场对"无足额抵押"稳定币高度警惕
  → 主流回归"有抵押"(法币/超额抵押, 见 2.2)
```

## 小结

- 算法稳定币不靠足额抵押，靠算法+套利维持锚定，想同时拿下稳定币三难——极难，多数失败。
- 典型机制（Terra UST+Luna 双代币套利）的致命缺陷：UST 价值靠 Luna 撑，互相依赖。
- 死亡螺旋：信心崩→套利机制狂铸 Luna→Luna 暴跌→UST 更没支撑→踩踏加速→双双归零（400 亿蒸发）。
- 脆弱根源：无外部锚（左脚踩右脚）、反身性、套利机制双刃、本质是无储备挤兑 + 高息诱因（Anchor 20%）。
- UST 崩盘 ≈ 固定汇率被狙击崩溃——锚定缺乏实力支撑被市场击垮，货币危机教训重演。
