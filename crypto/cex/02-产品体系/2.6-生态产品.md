# 2.6 生态产品（上币 / Launchpad / 平台币 / Web3）

> 深度：🟢🟡
> 这些产品把交易所从「交易场所」扩展成「生态帝国」，是头部所构筑护城河的关键。

## 1. 上币 / Listing

- **是什么**：让一个项目的代币在交易所上架交易。
- **对项目方**：上头部所 = 巨大流量 + 价格暴涨（俗称「上所利好」），是生死攸关的大事。
- **对交易所**：是权力、流量和收入来源（上币费 + 交易手续费）。
- 这是交易所**最有权力、也最灰色**的业务。详细内幕见 [04 上币机制内幕](../04-运营体系/4.2-上币机制内幕.md)。

## 2. Launchpad / Launchpool（新币发行平台）⭐

交易所的「打新股」业务，极受散户欢迎。

| 模式 | 玩法 | 类比 |
| --- | --- | --- |
| **Launchpad** | 用平台币（如 BNB）认购新项目代币，类似抽签打新 | 打新股 |
| **Launchpool** | 质押平台币/主流币，挖出新项目代币（免费领） | 矿池挖矿 |

- **对用户**：早期低价拿到新币，上线后常暴涨，是「薅羊毛」热门。
- **对项目方**：借交易所流量完成首发募资和冷启动。
- **对交易所**：给平台币赋能（要用 BNB 才能打新）+ 绑定用户 + 内容供给。

> 💡 **三赢设计（表面上）**：用户赚钱、项目融资、交易所获客和拉抬平台币。
> 但牛市末期常出现「上线即巅峰」——项目方和早期资本借散户热情出货，散户接盘。

## 3. 平台币（Exchange Token）⭐ 交易所的「股票」

如 **BNB（Binance）、OKB（OKX）、GT（Gate）、BGB（Bitget）**。

### 用途（赋能场景）
- 抵扣交易手续费（打折）。
- 参与 Launchpad/Launchpool 打新。
- VIP 等级、生态权益。
- （部分）作为自营公链的 gas 币（如 BNB 之于 BNB Chain）。

### 价值支撑机制：回购销毁（Buyback & Burn）⭐
- 交易所定期用**利润回购平台币并销毁**，减少流通量，支撑/推高币价。
- 💡 这相当于上市公司的「股票回购 + 注销」，把交易所的盈利能力**证券化**到平台币上。
- BNB 的长期上涨，本质是币安业务增长 + 持续销毁的结果。

> 💡 **深层逻辑**：平台币是交易所把自己「类股权化」的工具。
> 持有 BNB 某种程度上像持有「币安的影子股票」——业务好，回购销毁多，币价涨。
> 但它没有股票的法律权益（不分红、无投票权、无监管保护），是一种「监管灰色地带的类证券」。

## 4. Web3 / 自托管钱包

- 头部所推出**自托管钱包**（如 Binance Web3 Wallet、OKX Wallet、Coinbase Wallet）。
- **战略意图**：CEX 把用户往链上（DeFi、DEX、NFT）引导，但**留在自己的入口里**——既应对「自托管」趋势，又不丢失用户。
- 这是 CEX 应对 DEX 崛起的关键防御性布局。

## 5. 自营公链（Appchain）

- 如 **BNB Chain（币安）、Base（Coinbase 背后的 Coinbase）**。
- 交易所自建/扶持公链，构建链上生态，把交易所流量导入链上，形成「CEX + 公链」双轮。
- Base 的成功（低成本、Coinbase 导流）是这一战略的范本。

## 6. 加密借记卡 / 支付

- 如 Binance Card、Coinbase Card。
- 让用户**直接用加密资产消费**（刷卡时实时兑换成法币）。
- 战略意图：把加密资产接入现实世界支付场景，增强实用性和粘性。

## 7. NFT 市场、IEO、矿池等

- 各种围绕生态的延伸业务，随市场热点起落（NFT 热时上 NFT 市场，等等）。

## 把这些串起来：生态飞轮 ⭐

```
        ┌──────────────────────────────────────┐
        │                                        │
  上币/Launchpad ──带来──▶ 新用户+交易量          │
        │                      │                 │
        │                      ▼                 │
   平台币需求 ◀──打新要用──  用户买平台币          │
        │                      │                 │
        ▼                      ▼                 │
   回购销毁→币价涨 ──激励──▶ 用户更深绑定生态 ──────┘
        │
        ▼
   Web3钱包/公链/卡 ──留住用户，扩大版图
```

> **本质**：生态产品的目的，是把用户和资金**尽可能锁在交易所的闭环里**，
> 提高迁移成本，构筑护城河。这是头部所碾压二线所的关键。

## 小结

- 生态产品把交易所从「交易场所」升级为「生态帝国」。
- 上币是权力中心，Launchpad 是打新获客，平台币是「类股权化」工具（靠回购销毁支撑）。
- Web3 钱包、自营公链、支付卡是应对 DEX 趋势、扩大版图的布局。
- 所有生态产品共同构成「飞轮」，目的是锁住用户和资金，构筑护城河。
