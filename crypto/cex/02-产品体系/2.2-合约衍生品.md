# 2.2 合约衍生品（Derivatives）

> 深度：🟡 进阶 → 🔴 硬核
> ⭐ 这是 CEX 的利润大头，也是整个加密市场资金搅动最剧烈的地方。务必读懂。

## 为什么合约这么重要

- **交易量是现货的 3~5 倍**，手续费收入是交易所的核心利润。
- 高杠杆让小资金能撬动大仓位，放大了交易频率和市场波动。
- 大部分「行情剧烈波动、插针、踩踏」都和合约的强平连环反应有关。

> 一句话：**现货是加密市场的「现金」，合约是加密市场的「赌场」，而赌场永远比兑换点赚钱。**

## 三种主要衍生品

### 一、永续合约（Perpetual Futures / Perp）⭐ 最重要

加密市场最重要的金融创新，没有之一。

- **是什么**：一种**没有交割日**的期货，可以永远持有，赌价格涨跌。
- **杠杆（Leverage）**：可放大仓位。10x 杠杆 = 100U 本金开 1000U 的仓位，盈亏放大 10 倍。
- **做多/做空**：看涨开多（long），看跌开空（short）。合约能双向赚钱是它比现货吸引人的地方。

#### 关键机制 1：资金费率（Funding Rate）⭐
**问题**：永续合约没有交割日，凭什么它的价格会贴着现货价格走，而不是飞天？
**答案**：靠资金费率这个「锚」。

```
合约价 > 现货价(多头太热)  →  多头付钱给空头  →  抑制做多、鼓励做空 → 价格被拉回
合约价 < 现货价(空头太热)  →  空头付钱给多头  →  抑制做空、鼓励做多 → 价格被拉回
```

- 通常**每 8 小时结算一次**（各所略有不同，有的 1 小时/4 小时）。
- **这是多空之间的资金转移，交易所原则上不抽取**（但它鼓励了交易频率，间接增加手续费）。
- 💡 **套利玩法**：资金费率为正时，「现货买入 + 合约做空」对冲，可以稳吃资金费（delta 中性套利）。机构常做。详见 [05 博弈](../05-市场与博弈/5.3-典型玩法与套利.md)。

#### 关键机制 2：强平 / 爆仓（Liquidation）⭐
- 当价格朝反方向走，你的**保证金不足以维持仓位**时，系统强制平仓——这就是「爆仓」。
- 杠杆越高，离爆仓越近。100x 杠杆下，价格反向波动 1% 就爆仓。
- 爆仓后保证金归零（甚至更多）。**这是合约最核心的风险，也是散户亏钱的最主要方式。**

#### 关键机制 3：标记价格（Mark Price）
- 强平**不是用最新成交价**触发的，而是用「标记价」（基于多所现货指数算出来的公允价）。
- 为什么？防止有人用大单在某一瞬间砸出一个极端价（**插针**），恶意触发别人爆仓。

### 二、交割合约（Futures with Expiry）
- 有**固定到期日**（如当周、次周、季度合约）。
- 到期时按指数价**结算交割**，不能无限持有。
- 不同到期日的合约有价差（升水/贴水），反映市场对未来的预期。

### 三、期权（Options）
- 买卖「未来以某价格买入/卖出某资产的**权利**（而非义务）」。
- **看涨期权(Call)**：赌涨的权利；**看跌期权(Put)**：赌跌的权利。
- 更复杂，主要是机构和专业玩家用来对冲或做复杂策略。
- 加密期权市场以 Deribit 为代表，头部所也在布局。

## 保证金模式（进阶）

| 模式 | 含义 | 风险 |
| --- | --- | --- |
| **逐仓 Isolated** | 每个仓位独立保证金，爆仓只亏这一仓 | 风险隔离，新手友好 |
| **全仓 Cross** | 账户所有余额共享作保证金 | 抗波动强，但爆仓可能亏光全账户 |

- **U 本位合约**：用 USDT 等稳定币做保证金和结算（盈亏算 USDT，直观）。
- **币本位合约**：用币（如 BTC）做保证金（盈亏算币，适合长期持币者对冲）。

## 风控三道防线（交易所怎么防穿仓）

当价格剧烈波动，爆仓的用户亏损可能超过保证金（穿仓），谁来补这个窟窿？

```
第1道：强平引擎     → 在用户亏光前就接管平仓
第2道：保险基金     → 强平有盈余时存入；穿仓亏损时拿出来兜底
第3道：自动减仓ADL  → 保险基金也不够时，强制平掉对手方的盈利仓位补窟窿
```

> 这套机制保证**交易所不替用户的亏损买单、系统不产生坏账**。技术实现见 [03 风控系统](../03-技术架构/3.5-清算与风控.md)。

## 合约里资金怎么流动 ⭐

```
散户A做多 ←──── 对赌 ────→ 散户B/做市商做空
   │                          │
   ├── 双方都付手续费 → 交易所(稳赚)
   ├── 资金费率 → 多空之间互相转移
   └── 爆仓的钱 → 对手方盈利 + 保险基金
```

> 💡 **本质**：合约是**零和游戏**（一方赚=另一方亏），扣掉手续费后是**负和游戏**。
> 长期看，散户群体整体是亏的，赚走的是交易所（手续费）+ 专业玩家（技术优势）。这不是阴谋，是数学。

## ⚠️ 小白血泪误区

- **「高杠杆=高收益」**：高杠杆首先意味着高爆仓概率。100x 不是 100 倍收益，是 100 倍快速归零的风险。
- **「合约能以小博大翻身」**：绝大多数散户在合约市场长期是净亏损。它是交易所最赚钱的产品，恰恰因为大部分人在亏。
- **「设了止损就安全」**：极端行情下可能滑点击穿止损，或交易所卡顿无法操作。

## 小结

- 合约是 CEX 的利润核心，永续合约是其中最重要的。
- 三大机制：资金费率（锚定现货）、强平（核心风险）、标记价（防插针）。
- 交易所靠手续费旱涝保收，合约对散户是负和游戏。
- 风控三道防线（强平/保险基金/ADL）保证系统无坏账。
