# 04 · 清算交收基础设施 ⭐

> 深度：🔴 硬核
> 目标：理解「成交之后到钱货两清」这一整套基础设施——这是股市最独特、CEX/DEX 最不同的部分。

成交（trade）只是「达成协议」，真正的「一手交钱一手交货」叫**交收（settlement）**，要经过清算所、存管、登记一整套流程，可能要 1-2 天。这套「成交后基础设施（post-trade infrastructure）」是股市几百年沉淀的精华，也是普通人最不了解、却最重要的部分。

## 为什么这是最重要的对照 ⭐

```
股市:  成交(秒级) → 清算(CCP担保) → 交收(T+1/T+2, 钱货两清) → 存管登记
CEX:   成交 → 立刻改数据库余额(即时"交收", 但只是内部记账, 见 CEX 3.4)
DEX:   成交 = 链上结算(原子, 即时, 不可逆)
```

> ⭐ **三种"交收"哲学**：
> - **股市**：分离交易和交收，用独立机构（CCP+CSD）担保，慢但安全可追责、有违约管理。
> - **CEX**：交易即记账，快，但"交收"只是交易所数据库里改个数字（你信的是交易所）。
> - **DEX**：交易即链上结算，原子且不可逆，由区块链共识保证（你信的是代码）。
>
> **这是理解"为什么股市慢、CEX 快、DEX 不可逆"的钥匙，也是金融基础设施设计的核心。**
