# 2.3 杠杆交易与借贷

> 深度：🟡 进阶
> 介于现货和合约之间的产品。理解它有助于看清交易所「赚息差」的逻辑。

## 一、杠杆交易（Margin Trading）

### 是什么
**借入资金或币来放大现货交易。** 和合约的区别：杠杆交易交易的是**真实现货**（你真的持有/借入币），合约是衍生品（不持有标的）。

### 怎么玩
- **做多**：借 USDT 买更多 BTC（赌涨）。
- **做空**：借 BTC 卖出，等跌了买回还（赌跌）。
- 杠杆倍数通常比合约低（如 3x、5x、10x）。

### 例子
你有 1000 USDT，用 3x 杠杆：
- 借 2000 USDT，总共 3000 USDT 买 BTC。
- BTC 涨 10%，你赚 300 USDT（相当于本金的 30%）。
- 但要还 2000 USDT + 利息。
- BTC 跌太多触及维持保证金 → 同样会被强平。

### 杠杆 vs 合约
| | 杠杆交易 | 合约 |
| --- | --- | --- |
| 标的 | 真实现货（借来的真币） | 衍生品（不持有真币） |
| 杠杆倍数 | 较低(3-10x) | 高(可达 100x) |
| 成本 | 借币利息 + 手续费 | 手续费 + 资金费率 |
| 适合 | 想持有现货又想放大 | 纯赌涨跌、高频 |

## 二、借贷（Lending / Borrow）

### 是什么
- 用户抵押 A 币，借出 B 币。如抵押 BTC 借 USDT（不想卖币又想要流动性）。
- 典型场景：长期看好 BTC 不想卖，但临时需要 USDT 周转 → 抵押 BTC 借 USDT。

### 关键概念
- **抵押率 / LTV（Loan-to-Value）**：能借出的金额占抵押物价值的比例。如 LTV 70%，抵押 100U 的 BTC 最多借 70U。
- **清算线**：抵押物价值下跌到一定程度，触发强制清算（卖掉抵押物还债）。
- **利率**：借款人付利息，这是交易所/出借方的收益来源。

## 三、交易所怎么赚钱：息差模型 ⭐

这是理解交易所「类银行」业务的关键。

```
用户A 把币存进"理财" ──(低息)──▶ 交易所资金池 ──(高息)──▶ 借给用户B 做杠杆/借贷
        赚 2% 利息                                      付 8% 利息
                          交易所赚中间 6% 息差
```

> 💡 **这就是银行的生意**：低息吸储、高息放贷、赚中间差价。
> 交易所把用户存的闲置币，借给想做杠杆/借空的人，赚息差 + 借贷手续费。
> 这也解释了为什么交易所拼命推「理财」——它需要这些币作为「放贷的弹药」。

## 风险点

- **借贷利率浮动**：资金紧张时利率飙升，做空成本剧增。
- **连环清算**：抵押物（如 BTC）暴跌时，大量抵押贷款同时触及清算线，集中抛售加剧下跌。
- **借不到币**：极端行情下想做空却借不到币（可借量被借光）。

## 小结

- 杠杆交易 = 借钱/币放大现货交易，交易真实现货，杠杆倍数低于合约。
- 借贷 = 抵押一种币借另一种，关键看 LTV 和清算线。
- 交易所靠**息差**赚钱，逻辑和银行一样：低息吸储、高息放贷。
- 理财业务的本质是为借贷/杠杆提供「放贷弹药」。
