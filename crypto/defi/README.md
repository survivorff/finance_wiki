# 加密货币 · DeFi（去中心化金融）深度知识库

> Decentralized Finance，去中心化金融。
> 用智能合约在链上重建「稳定币、借贷、衍生品、收益」等金融功能，无需银行/券商等中介。

如果你读完了 [CEX](../cex/README.md)、[DEX](../dex/README.md)，以及传统的 [股票](../../stocks/README.md)、[外汇](../../forex/README.md)、[债券](../../bonds/README.md)，
DeFi 会把这一切**在链上重新拼装**：

> **[DEX](../dex/README.md) 解决了"链上怎么交易"，DeFi 则更进一步——
> 把"借贷（债券/银行）、稳定币（货币/外汇）、衍生品（合约）、收益（理财）"全部搬上链，
> 用智能合约像乐高一样拼成一个完整的、无需许可的"链上金融体系"。**

DeFi 最大的学习价值：

1. **它是传统金融的"链上重演"**：稳定币≈[货币/外汇](../../forex/README.md)，借贷≈[债券/银行](../../bonds/README.md)，DEX≈[交易所](../../stocks/README.md)，衍生品≈[合约](../cex/02-产品体系/2.2-合约衍生品.md)——你前面学的一切，在这里用代码重新实现。
2. **可组合性（货币乐高）**：协议像积木自由拼接，创新速度爆炸（也风险叠加）。
3. **稳定币是 DeFi 的"血液"**：理解稳定币，就理解了加密世界如何锚定法币、如何流动。
4. **它把"影子银行"搬上了链**：[借贷](../../bonds/05-市场与参与者/5.3-回购市场.md)、杠杆、挤兑——传统金融的风险以新形态重现。

这个知识库的目标：让**小白**理解「稳定币、链上借贷、收益挖矿到底是什么」，让**从业者/工程师**挖到清算机制、稳定币三难、收益来源、可组合性风险的深水细节。

## 🎯 读完应该建立的「认知锚点」⭐

1. **DeFi 是传统金融的链上重演**：稳定币、借贷、衍生品、理财，全是传统金融功能的智能合约版本。
2. **稳定币是 DeFi 的血液**：它是链上的"美元"，但有"三难困境"（稳定/资本效率/去中心化难兼得）。
3. **链上借贷靠超额抵押**：因为无法追债（匿名），必须抵押超过借款的资产——这是和 [传统借贷](../../bonds/01-行业全景/1.1-债券是什么.md) 的根本区别。
4. **清算是 DeFi 的安全阀**：抵押物价值下跌触发清算，由清算人机器人执行（对照 [CEX 强平](../cex/03-技术架构/3.5-清算与风控.md)）。
5. **可组合性是超能力也是炸药**：乐高式拼接催生创新，也让风险层层叠加、一处出事全线传染。
6. **DeFi 是"链上影子银行"**：高收益背后是高风险，挤兑、暴雷、踩踏以新形态重现（[Luna/Celsius](../cex/06-风险与监管/6.4-经典案例复盘.md)）。

## 一张图看懂 DeFi 是传统金融的"链上重演"

```
   传统金融                          DeFi（链上重演）
┌──────────────────┐            ┌──────────────────────┐
│ 货币/央行         │ ───链上──▶  │ 稳定币(USDT/USDC/DAI)  │
│ 银行/债券(借贷)   │ ───链上──▶  │ 借贷协议(Aave/Compound)│
│ 交易所            │ ───链上──▶  │ DEX(Uniswap, 见 DEX)   │
│ 期货/期权(衍生品) │ ───链上──▶  │ 衍生品 DEX(GMX/dYdX)   │
│ 资管/理财         │ ───链上──▶  │ 收益聚合(Yearn)        │
│ 影子银行          │ ───链上──▶  │ DeFi 借贷/杠杆套娃      │
└──────────────────┘            └──────────────────────┘
   中介+监管+牌照                     智能合约+可组合+无许可
```

> **关键洞察**：DeFi 没有发明新的金融功能——它把传统金融的每个功能用智能合约重新实现了一遍。
> 好处：无许可、可组合、透明、7×24。代价：代码风险、不可逆、无兜底、风险叠加。
> **你前面学的传统金融，就是理解 DeFi 的最好地图。**
