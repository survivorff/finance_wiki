# 05 · 央行与政策

> 深度：🟡 进阶 ⭐
> 目标：理解 FX 独有的"上帝玩家"——央行，及其货币政策、汇率制度、干预手段。

央行是 FX 区别于其他所有市场的关键存在：**它能印钞、设利率、直接干预，是唯一能"改变游戏规则"的玩家。**

## 核心收获 ⭐

```
央行的三大工具影响汇率:
1. 利率(货币政策): 加息→货币走强, 降息→走弱(见 4.1)
2. 印钞(QE/QT): 大量印钞→货币贬值压力
3. 直接干预: 用外汇储备买卖本币

汇率制度的"不可能三角": 固定汇率、自由资本流动、独立货币政策, 三者只能取其二
```
