# 3.3 久期与凸性

> 深度：🔴 硬核 ⭐
> 久期是债券最重要的风险指标——它量化"债券价格对利率的敏感度"。理解久期，就懂了债券的利率风险。

## 久期（Duration）⭐

```
久期 = 债券价格对利率变动的"敏感度"
  通俗: 利率变动 1%, 债券价格大约变动多少 %
  例: 久期 = 7 → 利率涨 1%, 价格大约跌 7%; 利率跌 1%, 价格涨 7%
```

### 久期的两种含义
```
麦考利久期(Macaulay Duration): 收回现金流的"加权平均时间"(年)
  → 你平均多久能收回投资(用现金流现值加权)
修正久期(Modified Duration): 价格对利率的敏感度(%)
  → 实务中说"久期"通常指这个(直接告诉你价格变动幅度)
```

## 什么影响久期 ⭐

```
1. 到期时间: 期限越长, 久期越长(利率风险越大)
   → 30 年债 比 2 年债 对利率敏感得多
2. 票息率: 票息越低, 久期越长
   → 零息债(无票息, 见 2.2)久期 = 到期时间(最长, 最敏感)
3. 收益率水平: 收益率越低, 久期越长
```

> ⭐ **关键直觉**：久期长的债（长期债、低息债、零息债）= 利率"放大器"。
> 利率小动一下，价格大动——这就是为什么 2022 年加息时，**长期债跌得最惨**（久期长）。

### 久期解释 2022 债券惨案 💡
```
2022 央行激进加息(利率涨好几个百分点):
  短期债(久期短): 跌得少
  长期债(久期长, 如 20-30 年国债): 暴跌(久期 × 利率涨幅 = 巨大跌幅)
  → 一些"安全"的长期国债基金跌了 20-30%+(久期惹的祸)
→ "买国债很安全"的人, 在利率风险上栽了跟头(信用安全≠没有利率风险)
```

## 久期的用途 ⭐

```
1. 风险衡量: 知道利率变动对组合的影响
2. 风险管理: 调整组合久期来控制利率风险
   - 怕利率涨 → 缩短久期(买短债/浮息债)
   - 赌利率跌 → 拉长久期(买长债, 放大收益)
3. 对冲(免疫): 让资产和负债的久期匹配 → 利率变动时两边抵消
   → 保险/养老金常用"久期匹配"管理利率风险
```

## 凸性（Convexity）⭐

久期是"线性近似"，但债券价格-利率关系实际是"弯的"（凸的）：

```
价格
 │＼
 │  ＼____ 实际价格曲线(凸向原点)
 │      ＼＿＿＿
 │  久期(直线)只是切线近似
 └──────────────── 利率
凸性 = 这条曲线的"弯曲程度"
```

### 凸性的意义 💡
```
正凸性(普通债券): 对投资者有利
  利率跌时: 实际价格涨得比久期预测的"更多"
  利率涨时: 实际价格跌得比久期预测的"更少"
  → 凸性是"额外的好处"(涨多跌少)
→ 久期估算大变动时不准, 要加凸性修正(见 08)
```

## 久期/凸性 vs 其他市场的"敏感度"💡

```
久期之于债券 ≈ 各市场的"风险敏感度"指标:
  期权的 Delta/Gamma(对标的价格的敏感度)
  → 久期(对利率), Delta(对价格), 都是"一阶敏感度"
  → 凸性(债券), Gamma(期权), 都是"二阶修正"
→ 同一套"敏感度 + 修正"的数学框架, 跨资产通用
```

- 💡 对照 [无常损失/做市的 gamma](../../crypto/dex/08-深水区/8.2-无常损失数学推导.md)：金融风险管理的核心都是"一阶敏感度 + 二阶凸性/gamma"的框架——债券、期权、做市、LP 共享同一套数学语言。

## 小结

- 久期 = 债券价格对利率的敏感度（久期 7 → 利率动 1%、价格动约 7%）。
- 麦考利久期=加权平均回收时间，修正久期=价格敏感度（实务常用）。
- 期限越长、票息越低、收益率越低 → 久期越长 → 利率风险越大（零息债久期最长）。
- 久期解释 2022 长期债惨跌（信用安全≠没利率风险）；用途是风险衡量、管理、久期匹配对冲。
- 凸性是久期的二阶修正（实际曲线是弯的），正凸性=涨多跌少对投资者有利。
- 久期/凸性 ≈ 期权 Delta/Gamma，是跨资产通用的"敏感度+修正"框架。
