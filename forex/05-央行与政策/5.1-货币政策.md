# 5.1 货币政策

> 深度：🟡 进阶 ⭐
> 央行的货币政策是汇率的最大中长期驱动，也是全球资金流动的总开关。

## 央行是什么、要干什么

```
央行(中央银行): 一国货币的发行者和管理者(美联储Fed、欧洲央行ECB、日本央行BOJ、中国人民银行PBOC)
核心目标(各国略有不同):
  - 控制通胀(物价稳定) ← 最核心
  - 促进就业/经济增长
  - 维护金融稳定
  - (部分)维持汇率稳定
工具: 利率、货币供应(印钞)、外汇干预、监管
```

## 工具一：利率（最重要）⭐

```
央行设定"基准利率"(政策利率):
  加息: 提高利率 → 抑制借贷和经济过热 → 压通胀; 同时吸引资金 → 本币走强
  降息: 降低利率 → 刺激借贷和经济 → 防衰退; 同时资金流出 → 本币走弱
对汇率: 加息→货币走强, 降息→货币走弱(见 4.1 利率驱动)
```

- 💡 **这是 FX 交易员最盯的东西**：每次央行利率决议都引发剧烈波动。
- 💡 "**预期**"比"实际"更重要：市场提前 price in 加息预期，真加息时反而可能"利好出尽"回落（见 [4.1](../04-汇率原理/4.1-什么决定汇率.md)）。

## 工具二：量化宽松/紧缩（QE/QT）⭐

```
QE(量化宽松): 经济危机/利率已降到 0 时, 央行"印钞买债券"
  → 向市场注入大量货币 → 压低长期利率、刺激经济
  → 货币供应大增 → 本币贬值压力 + 资产价格上涨
QT(量化紧缩): 反向, 央行回收流动性 → 收紧
```

- 💡 **2008 和 2020 后，主要央行大规模 QE**——印了天量货币。
- 💡 **这正是加密"反通胀"叙事的来源**：比特币的"固定 2100 万枚"是对"央行无限印钞"的回应（见 [1.4 简史](../01-行业全景/1.4-发展简史.md)）。理解 QE，就理解了加密"硬通货/数字黄金"叙事的现实背景。

## 工具三：外汇干预

```
央行直接下场买卖本币影响汇率(见 5.3):
  本币太强(伤出口) → 卖本币买外币 → 压低本币
  本币太弱(资本外逃/通胀) → 卖外储买本币 → 撑住本币
→ 用"外汇储备"作弹药
```

## 央行的"沟通"也是工具 💡

```
前瞻指引(Forward Guidance): 央行通过讲话"暗示"未来政策
  → 不用真动手, 光"说"就能影响市场预期 → 影响汇率
  "鹰派"(倾向收紧/加息) → 本币走强
  "鸽派"(倾向宽松/降息) → 本币走弱
→ 央行行长一句话能让汇率剧烈波动(市场解读字里行间)
```

## 美联储的全球影响 ⭐⭐

```
美联储(Fed)是"全球央行"(因为美元霸权, 见 1.3):
  Fed 加息 → 美元走强 + 全球美元回流美国
    → 新兴市场货币贬值、资本外流、偿还美元债压力大
    → 可能引发新兴市场危机(见 5.3)
  Fed 降息 → 美元走弱 + 资金流向新兴市场/风险资产
→ "美联储打喷嚏, 全球感冒"
```

> ⭐ **这是全球资金流动的总开关**：美联储的利率政策，通过美元，影响全世界的资金流向、汇率、股市、债市、大宗、甚至加密。
> 你最初想懂的「全球资金如何搅动」，**美联储的货币政策是那个最大的"总阀门"。**

## 货币政策的传导链 💡

```
央行利率 → 银行间利率 → 贷款/存款利率 → 借贷意愿 → 投资消费 → 经济/通胀
              ↓
            汇率(资金流) → 进出口、输入性通胀 → 反馈回经济
→ 货币政策通过"利率 + 汇率"两条腿影响经济
```

## 小结

- 央行管货币、控通胀，工具是利率、QE/QT（印钞）、外汇干预、沟通（前瞻指引）。
- 利率是汇率最大驱动（加息走强降息走弱），且"预期"比"实际"更重要。
- QE（印钞买债）带来货币贬值压力，是加密"反通胀/硬通货"叙事的现实背景。
- 美联储因美元霸权是"全球央行"，其政策通过美元成为全球资金流动总开关（"打喷嚏全球感冒"）。
