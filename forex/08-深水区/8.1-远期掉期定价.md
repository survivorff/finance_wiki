# 8.1 远期与掉期定价

> 深度：🔴🔴 极硬核
> 把覆盖利率平价（CIP）推导清楚，算出远期汇率和掉期点。
> 前置：[4.2 利率平价](../04-汇率原理/4.2-利率平价与购买力平价.md)、[2.2 远期](../02-参与者与产品/2.2-即期与远期.md)。

## 符号约定

```
S  = 即期汇率(spot), 报价为 1 单位基础货币 = S 单位报价货币
F  = 远期汇率(forward)
r_q = 报价货币利率(quote, 后面那个货币)
r_b = 基础货币利率(base, 前面那个货币)
T  = 期限(年)
以 EUR/USD 为例: base=EUR, quote=USD, S=1.10
```

## 覆盖利率平价（CIP）推导 ⭐

考虑两种等价的投资方式，结果必须相等（否则套利）：

```
方式 A: 持有 1 单位报价货币(USD), 存 T 年
  → 得到: 1 × (1 + r_q × T)   [美元利息]

方式 B: 用 1 USD 换成基础货币(EUR), 存 EUR 利率, 同时用远期锁定换回
  1. 1 USD 换成 EUR: 得到 1/S 单位 EUR
  2. 存 T 年: (1/S) × (1 + r_b × T) 单位 EUR
  3. 用远期 F 锁定换回 USD: (1/S) × (1 + r_b × T) × F

两者必须相等(都无风险, 否则套利):
  1 × (1 + r_q×T) = (1/S) × (1 + r_b×T) × F
```

解出远期汇率 F：

```
        1 + r_q × T
F = S × ─────────────        (CIP 公式)
        1 + r_b × T
```

## 直觉解读 ⭐

```
F/S = (1 + r_q·T)/(1 + r_b·T)

若 报价货币利率 r_q > 基础货币利率 r_b:
  → F > S (远期 > 即期, 基础货币"远期升水")
若 r_q < r_b:
  → F < S (基础货币"远期贴水")

记忆: 高利率货币 远期"贴水"(便宜), 低利率货币 远期"升水"(贵)
  → 远期的升贴水, 恰好抵消利率差 → 无套利(见 4.2)
```

## 算例

```
EUR/USD: S = 1.1000
USD 利率 r_q = 5%, EUR 利率 r_b = 3%, T = 1 年
F = 1.1000 × (1 + 0.05)/(1 + 0.03)
  = 1.1000 × 1.05/1.03
  = 1.1000 × 1.01942
  ≈ 1.1214
→ 1 年远期 EUR/USD ≈ 1.1214(EUR 远期升水, 因为 USD 利率更高)
远期点 = F − S = 1.1214 − 1.1000 = 0.0214 = 214 个点
```

## 掉期点定价（FX Swap）⭐

```
FX 掉期(见 2.3) = 即期 + 反向远期
掉期点(swap points) = F − S = 远期点
  → 掉期的"成本/收益"就是这个远期点, 由利率差决定
做掉期 = 借一种货币的利率, 付/收另一种货币的利率 → 净成本 = 利率差
```

> 💡 **核心结论**：FX 远期和掉期的价格，**完全由两国利率差决定**（在 CIP 成立下）。
> 这不是预测未来汇率，而是无套利锁定——和 [CEX 期现基差/资金费率](../../crypto/cex/08-深水区/8.2-永续合约数学全解.md) 是同一类无套利定价。
> **远期汇率 ≠ 市场对未来汇率的预测**，它只是"即期 + 利率差"的机械结果。

## CIP 偏离（现实中的细节）💡

```
理论上 CIP 严格成立(套利保证), 但 2008 后出现持续偏离:
  "Cross-Currency Basis": CIP 的小幅偏离
  原因: 银行资产负债表成本、美元融资紧张、监管约束 → 套利不再"免费"
→ 这个偏离本身成了一个交易/研究主题
→ 类比: 加密里 CIP 式套利(期现)也会因资金成本/风险有残留偏离
```

## 小结

- CIP 推导：两种等价投资必相等 → `F = S × (1+r_q·T)/(1+r_b·T)`。
- 高利率货币远期贴水、低利率货币远期升水，升贴水恰好抵消利率差（无套利）。
- FX 掉期点 = 远期点 = 由利率差决定，掉期净成本就是利率差。
- 远期汇率不是对未来的预测，是"即期+利率差"的机械结果（同 CEX 期现基差）。
- 现实中 CIP 有小幅偏离（cross-currency basis，因资金成本/监管），套利不再完全免费。
