# 6.3 风险与危机案例

> 深度：🟡 进阶
> 股市的风险类型，以及历次危机的教训。每次危机都催生了新的制度——和 [CEX 案例复盘](../../crypto/cex/06-风险与监管/6.4-经典案例复盘.md) 同构。

## 股市风险类型

```
1. 市场风险: 整体下跌(系统性), 个股波动(非系统性)
2. 流动性风险: 想卖卖不掉(冷门股/危机时)
3. 信用/对手方风险: 交收违约(由 CCP 缓解, 见 4.2)
4. 杠杆风险: 融资融券爆仓(见 2.5)
5. 系统性风险: 连锁崩溃、流动性枯竭、踩踏
6. 欺诈风险: 财务造假、操纵(见 6.2)
```

## 经典危机复盘

### 1929 大萧条股灾 → 现代监管诞生 ⭐
- 投机狂热 + 高杠杆 + 无监管 + 操纵横行 → 崩盘 → 大萧条。
- **催生**：SEC、《证券法》《证券交易法》、信息披露制度（见 [6.1](./6.1-监管体系.md)）。
- 💡 **金融监管的"创世事件"**——从蛮荒到规范的分水岭。

### 1987 黑色星期一 → 熔断机制
- 单日暴跌 22%（程序化交易"投资组合保险"加剧抛售）。
- **催生**：熔断机制（Circuit Breaker）——跌幅过大暂停交易降温。
- 💡 对照 [CEX 极端行情](../../crypto/cex/03-技术架构/3.7-高可用与安全工程.md)：股市用「主动熔断」防踩踏，加密多为高杠杆[连环爆仓](../../crypto/cex/03-技术架构/3.5-清算与风控.md)（无熔断）。

### 2000 互联网泡沫
- 科技股估值疯狂 → 泡沫破裂 → 大量公司归零。
- 💡 对照加密：和 [2017 ICO 泡沫、各种叙事炒作](../../crypto/cex/01-行业全景/1.5-发展简史与里程碑.md) 如出一辙——「新技术 + 过度投机 + 估值脱离基本面 → 泡沫破裂」是反复上演的剧本。

### 2008 全球金融危机
- 次贷 + 过度杠杆 + 衍生品 + 系统性关联 → 雷曼倒闭 → 全球危机。
- **催生**：多德-弗兰克法案、强化 CCP 和系统性风险监管、衍生品集中清算。
- 💡 雷曼倒闭时，**客户的证券因为在独立存管+隔离而基本无恙**（见 [1.3](../01-行业全景/1.3-分业制衡的产业链.md)、[4.3](../04-清算交收/4.3-中央存管CSD.md)）——这正是分业制衡保护投资者的实证，和 [FTX 用户血本无归](../../crypto/cex/06-风险与监管/6.4-经典案例复盘.md) 形成惨烈对比。

### 2010 闪电崩盘（Flash Crash）
- 高频交易互动 → 道指几分钟暴跌千点又反弹。
- **催生**：对高频交易的监控、更完善的熔断（见 [07 高频交易](../07-现代演化/7.2-高频交易.md)）。
- 💡 暴露了电子化、自动化市场的新型脆弱性——和 [加密的算法/清算瀑布](../../crypto/cex/03-技术架构/3.5-清算与风控.md) 类似。

### 2021 GameStop 模因股
- 散户在社交媒体（Reddit）抱团逼空机构，券商一度限制买入引发巨大争议。
- 暴露了 [PFOF、订单流](../02-参与者与产品/2.1-交易所与券商.md)、清算保证金（券商要向 CCP 交巨额保证金）等结构性问题。
- 💡 **社交媒体 + 散户情绪驱动行情**——和加密的社群文化、meme 币狂热遥相呼应。

## 危机的共同模式 ⭐

```
几乎每次危机都有相同配方:
  过度杠杆 + 过度投机 + 估值脱离基本面 + 监管滞后/盲区 + 某个触发点
  → 崩盘 → 反思 → 新监管补强
```

> 💡 **这个配方在股市和加密反复上演**：
> - 1929 股灾 ≈ 加密的各种泡沫破裂
> - 2008 雷曼（杠杆+衍生品+关联）≈ [FTX/Celsius 连环暴雷](../../crypto/cex/06-风险与监管/6.4-经典案例复盘.md)（杠杆+挪用+关联）
> - GME（社群+情绪）≈ 加密 meme 币狂热
>
> **历史不会简单重复，但会押韵。** 学股市的危机史，能让你在加密市场里更早识别"似曾相识"的危险信号。

## 风险管理工具（股市的应对）

| 工具 | 作用 |
| --- | --- |
| 熔断机制 | 暴跌时暂停交易降温 |
| 涨跌幅限制 | 限制单日波动（部分市场） |
| 保证金要求 | 限制杠杆（见 [2.5](../02-参与者与产品/2.5-相关产品.md)） |
| 卖空限制 | 危机时限制做空（争议） |
| CCP 违约管理 | 防交收违约（见 [4.2](../04-清算交收/4.2-中央对手方CCP.md)） |
| 信息披露+监管 | 防欺诈（见 [6.1](./6.1-监管体系.md)） |

## 小结

- 股市风险：市场、流动性、对手方、杠杆、系统性、欺诈。
- 经典危机：1929（催生 SEC）、1987（催生熔断）、2000（互联网泡沫）、2008（催生多德-弗兰克）、2010 闪电崩盘、2021 GME。
- 2008 雷曼倒闭时客户证券因隔离存管基本无恙，与 FTX 用户血本无归形成对比，印证分业制衡价值。
- 危机的共同配方（过度杠杆+投机+估值脱节+监管盲区）在股市和加密反复上演——历史押韵。
