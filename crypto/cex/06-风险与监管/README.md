## 一条贯穿全行业的主线 ⭐

> **CEX 的历史就是一部「信任崩塌 → 制度补强」的循环史。**
> 每一项你今天看到的「标配设计」（冷热钱包、储备金证明、资产隔离监管），
> 背后都站着一次惨痛的事故。
>
> CEX 全部的技术和监管设计，本质都在回答同一个问题：
> **「如何让一个中心化的钱袋子，值得被信任？」** 这个问题至今没被完美解决——正是 DEX 叙事的起点。

## 风险与防线对照速查

```
风险                 → 对应的技术/制度防线
──────────────────────────────────────────────
被黑被盗             → 冷热钱包分离、多签/MPC、签名验证、提币风控
挪用用户资产         → 客户资产隔离(监管)、储备金证明、第三方审计
穿仓/连环爆仓         → 标记价、强平引擎、保险基金、ADL
市场操纵/插针         → 多所加权指数价、深度建设、市场监控
洗钱/非法资金         → KYC、AML、链上分析、制裁筛查
偿付能力不透明        → PoR + 默克尔树 + 持续审计
```
