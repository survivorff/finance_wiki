# 外汇市场（Forex / FX）深度知识库

> Foreign Exchange，外汇市场。全球最大、流动性最高的金融市场。
> 没有中央交易所，是一个由银行间网络构成的「场外（OTC）市场」。

如果你读完了 [股票](../stocks/README.md)、[CEX](../crypto/cex/README.md)、[DEX](../crypto/dex/README.md)，
外汇市场会给你一个震撼的新视角：

> **这是一个日交易量 7-8 万亿美元的庞然大物（比全球股市加起来的日交易量还大几倍），
> 却没有一个"交易所"——它是由全球各大银行用电话和电子网络织成的、24 小时不眠的去中心化网络。**

外汇市场最大的学习价值：

1. **没有中央交易所的"去中心化"**：和股市的「中央交易所」完全相反，FX 天然是分散的银行间网络——这其实和 [DEX 的去中心化理想](../crypto/dex/README.md) 遥相呼应，只是用的是"机构信任网络"而非"代码"。
2. **货币本身就是商品**：股票背后是公司，FX 交易的是「国家的货币」——价格背后是利率、通胀、国际收支、地缘政治。
3. **杠杆的极致**：FX 是杠杆交易的鼻祖（100:1 杠杆常见），理解它能看懂 [CEX 合约杠杆](../crypto/cex/02-产品体系/2.2-合约衍生品.md) 的源头。
4. **央行是终极玩家**：股市没有"中央股票发行者能直接干预"，但 FX 里央行能印钱、能干预汇率——这是其他市场没有的"上帝之手"。

这个知识库的目标：让**小白**理解「汇率到底怎么回事、换汇背后发生了什么」，让**从业者/工程师**挖到做市、远期掉期、套息交易、央行干预的深水细节。

## 🎯 读完应该建立的「认知锚点」⭐

1. **FX 没有中央交易所**：它是全球银行间的 OTC 网络，24×5 不间断。这是和股市最根本的结构差异。
2. **汇率是两种货币的"相对价格"**：EUR/USD = 1.10 意思是 1 欧元换 1.10 美元。汇率永远是"成对"的。
3. **汇率的根本驱动是利率**：高利率货币吸引资金流入（套息），利率差驱动了 FX 最大的交易策略——这是理解 FX 的钥匙。
4. **央行是终极玩家**：央行通过利率、印钞、直接干预影响汇率——其他市场没有这种"能改变游戏规则的玩家"。
5. **FX 是杠杆的鼻祖**：100:1 杠杆司空见惯，零售外汇是散户亏损的重灾区。
6. **FX 是所有市场的"地基货币层"**：股票、债券、大宗、加密的跨境交易，最终都要经过 FX 换汇——它是全球资金流动的底层管道。

## 一张图看懂外汇市场的"无中心"结构

```
   股票市场(有中央交易所)          外汇市场(无中央交易所)
   ┌──────────────────┐         ┌────────────────────────────┐
   │   中央交易所       │         │   没有中心! 是一张银行间网络   │
   │  (所有订单汇集)    │         │                              │
   │       ▲           │         │  花旗 ←──→ 摩根 ←──→ 德银      │
   │   ┌───┼───┐       │         │   ▲    ╲   ▲   ╱    ▲         │
   │  券商 券商 券商     │         │   │     ╲ │ ╱      │          │
   └──────────────────┘         │  汇丰 ←──→ 瑞银 ←──→ 巴克莱     │
                                 │  (各大银行互相报价、成交)       │
                                 │  电子平台(EBS/Reuters)聚合       │
                                 └────────────────────────────┘
```

> **关键洞察**：FX 没有"唯一的价格"——理论上每家银行报的价都可能略有不同，
> 靠套利和电子平台聚合趋于一致（很像 [DEX 多池 + 套利校准](../crypto/dex/02-产品与机制/2.1-AMM是什么.md)）。
> **一个没有中央交易所、却能高效运转几十年的市场，本身就是"去中心化金融"最古老的成功案例。**
