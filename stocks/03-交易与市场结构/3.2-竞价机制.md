# 3.2 竞价机制

> 深度：🟡 → 🔴
> 股市有两种撮合方式：连续竞价和集合竞价。集合竞价（尤其开收盘）是股市独有、CEX 没有的精妙设计。

## 两种竞价方式

### 连续竞价（Continuous Trading）
```
盘中大部分时间: 来一个订单就立刻尝试撮合, 实时成交
→ 和 CEX 的撮合一样(见 3.1)
```

### 集合竞价（Call Auction）⭐ 股市独有
```
在某个时点, 集中收集一段时间内的所有买卖订单, 一次性统一定价撮合
→ 找出一个"成交量最大化"的价格, 所有该轮成交都按这一个价格成交
用于: 开盘、收盘、（部分市场）盘中临时停牌后复牌
```

## 集合竞价怎么定价 ⭐

```
原理: 找一个价格 P, 使得"能成交的数量最大"
做法:
  1. 收集所有买单(各价位想买多少)和卖单
  2. 对每个候选价格, 算出该价格下能撮合的总量
  3. 选"成交量最大"的价格作为集合竞价价
  4. 该轮所有成交都按这一个统一价格 → 公平(同一时刻同一价)
```

### 一个简化例子
```
若价格定 100: 买方愿买 1000 股, 卖方愿卖 600 股 → 成交 600
若价格定 101: 买方愿买 800 股, 卖方愿卖 900 股 → 成交 800
若价格定 102: 买方愿买 500 股, 卖方愿卖 1100 股 → 成交 500
→ 选 101(成交量最大 800) 作为集合竞价价
```

## 为什么需要集合竞价 ⭐

```
开盘时: 隔夜积累了大量信息和订单(公司发财报、隔夜外盘大涨)
  → 如果直接连续竞价, 第一笔就可能乱跳、被操纵
  → 集合竞价集中所有订单, 形成一个"公允的开盘价", 更稳定抗操纵

收盘时: 收盘价极重要(用于结算、估值、指数计算、衍生品交割)
  → 集合竞价让收盘价更不容易被"尾盘一笔大单"操纵
```

> 💡 **这是 CEX 缺失的精妙设计**：CEX 7×24 连续交易，没有「开收盘集合竞价」，
> 所以 CEX 的价格更容易被[瞬时大单插针](../../crypto/cex/05-市场与博弈/5.4-市场操纵与黑幕.md)。
> 股市用「集合竞价」在关键时点（开收盘）**集中流动性、统一定价**，提升了抗操纵性和价格质量。
> 💡 有意思的是，[DEX 的 CoW 批量拍卖](../../crypto/dex/02-产品与机制/2.4-聚合器与意图.md) 和[抗 MEV 的批量竞价](../../crypto/dex/08-深水区/8.4-MEV实战剖析.md) 思路，本质就是重新发明了股市的「集合竞价」来对抗操纵/MEV！

## 收盘集合竞价的重要性 💡

- 收盘价是「官方价格」：用于基金估值、指数计算、衍生品结算、业绩考核。
- 所以**收盘集合竞价的成交量极大**（很多被动基金、ETF 在收盘时调仓）。
- 这也是一天中流动性最集中的时刻之一。

## 对照三个市场的「定价时点」

| | 关键定价机制 |
| --- | --- |
| 股市 | 开收盘集合竞价（集中、统一价、抗操纵） |
| [CEX](../../crypto/cex/02-产品体系/2.1-现货交易.md) | 7×24 连续撮合，无集合竞价 |
| [DEX](../../crypto/dex/02-产品与机制/2.4-聚合器与意图.md) | 连续(AMM)，但 CoW 等用批量拍卖（≈集合竞价）抗 MEV |

## 小结

- 连续竞价（盘中实时撮合，同 CEX）+ 集合竞价（开收盘集中统一定价）两种方式。
- 集合竞价找「成交量最大化」的统一价格，该轮所有成交同价，公平抗操纵。
- 开盘集合竞价消化隔夜信息形成公允开盘价；收盘集合竞价保护重要的收盘价。
- 这是 CEX 缺失的设计；而 DEX 的批量拍卖（抗 MEV）本质是重新发明了集合竞价。
