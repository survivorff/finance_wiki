## 核心收获 ⭐

```
收益率曲线 = 把"不同期限国债的收益率"连成一条线
  X 轴: 期限(3个月、2年、10年、30年...)
  Y 轴: 对应收益率
形状告诉你:
  正常(向上倾斜): 长期利率 > 短期 → 经济正常
  倒挂(向下倾斜): 短期利率 > 长期 → 衰退警报! ⚠️
  平坦: 转折期
```

> ⭐ **收益率曲线是经济的"水晶球"**：它综合了市场对未来利率、通胀、增长、衰退的全部预期。
> 「收益率曲线倒挂」是历史上最可靠的衰退先行指标之一——理解它，你就有了一个观察经济的强大工具。
