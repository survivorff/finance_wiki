# 6.3 全球监管格局

> 深度：🟡 进阶
> 监管是 CEX 的「边界和闸门」。理解监管，才能理解为什么交易所在不同地区行为各异。

## 监管核心关注什么

无论哪个国家，加密监管核心就盯这几件事：

```
1. 反洗钱(AML) + 实名(KYC)  ── 防止加密成为洗钱/犯罪工具
2. 用户资产保护             ── 防止 FTX 式挪用(资产隔离)
3. 市场操纵                 ── 防止插针、刷量、内幕交易
4. 偿付能力                 ── 储备充足、能兑付
5. 证券属性                 ── 哪些币是"证券"，要按证券法管
6. 稳定币                  ── 储备充足、可赎回(系统重要性)
```

## 各地区监管格局

| 地区 | 框架/态度 | 关键点 |
| --- | --- | --- |
| 🇺🇸 **美国** | 多头监管、执法严厉 | SEC(证券)、CFTC(商品/衍生品)、FinCEN(AML)。监管边界不清，靠"执法定规则" |
| 🇪🇺 **欧盟** | **MiCA** 统一框架 | 全球首个全面加密法规，发牌+稳定币+消费者保护 |
| 🇬🇧 英国 | FCA 监管 | 严格限制加密广告，推进牌照制 |
| 🇸🇬 **新加坡** | MAS 牌照制 | 稳健、对机构友好，但对散户营销限制 |
| 🇭🇰 **香港** | VASP 发牌制度 | 主动拥抱合规，发放交易所牌照，想做亚洲枢纽 |
| 🇯🇵 日本 | FSA 严格 | Mt.Gox/Coincheck 后立法严，用户资产隔离要求高 |
| 🇰🇷 韩国 | 实名银行账户制 | 交易所须绑实名银行账户、强 KYC |
| 🇦🇪 迪拜/阿联酋 | VARA | 主动招揽、监管明确，吸引交易所设点 |
| 🇨🇳 中国大陆 | 全面禁止 | 禁止交易所运营和加密交易 |

## 重点：美国的「监管靠执法」模式 💡

- 美国没有统一的加密立法，主要靠 **SEC 和 CFTC 的执法行动**来"划线"。
- **核心争议：什么是证券？** SEC 用「Howey 测试」认定很多代币是「未注册证券」。
  - 这是 SEC 起诉 Coinbase、Binance 的核心——指控它们交易「未注册证券」。
- 这种「不立法、靠起诉」的模式被业界批评为「monitoring by enforcement（用执法代替规则）」，导致合规预期不明确。

## 重点：欧盟 MiCA（全球范本）

- **MiCA（Markets in Crypto-Assets）**：全球首个全面、统一的加密监管框架。
- 覆盖：交易所发牌、稳定币规则、消费者保护、市场操纵防范。
- 意义：给了行业**明确的合规路径**（相比美国的模糊），很多国家参考。

## 监管的几个核心要求趋势 ⭐

1. **客户资产隔离**：用户资产必须与交易所自有资产分开存放，禁止挪用（直击 FTX 病根）。
2. **强制 KYC/AML**：实名 + 可疑交易上报 + 制裁名单筛查 + 旅行规则（Travel Rule）。
3. **牌照准入**：未持牌不得对当地用户提供服务。
4. **稳定币专项**：储备充足、可赎回、定期审计（MiCA、美国相关立法）。
5. **披露与审计**：储备金、风险、利益冲突的强制披露。
6. **衍生品限制**：很多地区限制或禁止向散户提供高杠杆合约。

## 监管套利的终结 💡

- 过去交易所把主体注册在宽松法域（马耳他、塞舌尔），服务全球用户，做「监管套利」。
- **Binance 43 亿美元和解（2023）是标志性转折**：再大的所也躲不过主要市场的监管。
- 趋势：**全球监管协同收紧，监管套利的空间越来越小**，合规成为头部所的生命线和护城河。

## 监管对行业的双重影响

| 正面 | 负面 |
| --- | --- |
| 保护用户、增强信任 | 合规成本高，挤压小所 |
| 吸引机构和传统资金 | 产品受限(高杠杆被禁) |
| 行业长期健康 | 可能扼杀创新、把业务赶向离岸/DEX |

> 💡 监管是把双刃剑：它让行业更安全、更主流，但也提高门槛、限制玩法，
> 并把一部分需求"挤"向去中心化（DEX 抗审查的吸引力部分来自此）。

## 小结

- 监管核心盯：AML/KYC、资产保护、市场操纵、偿付、证券属性、稳定币。
- 美国靠「执法划线」（证券属性争议是焦点），欧盟 MiCA 是统一范本。
- 核心趋势：资产隔离、强 KYC/AML、牌照准入、稳定币专项、强制披露。
- 监管套利时代终结（Binance 和解为标志），合规成为头部所护城河。
- 监管是双刃剑：增强信任 vs 提高门槛、限制玩法、把需求挤向 DEX。
