# CEX 术语表（Glossary）

> 按主题分组的 CEX 常用术语。看正文遇到不懂的名词来这查。
> 带 ⭐ 的是最核心、最高频的术语。

## 交易所类型与基础
- **CEX（Centralized Exchange）** ⭐：中心化交易所，公司运营、托管用户资产、中心化撮合。
- **DEX（Decentralized Exchange）**：去中心化交易所，用户自持私钥，链上合约撮合。
- **托管 / Custodial** ⭐：交易所替用户保管资产（私钥）。
- **自托管 / Non-custodial**：用户自己掌握私钥。
- **Not your keys, not your coins** ⭐：不掌握私钥就不真正拥有币（交易所余额是债权）。
- **KYC（Know Your Customer）**：实名认证。
- **AML（Anti-Money Laundering）**：反洗钱。

## 交易机制
- **订单簿 / Order Book** ⭐：所有未成交买卖挂单的集合。
- **交易对 / Trading Pair** ⭐：如 BTC/USDT，前为基础币(base)、后为计价币(quote)。
- **盘口深度 / Depth** ⭐：盘口挂单的厚度，反映流动性。
- **价差 / Spread**：最优买价与最优卖价之差。
- **滑点 / Slippage**：实际成交价与预期价的偏差，深度差时滑点大。
- **流动性 / Liquidity** ⭐：资产能被快速买卖而不显著影响价格的能力。
- **价格发现 / Price Discovery**：市场通过交易形成公允价格的过程。

## 订单类型
- **限价单 / Limit Order**：指定价格挂单，不满足则等待。
- **市价单 / Market Order**：按当前最优价立即成交。
- **止损/止盈 / Stop / Take-Profit**：价格触发后转为市价或限价单。
- **IOC**：立即成交剩余取消。**FOK**：全部成交否则取消。**Post-Only**：只做 maker。
- **冰山订单 / Iceberg**：大单拆成小单隐藏真实数量。

## 角色
- **Maker（挂单方）** ⭐：提供流动性，费率低/有返佣。
- **Taker（吃单方）** ⭐：消耗流动性，费率高。
- **做市商 / Market Maker (MM)** ⭐：持续双边挂单提供流动性、赚价差的专业机构。
- **OTC**：场外大宗交易。**C2C / P2P**：用户间点对点法币交易。

## 衍生品
- **永续合约 / Perpetual (Perp)** ⭐：无交割日的期货。
- **交割合约 / Futures**：有到期日的期货。
- **期权 / Options**：买卖未来以某价买/卖的权利。
- **杠杆 / Leverage** ⭐：放大仓位倍数（如 10x）。
- **保证金 / Margin**：开合约仓位需抵押的本金。
- **逐仓 / 全仓 (Isolated/Cross)**：保证金隔离 / 共享模式。
- **U本位 / 币本位**：用稳定币 / 用币 作保证金结算。
- **资金费率 / Funding Rate** ⭐：永续合约锚定现货的多空周期性付费机制。
- **强平 / 爆仓 / Liquidation** ⭐：保证金不足时被强制平仓。
- **穿仓**：亏损超过本金（账户变负）。
- **保险基金 / Insurance Fund**：兜底穿仓损失的资金池。
- **ADL（Auto-Deleveraging）**：自动减仓，保险基金不足时强平对手盈利仓位。
- **指数价 / Index Price**：多所现货加权均价。
- **标记价 / Mark Price** ⭐：用于计算盈亏和触发强平的参考价，防插针误伤。
- **基差 / Basis**：期货价与现货价之差。
- **持仓量 / Open Interest (OI)**：未平仓合约的总价值。

## 钱包与链上
- **冷钱包 / Cold Wallet** ⭐：离线存储，安全性高。
- **热钱包 / Hot Wallet** ⭐：在线，应对日常提币。
- **多签 / Multisig**：需多把私钥共同签名。
- **MPC（多方安全计算）** ⭐：私钥分片协同签名，从不完整存在于一处。
- **HSM**：硬件安全模块，防篡改的私钥存储硬件。
- **盲签 / Blind Signing**：在看不清内容的情况下签名（安全大忌）。
- **充币/提币 / Deposit/Withdraw**：链上资产转入/转出交易所。
- **确认数 / Confirmations**：交易被打包后经过的区块数，防双花。
- **归集 / Sweeping**：把分散地址资金集中到主钱包。
- **重组 / Reorg**：区块链回滚已打包区块的现象。
- **双花 / Double Spend**：同一笔币被花费两次的攻击。

## 业务与生态
- **上币 / Listing** ⭐：代币上架交易所交易。
- **下币 / Delisting**：代币被交易所下架。
- **Launchpad / Launchpool** ⭐：新币首发/打新平台。
- **平台币 / Exchange Token** ⭐：如 BNB、OKB，抵扣手续费、参与生态。
- **回购销毁 / Buyback & Burn**：用利润回购并销毁平台币以支撑价值。
- **理财 / Earn**：存币赚收益。**质押 / Staking**：参与 PoS 公链质押获奖励。
- **双币投资 / Dual Investment**：变相卖期权的结构化产品，不保本。
- **法币出入金 / On-Off Ramp** ⭐：法币与加密币的兑换通道。
- **返佣 / Referral**：邀请返手续费的获客机制。
- **跟单交易 / Copy Trading**：复制高手操作的功能。

## 风险与合规
- **储备金证明 / PoR（Proof of Reserves）** ⭐：证明交易所持有足额资产。
- **默克尔树 / Merkle Tree** ⭐：PoR 中用于让用户验证余额被计入负债的数据结构。
- **客户资产隔离**：用户资产与交易所自有资产分开存放，禁止挪用。
- **MiCA**：欧盟加密资产市场统一监管框架。
- **VASP**：虚拟资产服务提供商（香港等地的牌照对象）。
- **Howey 测试**：美国判断某资产是否为「证券」的法律标准。
- **挤兑 / Bank Run** ⭐：用户集中提现导致交易所流动性崩溃。
- **监管套利 / Regulatory Arbitrage**：把主体注册在监管宽松法域。
- **旅行规则 / Travel Rule**：转账需附带收发方身份信息的 AML 要求。

## 市场操纵（识别用，勿实施）
- **插针 / Stop Hunting** ⭐：制造瞬间极端价触发强平。
- **拉高出货 / Pump and Dump** ⭐：拉盘吸引跟风后高位抛售。
- **刷量 / Wash Trading** ⭐：自买自卖制造虚假交易量。
- **幌骗 / Spoofing**：挂假大单诱导后撤单。
- **抢跑 / Front-running**：利用提前知道的订单信息抢先交易。
- **FUD**：散布恐惧/不确定/怀疑，打击对手或制造恐慌。
- **连环爆仓 / Liquidation Cascade**：强平引发的踩踏式瀑布行情。

## 套利与策略
- **套利 / Arbitrage** ⭐：利用价差低买高卖。
- **跨所套利**：不同交易所间的价差套利。
- **资金费率套利**：现货+合约对冲赚资金费（delta 中性）。
- **三角套利**：同所内三个交易对价格不自洽的套利。
- **泡菜溢价 / Kimchi Premium**：韩国市场因资本管制产生的币价溢价。
- **delta 中性**：通过对冲使组合对价格涨跌不敏感。

## 工程概念
- **撮合引擎 / Matching Engine** ⭐：匹配买卖订单的核心系统。
- **价格-时间优先 / Price-Time Priority** ⭐：撮合规则（价格优先，同价时间优先）。
- **事件溯源 / Event Sourcing** ⭐：记录所有输入指令以便重放恢复状态。
- **状态机复制 / State Machine Replication**：相同输入喂多副本得一致状态，实现高可用。
- **复式记账 / Double-Entry Bookkeeping** ⭐：每笔变动有借贷两方、账目恒平。
- **WAL（Write-Ahead Log）**：预写日志，先持久化再处理。
- **清算与结算 / Clearing & Settlement** ⭐：撮合后改余额、扣费、结算盈亏。
- **对账 / Reconciliation**：核对账面余额与链上实际资产。
- **预冻结 / Freeze**：下单时先冻结资金，成交后再扣减。
- **幂等性 / Idempotency**：同操作重复执行结果一致。
- **LMAX Disruptor**：金融业经典的高性能单线程队列架构。
- **WebSocket**：行情实时推送用的长连接协议。
- **增量推送 / Delta Update**：只推送变化部分以省带宽。

## 其他
- **稳定币 / Stablecoin** ⭐：锚定法币的加密币（USDT/USDC），加密世界的「美元」。
- **RWA（Real World Assets）**：真实世界资产代币化。
- **ETF**：交易所交易基金；比特币现货 ETF 是机构入场里程碑。
- **CeDeFi**：中心化与去中心化融合的混合模式。
- **反身性 / Reflexivity**：价格与情绪互相强化的现象。
- **AUM（Assets Under Management）**：管理的资产规模。
- **CAC / LTV**：获客成本 / 用户生命周期价值。
