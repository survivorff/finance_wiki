# 4.3 中央证券存管（CSD）

> 深度：🔴 硬核
> 你买的股票到底「存」在哪里？答案是 CSD。这一篇揭示「证券保管」的真相，并和 CEX 托管做最关键的对照。

## 你的股票存在哪里 ⭐

```
你以为: 股票"在我的券商账户里"
真相:   股票登记在"中央证券存管(CSD)", 通过多层"名义持有"链条到你
```

## 名义持有链条（Custody Chain）⭐

```
中央证券存管(CSD) — 登记股票的最终归属
   │ (券商作为参与者, 在 CSD 开账户)
   ▼
券商(Broker) — 在 CSD 持有"一个大账户", 里面是所有客户的股票
   │ (券商内部记账, 区分哪些是哪个客户的)
   ▼
你 — 在券商的记录里, 拥有 100 股
```

- 💡 **关键**：在很多市场，股票以**"名义持有(street name / nominee)"** 方式登记——
  CSD 层面看到的是「券商持有 X 股」，券商内部记账区分「这 X 股里哪些属于哪个客户」。
- 你是**受益所有人（beneficial owner）**，券商是**名义持有人（nominee）**。

## 这听起来像 CEX 托管？关键区别在哪 ⭐⭐

> ⚠️ 表面上，「券商代你持有股票」和 [CEX 替你保管币](../../crypto/cex/01-行业全景/1.1-CEX是什么.md) 很像。
> 但背后的信任结构**天差地别**：

| | 股市（券商代持） | CEX（交易所托管） |
| --- | --- | --- |
| 最终登记在哪 | **独立的 CSD**（券商动不了） | 交易所自己的数据库 |
| 客户资产隔离 | **法律强制**（见 [1.3](../01-行业全景/1.3-分业制衡的产业链.md)） | FTX 后才开始强调 |
| 券商能挪用吗 | 受严格监管+审计，违法 | 技术上可以（FTX 就挪用了） |
| 券商破产 | 客户股票在 CSD，可转移到别的券商 | 可能血本无归（资产被挪空） |
| 独立核对 | CSD/监管可独立核对 | 只有交易所自己知道 |
| 兜底 | 投资者保护基金（如 SIPC） | 一般没有 |

> ⭐ **核心洞察**：股市和 CEX 都是「中介代你持有」，但股市有**独立存管 + 法律强制隔离 + 监管核对 + 保护基金**的多重保险，
> 而 CEX 是「交易所既保管又记账又能动用，全凭它自律」。
> **这就是为什么券商倒闭（如雷曼）客户股票基本无恙，而 FTX 倒闭用户血本无归。**
> 「代持」不可怕，可怕的是「代持方能不受制约地动用你的资产」——这正是 [分业制衡](../01-行业全景/1.3-分业制衡的产业链.md) 要解决的。

## 登记结算与公司行动

CSD 还负责：
- **登记股东名册**：谁持有多少股。
- **执行公司行动**：分红派息、配股、拆股时，按持股自动分配（见 [05](../05-发行与公司行动/README.md)）。
- **行使权利**：投票权的传递（虽然名义持有下，投票权传递较复杂）。

## 无纸化（Dematerialization）💡

```
历史: 股票是纸质凭证, 要physically 保管和转移(易丢、易伪造、转移慢)
现代: 股票"无纸化", 变成 CSD 系统里的电子记录
→ 这其实就是"中心化的电子账本"
```

> 💡 **有趣的对照**：CSD 本质是「一个中心化的、记录证券归属的电子账本」。
> 而区块链是「一个去中心化的、记录资产归属的账本」。
> **DEX/区块链某种意义上是想用「去中心化账本」替代「CSD 这个中心化账本」**——
> 让资产归属直接记在链上、用户自托管，去掉 CSD 和券商的代持层。
> 这就是「代币化证券（tokenized securities）」的核心想法（见 [07](../07-现代演化/7.3-代币化与未来.md)）。

## 真实例子

- 美国：**DTC（Depository Trust Company）**，DTCC 旗下。
- 欧洲：Euroclear、Clearstream。
- 中国：中国证券登记结算公司（中国结算）。

## 小结

- 你的股票登记在中央存管(CSD)，通过「CSD→券商→你」的名义持有链条，你是受益所有人。
- 表面像 CEX 托管，但股市有独立 CSD + 法律强制隔离 + 监管核对 + 保护基金，信任结构完全不同。
- 这就是券商倒闭客户股票无恙、而 FTX 倒闭用户血本无归的根本原因。
- CSD 本质是中心化电子账本；区块链/代币化证券想用去中心化账本+自托管替代它。
