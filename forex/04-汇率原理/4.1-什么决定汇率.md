# 4.1 什么决定汇率

> 深度：🟡 进阶 ⭐
> 这是 FX 最核心的问题。汇率不是随机的，背后有清晰的驱动因素，只是不同时间尺度看不同的东西。

## 汇率的本质：货币的供需 ⭐

```
汇率 = 两种货币的相对价格 = 由两种货币的"相对供需"决定
  对某货币需求↑(大家想要它) → 它升值
  对某货币供给↑(大家想抛它) → 它贬值
→ 一切因素, 最终都通过"影响供需"来影响汇率
```

那什么影响供需？分时间尺度看：

## 一、利率（中期最重要）⭐⭐

```
某国利率↑(相对别国):
  → 持有该货币的资产能赚更高利息
  → 全球资金涌入该货币(买它的债券/存款)赚利息
  → 对该货币需求↑ → 该货币升值
例: 美联储加息 → 美元利率高 → 资金涌入美元 → 美元走强
```

- 💡 **这是 FX 最核心的驱动**：**利率差驱动资金流动，资金流动驱动汇率。**
- 这也是 [套息交易](./4.3-套息交易.md) 的基础，是 FX 交易员最盯央行利率的原因。
- 详见 [4.2 利率平价](./4.2-利率平价与购买力平价.md)。

## 二、通胀（长期）⭐

```
某国通胀↑(货币购买力下降):
  → 长期看该货币会贬值(同样的钱能买的东西变少)
  → 购买力平价(PPP)理论: 长期汇率向"两国购买力相等"靠拢
例: 高通胀国家的货币长期趋于贬值(如历史上的恶性通胀货币)
```

- 💡 通胀和利率often 反向操作：高通胀 → 央行加息 → 短期货币可能因高利率走强，但长期因通胀走弱。**短期利率效应和长期通胀效应可能打架**——这是汇率分析的复杂之处。

## 三、国际收支（中长期）

```
经常账户(贸易):
  贸易顺差(出口>进口) → 外国要买你的货 → 需要你的货币 → 货币升值压力
  贸易逆差 → 货币贬值压力
资本账户(投资):
  外资流入(买你的股债) → 货币升值; 资本外逃 → 货币贬值
```

- 💡 出口大国（如历史上的德国、日本、中国）的货币常有升值压力，但会被央行干预/政策对冲。

## 四、避险情绪（短期）⭐

```
风险事件(危机/战争/恐慌):
  资金涌入"避险货币"(USD、JPY、CHF, 见 1.3) → 避险货币升值
  抛售"风险货币"(新兴市场、商品货币) → 它们贬值
→ "risk-off"(避险)时, 美元/日元/瑞郎走强
→ "risk-on"(乐观)时, 资金流向高息/风险货币
```

- 💡 这是 FX 独有的「情绪开关」——和 [加密的恐慌/贪婪](../../crypto/cex/05-市场与博弈/5.5-资金流动全景.md) 类似，但 FX 的避险有明确的"安全港"货币。

## 五、预期与消息（短期）

```
市场是"预期"的游戏:
  - 经济数据(GDP/就业/通胀)好于/差于预期 → 汇率立刻反应
  - 央行讲话/利率决议 → 剧烈波动
  - "买预期卖事实": 加息预期推升货币, 真加息时反而回落(已price in)
```

## 六、央行干预（任何时候）

```
央行可以直接下场买卖本币影响汇率(见 05)
  → 这是其他市场没有的"上帝之手"
```

## 综合：不同时间尺度看不同因素 ⭐

```
时间尺度        主导因素
─────────────────────────────────
分钟/小时    消息、数据、订单流、避险情绪
日/周        利率预期、央行讲话、资金流
月/季        利率差(套息)、经济基本面
年/多年      通胀差(购买力平价)、国际收支、结构性因素
```

> ⭐ **记住这张表**：当有人问"为什么这货币涨了"，答案取决于你看的时间尺度。
> 短期可能是消息驱动的噪音，长期才是基本面。**混淆时间尺度是汇率分析最常见的错误。**

## ⚠️ 汇率预测极难

```
即使懂了所有因素, 汇率预测依然极难:
  - 因素互相打架(利率 vs 通胀)
  - 市场已 price in 预期(消息出来反而反向)
  - 央行干预、黑天鹅难预料
→ 顶级机构都经常预测错, 散户更不用说
→ 这也是零售外汇亏损率高的原因之一(见 06)
```

## 小结

- 汇率本质是货币的相对供需，一切因素通过影响供需起作用。
- 核心驱动按时间尺度：短期（消息/避险/资金流）、中期（利率差，最重要）、长期（通胀/国际收支）。
- 利率是贯穿核心（利率差驱动资金流→驱动汇率），通胀长期主导（购买力平价）。
- 避险情绪驱动 risk-on/risk-off（USD/JPY/CHF 是安全港）。
- 汇率预测极难（因素打架、预期已 price in、干预难料），混淆时间尺度是常见错误。
