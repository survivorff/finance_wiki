# 2.5 法币与出入金（Fiat On/Off Ramp）

> 深度：🟢 入门
> 这是用户进入加密世界的「第一道门」，也是交易所最敏感、最容易被监管掐断的环节。

## 为什么出入金是「命脉」

- 再好的交易系统，如果用户的法币进不来、币换不回法币提不出去，都是空中楼阁。
- **法币通道是合规的最前线**：监管打击交易所，第一刀往往就是切断它的银行通道。
- 不同地区的出入金难易，直接决定了交易所在当地能不能活。

## 主要出入金方式

### 1. C2C / P2P（点对点交易）⭐ 最主流（尤其新兴市场）
- **怎么运作**：用户之间直接用法币买卖币（你转钱给卖家的银行卡，平台放币给你）。
- **平台角色**：做**担保中介**——锁定卖家的币（托管），确认买家付款后放币，防止欺诈。
- **为什么流行**：在银行不直接支持加密的地区，C2C 是事实上的主要法币入口。
- ⚠️ **风险**：可能收到「黑钱」（涉诈骗资金的银行卡转账），导致自己银行卡被冻结。这是中国大陆用户的高发风险。

### 2. 第三方支付通道（On-Ramp 服务商）
- 如 **MoonPay、Simplex、Banxa**，把「信用卡/银行卡买币」这件事外包给专业合规公司。
- 用户用 Visa/Mastercard 直接买币，服务商处理合规和支付，收较高手续费。

### 3. 银行电汇 / 卡充值
- 合规交易所（如 Coinbase、Kraken）直接对接银行，支持电汇、ACH、SEPA 等。
- 需要强 KYC，受当地银行合作关系制约。

### 4. 一键买币（Quick Buy）
- 给小白的「傻瓜式」入口：输入金额，刷卡，直接到账币。背后整合了上述通道。
- 体验最简单，费率通常最高（小白不敏感，交易所利润高）。

## 出入金的资金流

```
入金:  用户法币 ──银行/支付/C2C──▶ 法币到达交易所/对手方 ──▶ 交易所给账户记上余额
出金:  交易所扣减账户余额 ──▶ 通过银行/C2C 把法币打给用户
```

> 💡 注意：法币的出入金**不经过区块链**（法币不在链上），走的是传统银行/支付网络。
> 只有「币」的充提才走区块链（见 [2.1 充提](./2.1-现货交易.md) 与 [03 钱包系统](../03-技术架构/3.3-钱包与充提.md)）。

## 出入金为什么这么难（合规深水区）

| 难点 | 说明 |
| --- | --- |
| **银行不愿合作** | 银行怕洗钱风险，常拒绝给加密交易所开户 |
| **AML/KYC 要求** | 大额出入金必须实名、可疑交易要上报 |
| **黑钱风险** | C2C 收到诈骗/赌博资金，卡被司法冻结 |
| **跨境与外汇管制** | 涉及各国外汇政策，灰色地带多 |

> 💡 **内幕**：交易所的「法币通道团队」是和银行、支付商常年博弈的角色。
> 通道经常「这个月好用、下个月就断」，是交易所运营中最不稳定的环节之一。
> 一些交易所靠不断更换、新增通道（甚至灰色通道）来维持出入金顺畅。

## ⚠️ 用户风险提示

- **C2C 收黑钱被冻卡**：是中文区用户最常见的真实损失。尽量选信誉好的商家、保留转账记录。
- **出金困难是暴雷前兆**：如果一个交易所突然「出金延迟/暂停」，要高度警惕。

## 小结

- 出入金是用户进入加密的第一道门，也是合规最前线、最易被掐断的命脉。
- 主流方式：C2C（最普及）、第三方支付、银行电汇、一键买币。
- 法币出入金走传统银行/支付网络，不经过区块链。
- 银行合作、AML、黑钱、外汇管制是四大难点；C2C 收黑钱冻卡是用户高发风险。
