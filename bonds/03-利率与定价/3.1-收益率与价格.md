# 3.1 收益率与价格

> 深度：🟡 进阶 ⭐
> 债券最核心、最反直觉的关系：价格和利率反向。搞懂这个，债券就入门了。

## 票息率 vs 收益率（别混淆）⭐

```
票息率(Coupon Rate): 债券"印在票面上"的固定利率(发行时定死, 不变)
  例: 面值 100, 票息率 3% → 每年永远付 3 元
收益率(Yield): 市场"实际"给你的回报率(随债券价格变动)
  → 你按市场价买入后, 实际能获得的回报
```

- 💡 **票息率是死的（票面固定），收益率是活的（随价格变）。** 市场用"收益率"给债券定价。

## 到期收益率（YTM, Yield to Maturity）⭐

```
YTM = 如果你"现在按市场价买入并持有到期", 能获得的年化总回报
  → 考虑了: 买入价、所有票息、到期还本
  → 是债券最重要的"价格指标"(债市报价主要看 YTM, 不是价格)
```

## 价格与利率反向（最核心）⭐⭐

这是债券最反直觉的关系，用例子说清：

```
你持有一张债券: 面值 100, 票息率 3%(每年付 3 元), 还有 10 年到期
情况 A — 市场利率涨到 5%:
  → 新发的债券都付 5% 了, 你这张只付 3% 的"不香了"
  → 没人愿按 100 买你的(谁要 3% 当能买 5%)
  → 你只能降价卖(比如卖 85), 让买家的"收益率"达到 5%
  → 债券价格跌! (利率↑ → 价格↓)

情况 B — 市场利率跌到 1%:
  → 你这张付 3% 的"真香"(比新债的 1% 高)
  → 大家抢着要 → 价格涨到 100 以上(比如 118)
  → 债券价格涨! (利率↓ → 价格↑)
```

> ⭐ **记住这个铁律：利率涨→债券价格跌，利率跌→债券价格涨。**
> 直觉：债券票息固定，市场利率变了，只能靠"调整价格"让收益率追上市场——所以价格和利率永远反向。

## 为什么这关系如此重要 ⭐

```
1. 解释债券的"价格波动": 不持有到期就卖, 价格随利率波动 → 可能亏(利率涨时)
2. 解释 2022 年债券大跌: 央行激进加息(利率↑) → 债券价格暴跌(见 1.3)
3. 是"久期"概念的基础(见 3.3): 期限越长, 这个反向关系越剧烈
4. 连接到"利率是万物之锚"(见 07): 同样的逻辑, 利率涨压制所有资产价格
```

## 折价、平价、溢价

```
债券市场价 vs 面值:
  平价(Par): 市场价 = 面值(收益率 = 票息率)
  折价(Discount): 市场价 < 面值(利率涨了, 收益率 > 票息率)
  溢价(Premium): 市场价 > 面值(利率跌了, 收益率 < 票息率)
```

## 当前收益率 vs YTM

```
当前收益率(Current Yield) = 年票息 / 当前市场价 (简单, 但忽略资本利得/损失)
YTM = 更完整(含到期还本的价差) → 更准确的回报衡量
```

## ⚠️ 小白误区

- **"买债券就保本"**：只有"持有到期且不违约"才拿回面值。**中途卖出，价格随利率波动，可能亏损**（2022 很多债券基金大亏就是这个）。
- **"票息率高就回报高"**：不一定。要看你买入的"价格"和对应的"收益率"，不是看票面票息率。

## 小结

- 票息率（票面固定）≠ 收益率（随价格变，市场用它定价）；YTM 是最重要的价格指标。
- 核心铁律：**利率涨→债券价格跌，利率跌→债券价格涨**（票息固定，靠调价让收益率追市场）。
- 这解释了债券价格波动、2022 债券大跌、久期、以及"利率是万物之锚"。
- 折价/平价/溢价对应利率高于/等于/低于票息率。
- 误区：债券非绝对保本（中途卖随利率波动）；回报看买入收益率而非票面票息率。
