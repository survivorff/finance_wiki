# 债券与利率市场（Bonds & Rates）深度知识库

> 债券：人类最古老的金融工具之一，规模比股市还大。
> 利率：金融世界的「地心引力」——它影响一切资产的价格。

如果你读完了 [股票](../stocks/README.md)、[外汇](../forex/README.md)、[CEX](../crypto/cex/README.md)、[DEX](../crypto/dex/README.md)，
债券市场会给你金融世界最底层、最重要的一块拼图：

> **债券市场比股市更大（全球债券市值上百万亿美元），但更"低调"。
> 而它背后的"利率"，是整个金融世界的定价基准——
> 股票估值、汇率、房价、加密、一切资产的价格，都建立在"利率"这个地基之上。**

债券与利率市场最大的学习价值：

1. **利率是万物定价的锚**：所有资产都在和「无风险利率」竞争。利率一动，所有资产价格重估——这是理解 2022 年加密/股票同步暴跌的钥匙（美联储加息）。
2. **债券是"借钱"的市场**：股票是"卖股权"，债券是"借债"。理解债券，就理解了政府和企业怎么融资、国家债务怎么运转。
3. **收益率曲线是经济的"水晶球"**：它预示衰退、通胀、央行政策。
4. **它连接了 [外汇的利率](../forex/04-汇率原理/README.md) 和一切**：FX 的套息、央行政策，最终都体现在债券利率上。

这个知识库的目标：让**小白**理解「债券、利率、收益率到底是什么」，让**从业者/工程师**挖到久期、收益率曲线、定价数学、回购市场的深水细节。

## 🎯 读完应该建立的「认知锚点」⭐

1. **债券是"借条"**：你买债券=借钱给发行人，到期还本金 + 期间付利息（票息）。
2. **债券价格和利率反向**：利率涨→已发行债券价格跌（反之亦然）。这是债券最反直觉、最核心的关系。
3. **收益率（yield）才是关键**，不是票面利率：市场用"到期收益率"给债券定价。
4. **利率是万物之锚**：无风险利率是所有资产估值的基准——利率一动，股票/房产/加密全部重估。
5. **收益率曲线倒挂预示衰退**：短期利率高于长期，是经济的著名警报。
6. **国债是"无风险资产"的基石**：整个金融体系建立在"政府不会赖账"的假设上——这个假设一旦动摇就是系统性危机。

## 一张图看懂「利率是万物之锚」

```
                    无风险利率(国债收益率)
                          │ 是一切的"地基"
        ┌─────────────────┼─────────────────┐
        ▼                 ▼                 ▼
     股票估值           房地产            其他资产
   (折现率↑→估值↓)    (房贷利率↑→房价↓)   (加密/大宗等)
        │                 │                 │
        └──── 利率↑ = 所有资产的"重力↑" → 价格承压 ────┘

  2022 年实例: 美联储激进加息 → 利率飙升
            → 股票、债券、加密 同步暴跌(万物重估)
```

> **关键洞察**：利率就像金融世界的"地心引力"——
> 利率低，万物轻飘飘往上涨（2020-2021 的资产狂欢）；
> 利率高，万物被往下拽（2022 的全面下跌）。
> **理解利率，你就理解了为什么所有市场会"一起涨、一起跌"——它们共用同一个地基。**
